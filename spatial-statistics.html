空間自相關與 Gaussian Process

Moran's I、Geary's C、SpatialDE、SPARK、nnSVG、BayesSpace——找「空間變異基因」(SVG) 的統計工具箱。

Moran's I, Geary's C, SpatialDE, SPARK, nnSVG, BayesSpace — the statistical toolbox for spatially variable genes (SVGs).

核心概念

Moran's I 與 Geary's C

空間統計來自地理學。Moran's I 衡量「鄰近地點是否傾向有相似值」：

I = (n/W) · Σᵢⱼ wᵢⱼ(xi − x̄)(xj − x̄) / Σ(xi − x̄)²

其中 wij 是空間權重（鄰居 = 1，否則 0）、W = Σwij。I > 0 → 正空間自相關（聚集）；I < 0 → 棋盤狀；I ≈ 0 → 隨機。

Geary's C：用「鄰居差異」而非「鄰居協方差」測自相關，數值上 C ∈ [0, 2]，C < 1 為正自相關。對局部變化更敏感。

Spatial statistics originate from geography. Moran's I measures whether neighboring locations tend to have similar values:

I = (n/W) · Σᵢⱼ wᵢⱼ(xi − x̄)(xj − x̄) / Σ(xi − x̄)²

where wij is spatial weight (1 if neighbors, 0 else); W = Σwij. I > 0 → positive autocorrelation (clustering); I < 0 → checkerboard; I ≈ 0 → random.

Geary's C uses "neighbor differences" instead of covariance; C ∈ [0, 2], C < 1 = positive autocorrelation. More sensitive to local change.

📍

全域 vs 局部

Global I 是一個總體分數。Local Moran's I (LISA) 給每個 spot 一個 score——找熱點/冷點。

Global I is one summary. Local Moran's I (LISA) gives per-spot scores — find hotspots/coldspots.

Hotspot detector that targets clusters of high values.

進階方法

代表性方法

SpatialDE (Svensson 2018)

用 Gaussian Process 模型分解基因表現為「空間相關」+「隨機」兩部分。先導性的 SVG 方法，但對 droput 與 over-dispersion 敏感。

Decomposes gene expression into "spatial" + "noise" via Gaussian Process. Pioneering SVG method but sensitive to dropout and over-dispersion.

SPARK / SPARK-X (Zhu 2021/2024)

基於 GLM + 多個 kernel 的空間檢定，SPARK-X 用核–線性等價技巧把計算速度提升 100×，可擴展到 10⁵ spots。

GLM + multiple-kernel test; SPARK-X uses kernel–linear equivalence for 100× speedup, scales to 10⁵ spots.

nnSVG (Weber 2023)

每個基因 fit 一個 nearest-neighbor GP，給出基因專屬的「effective range」。對於異質長度尺度的 SVG 偵測表現好。

Fits a nearest-neighbor GP per gene; gives gene-specific "effective range." Excels when SVGs have heterogeneous length scales.

BayesSpace (Zhao 2021)

階層貝氏模型在「鄰居應屬同一空間域」的先驗下做分群、解析度提升 (subspot)。

Hierarchical Bayes with "neighbors tend to share domain" prior — clustering and subspot resolution enhancement.

MERINGUE (Miller 2021)

融合空間自相關與 cross-correlation——找「空間共定位」基因對。

Combines spatial autocorrelation + cross-correlation — find spatially co-localized gene pairs.

GASTON / SOTIP (2024)

最新一代用神經網路或拓撲方法擴展，處理跨切片整合與三維重建。

Latest gen using neural networks / topology for cross-slice integration and 3D reconstruction.

深入解析

深入解析：權重矩陣、變異函數、GP 與 SVG 偵測

① 空間權重矩陣 (Spatial Weight Matrix) 的設計

所有空間自相關統計都依賴一個 n × n 的權重矩陣 W，定義「誰是誰的鄰居」。常見設計：

  • Rook contiguity：共享一條邊（grid 上的上下左右，4 鄰居）。適合規則網格如早期 array-based ST。
  • Queen contiguity：共享邊或角（8 鄰居）。仍是規則網格用，但對「對角訊號」更敏感。
  • Distance-based：w_{ij} = 1 if d_{ij} < threshold else 0。優點：適用不規則點分布；缺點：threshold 需手調。
  • k-Nearest Neighbors：每個 spot 連到最近 k 個鄰居。優點：density-adaptive；缺點：W 不對稱（需 symmetrize）。Visium 通常用 k = 6（六角網格）。
  • Kernel-based：w_{ij} = exp(−d_{ij}² / σ²)，連續權重，與 Gaussian Process 接軌。

權重矩陣需要 row-normalize（每列總和=1），讓統計量在不同度數節點上可比。

② Moran's I 的虛無分布：Randomization vs Permutation

Moran's I = (n / S₀) · (xᵀ W x) / (xᵀ x)（中心化後）。其虛無分布有兩種假設：

  • Normality assumption (Normal H₀): 假設 x ~ N(μ, σ²) i.i.d.，可導出 E[I] = −1/(n−1)，Var[I] 有閉式解 (Cliff & Ord 1981)。z-score 適用大樣本但若資料有 outlier 會失準。
  • Randomization assumption (Random H₀): 假設 x 的「值集合」固定，但位置隨機重排。permutation 999 次計算 empirical p-value。對非常態資料較穩健，但對非常大樣本 (n > 10000) 太貴。

實務經驗：基因表現多為 zero-inflated count，建議 (a) 先 log1p normalize，(b) 用 permutation，(c) 對全基因組做 FDR (Benjamini-Hochberg)。

③ Local Moran's I (LISA, Anselin 1995)

Global Moran's I 只給一個總體數字；Anselin 把它分解到每個 spot：I_i = (x_i − x̄)/σ² · Σⱼ w_{ij}(x_j − x̄)。可分類為四種：

  • HH (High-High): 自己高，鄰居也高 — hot spot
  • LL (Low-Low): 自己低，鄰居也低 — cold spot
  • HL (High-Low): 自己高、鄰居低 — outlier / spatial discontinuity
  • LH (Low-High): 自己低、鄰居高 — 反向 outlier

對於 ST：HH/LL 找 niche 中心；HL/LH 找 niche 邊界或浸潤前緣（例如腫瘤邊界、傷口邊緣）。

④ Geary's C 與 Getis-Ord G* 的差異

三者並非等價：

  • Moran's I：對「均值偏移」的 spatial pattern 最敏感。
  • Geary's C：C = ((n−1) / 2S₀) · ΣᵢΣⱼ w_{ij}(x_i − x_j)² / Σᵢ(x_i − x̄)²。對「鄰近 pair 差異」直接量化，對 local heterogeneity 比 Moran 敏感。C 範圍 [0, 2]，C < 1 表正自相關。
  • Getis-Ord G*：G_i* = Σⱼ w_{ij} x_j / Σⱼ x_j。只看絕對值，能區分 high cluster 與 low cluster（Moran 無法）。常用於犯罪 hotspot 分析，ST 中可用於 marker gene hotspot 偵測。

⑤ Variogram 與 Covariogram

半變異函數 γ(h) = ½ E[(Z(x+h) − Z(x))²] 是地統計學核心工具，衡量「距離 h 的兩點之差異變異度」。三個關鍵參數：

  • Nugget (τ²): γ(0⁺)，反映微觀變異 + 量測誤差。在 ST 中 dropout 會讓 nugget 偏高。
  • Sill (σ² + τ²): γ(∞)，總變異。
  • Range (a): 達到 sill 的距離，物理意義是「spatial 影響範圍」。基因 range = 50 µm 表示其表現只在 50 µm 內有 spatial 結構。

covariogram C(h) = Var(Z) − γ(h)。在 stationary GP 下兩者等價。

⑥ Gaussian Process Kernel 的選擇

GP 模型 Z(x) ~ GP(μ(x), k(x, x'))。kernel 決定「平滑度」：

  • RBF / Squared Exponential: k(d) = σ² exp(−d²/2ℓ²)。infinitely smooth，但太平滑常不合生物資料。
  • Matérn 3/2: k(d) = σ²(1 + √3 d/ℓ) exp(−√3 d/ℓ)。一次可微，介於 exponential 與 RBF 之間。對 ST 多數基因合適。
  • Matérn 5/2: k(d) = σ²(1 + √5 d/ℓ + 5d²/3ℓ²) exp(−√5 d/ℓ)。二次可微，比 3/2 更平滑。
  • Periodic kernel: k(d) = σ² exp(−2 sin²(πd/p) / ℓ²)。捕捉週期性 spatial pattern（如皮層柱、肝小葉）。
  • Composite kernels: RBF + Periodic + Linear，模擬「全局趨勢 + 區域結構 + 週期性」。

⑦ SpatialDE (Svensson 2018, Nature Methods) 的數學

對每個基因 g，建立兩個 GP 模型：

  • H₁ (spatial): y_g(x) ~ GP(0, σ²_s K_s(x, x') + σ²_n I)，K_s 為 spatial kernel。
  • H₀ (non-spatial): y_g ~ N(0, σ²_n I)，純噪音。

用 maximum likelihood 估計參數，做 likelihood ratio test：LLR = log L(H₁) − log L(H₀)，依 χ² 分布算 p-value。最後 BH-FDR 校正。優點：每個 SVG 還能得到 length scale ℓ_s，反映 spatial 結構尺度。

⑧ SPARK (Sun 2020, Nat Methods) 與 SPARK-X

SPARK 改用 Poisson / Negative Binomial GLM + spatial random effect（不像 SpatialDE 假設 Gaussian），對 raw count 更合適。並且同時 fit 多個 kernel（不同長度尺度）做集成，找對「任何 spatial 結構」都敏感。SPARK-X (Zhu 2021, Genome Biology) 是 scalable 版本：用 covariance test 取代 likelihood，O(n) 即可，能處理 Slide-seqV2 百萬 spot 資料。

⑨ nnSVG (Weber 2023, Nat Commun) 的 Nearest-neighbor GP

使用 Nearest-Neighbor Gaussian Process (NNGP, Datta 2016)：每個 spot 只與最近 m 個鄰居（如 m=10）建模，O(nm³) 而非 O(n³)。對每個基因 fit 不同 length scale ℓ_g，能找出「短程 vs 長程 spatial 變化」基因。對 Visium DLPFC 資料能精準分層皮層 layer 特異基因。

⑩ BayesSpace (Zhao 2021, Nat Biotechnol) 的 Hierarchical Model

不是 SVG 偵測，而是 spatial domain 分群。用 hierarchical Bayesian：

  • Layer 1: y_i ~ N(μ_{z_i}, Σ_{z_i})，z_i 為 spot i 的 cluster 標籤。
  • Layer 2: z 服從 Potts model（Ising 推廣），鼓勵相鄰 spot 同 cluster：P(z) ∝ exp(γ Σ_{i~j} I(z_i = z_j))。
  • 用 MCMC 採樣。

還能做 sub-spot resolution enhancement：把每個 Visium spot 細分為更小的 sub-spot。

⑪ MERINGUE (Miller 2021, Genome Biology)

結合 spatial autocorrelation 與 cross-correlation。先用 Moran's I 找 SVG，再算 SVG pair 之間的 spatial cross-correlation，找出「在相同空間區域共表現」的基因模組。對 ligand-receptor 共定位特別有用。

⑫ Sparse / Inducing-point GP

標準 GP 訓練 O(n³)，記憶體 O(n²)，n > 5000 即不可行。Sparse GP (Titsias 2009)：選 m « n 個 inducing points u，用 q(f, u) ≈ p(u) q(f|u)，把複雜度降到 O(nm²)。對 Stereo-seq 等百萬 spot 資料必用。最新進展：SVGP (Hensman 2013)、Deep GP (Damianou 2013) 等。

① Spatial Weight Matrix Design

All spatial autocorrelation statistics need an n × n weight matrix W defining "who is whose neighbor". Common choices:

  • Rook contiguity: share an edge (4 neighbors on a grid). Good for regular grids like early array-based ST.
  • Queen contiguity: share edge or corner (8 neighbors). More sensitive to diagonal patterns.
  • Distance-based: w_{ij} = 1 if d_{ij} < threshold else 0. Adapts to irregular layouts but threshold must be tuned.
  • k-Nearest Neighbors: each spot connects to k nearest. Density-adaptive but W is non-symmetric (symmetrize). Visium typically uses k = 6 (hex grid).
  • Kernel-based: w_{ij} = exp(−d_{ij}² / σ²), continuous weights, bridges to Gaussian Processes.

Row-normalize W (each row sums to 1) for comparability across degree-heterogeneous nodes.

② Moran's I Null: Randomization vs Permutation

Moran's I = (n / S₀) · (xᵀ W x) / (xᵀ x) (centered). Two null hypotheses:

  • Normality (Normal H₀): x ~ N(μ, σ²) i.i.d., gives E[I] = −1/(n−1) and closed-form Var[I] (Cliff & Ord 1981). z-score works for large samples but breaks with outliers.
  • Randomization (Random H₀): value set fixed, locations permuted; compute empirical p over 999 permutations. Robust to non-normality but expensive for n > 10000.

For zero-inflated gene counts: (a) log1p normalize, (b) use permutation, (c) FDR via Benjamini-Hochberg.

③ Local Moran's I (LISA, Anselin 1995)

Global I gives one number; LISA decomposes to each spot: I_i = (x_i − x̄)/σ² · Σⱼ w_{ij}(x_j − x̄). Four quadrants:

  • HH: self high, neighbors high — hotspot
  • LL: self low, neighbors low — coldspot
  • HL: self high, neighbors low — outlier / discontinuity
  • LH: self low, neighbors high — reverse outlier

In ST: HH/LL find niche centers; HL/LH find boundaries or invasion fronts (e.g. tumor margins, wound edges).

④ Geary's C and Getis-Ord G*

  • Moran's I: most sensitive to mean-shift spatial patterns.
  • Geary's C: C = ((n−1) / 2S₀) · ΣᵢΣⱼ w_{ij}(x_i − x_j)² / Σᵢ(x_i − x̄)². Directly quantifies neighbor differences, more sensitive to local heterogeneity. Range [0, 2]; C < 1 ⇒ positive autocorrelation.
  • Getis-Ord G*: G_i* = Σⱼ w_{ij} x_j / Σⱼ x_j. Looks at absolute level, distinguishes high vs low clusters (Moran cannot). Common in crime hotspot analysis; useful in ST for marker-gene hotspot detection.

⑤ Variogram and Covariogram

γ(h) = ½ E[(Z(x+h) − Z(x))²]. Three key parameters:

  • Nugget (τ²): γ(0⁺), micro-scale variability + measurement error. ST dropout inflates nugget.
  • Sill (σ² + τ²): γ(∞), total variance.
  • Range (a): distance at which γ reaches sill — "spatial influence range". A gene with range = 50 µm only shows spatial structure within 50 µm.

Covariogram C(h) = Var(Z) − γ(h). Equivalent to variogram under stationary GP.

⑥ Choosing a Gaussian Process Kernel

Z(x) ~ GP(μ(x), k(x, x')). Kernel sets smoothness:

  • RBF / Squared Exponential: k(d) = σ² exp(−d²/2ℓ²). Infinitely smooth — often too smooth for biology.
  • Matérn 3/2: k(d) = σ²(1 + √3 d/ℓ) exp(−√3 d/ℓ). Once-differentiable, between exponential and RBF — good default for ST.
  • Matérn 5/2: k(d) = σ²(1 + √5 d/ℓ + 5d²/3ℓ²) exp(−√5 d/ℓ). Twice-differentiable.
  • Periodic: k(d) = σ² exp(−2 sin²(πd/p) / ℓ²). Captures periodic spatial patterns (cortical columns, hepatic lobules).
  • Composite: RBF + Periodic + Linear models global trend + regional structure + periodicity.

⑦ SpatialDE Math (Svensson 2018, Nature Methods)

For each gene g, fit two GPs:

  • H₁ (spatial): y_g(x) ~ GP(0, σ²_s K_s(x, x') + σ²_n I).
  • H₀ (non-spatial): y_g ~ N(0, σ²_n I), noise only.

MLE the parameters; LLR test gives p-value via χ², then BH-FDR. Bonus: each SVG returns its length scale ℓ_s indicating spatial scale.

⑧ SPARK (Sun 2020) and SPARK-X

SPARK uses Poisson / NB GLM + spatial random effects (vs SpatialDE's Gaussian assumption) — better for raw counts. Fits multiple kernels with different length scales as ensemble. SPARK-X (Zhu 2021, Genome Biology) is the scalable version using a covariance test instead of likelihood, O(n) — handles Slide-seqV2 million-spot data.

⑨ nnSVG (Weber 2023, Nat Commun)

Nearest-Neighbor GP (NNGP, Datta 2016): each spot models only the m nearest neighbors (e.g. m=10), O(nm³) not O(n³). Per-gene length scale ℓ_g distinguishes short- vs long-range spatial genes. Accurately layers cortical layer-specific genes on Visium DLPFC.

⑩ BayesSpace (Zhao 2021, Nat Biotechnol)

Not an SVG detector but a spatial-domain clusterer. Hierarchical Bayesian:

  • Layer 1: y_i ~ N(μ_{z_i}, Σ_{z_i}).
  • Layer 2: z follows a Potts model (Ising generalization), neighbors prefer same cluster: P(z) ∝ exp(γ Σ_{i~j} I(z_i = z_j)).
  • MCMC sampling.

Also performs sub-spot resolution enhancement (subdivides each Visium spot).

⑪ MERINGUE (Miller 2021, Genome Biology)

Combines spatial autocorrelation + cross-correlation. First Moran's I finds SVGs, then SVG pairs' spatial cross-correlation finds gene modules co-expressed in the same region. Especially useful for ligand-receptor co-localization.

⑫ Sparse / Inducing-point GP

Standard GP is O(n³) train, O(n²) memory — infeasible for n > 5000. Sparse GP (Titsias 2009): pick m « n inducing points u, q(f, u) ≈ p(u) q(f|u), reducing to O(nm²). Must for Stereo-seq's million spots. State of the art: SVGP (Hensman 2013), Deep GP (Damianou 2013).

詳細已知應用

Spatial Statistics 在生物資訊的具體案例

🧬 案例 1：Svensson et al. 2018, Nature Methods — SpatialDE on Mouse Olfactory Bulb

SpatialDE 原論文。在 Visium 前身的 ST (Stahl 2016) 資料上分析小鼠嗅球，找到 67 個 SVG (FDR < 0.05)，包括已知的 layer-specific markers（Olig1, Mbp）以及新發現的「短程 spatial pattern」基因。同時建立 FSV (Fraction of Spatial Variance) 指標：σ²_s / (σ²_s + σ²_n)，量化各基因的 spatial 結構強度。

🧬 案例 2：Sun et al. 2020, Nature Methods — SPARK on Mouse Cerebellum

SPARK 論文。在 mouse cerebellum Visium 資料上比 SpatialDE 多找出 40% SVG，包括 Calb1, Calb2 等 Purkinje 細胞特異基因。重要創新：用 multi-kernel ensemble（同時 fit Gaussian + cosine + linear kernels），對各種 spatial pattern 都敏感。Zhu et al. 2021 後續的 SPARK-X 把計算複雜度降到 O(n)，可處理 Slide-seqV2 等高解析度資料。

🧬 案例 3：Zhu et al. 2024, Genome Biology — SPARK-X on Slide-seqV2 Atlas

用 SPARK-X 分析 1.5 M spot 的 Slide-seqV2 mouse hippocampus，找到 3000+ SVG 並做 spatial clustering，揭示 dentate gyrus 不同 sublayer 的 marker。傳統 SpatialDE/SPARK 在這規模都 OOM；SPARK-X 在 1 個 CPU 約 30 分鐘完成。

🧬 案例 4：Weber et al. 2023, Nature Communications — nnSVG on Visium DLPFC

nnSVG 論文。在 Maynard 2021 的 12 個 DLPFC Visium 樣本上，nnSVG 比 SpatialDE/SPARK 對「跨樣本一致 SVG」的偵測更準確（measured by replicates' rank correlation）。重要創新：per-gene length scale 估計能區分「皮層 layer 特異」（短 ℓ）與「白質-灰質對比」（長 ℓ）基因，提供生物學上有用的「scale 註解」。

🧬 案例 5：Zhao et al. 2021, Nature Biotechnology — BayesSpace

BayesSpace 用 hierarchical Bayesian + Potts model 做 spatial clustering，並能 sub-spot enhancement（55 µm Visium → 25 µm sub-spot）。在 DLPFC, melanoma, breast cancer 資料上的 layer/domain 識別 ARI 都比 Seurat、Giotto 高 20–40%。後續還有 BayesTME (Zhang 2023) 整合 cell-type deconvolution。

🧬 案例 6：Miller et al. 2021, Genome Biology — MERINGUE

MERINGUE 結合 spatial autocorrelation 與 cross-correlation 找 ligand-receptor 共定位。在 Slide-seq 小鼠肝臟資料上找到 Igf1-Igf1r、Wnt2-Fzd1 等 paracrine 訊號軸，並對應到 portal vein vs central vein 不同 spatial zone。對 cell-cell communication 研究有獨特價值。

🧬 案例 7：DeTomaso & Yosef 2021, Cell Systems — Hotspot

把 spatial autocorrelation 推廣到 single-cell：把每個 cell 當作 node，邊由 KNN graph 在 PCA / latent space 上建構。然後對每個基因算「graph autocorrelation」，找 informative gene。可視為 scRNA-seq 的 Moran's I 推廣。最後用 hotspot graph 找「同變化」基因 module。已整合進 Scanpy/scvi-tools 生態系。

🧬 案例 8：Varrone et al. 2024, Nature Methods — CellCharter

用 spatial neighborhood enrichment 與 Gaussian Mixture Model + Markov Random Field 做 niche 分析。能自動找出「在空間上 enriched」的 cell-type combination（spatial niche），並對多病人 ST cohort 做 niche 比較。在 1000+ 病人胃腸道癌 ST atlas 上找到多個 prognostic niche。

🧬 案例 9：Palla et al. 2022, Nature Methods — Squidpy

Squidpy 是 ST 標準工具包，提供 Moran's I, Geary's C, Ripley's K, neighborhood enrichment 等多種 spatial statistics 介面。已整合到 Scanpy / AnnData 生態系。雖然不是新方法，但作為「ST analysis 統一接口」對社群影響極大。

🧬 案例 10：Kats et al. 2022, bioRxiv — SpatialDE2

SpatialDE 升級版，整合多模態（gene expression + cell type + morphology），並對多切片做 joint inference。在多 batch ST 資料上比 SpatialDE 更穩健，且能找到「跨切片一致」的 SVG。

🧬 Case 1: Svensson et al. 2018, Nature Methods — SpatialDE on Mouse Olfactory Bulb

The original SpatialDE paper. On mouse olfactory bulb (Stahl 2016 ST data), found 67 SVGs (FDR < 0.05) including known layer-specific markers (Olig1, Mbp) and short-range spatial genes. Introduced FSV = σ²_s / (σ²_s + σ²_n) to quantify spatial structure strength.

🧬 Case 2: Sun et al. 2020, Nature Methods — SPARK on Mouse Cerebellum

The SPARK paper. On mouse cerebellum Visium, found 40% more SVGs than SpatialDE, including Calb1/Calb2 Purkinje markers. Key innovation: multi-kernel ensemble (Gaussian + cosine + linear) sensitive to varied spatial patterns. Follow-up SPARK-X (Zhu 2021) lowered cost to O(n) for high-res data.

🧬 Case 3: Zhu et al. 2024, Genome Biology — SPARK-X on Slide-seqV2 Atlas

SPARK-X on 1.5M-spot Slide-seqV2 hippocampus, found 3000+ SVGs revealing sublayer markers in dentate gyrus. SpatialDE / SPARK OOM at this scale; SPARK-X finishes in ~30 min on one CPU.

🧬 Case 4: Weber et al. 2023, Nature Communications — nnSVG on Visium DLPFC

The nnSVG paper. On Maynard 2021's 12 DLPFC samples, more accurate cross-sample SVG detection than SpatialDE/SPARK (measured by replicate rank correlation). Per-gene length scale separates layer-specific (short ℓ) from white-gray contrast (long ℓ) genes — biologically useful scale annotation.

🧬 Case 5: Zhao et al. 2021, Nature Biotechnology — BayesSpace

Hierarchical Bayesian + Potts spatial clustering, plus sub-spot enhancement (55 µm → 25 µm). 20–40% higher ARI on DLPFC, melanoma, breast cancer vs Seurat/Giotto. Follow-up BayesTME (Zhang 2023) integrates cell-type deconvolution.

🧬 Case 6: Miller et al. 2021, Genome Biology — MERINGUE

Combines spatial autocorrelation + cross-correlation for L-R co-localization. Found Igf1-Igf1r, Wnt2-Fzd1 paracrine axes on Slide-seq liver, matched to portal vs central vein zones. Unique value for cell-cell communication.

🧬 Case 7: DeTomaso & Yosef 2021, Cell Systems — Hotspot

Generalizes spatial autocorrelation to single-cell: cells are nodes, edges from KNN graph in PCA/latent space. Computes "graph autocorrelation" per gene to find informative genes — a Moran's I for scRNA-seq. Then finds co-varying gene modules. Integrated into Scanpy/scvi-tools.

🧬 Case 8: Varrone et al. 2024, Nature Methods — CellCharter

Spatial neighborhood enrichment + Gaussian Mixture Model + Markov Random Field for niche analysis. Automatically finds spatially-enriched cell-type combinations (niches) and compares across multi-patient ST cohorts. Identified prognostic niches on 1000+ GI cancer ST atlas.

🧬 Case 9: Palla et al. 2022, Nature Methods — Squidpy

Squidpy is the standard ST toolkit: Moran's I, Geary's C, Ripley's K, neighborhood enrichment APIs, integrated with Scanpy/AnnData. Not a new method, but the unified interface massively boosted ST community adoption.

🧬 Case 10: Kats et al. 2022, bioRxiv — SpatialDE2

SpatialDE upgrade integrating multi-modal data (gene expression + cell type + morphology), joint inference across slices. More robust to batch effects, finds "cross-slice consistent" SVGs.

潛在改良應用

Spatial Statistics 還能改良哪些任務？

🎯 1. Sub-cellular Spatial Transcriptomics 的 GP 解析

現有限制：SpatialDE/SPARK 設計給 Visium (55 µm spot)；對 Stereo-seq、MERFISH 等 sub-cellular 解析度資料，spot 間距離很小，傳統 length scale 估計不穩定。新提案：設計 multi-scale Matérn kernel：k(d) = Σ_k w_k Matérn_νk(d / ℓ_k)，同時 fit 多個 length scale；用 sparse GP 加速。預期改進：能同時找出「胞器尺度」（< 1 µm，如核仁基因）與「組織尺度」（> 100 µm，如皮層 layer）兩種 spatial pattern，傳統方法無法同時偵測。

🎯 2. Multi-scale Spatial Autocorrelation

現有限制：Moran's I 只給一個 scale 的整體統計，無法說「這個基因在小尺度 (10 µm) 自相關但大尺度 (100 µm) 隨機」。新提案：用 wavelet decomposition + spatial autocorrelation：對 expression field 做 spatial wavelet，每個 scale 各算一個 Moran's I。預期改進：產生 multi-scale autocorrelation spectrum，類似時間序列的 wavelet coherence。能找出 nested spatial pattern（如 tumor 內的免疫 hotspot 又包含更細的 T cell sub-cluster）。

🎯 3. 疾病 vs 健康 Spatial Pattern 對比

現有限制：多數 SVG 偵測在單一條件內做；跨條件比較通常是「在各條件分別找 SVG 再取交集/差集」，缺乏統計嚴謹性。新提案：設計 differential spatial pattern test：H₀ = spatial structure 在 disease vs healthy 相同；H₁ = 兩條件 spatial pattern 不同。用 multi-condition GP 或 Bayesian model comparison。預期改進：能找到「健康時 spatially organized 但疾病時 spatial 結構崩潰」的基因——這類 disease-disrupted spatial pattern 是潛在診斷標誌。

🎯 4. Dynamic Spatial Pattern（多時間點 ST）

現有限制：STARmap、CONFINIT 等開始有 time-course ST，但目前分析各時間點獨立做 SVG，無時序整合。新提案：spatio-temporal GP：k((x,t), (x',t')) = k_spatial(x, x') · k_time(t, t')；可分離 spatial vs temporal length scale，並偵測「spatial pattern 隨時間演化」的基因。預期改進：能量化「pattern 形成速度」，對發育、傷口癒合、腫瘤侵襲時序研究關鍵。

🎯 5. 三維 Spatial GP（連續切片 3D 重建）

現有限制：ST 多為 2D 切片；3D 整合（如 Wang et al. 3D STARmap）目前用 alignment 後再 2D 分析，沒有真正的 3D spatial structure 偵測。新提案：3D GP with anisotropic kernel：k(d_x, d_y, d_z) 在 z 方向（切片間）可有不同 length scale。可整合 H&E 影像做 anatomical alignment。預期改進：找出「真正三維 niche」，例如腎絲球的球狀結構、神經元 dendritic field。對器官級理解（whole-organ atlas）重要。

🎯 6. 跨病人 Spatial Pattern 對齊

現有限制：每個病人 ST 的座標系不同，不能直接做 spatial 比較。CellCharter 用 niche label 比較，但不保留 spatial 連續性。新提案：用 functional Bayesian alignment：把每個病人的 SVG field 視為「空間函數」，用 functional PCA / GP registration 對齊到 canonical 坐標系。預期改進：能做 population-level spatial 分析，例如「100 個乳癌病人 ST 的腫瘤邊界區域共同表現哪些基因」，遠超目前 cell-type 比較的層次。

🎯 7. nnSVG + RNA Velocity = 動態 Spatial Hotspot

現有限制：nnSVG 找 spatial 結構，scVelo 找 RNA velocity，但分開做，無法回答「spatial pattern 是穩態還是動態形成中」。新提案：對 velocity vector field（每 spot 有 ds/dt）跑 spatial autocorrelation：高 Moran's I 表示「velocity 也有 spatial 結構」，這就是「動態 hotspot」（不只表現高，還在快速變化的區域）。預期改進：找出「即將分化」、「即將遷移」的 spatial niche，對 wound healing、發育時序研究極有價值。

🎯 8. 用 Spatial Autocorrelation 修正 Cell-Cell Communication

現有限制：CellChat、NicheNet 用平均表現算 L-R signaling，但這假設 ligand 與 receptor 細胞「均勻混合」。實際上若 ligand 細胞群集在某 spatial 區域，rest 細胞接觸不到 ligand，CellChat 會高估 communication。新提案：用 spatial colocalization-weighted CellChat：計算 ligand-receptor pair 的 spatial cross-correlation（如 MERINGUE），用以加權 CellChat 的 LR score。預期改進：能區分「真正空間鄰近的 paracrine 訊號」vs「兩細胞型別都表現但物理上分開」的假訊號，大幅減少 false positive。

🎯 9. H&E Imaging + Spatial GP 多模態整合

現有限制：多數 SVG 方法只用 ST expression，忽略 H&E 提供的組織結構資訊。DeepST 用 image，但是 GNN-based，缺乏 GP 的不確定性量化。新提案：multi-output GP：兩個 output channel（expression + image embedding），共享 spatial kernel。可整合 cross-modal correlation。預期改進：能找「組織學上明顯但 expression 弱訊號」的 niche（例如某些 cancer stroma），並提供 uncertainty interval 指導後續實驗。

🎯 10. Senescence 空間漸層分析

現有限制：傳統 senescence 研究用 bulk 或單細胞，不知道 senescent cell 在組織中的 spatial pattern。新提案：用 Local Moran's I 在 senescence score（如 SenMayo gene signature）上找 senescence hotspot；用 GP 量化「senescence 隨距離某 niche 的漸層」（如離血管的距離）。預期改進：能找出 senescence「擴散源頭」，對 anti-aging therapy 設計（如 senolytic drug 局部投藥）有實用意義。

🎯 11. Tumor Margin 的 GP-based 自動偵測

現有限制：病理學家手動標 tumor margin 費時且主觀；BayesSpace 等用 hard cluster，邊界不平滑。新提案：用 GP-based segmentation：把 tumor probability 視為 GP，邊界 = probability gradient 最大處。GP 能給出 boundary uncertainty band。預期改進：提供「定量的、含信心區間的」tumor margin，並可量化「margin 寬度」與預後關聯——這在腫瘤手術切除邊界規劃有臨床價值。

🎯 1. GP-based Sub-cellular Spatial Transcriptomics

Current limit: SpatialDE/SPARK designed for Visium (55 µm). For Stereo-seq, MERFISH at sub-cellular resolution, spot spacing is tiny and length-scale estimation is unstable. Proposal: multi-scale Matérn kernel k(d) = Σ_k w_k Matérn_νk(d / ℓ_k), jointly fitting multiple length scales with sparse GP for speed. Expected gain: simultaneously detect organelle-scale (< 1 µm, e.g. nucleolar genes) and tissue-scale (> 100 µm, cortical layers) spatial patterns — currently impossible to do both.

🎯 2. Multi-scale Spatial Autocorrelation

Current limit: Moran's I gives one number per gene, can't say "autocorrelated at 10 µm but random at 100 µm". Proposal: spatial wavelet decomposition + per-scale Moran's I, producing a multi-scale autocorrelation spectrum (analogous to wavelet coherence). Expected gain: reveal nested spatial patterns — e.g. an immune hotspot in a tumor that itself contains finer T-cell sub-clusters.

🎯 3. Differential Spatial Pattern: Disease vs Healthy

Current limit: SVG detection is within-condition; cross-condition comparison is usually "find SVGs separately and intersect", lacking rigor. Proposal: differential spatial pattern test — H₀: same spatial structure in disease vs healthy; H₁: different. Multi-condition GP or Bayesian model comparison. Expected gain: find genes whose spatial organization is disrupted in disease — potential diagnostic markers.

🎯 4. Dynamic Spatial Patterns (time-course ST)

Current limit: STARmap, CONFINIT now offer time-course ST; current analysis does SVG per time point with no temporal integration. Proposal: spatio-temporal GP k((x,t), (x',t')) = k_spatial(x, x') · k_time(t, t'), separable spatial vs temporal length scales, detect "pattern evolves over time" genes. Expected gain: quantify "pattern formation rate" — key for development, wound healing, tumor invasion.

🎯 5. 3D Spatial GP for Serial-section Reconstruction

Current limit: ST is mostly 2D; 3D works (e.g. 3D STARmap) align then analyze in 2D — no true 3D spatial detection. Proposal: 3D GP with anisotropic kernel k(d_x, d_y, d_z), different z-direction length scale. Integrate H&E for anatomical alignment. Expected gain: identify truly 3D niches like glomerular spheres or neuronal dendritic fields — essential for whole-organ atlases.

🎯 6. Cross-patient Spatial Pattern Alignment

Current limit: each patient's ST has its own coordinate frame, can't be directly compared. CellCharter uses niche labels but loses spatial continuity. Proposal: functional Bayesian alignment — treat each patient's SVG field as a function, use functional PCA / GP registration to a canonical frame. Expected gain: population-level spatial analysis — e.g. "in 100 breast cancer ST, what genes are co-expressed at tumor margins?" — far richer than current cell-type comparisons.

🎯 7. nnSVG + RNA Velocity = Dynamic Spatial Hotspot

Current limit: nnSVG finds spatial structure, scVelo finds velocity — separately, can't say "is the spatial pattern steady or actively forming?" Proposal: run spatial autocorrelation on the velocity vector field (ds/dt per spot); high Moran's I ⇒ velocity itself is spatially structured ⇒ a dynamic hotspot. Expected gain: identify niches that are "about to differentiate" or "about to migrate" — invaluable for wound healing and developmental kinetics.

🎯 8. Spatial-aware Correction of Cell-Cell Communication

Current limit: CellChat/NicheNet compute L-R signaling from mean expression, assuming uniform mixing. If ligand cells cluster spatially, others never contact ligand — CellChat overestimates. Proposal: spatial colocalization-weighted CellChat — compute LR pair spatial cross-correlation (à la MERINGUE), weight CellChat scores accordingly. Expected gain: distinguish genuinely adjacent paracrine signals from "both cell types express but are physically separated", dramatically reducing false positives.

🎯 9. Multi-modal H&E + Spatial GP

Current limit: most SVG methods use only ST expression, ignore H&E tissue context. DeepST uses image but is GNN-based, lacking GP uncertainty quantification. Proposal: multi-output GP — two channels (expression + image embedding) sharing a spatial kernel; integrate cross-modal correlation. Expected gain: find niches with strong histology but weak expression signal (e.g. some cancer stroma), with uncertainty intervals to guide experiments.

🎯 10. Spatial Senescence Gradient Analysis

Current limit: classical senescence studies use bulk or single-cell, ignoring tissue spatial pattern. Proposal: Local Moran's I on senescence scores (e.g. SenMayo signature) finds senescence hotspots; GP quantifies gradient with distance from a niche (e.g. from vasculature). Expected gain: identify senescence diffusion sources, practical for anti-aging therapy (localized senolytic delivery).

🎯 11. GP-based Automatic Tumor Margin Detection

Current limit: manual margin annotation is laborious and subjective; BayesSpace uses hard clusters with non-smooth boundaries. Proposal: GP-based segmentation — model tumor probability as a GP, boundary = max-gradient location, with uncertainty band. Expected gain: quantitative tumor margins with confidence intervals, supporting margin-width vs prognosis studies — clinically valuable for surgical planning.
