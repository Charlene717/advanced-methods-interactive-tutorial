Energy-Based Models 與 Direct Coupling Analysis

EBM 把任意函數轉成機率：p(x) ∝ exp(−E(x))。應用到 protein 演化序列分析就是 Direct Coupling Analysis (DCA)——從 MSA 解出殘基間的直接耦合，重建蛋白接觸圖。Morcos 2011 (PNAS) 的 DCA 是 AlphaFold 之前最強的接觸預測法，也是 EVcouplings、GREMLIN、ESM-MSA 的基礎。

EBM turns any function into a probability: p(x) ∝ exp(−E(x)). Applied to protein evolutionary sequences, this is Direct Coupling Analysis (DCA) — extracting direct residue-residue couplings from an MSA to reconstruct contact maps. Morcos 2011 (PNAS)'s DCA was the strongest contact predictor before AlphaFold, and underlies EVcouplings, GREMLIN, and ESM-MSA.

核心概念

EBM 的統一視角與 DCA 的特例

Energy-Based Model (LeCun 2006)：選一個能量函數 E_θ(x) : ℝ^d → ℝ，定義機率

p_θ(x) = exp(−E_θ(x)) / Z(θ)，   Z(θ) = ∫ exp(−E_θ(x)) dx

低能量 = 高機率。強大之處：E_θ 可以是任意 NN，無需 normalization 約束（VAE / flow 都需要）。麻煩之處：Z 通常算不出來；訓練與採樣都需要近似（contrastive divergence、Langevin、score matching）。

Boltzmann machine：EBM 的歷史前身。對 binary x ∈ {0,1}^d，E(x) = −Σ h_i x_i − Σ J_ij x_i x_j——即「外場 + 兩體耦合」的 Ising / Potts 模型。所以 EBM 本質就是廣義的 Potts/Ising。

Direct Coupling Analysis (Morcos 2011, PNAS)：對蛋白 MSA（每列一物種、每行一殘基位置），fit 一個 Potts 模型

p(σ₁, ..., σ_L) ∝ exp(Σ_i h_i(σ_i) + Σ_(i<j) J_ij(σ_i, σ_j))

其中 σ_i ∈ {20 amino acids + gap}。J_ij 大就代表「殘基 i 與 j 直接耦合」，通常表示空間接觸（或功能性 epistasis）。一階共變 (MI 或 mutual info) 會被 indirect chains 污染；DCA 用全模型 fit 把 direct vs indirect 分開——這就是「Direct」的由來。

Energy-Based Model (LeCun 2006): pick an energy function E_θ(x) : ℝ^d → ℝ; define

p_θ(x) = exp(−E_θ(x)) / Z(θ),   Z(θ) = ∫ exp(−E_θ(x)) dx

Low energy = high probability. Strength: E_θ can be any NN — no normalization constraint (unlike VAE / flow). Pain: Z is generally intractable; training and sampling rely on approximations (contrastive divergence, Langevin, score matching).

Boltzmann machine: historical ancestor. For binary x ∈ {0,1}^d, E(x) = −Σ h_i x_i − Σ J_ij x_i x_j — external field + pairwise coupling, an Ising / Potts model. Hence EBM is essentially a generalized Potts/Ising.

Direct Coupling Analysis (Morcos 2011, PNAS): on a protein MSA (rows = species, columns = residue positions), fit a Potts model

p(σ₁, ..., σ_L) ∝ exp(Σ_i h_i(σ_i) + Σ_(i<j) J_ij(σ_i, σ_j))

with σ_i ∈ {20 AAs + gap}. Large J_ij ⇒ direct coupling between residues i and j, usually spatial contact (or functional epistasis). Pairwise co-evolution metrics (e.g., MI) get contaminated by indirect chains. DCA's full-model fit cleanly separates direct from indirect — hence "Direct".

演算法

EBM 訓練、Langevin 採樣、mfDCA / plmDCA

① EBM 訓練：Maximum Likelihood

∇_θ log p_θ(x) = −∇_θ E_θ(x) + 𝔼_(x'∼p_θ) [∇_θ E_θ(x')]

第一項好算（資料樣本上的梯度）；第二項要對 model distribution 做期望——這要 sampling，通常是麻煩來源。

② Contrastive Divergence (Hinton 2002, Neural Comp.)：跑 k 步 MCMC（k 通常很小 ~1）從資料點出發，得到 x'；用 x' 近似 model expectation。雖然 biased，但實務中 work。

③ Langevin 採樣：連續 EBM 用 Langevin dynamics：

x_{k+1} = x_k − (ε/2) ∇_x E_θ(x_k) + √ε · z_k，   z_k ~ N(0, I)

k → ∞ 時 x_k → p_θ。可作為「神經 EBM」的標準採樣器（如 IGEBM, Du 2019）。

④ Score Matching (Hyvärinen 2005)：避開 Z 直接學 ∇log p_θ = −∇E_θ。denoising score matching（Vincent 2011）是 diffusion 的根源。

⑤ DCA 的兩大實作：

  • mfDCA (mean-field, Morcos 2011)：Boltzmann inversion 近似——估計經驗 covariance C，取 J ≈ −C⁻¹。快速但近似。
  • plmDCA (pseudo-likelihood, Ekeberg 2013, Phys. Rev. E)：對每個 column 做 conditional logistic regression（給定其他位置預測當前位置），組合得到 J。準確度比 mfDCA 高，現代 DCA 主流。
  • bmDCA (Boltzmann machine learning)：真正 MCMC fit，最精確但慢。

⑥ 接觸分數：對 J_ij ∈ ℝ^(q×q)（q=21），取 Frobenius norm

F_ij = ‖J_ij‖_F

並做 Average Product Correction (APC) 去除背景熵：F'_ij = F_ij − (F_i · F_j / F̄)。F'_ij 最大的 (i,j) pair 即預測接觸。

① EBM training: Maximum Likelihood

∇_θ log p_θ(x) = −∇_θ E_θ(x) + 𝔼_(x'∼p_θ) [∇_θ E_θ(x')]

The first term is easy (data-sample gradient); the second requires expectation under the model — sampling is the bottleneck.

② Contrastive Divergence (Hinton 2002, Neural Comp.): run k-step MCMC (k often 1) from data points to get x'; approximate model expectation with x'. Biased but practical.

③ Langevin sampling: for continuous EBMs, use Langevin dynamics:

x_{k+1} = x_k − (ε/2) ∇_x E_θ(x_k) + √ε · z_k,   z_k ~ N(0, I)

k → ∞ ⇒ x_k → p_θ. Used as the standard sampler for neural EBMs (e.g., IGEBM, Du 2019).

④ Score Matching (Hyvärinen 2005): sidestep Z by learning ∇log p_θ = −∇E_θ directly. Denoising score matching (Vincent 2011) became the foundation of diffusion models.

⑤ Two main DCA implementations:

  • mfDCA (mean-field, Morcos 2011): Boltzmann inversion — estimate empirical covariance C, set J ≈ −C⁻¹. Fast but approximate.
  • plmDCA (pseudo-likelihood, Ekeberg 2013, Phys. Rev. E): conditional logistic regression per column (predict residue given others); combine into J. More accurate than mfDCA; modern default.
  • bmDCA (Boltzmann machine learning): true MCMC fit; most accurate but slow.

⑥ Contact score: for J_ij ∈ ℝ^(q×q) (q=21), take Frobenius norm

F_ij = ‖J_ij‖_F

then apply Average Product Correction (APC) to remove background entropy: F'_ij = F_ij − (F_i · F_j / F̄). Top (i,j) pairs by F' predict spatial contacts.

歷史脈絡：共變分析的概念可追溯到 Göbel et al. 1994 (Proteins)，但早期方法只用 pairwise 統計（如 mutual information），會被 indirect coupling 嚴重污染。Weigt et al. 2009 (PNAS) 提出 mfDCA 雛形；Morcos 2011 系統化並用於 ~150 PDB 接觸預測。AlphaFold（2018, 2020）後 DCA 不再是 contact 預測 SOTA，但仍是 ESM-MSA、MSA Transformer 等模型的概念基礎。
Historical context: Co-evolutionary analysis dates back to Göbel et al. 1994 (Proteins), but early pairwise statistics (e.g., MI) were contaminated by indirect couplings. Weigt 2009 (PNAS) introduced mfDCA; Morcos 2011 systematized it for ~150 PDB contact predictions. After AlphaFold (2018, 2020), DCA is no longer SOTA for contacts, but it remains the conceptual basis of ESM-MSA, MSA Transformer, etc.

生物應用

EBM & DCA 在蛋白與基因組

EVcouplings — 變異效應預測

Hopf et al. (2017, Nat. Biotech.) 用 plmDCA fit MSA 後，能量差 ΔE(mutant, WT) 直接預測 missense variant 的功能影響（fitness）。在 deep mutational scanning benchmark 與 clinical variant 上至今仍是強 baseline。

Hopf et al. (2017, Nat. Biotech.) fit plmDCA to an MSA; energy gap ΔE(mutant, WT) directly predicts missense variant fitness. Still a strong baseline on deep mutational scanning and clinical variant benchmarks.

GREMLIN — 接觸與蛋白家族

Kamisetty et al. (2013, PNAS) 提出 GREMLIN，採 group-L1 正則化 plmDCA，在 Pfam family contacts 預測上比 mfDCA 大幅改進。也是 RoseTTAFold 早期 trRosetta 的訓練輸入之一。

Kamisetty et al. (2013, PNAS) introduced GREMLIN with group-L1 regularized plmDCA — major improvement over mfDCA on Pfam contacts. Also fed into early trRosetta / RoseTTAFold.

AlphaFold 與 ESM-MSA

AlphaFold2 (Jumper 2021, Nature) 透過 MSA Transformer attention 學 coevolution，但其概念可追溯到 DCA。ESM-MSA (Rao 2021, ICML) 用 row + column attention 替代 plmDCA，效果更好且不需 explicit pairwise feature。

AlphaFold2 (Jumper 2021, Nature) learns coevolution via MSA Transformer attention, conceptually rooted in DCA. ESM-MSA (Rao 2021, ICML) replaces plmDCA with row/column attention, outperforming explicit pairwise features.

ProteinMPNN — 蛋白序列設計

Dauparas et al. (2022, Science) 的 ProteinMPNN 用 autoregressive 設計 sequence 給定 structure。同期 Anand 2022、Frey 2024 把 EBM 應用於 sequence-structure joint energy，可條件式生成 functional protein。

Dauparas et al. (2022, Science) — ProteinMPNN autoregressively designs sequences given structure. Anand 2022 / Frey 2024 push further with explicit EBM over joint sequence-structure energy for conditional functional protein generation.

RNA 結構 — DCA 變種

De Leonardis et al. (2015, NAR) 把 DCA 拓展到 RNA MSA，用 4-letter alphabet (A,U,G,C) 預測 base-pair 接觸。在 16S/23S rRNA 與 ncRNA 上 work，是 RNAcontact 等模型的前身。

De Leonardis et al. (2015, NAR) extended DCA to RNA MSAs (A, U, G, C alphabet) for base-pair contacts. Worked on 16S/23S rRNA and ncRNA; precursor to RNAcontact-style models.

轉錄因子結合的 EBM

DeepBind (Alipanahi 2015, Nat. Biotech.) 雖採 CNN，但其分數函數可解讀為 EBM。後續 BPNet、Enformer 將其推廣到全基因組——能量函數 → binding probability。

DeepBind (Alipanahi 2015, Nat. Biotech.) — although CNN-based, its score is interpretable as an EBM. BPNet, Enformer extend this genome-wide: energy → binding probability.

深入解析

EBM 與 DCA 進階主題

① Partition function Z 的詛咒

Z = ∫ exp(−E_θ(x)) dx 對任何 NN-based E_θ 都不可解析。所有實務 EBM 訓練都繞著「如何避開 Z」展開：CD-k 用 short-chain MCMC、score matching 對 ∇E 而非 E、noise-contrastive estimation (Gutmann 2010) 把 EBM 訓練變成二元分類。

② 與 Diffusion 的本質關係

Diffusion model 學的是 ∇log p_t(x)，這就是 −∇E_t(x)：每個 noise level 都對應一個 EBM。Diffusion 的 success 啟示：把 EBM 學成一族隨時間平滑的能量比直接學單一 sharp 能量好。Song & Ermon (2019, NeurIPS) 的 NCSN 把這點明確化。

③ DCA 的 epistasis 詮釋

J_ij 大不僅可能是 spatial contact，也可能是 functional epistasis（如 active site 內遠程協同）。真正的接觸 → 短程 J；epistasis → 可以遠程。EVcouplings 同時利用兩者預測變異效應。

④ MSA depth 與 effective sequences

DCA 需要 large MSA（通常 N_eff > 100 為佳）。N_eff 計算需做 sequence reweighting（80% identity 內合併）。對 orphan family（少同源）DCA 失效——這也是 ESM-2 等 language model 出現的動機（從單序列推結構）。

⑤ Pseudo-likelihood 的數學

真正的 likelihood p(σ) = (1/Z) exp(−E)；pseudo-likelihood 取代為 Π_i p(σ_i | σ_{−i})——每個位置在「給定其它位置」的條件機率，無需 Z（因為每個 conditional 都有自己的 normalization Σ_{σ_i})）。Ekeberg 2013 證明 plmDCA consistent under mild conditions。

⑥ Group sparsity 正則化

J_ij ∈ ℝ^(q×q) 是 21×21 矩陣；用 group-L1 (Frobenius norm penalty) 讓「整個 J_ij 一起為零」——避免單個 entry 為 noise 而 matrix 仍記入。這是 GREMLIN 比 mfDCA 高一截的關鍵。

⑦ APC 與背景訊號

Phylogenetic bias 與保守度差異會在 J_ij 上引入「背景」訊號——某些行 entropy 一致偏高。Average Product Correction (APC, Dunn 2008)：F'_ij = F_ij − (F_i· · F_·j / F̄)，把每行每列的平均強度當背景扣除。常見的進一步修正還包括 PSI-APC 與 sequence weighting。

⑧ 從 DCA 到 MSA Transformer

plmDCA 等價於「對每個 column 做 softmax logistic regression with linear features」。MSA Transformer (Rao 2021, ICML) 用 attention 學「列方向 ✕ 行方向」的非線性互動——可視為 plmDCA 的非線性、共享參數版本。AF2 / RoseTTAFold 沿用此 attention 結構。

① The partition-function curse

Z = ∫ exp(−E_θ(x)) dx is intractable for any NN-based E_θ. All practical EBM training centers on circumventing Z: CD-k uses short MCMC, score matching learns ∇E instead of E, noise-contrastive estimation (Gutmann 2010) reduces to binary classification.

② Essential relation to diffusion

Diffusion models learn ∇log p_t(x) = −∇E_t(x): each noise level corresponds to an EBM. Insight: learning a smooth family of energies over time beats a single sharp energy. Song & Ermon (2019, NeurIPS)'s NCSN crystallized this.

③ DCA's epistasis reading

Large J_ij is not only spatial contact but possibly functional epistasis (e.g., long-range cooperativity in an active site). True contacts → short-range J; epistasis → can be long-range. EVcouplings leverages both.

④ MSA depth and effective sequences

DCA needs deep MSAs (N_eff > 100 is a good rule). N_eff requires sequence reweighting (merge ≥80% identity). DCA fails for orphan families — motivating ESM-2 etc. (single-sequence structure inference).

⑤ Pseudo-likelihood math

True likelihood p(σ) = exp(−E)/Z; pseudo-likelihood uses Π_i p(σ_i | σ_{−i}) — each position conditional on others, requiring no Z (each conditional has its own normalization Σ_{σ_i}). Ekeberg 2013 proved consistency of plmDCA under mild conditions.

⑥ Group-sparsity regularization

J_ij ∈ ℝ^(q×q) is a 21×21 matrix; group-L1 (Frobenius penalty) makes the entire J_ij zero when uninformative — avoids individual noisy entries leaking through. Key to GREMLIN beating mfDCA.

⑦ APC and background signal

Phylogenetic bias and conservation differences inject background structure into J_ij. Average Product Correction (APC, Dunn 2008): F'_ij = F_ij − (F_i· · F_·j / F̄). Further refinements: PSI-APC and sequence reweighting.

⑧ From DCA to MSA Transformer

plmDCA is equivalent to "softmax logistic regression with linear features per column". MSA Transformer (Rao 2021, ICML) uses attention over "row × column" with nonlinear shared params — a nonlinear, parameter-sharing plmDCA. AF2 / RoseTTAFold extend this attention.

潛在改良應用

還能改良哪些任務？

🎯 1. 抗體 paratope-epitope coupling

把 antibody-antigen pair MSA 做 cross-DCA，挑出 paratope-epitope 接觸對。提案：整合 OAS (Observed Antibody Space) 抗體庫，可用於疫苗 epitope 設計。

🎯 2. RNA-蛋白互動的 cross-DCA

RBPs (RNA-binding proteins) 與 target mRNA 共演化。提案：對 paralog 對應的 protein–RNA MSA 做 cross-DCA 找 contact，補充 CLIP-seq 結構訊息。

🎯 3. cis-regulatory module 的能量模型

把 TF binding 與 chromatin accessibility 視為 EBM。提案：用 Enformer-style + 顯式 EBM 學「sequence + cellular context → expression」的能量，可生成 synthetic regulatory sequences。

🎯 4. scLLM 訓練的 EBM 補充

scGPT / Geneformer 採 MLM 目標；可加入 EBM-style contrastive loss（正樣本 vs corrupted）。提案：scEBM joint training，提升 perturbation 預測。

🎯 5. Variant effect prediction 的多任務 EBM

EVcouplings 只用 single protein MSA；可整合「sequence + structure + cellular context」EBM 預測 missense / synonymous variant 在不同 tissue 的效應。提案：MultiEBM-Variant。

🎯 6. 蛋白設計的「inverse DCA」

給定 target functional energy profile（如「能結合 ATP，不結合 GTP」），找出符合的 sequence。提案：結合 plmDCA + gradient-based sequence optimization。

🎯 7. 單細胞動態的 EBM

細胞 trajectory 看成 stochastic process on energy landscape (Waddington)。提案：用 neural EBM 學 cell-state energy，可解讀 attractor、bifurcation；與 RNA velocity 互補。

🎯 8. 抗藥性突變的 DCA-guided 預測

結合 plmDCA 與藥物結合能量。提案：給定 target protein + drug, 預測 J 上「最可能突變且仍保功能」的位點——可預測 resistance hotspots。

🎯 1. Antibody paratope-epitope coupling

Cross-DCA on antibody-antigen MSA pairs to identify paratope-epitope contacts. Proposal: integrate OAS (Observed Antibody Space) for epitope-driven vaccine design.

🎯 2. RNA-protein interaction via cross-DCA

RBPs and target mRNAs coevolve. Proposal: cross-DCA on paralog-matched protein-RNA MSAs to find contacts, complementing CLIP-seq.

🎯 3. cis-regulatory module energy model

Treat TF binding + chromatin accessibility as EBM. Proposal: Enformer-style + explicit EBM for "sequence + cell context → expression"; enables synthetic regulatory sequence design.

🎯 4. EBM augmentation for scLLM training

scGPT / Geneformer use MLM; add EBM-style contrastive loss (positive vs corrupted). Proposal: scEBM joint training for better perturbation prediction.

🎯 5. Multi-task EBM for variant effect

EVcouplings uses one MSA only. Integrate sequence + structure + cellular context EBM to predict missense/synonymous effects per tissue. Proposal: MultiEBM-Variant.

🎯 6. "Inverse DCA" for protein design

Given a target functional energy profile (e.g., "binds ATP but not GTP"), find compatible sequences. Proposal: plmDCA + gradient-based sequence optimization.

🎯 7. Single-cell dynamics as EBM

Cell trajectories as stochastic processes on a Waddington-style energy landscape. Proposal: neural EBM for cell-state energy — interpretable attractors and bifurcations; complements RNA velocity.

🎯 8. DCA-guided drug resistance prediction

Combine plmDCA with drug binding energy. Proposal: given target protein + drug, predict sites where mutations are likely and retain function — identifies resistance hotspots.

    biomolecule='protein',

for i, j, s in contacts[:20]:

from evcouplings.couplings import CouplingsModel

# Loads plmDCA-trained model file (.model)
m = CouplingsModel('PF00071.model')

# 1) Predicted contacts (sorted by CN/APC score)
df = m.ecs                        # DataFrame: i, j, score
top_contacts = df.head(100)

# 2) Mutation effect: ΔE between mutant and WT
# Hamiltonian = -(Σ h + Σ J) → larger H = less probable
wt_seq = 'MAVKL...'
mut_seq = 'MAVKM...'     # single mutation at pos 4
H_wt  = m.hamiltonians([wt_seq])[0, 0]
H_mut = m.hamiltonians([mut_seq])[0, 0]
print(f"ΔE = {H_mut - H_wt:.3f}")         # > 0 ⇒ deleterious
# Neural EBM with Langevin sampling — Du & Mordatch 2019 style
import torch, torch.nn as nn

class EBM(nn.Module):                       # E_theta(x)
    def __init__(self, d=2):
        super().__init__()
        self.net = nn.Sequential(
            nn.Linear(d, 256), nn.SiLU(),
            nn.Linear(256, 256), nn.SiLU(),
            nn.Linear(256, 1))
    def forward(self, x):
        return self.net(x).squeeze(-1)

def langevin(model, x, steps=60, eps=0.01):
    x = x.clone().requires_grad_(True)
    for _ in range(steps):
        E = model(x).sum()
        grad = torch.autograd.grad(E, x)[0]
        x = x.detach() - 0.5*eps*grad + (eps**0.5)*torch.randn_like(x)
        x = x.clamp(-3, 3).requires_grad_(True)
    return x.detach()

ebm = EBM(); opt = torch.optim.Adam(ebm.parameters(), lr=1e-4)
for step in range(10_000):
    x_pos = sample_data(128)                # real samples
    x_neg = langevin(ebm, torch.randn_like(x_pos))
    loss = (ebm(x_pos) - ebm(x_neg)).mean() \
         + 0.1*(ebm(x_pos)**2 + ebm(x_neg)**2).mean()
