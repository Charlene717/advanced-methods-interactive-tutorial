Sheaf 與 Cellular Sheaves

Graph Laplacian 的代數幾何推廣——當「節點之間有 disagreement」時，sheaf 把不同模態、不同切片、不同標籤的訊號縫合起來，並偵測無法調和的衝突。

An algebraic-geometric generalization of the graph Laplacian — when nodes "disagree," sheaves stitch signals across modalities, slices, and labels, while flagging irreconcilable inconsistencies.

核心概念

從 graph 到 sheaf

一張 graph 在每個節點放一個 純量（基因表現、温度），透過 graph Laplacian L = D − A 可做 smoothing、Fourier 分析（Ch 9）。但生物資料 每個節點通常是一個向量——一群基因、一個細胞的 multi-omics 多模態 profile——而且邊上兩端的「向量空間」常常不一樣（節點 A 是 RNA 維度、節點 B 是 ATAC 維度）。

Cellular sheaf (Hansen & Ghrist 2019) 解決這個問題：

  • 在每個節點 v 上指派一個 vector space F(v)（稱為 stalk）。
  • 在每條邊 e = (u, v) 上指派一條 線性對映 Fu≤e: F(u) → F(e) 和 Fv≤e: F(v) → F(e)，稱為 restriction map。
  • 一個 section（全域訊號）是每節點各選一個向量 xv ∈ F(v)，使得邊上兩端透過 restriction map 後相等：Fu≤e(xu) = Fv≤e(xv)。

當邊兩端 restriction map 完全一致時，sheaf 就退化回普通 graph。當不一致時，sheaf 量化「整體有多協調」——這就是 sheaf cohomology。

A graph puts a scalar at each node (gene expression, temperature) and analyzes it with the graph Laplacian L = D − A (Ch 9). But biological data typically has a vector at each node: a gene module, a multi-omics cell profile — and the "spaces" on either side of an edge may differ (node A is in RNA space, node B in ATAC space).

Cellular sheaves (Hansen & Ghrist 2019) handle this:

  • Assign each node v a vector space F(v) (called the stalk).
  • For each edge e = (u, v), give two linear maps Fu≤e: F(u) → F(e) and Fv≤e: F(v) → F(e), the restriction maps.
  • A section (global signal) is a choice of xv ∈ F(v) per node, with Fu≤e(xu) = Fv≤e(xv) on every edge.

When all restriction maps agree, sheaves reduce to ordinary graphs. When they disagree, sheaves quantify "how globally consistent" — that's sheaf cohomology.

演算法

Sheaf Laplacian 與計算

Sheaf Laplacian (Hansen & Ghrist 2019, J Appl Comput Topol)：類比 graph Laplacian，定義 coboundary 算子 δ: C⁰ → C¹，把節點上的訊號 x = (xv)v 對映到邊上的 mismatch：(δx)e = Fv≤e(xv) − Fu≤e(xu)。Sheaf Laplacian 即 LF = δᵀδ，是個半正定對稱矩陣。

  1. H⁰(F) = ker(LF)：「全域 section 的空間」——所有邊都調和的訊號。dim H⁰ 量「獨立的全域共識通道」。
  2. H¹(F) = ker(δᵀ) / im(δ)：「衝突空間」——當有「在每條邊看起來都自洽，但全域湊不起來」的訊號時 H¹ > 0。生物上代表「局部一致、全域矛盾」。
  3. Sheaf diffusion：ẋ = −LF x — 把訊號 smooth 到 ker(LF) 上，類比 heat equation。
  4. Sheaf eigendecomposition：LF 的小特徵值對應 sheaf-aware 的低頻成分——sheaf 版的 spectral clustering。

Sheaf Neural Network (Bodnar et al. 2022, NeurIPS)

把 sheaf 結構嵌入 GNN：每層更新規則是 x ← x − α·LF·MLP(x)，且 restriction map 是 學習出來 的而非固定。實證上比普通 GCN 在 heterophilic graph（鄰居 label 常不同）上勝出 5–15 個百分點，且能在「over-smoothing」的層數下仍保留鑑別力。

Sheaf Laplacian (Hansen & Ghrist 2019, J Appl Comput Topol): analogous to the graph Laplacian, define coboundary δ: C⁰ → C¹ mapping a node signal x = (xv) to the per-edge mismatch (δx)e = Fv≤e(xv) − Fu≤e(xu). Then LF = δᵀδ, positive-semidefinite and symmetric.

  1. H⁰(F) = ker(LF): the space of global sections — signals agreeing on every edge. dim H⁰ counts independent global-consensus channels.
  2. H¹(F) = ker(δᵀ) / im(δ): the inconsistency space — non-zero when "every edge looks fine but globally nothing fits." Biologically: locally consistent, globally contradictory.
  3. Sheaf diffusion: ẋ = −LF x — smoothing toward ker(LF), the sheaf heat equation.
  4. Sheaf eigendecomposition: small eigenvalues of LF give sheaf-aware low-frequency components — sheaf spectral clustering.

Sheaf Neural Network (Bodnar et al. 2022, NeurIPS)

Embed sheaves in GNNs: per-layer update x ← x − α·LF·MLP(x), where restriction maps are learned, not fixed. Empirically beats plain GCN by 5–15 points on heterophilic graphs (neighbors with different labels) and resists over-smoothing across deep stacks.

直覺：graph Laplacian 假設「鄰居應該相似」；sheaf Laplacian 假設「鄰居應該透過某個學到的對映 R 相似」——這非常符合多模態資料（RNA → ATAC 的對應不是 identity，而是某種已知或學到的線性對映）。
Intuition: the graph Laplacian assumes "neighbors should be similar"; the sheaf Laplacian assumes "neighbors should be similar after some learned map R" — perfectly matching multi-modal data (RNA → ATAC is not identity but a learned linear map).

🌳 什麼時候用 sheaf？

情境1:
異質節點：節點上是不同類型的資料（RNA + ATAC + protein），不能直接平均。
情境2:
多視角整合：多個 batch / patient / 切片各自有 cell-type，要找全域一致 labeling。
情境3:
偵測「不一致」：H¹ > 0 直接指出哪些邊或迴圈無法調和——可作為 QC 工具。
Case 1:
Heterogeneous nodes: different data types per node (RNA + ATAC + protein) — can't be averaged directly.
Case 2:
Multi-view integration: multiple batches / patients / slices each with their own cell types — find a globally consistent label.
Case 3:
Inconsistency detection: H¹ > 0 pinpoints irreconcilable edges or loops — a QC tool.
生物應用

Sheaf 在 omics 的應用

Multi-omics consistency

Wang et al. (2023) 把 paired scRNA + scATAC 視為 sheaf：節點是細胞、stalk 是 (RNA, ATAC) 向量、restriction map 學習「peak → gene」對應。H¹ 量化每個區域的 RNA-ATAC 不一致程度，幫助找「regulatory mismatch」（peak 開但 gene 不表現）。

Wang et al. (2023) treat paired scRNA + scATAC as a sheaf: nodes are cells, stalk = (RNA, ATAC) vector, restriction map learns peak→gene mapping. H¹ quantifies regional RNA-ATAC mismatch, finding "regulatory dissonance" (peak open but gene silent).

空間訊號處理

Stereo-seq / MERFISH 的空間轉錄體：邊 = 物理相鄰、stalk = 多基因表現向量、restriction map 學習組織型別間的「轉換邏輯」。sheaf low-pass filter 比普通 spatial Laplacian smoothing 更好保留 tissue boundary。

For Stereo-seq / MERFISH spatial transcriptomics: edges = physical adjacency, stalk = multi-gene vector, restriction map learns tissue-type transitions. Sheaf low-pass preserves tissue boundaries better than ordinary spatial Laplacian.

PPI 不一致性

在 protein-protein interaction network 上以 protein domain composition 為 stalk，restriction map 為 domain compatibility。H¹ 找出「sub-network 內報告衝突的實驗結果」，輔助 STRING / BioGRID 資料品控。

On PPI networks with domain composition as stalk and domain compatibility as restriction map, H¹ pinpoints subnetworks with conflicting experimental reports — QC for STRING / BioGRID.

TCR-pMHC sheaf

免疫識別建模：T cell receptor 的 CDR3 序列空間與 pMHC 結合模式空間以 sheaf 連結，restriction map 為「結合幾何」對映。H⁰ 給出穩定的識別語意。

Immune recognition: link TCR CDR3 sequence space and pMHC binding-pose space via a sheaf, with restriction maps as binding-geometry projections. H⁰ yields stable recognition semantics.

跨物種對齊

跨物種 GRN 對齊：節點是 ortholog gene、stalk 是物種特定的表現向量、restriction map 學「物種間映射」。H¹ 找出「物種間調控不保留」的迴路——演化生物學新工具。

Cross-species GRN alignment: nodes are ortholog genes, stalks are species-specific expression, restriction maps learn cross-species translation. H¹ finds regulatory loops not conserved across species — a new evolutionary tool.

連結組學

神經迴路拓撲分析：每個神經元的活動模式為 stalk、突觸的訊號轉導為 restriction map，sheaf cohomology 量化「迴路一致 firing pattern」。

Neural-circuit topology: each neuron's activity pattern is a stalk; synaptic signal transduction is the restriction map; sheaf cohomology quantifies consistent firing patterns.

潛在改良應用

Sheaf 還能解決哪些難題？

🎯 1. scRNA-ATAC paired consistency QC

用 sheaf cohomology 自動偵測「sample 內 RNA 與 ATAC 矛盾的細胞」——這些常是 doublet 或污染。比目前的 quality filter（總 UMI、percent_mt）更具結構資訊。

🎯 2. 跨切片 spatial sheaf

連續組織切片之間 sheaf 對齊（取代或補充 PASTE 的 Gromov-Wasserstein）。H¹ 顯示哪些區域「無法 3D 重建」——可能是組織損傷或 acquisition 誤差。

🎯 3. 多病人 atlas sheaf

把每個病人視為一個 stalk，restriction map 學「病人間 cell-type 對映」。H¹ > 0 的迴路指出「在某些病人組合下無法找到一致 cell-type」——可能對應疾病亞型或 patient-specific subset。

🎯 4. 跨物種 GRN inconsistency

結合 evolutionary biology 與 sheaf — 找「人/鼠/魚的 GRN 在哪些 module 上保留結構但 strength 不同」、「在哪些 module 上連結構都改變」。

🎯 5. Cell-type assignment uncertainty

把 scLLM 的 cell-type 預測視為 sheaf section；用 H¹ 偵測「鄰近細胞的 label 互相衝突區域」——這正是需要人工標註的 boundary region。比 confidence score 提供更豐富的不確定性訊號。

🎯 1. scRNA-ATAC paired consistency QC

Sheaf cohomology automatically flags cells with internal RNA-ATAC contradictions — often doublets or contamination. Structurally richer than total-UMI / pct_mt filters.

🎯 2. Cross-slice spatial sheaf

Sheaf alignment across serial slices (replacing/complementing PASTE's GW). H¹ reveals regions that cannot be 3D-reconstructed — likely tissue damage or acquisition error.

🎯 3. Multi-patient atlas sheaf

Treat each patient as a stalk; restriction maps learn cross-patient cell-type correspondence. Non-zero H¹ on a loop indicates "no consistent cell type across this patient combination" — may correspond to disease subtypes.

🎯 4. Cross-species GRN inconsistency

Combine evolutionary biology with sheaves to find modules whose structure is conserved across human/mouse/fish but with different strengths — and modules where even structure changes.

🎯 5. Cell-type assignment uncertainty

Treat scLLM cell-type predictions as a sheaf section; H¹ pinpoints regions where neighboring labels conflict — exactly the boundary cells worth manual annotation. Richer than a confidence score.

實作範例

# Build sheaf Laplacian by hand: nodes have d-dim stalks
import numpy as np

def sheaf_laplacian(n_nodes, d, edges, R):
    # edges: list of (u, v); R[e] = (R_u, R_v) restriction maps
    rows = len(edges) * d
    cols = n_nodes * d
    delta = np.zeros((rows, cols))
    for e, (u, v) in enumerate(edges):
        R_u, R_v = R[e]
        delta[e*d:(e+1)*d, u*d:(u+1)*d] = -R_u
        delta[e*d:(e+1)*d, v*d:(v+1)*d] =  R_v
    L = delta.T @ delta            # sheaf Laplacian
    return L, delta

import torch
import torch.nn as nn

class SheafLayer(nn.Module):
    def __init__(self, d, hidden):
        super().__init__()
        self.predict_R = nn.Sequential(   # learn restriction maps
            nn.Linear(2*d, hidden), nn.ReLU(),
            nn.Linear(hidden, d*d),
        )
        self.W = nn.Linear(d, d)

    def forward(self, x, edge_index):
        # x: (N, d). For each edge predict R_uv, R_vu (d x d)
        u, v = edge_index
        h_uv = torch.cat([x[u], x[v]], dim=-1)
        R = self.predict_R(h_uv).view(-1, d, d)
        # Sheaf diffusion step: x ← x - α L_F W(x)
        msg = torch.einsum("eij,ej->ei", R, self.W(x[u]))
        out = x.clone()
        out.index_add_(0, v, msg - self.W(x[v]))
        return out
