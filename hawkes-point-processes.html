Hawkes 過程與點過程

事件不獨立、且過去事件激發未來——這是生物與社會系統最常見的時序性質。Hawkes (1971) 過程用「過去事件指數衰減的自激發 intensity」捕捉此現象。從神經元 spike train 到 cancer mutation timing、HIV evolution、CRISPR 變異熱點，Hawkes 是時序事件建模的核心工具，且已被神經網路化（Neural Hawkes, Transformer Hawkes）。

Events are not independent — past events excite future ones. This is the most common temporal property in biological and social systems. Hawkes (1971) processes capture this via "self-exciting intensity with exponentially decaying impact of past events". From neuronal spike trains to cancer mutation timing, HIV evolution, and CRISPR hotspots, Hawkes is a workhorse for event-time modeling — and is now neural (Neural Hawkes, Transformer Hawkes).

核心概念

點過程：從 Poisson 到 Hawkes

點過程：時間軸上隨機出現的事件序列 {t_1 < t_2 < ... < t_n}，由 intensity λ(t)（單位時間內事件出現的瞬時概率）描述：

P(事件 ∈ [t, t+dt) | 過去 ℋ_t) = λ(t | ℋ_t) dt

ℋ_t 是 t 之前的事件歷史。若 λ 不依賴歷史 → memoryless。

Poisson process：λ(t) = μ（常數）。事件間隔 ~ Exp(μ)，獨立且 memoryless。最基本的 baseline。

非齊次 Poisson：λ(t) 隨時間變化但仍不看歷史。可建模 circadian 或 stimulus-driven 事件。

Hawkes process (Hawkes 1971, J. Roy. Stat. Soc.)：核心是「self-exciting」

λ(t) = μ + Σ_(t_i < t) α · exp(−β (t − t_i))

μ = 基線；每次事件 t_i 都讓 intensity 瞬間增加 α，然後以 β 速率指數衰減。過去事件激發未來事件——導致 burst（事件成簇）。

條件穩定性：當 α/β < 1（branching ratio）時 Hawkes 平穩；否則 intensity 爆炸（如 emergency outbreaks 早期）。

Multivariate Hawkes：K 種 process，互相激發：

λ_k(t) = μ_k + Σ_l Σ_(t_i^l < t) α_{k,l} exp(−β (t − t_i^l))

α_{k,l} 是「process l 激發 process k 的強度」——可用於 cross-excitation 分析（e.g. 某神經元放電激發其他神經元）。

Point process: a random sequence {t_1 < t_2 < ... < t_n} along time, described by an intensity λ(t):

P(event ∈ [t, t+dt) | history ℋ_t) = λ(t | ℋ_t) dt

ℋ_t is event history before t. If λ does not depend on history → memoryless.

Poisson process: λ(t) = μ (constant). Inter-event times ~ Exp(μ), independent and memoryless. The basic baseline.

Inhomogeneous Poisson: λ(t) varies but is history-independent. Models circadian or stimulus-driven events.

Hawkes process (Hawkes 1971, J. Roy. Stat. Soc.): the key idea is self-exciting:

λ(t) = μ + Σ_(t_i < t) α · exp(−β (t − t_i))

μ = baseline; each event t_i raises intensity by α, decaying exponentially at rate β. Past events excite future ones → bursty event clusters.

Stationarity: for α/β < 1 (branching ratio), Hawkes is stationary; otherwise intensity explodes (e.g., outbreak onset).

Multivariate Hawkes: K processes, mutually exciting:

λ_k(t) = μ_k + Σ_l Σ_(t_i^l < t) α_{k,l} exp(−β (t − t_i^l))

α_{k,l} = strength of process l exciting process k — used in cross-excitation analysis (e.g., one neuron's firing exciting others).

⚡

自激發

過去事件抬高未來 intensity；事件天然成簇 (burst)，符合許多生物現象。

Past events raise future intensity; events naturally cluster — matches many biological phenomena.

🔁

可分解為 cascade

每個事件視為前一事件「後代」(immigrant + offspring) — 對應 branching process。

Each event is an "offspring" of an earlier one (immigrant + offspring) → branching process formulation.

🧠

多元互激發

Multivariate Hawkes 可推 cross-excitation 網路——例如哪個神經元觸發誰、哪個 mutation 觸發誰。

Multivariate Hawkes infers cross-excitation networks — e.g., which neuron triggers which, which mutation triggers which.

演算法

MLE、Neural Hawkes、Transformer Hawkes

① Log-likelihood：對觀測序列 {t_1, ..., t_n} 在 [0, T]：

log L = Σ_i log λ(t_i) − ∫_0^T λ(s) ds

前者是 event-time 處 intensity 的 log；後者是「沒有事件的期間 intensity 累積」penalty。對 exponential kernel 的 Hawkes，這個積分有 closed form recursion，可 O(n) 計算。

② MLE for parametric Hawkes：對 (μ, α, β) 做 gradient ascent。Ozaki (1979) 給出 closed-form recursive evaluation；現代 tick 套件用 Quasi-Newton + L1 reg。

③ Mark / Marked point process：每個事件帶 mark m_i ∈ ℳ（如 mutation 種類、neuron index、disease 嚴重度）。intensity 變

λ(t, m) = λ(t) · f(m | t)

分成「何時發生」+「發生何種事件」兩部分，可分別建模。

④ Neural Hawkes (Mei & Eisner 2017, NeurIPS)：把 intensity 改成 continuous-time LSTM 的輸出

λ_k(t) = softplus(W_k^T h(t))

其中 h(t) 是「在事件之間連續演化」的 hidden state（用 continuous-time LSTM 或 ODE）。無 exponential kernel 限制，可學任意 history dependence。

⑤ Transformer Hawkes (Zuo 2020, ICML)：用 self-attention 對事件序列建模，比 Neural Hawkes 更穩、更平行化。對長序列特別好。

⑥ Goodness-of-fit：時間轉換 (time-rescaling)。 對任意 (univariate) point process，若 model 對，則 Λ(t_i) = ∫_0^t_i λ(s)ds 後事件變成 unit-rate Poisson；可用 Kolmogorov-Smirnov test 驗證。

① Log-likelihood: for observations {t_1, ..., t_n} on [0, T]:

log L = Σ_i log λ(t_i) − ∫_0^T λ(s) ds

First term = log-intensity at event times; second = accumulated intensity penalty over event-free intervals. For exponential-kernel Hawkes, the integral has closed-form recursion in O(n).

② MLE for parametric Hawkes: gradient ascent on (μ, α, β). Ozaki (1979) provides closed-form recursive evaluation; modern tick uses Quasi-Newton + L1 reg.

③ Mark / Marked point process: each event carries mark m_i ∈ ℳ (mutation type, neuron index, disease severity). Intensity becomes

λ(t, m) = λ(t) · f(m | t)

Splits "when does it occur" from "what type" — modeled separately.

④ Neural Hawkes (Mei & Eisner 2017, NeurIPS): replace intensity with continuous-time LSTM output:

λ_k(t) = softplus(W_k^T h(t))

h(t) evolves continuously between events (continuous-time LSTM or ODE). No exponential-kernel constraint; can learn arbitrary history dependence.

⑤ Transformer Hawkes (Zuo 2020, ICML): self-attention over the event sequence; more stable and parallel than Neural Hawkes; great for long sequences.

⑥ Goodness-of-fit: time-rescaling. For any univariate point process, if the model is correct, Λ(t_i) = ∫_0^t_i λ(s)ds gives a unit-rate Poisson; verify via Kolmogorov-Smirnov.

歷史補充：Hawkes 過程最初應用於地震 aftershock 建模（Ogata 1988 在統計地震學的 ETAS 模型）。後來在金融市場（high-frequency trading order arrivals）大放異彩。生物醫學的 Hawkes 復興主要在 2010 年代後——neural spike train、somatic mutation timing、流行病擴散、社群網路訊息傳播都使用 Hawkes 變體。
Historical note: Hawkes was first applied to earthquake aftershocks (Ogata 1988's ETAS model in statistical seismology). It exploded in finance for high-frequency trading order arrivals. Biomedical Hawkes revivals took off post-2010 — neural spike trains, somatic mutation timing, epidemic spread, and social-message propagation all use Hawkes variants.

🌳 何時用 Hawkes vs Poisson？

情境1:
事件成簇明顯——例如 neural burst、相連 mutation、epidemic cluster：Hawkes 是 default。
情境2:
事件相互影響、要 infer 因果網路——multivariate Hawkes 可推 α_{k,l} 矩陣作為 functional connectivity。
情境3:
有複雜歷史依賴（非 exponential decay）——用 Neural Hawkes 或 Transformer Hawkes。
不適合:
事件真正獨立（如 Poisson 散射）；或事件密度極低（資料不足以擬合）。
Case 1:
Clear event clustering — neural bursts, linked mutations, epidemic clusters: Hawkes is the default.
Case 2:
Events influence each other; want to infer a causal network — multivariate Hawkes recovers α_{k,l} as functional connectivity.
Case 3:
Complex history dependence (non-exponential decay) — use Neural Hawkes or Transformer Hawkes.
Not for:
Truly independent events (Poisson); or extremely sparse data (insufficient to fit).
生物應用

Hawkes / 點過程在生物醫學

神經元 spike train (Truccolo 2005)

Truccolo et al. (2005, J. Neurophysiol.) 提出 generalized linear model (GLM) point process 框架建模 spike trains——同時納入 self-excitation, refractory, 與 cross-coupling。後續神經科學的 functional connectivity 分析多基於此。Pillow et al. (2008, Nature) 進一步建立 retinal ganglion cell network 模型。

Truccolo et al. (2005, J. Neurophysiol.) introduced a GLM point-process framework for spike trains — capturing self-excitation, refractoriness, and cross-coupling. Foundation of much functional connectivity work. Pillow et al. (2008, Nature) modeled retinal ganglion cell networks similarly.

癌症突變時序 (Greenman 2007)

Greenman et al. (2007, Nature) 等先驅工作把 cancer somatic mutation 視為時序事件——某些 driver mutation 出現後激發後續 passenger 與 second hit。後續 Bolli et al. (2014, Nat. Comm.)、PCAWG (2020, Nature) 等大型分析用 Hawkes 變體追蹤 clonal evolution timing。

Greenman et al. (2007, Nature) and follow-ups treated cancer somatic mutations as temporal events — driver mutations excite subsequent passenger / second-hit mutations. Bolli et al. (2014, Nat. Comm.) and PCAWG (2020, Nature) used Hawkes variants to track clonal evolution timing.

HIV evolution timing

HIV 在 host 內快速演化、escape mutation 對應免疫壓力衝擊。Hawkes 模型用來推 escape 與 CTL 攻擊事件的時序耦合。tick 套件常被用於免疫–病毒共演化資料的時序分析。

HIV evolves rapidly within hosts; escape mutations correspond to immune pressure shocks. Hawkes models infer the timing coupling between escape and CTL attack events. The tick library is often used for immune-virus coevolution time-series.

TCR repertoire dynamics

T cell clonal expansion 是時序事件——某 clonotype 在某時刻爆發擴增。Hawkes 過程可建模「clone i 擴增激發 clone j 擴增」的 cross-excitation，揭示 immune coordination。應用於 COVID-19、癌症免疫追蹤。

T-cell clonal expansion is a temporal event — clones burst at specific moments. Hawkes models cross-excitation "clone i expansion excites clone j" to reveal immune coordination. Applied to COVID-19 and cancer immune tracking.

CRISPR 變異熱點 / 微同調

CRISPR-Cas 編輯後 DNA repair 在某些位置容易再次發生 indels；Hawkes 把這些「再次擾動」建模為自激發過程，可預測 indel hotspot。用於 base editing 與 prime editing 的 off-target risk 評估。

After CRISPR-Cas editing, certain sites are prone to repeat indels; Hawkes models these as self-exciting events, predicting indel hotspots. Used in off-target risk assessment for base / prime editing.

Chromothripsis 與時序簇突變

Chromothripsis 是「一次性多斷裂事件」，與漸進演化突變不同。Hawkes 可區分「強短期 burst」（chromothripsis）與「慢長期 trend」（漸進突變），透過 timing-of-mutation 與 multiplicity 分析（Stephens 2011, Cell）。

Chromothripsis is a "single catastrophic event" distinct from gradual evolution. Hawkes distinguishes "strong short bursts" (chromothripsis) from "slow long trends" (gradual mutation) via timing and multiplicity analysis (Stephens 2011, Cell).

藥物不良反應監測

EHR 系統的 adverse event 報告呈現「藥物使用後激發後續事件」結構。Hawkes 模型用來偵測 drug-event signal（FDA AERS, Reps et al. 2014 等）——比 disproportionality methods 更動態。

Adverse-event reports in EHRs exhibit "drug use excites subsequent events". Hawkes models detect drug-event signals (FDA AERS, Reps et al. 2014) — more dynamic than disproportionality methods.

ECG / EEG burst 偵測

心律不整、癲癇 burst 都是 self-exciting temporal events。Neural Hawkes 在 wearable monitoring data 上做 burst 偵測與分類，比 sliding-window threshold 更穩。

Cardiac arrhythmia and epileptic bursts are self-exciting events. Neural Hawkes on wearable monitoring data detects and classifies bursts more robustly than sliding-window thresholds.

深入解析

Hawkes 進階主題

① Branching ratio 與 cluster size

α/β 是「每個事件預期激發的後代數量」。α/β = 0.5 時，每個事件平均產生 0.5 個直接後代、0.25 個孫代...級聯總大小為幾何級數 1/(1−α/β)。生物應用中，branching ratio 接近 1 對應「臨界 burst 狀態」——常見於 epileptic seizure、cytokine storm 等。

② Hawkes 與 epidemic SIR 的對應

Hawkes intensity μ + Σ α exp(−β·Δt) 與 SEIR 在 mean-field limit 下對應：α 對應 R₀、1/β 對應 infectious duration。Park et al. 2020 / Bertozzi 2020 (PNAS) 在 COVID-19 早期用 Hawkes 估算 R_t 並比 SEIR 更靈活地處理 onset noise。

③ Inference via EM

對 Hawkes，可用 latent "parent assignment"（每個事件被誰激發）做 EM：E-step 計算每個事件被歷史每個事件激發的後驗機率；M-step 重估 (μ, α, β)。Lewis & Mohler 2011 的 EM 演算法應用於 crime data，也適用於生物時序。

④ Non-parametric kernel

不必假設 exponential decay；可用 splines 或 kernel methods（Lewis 2011；Bacry 2014, Quant. Finance）非參數估計 trigger function φ(t)。對未知 dynamics 的生物事件更彈性。

⑤ Sparse multivariate Hawkes

K 個 process 時 α_{k,l} 是 K×K matrix。用 L1 / nuclear-norm 正則化（Bacry 2015）可推稀疏 / 低秩 cross-excitation network——對應生物學中「少數 master regulator」的假設。

⑥ Neural Hawkes 的優劣

優：學任意 history dependence（非 exp decay）、自動處理 marks、可端對端 train。劣：缺乏 closed-form intensity 積分 → 需 Monte Carlo 估計 ∫λ dt；訓練資料需求高；參數可解釋性差。

⑦ Time-rescaling 與診斷

對 fit 完的 Hawkes 算 Λ(t_i)，序列應為 unit-rate Poisson；對 Λ(t_i) − Λ(t_{i−1}) 做 QQ-plot vs Exp(1)。偏離直線 = misfit。對 multivariate, 同樣對每個 process 個別做。

⑧ 與 RNN / Transformer 序列模型的關係

標準 RNN 處理 fixed-step sequence；Hawkes 處理 irregularly-timed events。Neural Hawkes 用 continuous-time hidden state 把這兩者銜接。Transformer Hawkes 進一步把時間差編進 attention，可建模 long-range temporal dependency。

① Branching ratio and cluster size

α/β = expected number of offspring per event. At α/β = 0.5, each event triggers 0.5 direct, 0.25 grandchild, ... total cluster size geometric series 1/(1 − α/β). In biology, near-1 branching ratio is the "critical bursting state" — seen in epileptic seizures, cytokine storms, etc.

② Hawkes vs. epidemic SIR

Hawkes intensity μ + Σ α exp(−β·Δt) matches SEIR in the mean-field limit: α ↔ R₀, 1/β ↔ infectious duration. Park et al. 2020 / Bertozzi 2020 (PNAS) used Hawkes for early-COVID R_t with more flexibility than SEIR around noisy onset.

③ Inference via EM

With latent "parent assignment" (which earlier event triggered the current), do EM: E-step computes posterior triggering probabilities; M-step re-estimates (μ, α, β). Lewis & Mohler 2011's EM algorithm for crime data transfers cleanly to biological time series.

④ Non-parametric kernel

No need to assume exponential decay; use splines or kernels (Lewis 2011; Bacry 2014, Quant. Finance) for non-parametric trigger φ(t). More flexible for unknown biological dynamics.

⑤ Sparse multivariate Hawkes

For K processes, α_{k,l} is a K×K matrix. L1 / nuclear-norm regularization (Bacry 2015) yields sparse / low-rank cross-excitation networks — matching biology's "few master regulators".

⑥ Neural Hawkes pros and cons

Pros: arbitrary history dependence (not just exp); native marks; end-to-end training. Cons: no closed-form ∫λ dt — needs Monte Carlo; data-hungry; less interpretable parameters.

⑦ Time-rescaling and diagnostics

For a fitted Hawkes, compute Λ(t_i); the rescaled times should be unit-rate Poisson; do a QQ-plot of Λ(t_i) − Λ(t_{i−1}) against Exp(1). Deviation = misfit. For multivariate, do this per process.

⑧ Relation to RNN / Transformer sequence models

Standard RNNs handle fixed-step sequences; Hawkes handles irregularly-timed events. Neural Hawkes bridges via continuous-time hidden state. Transformer Hawkes folds inter-event times into attention for long-range temporal dependencies.

潛在改良應用

還能改良哪些任務？

🎯 1. 單細胞 in-vivo lineage timing

CellTagging 與 lineage barcode 已可記錄 lineage tree，但缺少「事件發生時序」。提案：multivariate Hawkes 對 lineage branching events 建模，推 fate decision cross-excitation。

🎯 2. cancer subclonal expansion 預測

結合 Hawkes + clonal copy-number timing。提案：Hawkes-aware MutTime — 不只給每個 mutation 一個 time point，而是給「此 mutation 引發後續多少 cascade」的解析。

🎯 3. 免疫 cytokine storm 預警

Cytokine network 是強自激發系統。提案：multivariate Neural Hawkes 對 longitudinal cytokine panel 建模，提前 24h 偵測 storm onset 對 sepsis / COVID-19 ICU 有實用價值。

🎯 4. EHR 多事件預測

住院、用藥、檢驗、轉院都是 events。提案：Transformer Hawkes 整合 EHR 全部時序，用於 readmission 預測與 adverse-event risk。比現有 RETAIN、Med-BERT 在 irregular timing 處理上更原生。

🎯 5. 神經電生理 spike sorting 後的 connectivity

大規模 Neuropixels 記錄上萬 neuron。提案：sparse multivariate Hawkes 重建 functional connectivity matrix，可揭露 microcircuit 動態，比 cross-correlation 更區分 direct vs indirect。

🎯 6. CRISPR screen 時序解析

Perturb-seq 多時間點測量，但目前少用 temporal model。提案：Hawkes 對 perturbation-response 事件建模，可推 「擾動何時開始造成 cascade」與 「哪些 perturbations 互相激發」。

🎯 7. Microbiome dynamics

菌群 bloom 是強自激發事件。提案：multivariate Hawkes 對 longitudinal 16S/metagenomics 建模 species-species excitation 網路；對 C. diff infection、IBD flare 預測有用。

🎯 8. 結合 RL 的處方推薦

每次處方 → 後續臨床事件。提案：Hawkes-aware RL，把 reward 與 policy 設計考慮「處方激發未來事件」的時序結構，避免短視 myopic decision。

🎯 1. Single-cell in vivo lineage timing

CellTagging / lineage barcodes capture lineage trees but lack "when did each branching happen". Proposal: multivariate Hawkes over lineage branching events to infer fate-decision cross-excitation.

🎯 2. Cancer subclonal expansion forecast

Combine Hawkes + clonal copy-number timing. Proposal: Hawkes-aware MutTime — not just per-mutation timing but "how much cascade did this mutation trigger".

🎯 3. Cytokine-storm early warning

Cytokine networks are strongly self-exciting. Proposal: multivariate Neural Hawkes over longitudinal cytokine panels to detect storm onset 24 h ahead — valuable for sepsis / COVID-19 ICU triage.

🎯 4. EHR multi-event prediction

Admissions, medications, labs, transfers are all events. Proposal: Transformer Hawkes ingesting full EHR timelines for readmission / adverse-event risk — more native at irregular timing than RETAIN / Med-BERT.

🎯 5. Functional connectivity from large-scale spike sorting

Neuropixels record tens of thousands of neurons. Proposal: sparse multivariate Hawkes for functional connectivity — better at separating direct vs indirect than cross-correlation.

🎯 6. Temporal CRISPR screens

Perturb-seq is increasingly time-resolved but rarely temporally modeled. Proposal: Hawkes for perturbation-response events to infer "when does the cascade start" and "which perturbations mutually excite".

🎯 7. Microbiome dynamics

Microbial blooms are strongly self-exciting. Proposal: multivariate Hawkes on longitudinal 16S/metagenomics to recover species-species excitation networks — useful for C. diff infection and IBD flare prediction.

🎯 8. RL with Hawkes-aware reward

Each prescription excites future events. Proposal: Hawkes-aware RL with rewards and policies modeling "prescription excites future events" — avoids myopic short-sighted recommendations.

# pip install tick
from tick.hawkes import SimuHawkesExpKernels, HawkesExpKern

# Simulate a 2D Hawkes with exponential kernel
import numpy as np
adjacency = np.array([[0.4, 0.1], [0.2, 0.5]])     # α_kl matrix
decays    = np.array([[1.0, 1.0], [1.0, 1.0]])     # β
baseline  = np.array([0.05, 0.05])                   # μ

sim = SimuHawkesExpKernels(adjacency, decays, baseline=baseline,
                            end_time=1000, seed=42, verbose=False)
sim.simulate()
events = sim.timestamps          # list of arrays per process

# Estimate parameters from observed events
est = HawkesExpKern(decays=1.0, penalty='l1', C=100)
est.fit(events)
print("baseline", est.baseline)
print("adjacency", est.adjacency)

import numpy as np
import pyhawkes.models as hm

K = 4                                  # number of processes
dt_max = 2.0                            # trigger horizon
model = hm.DiscreteTimeNetworkHawkesModelSpikeAndSlab(
    K=K, dt=0.1, dt_max=dt_max)

# observed events: list of (t, k) tuples discretized into spike matrix
S = np.random.rand(10000, K) < 0.05      # demo
model.add_data(S.astype(int))
# Gibbs sampling for posterior
for i in range(200):
    model.resample_model()
print("posterior weights", model.weight_model.W)

import torch, torch.nn as nn

class NeuralHawkes(nn.Module):
    def __init__(self, K, h=64):
        super().__init__()
        self.K = K
        self.lstm = nn.LSTMCell(K+1, h)         # event-type one-hot + Δt
        self.W = nn.Linear(h, K)
    def intensity(self, h):
        return torch.nn.functional.softplus(self.W(h))
    def forward(self, events):
        # events: list of (Δt, k_type) tensors
        h = torch.zeros(1, 64); c = torch.zeros(1, 64)
        log_lik = 0.
        for dt, k in events:
            # Between-events: integrate intensity (Monte Carlo)
            t_samps = torch.linspace(0, dt, 10)
            lam_int = self.intensity(h).sum() * dt
            # At event: update LSTM
            one_hot = torch.zeros(1, self.K+1)
            one_hot[0, k] = 1.0; one_hot[0, -1] = dt
            h, c = self.lstm(one_hot, (h, c))
            log_lik += torch.log(self.intensity(h)[0, k]+1e-8) - lam_int
        return -log_lik          # negative log-likelihood
