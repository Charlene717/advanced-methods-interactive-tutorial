隱式神經表示 (Implicit Neural Representations)

把一張影像、一個 3D 場、或一段蛋白質序列當作「座標 → 值」的函數，並讓神經網路自己學會這個函數——這就是 INR。SIREN (Sitzmann 2020) 用 sine activation、NeRF (Mildenhall 2020) 開啟 3D 重建新時代。解析度連續、記憶體緊湊、可微分——是 cryo-EM、spatial transcriptomics 的下一代表示。

Treat an image, 3D scene, or protein sequence as a function "coordinate → value", and let a neural net learn it — that's INR. SIREN (Sitzmann 2020) used sine activations; NeRF (Mildenhall 2020) opened a new era of 3D reconstruction. Continuous resolution, compact memory, fully differentiable — the next-generation representation for cryo-EM and spatial transcriptomics.

核心概念

從 grid 到 function：INR 的根本轉換

傳統表示 (explicit)：影像 = pixel grid（H × W × C）、3D 體 = voxel grid、protein = atom 座標 list。固定解析度，記憶體與解析度同步爆炸，且離散不可微。

INR (implicit) 表示：學一個 MLP f_θ : ℝ^d → ℝ^k，把座標 x 映到值（顏色、密度、SDF、表現量等）。整個資料 = 神經網路的權重 θ。

  • 任意解析度：可在任何 x 查詢——「512×512」與「4K」用相同 θ。
  • 記憶體緊湊：常見 MLP 約 0.5–5 MB，可比原始資料壓縮 10–100×。
  • 連續可微：可直接做 gradient、Laplacian——對物理約束、PDE-aware 應用很重要。

關鍵挑戰：標準 ReLU MLP 偏低頻 (spectral bias, Rahaman 2019)。用普通 MLP 學影像會得到模糊版——因為 ReLU 對高頻訊號收斂極慢。三個突破：

  1. Positional encoding (NeRF, Mildenhall 2020, ECCV)：把座標 x 用 γ(x) = [sin(2^k π x), cos(2^k π x)]_{k=0..L} 映射到高頻 Fourier feature。
  2. Fourier features (Tancik 2020, NeurIPS)：用隨機 Gaussian B 矩陣作 γ(x) = [sin(B x), cos(B x)]——Bandwidth 由 B 的方差控制。
  3. SIREN (Sitzmann 2020, NeurIPS)：不用 positional encoding，直接把 activation 換成 sin(ω₀ · Wx + b)，配合特殊初始化。可同時表現高頻 + 連續導數，能解 PDE（如 wave / Helmholtz equation）。

Traditional (explicit) representation: image = pixel grid (H × W × C); 3D volume = voxel grid; protein = atom coordinate list. Fixed resolution; memory scales with resolution; discrete and non-differentiable.

INR (implicit) representation: learn an MLP f_θ : ℝ^d → ℝ^k mapping coordinate x to value (color, density, SDF, expression, etc.). The entire dataset = network weights θ.

  • Arbitrary resolution: queryable at any x — "512×512" and "4K" use the same θ.
  • Compact memory: typical MLPs are 0.5–5 MB — 10–100× compression over raw data.
  • Continuous, differentiable: direct access to gradients, Laplacians — important for physics-aware / PDE applications.

Key challenge: standard ReLU MLPs are biased toward low frequencies (spectral bias, Rahaman 2019). A plain MLP yields blurry images. Three breakthroughs:

  1. Positional encoding (NeRF, Mildenhall 2020, ECCV): map x to γ(x) = [sin(2^k π x), cos(2^k π x)]_{k=0..L} — high-frequency Fourier features.
  2. Fourier features (Tancik 2020, NeurIPS): random Gaussian matrix B, γ(x) = [sin(B x), cos(B x)] — bandwidth controlled by var(B).
  3. SIREN (Sitzmann 2020, NeurIPS): drop positional encoding, replace activation with sin(ω₀ · Wx + b) with special initialization. Represents high frequencies + continuous derivatives — can solve PDEs (wave, Helmholtz).

演算法

SIREN、NeRF、Hash grid、解 PDE

① NeRF (Mildenhall 2020)：把 5D 座標 (x, y, z, θ, φ)（3D 位置 + 視角）映到 (RGB, σ)（顏色 + density）。給定多張影像，沿視線做 volume rendering：

C(r) = ∫ T(t) σ(r(t)) c(r(t), d) dt, T(t) = exp(−∫_0^t σ ds)

對 rendered C 與 ground truth pixel 比 MSE，反傳訓練 MLP。結果：從 ~100 張照片重建完整 3D 場景。生物應用：CryoDRGN 重建 cryo-EM 異質構象。

② SIREN (Sitzmann 2020)：activation = sin(ω₀ · Wx + b)，ω₀ ≈ 30。初始化：first layer W ~ U(−1/n, 1/n)、後續層 W ~ U(−√(6/n)/ω₀, √(6/n)/ω₀)——確保 forward signal 保持 N(0, 1)。SIREN 不僅表現高頻，導數也是 SIREN，可用 supervise 各階梯度（如 Eikonal equation: ‖∇f‖ = 1 學 SDF）。

③ Hash grid encoding (Müller et al. 2022, SIGGRAPH; Instant-NGP)：NeRF 訓練 hours~days；hash grid 用多解析度 spatial hash table 儲存 trainable feature vector，MLP 變成 tiny（2-layer）。訓練從 hours 縮短到秒級。原理：把 spectral bias 問題轉嫁給 hash grid（不同 hash resolution 對應不同頻率）。

④ Solving PDEs with INR (Physics-Informed NN, PINN, Raissi 2019; SIREN PDE)：把解 u(x, t) 表為 SIREN，loss = data fitting + PDE residual ‖∂u/∂t − F(u, ∇u)‖²。INR 的連續可微性讓 ∇u, Δu 直接由 autodiff 取得——無需 grid finite-difference。

⑤ Latent INR / Hyper-network：單一 INR 只能編一個信號。若有 N 個信號（如 N 張影像、N 個 cryo-EM particle），可用 hyper-network 預測 θ_i = h_φ(z_i)，z_i 為 latent code。CryoDRGN 即此架構，學連續異質性。

① NeRF (Mildenhall 2020): map 5D coordinate (x, y, z, θ, φ) (3D position + view) to (RGB, σ) (color + density). Given posed images, volume-render along rays:

C(r) = ∫ T(t) σ(r(t)) c(r(t), d) dt, T(t) = exp(−∫_0^t σ ds)

MSE between rendered C and ground-truth pixels back-propagates through the MLP. Result: full 3D reconstruction from ~100 photos. Bio: CryoDRGN for heterogeneous cryo-EM.

② SIREN (Sitzmann 2020): activation = sin(ω₀ · Wx + b), ω₀ ≈ 30. Initialization: first layer W ~ U(−1/n, 1/n); deeper W ~ U(−√(6/n)/ω₀, √(6/n)/ω₀) — keeps forward signals N(0,1). SIREN represents high frequencies, and its derivatives are SIREN too — supervise any-order gradients (e.g., Eikonal ‖∇f‖ = 1 to learn SDFs).

③ Hash grid encoding (Müller et al. 2022, SIGGRAPH; Instant-NGP): NeRF takes hours-days; hash grids store trainable feature vectors in multi-resolution spatial hash tables, making the MLP tiny (2-layer). Training drops from hours to seconds. The spectral bias is delegated to the hash grid (different hash resolutions → different frequencies).

④ Solving PDEs with INR (PINN, Raissi 2019; SIREN PDE): represent u(x, t) as SIREN, loss = data fitting + PDE residual ‖∂u/∂t − F(u, ∇u)‖². INR continuity lets autodiff compute ∇u, Δu — no finite-difference grid needed.

⑤ Latent INR / hyper-networks: a single INR encodes one signal. For N signals (N images, N cryo-EM particles), a hyper-network predicts θ_i = h_φ(z_i) with z_i a latent code. CryoDRGN uses this for continuous heterogeneity.

為何 sine activation 行得通？(1) sin 的任意階導數仍是 sin（差個相位），所以高階導數可學習；(2) sin 是 oscillatory，可表現高頻；(3) Sitzmann 證明 SIREN 是 universal approximator——理論上等同 ReLU MLP。實務上 SIREN 對影像 PSNR 比 ReLU + PE 高 5–10 dB。
Why does sine activation work? (1) Derivatives of sin are sin (phase-shifted), so higher-order derivatives stay trainable; (2) sin is oscillatory, captures high frequencies; (3) Sitzmann proved SIREN is a universal approximator — theoretically on par with ReLU. In practice SIREN beats ReLU+PE by 5–10 dB PSNR on images.

🌳 什麼時候用 INR？

情境1:
需要任意解析度——super-resolution、cryo-EM 連續解析、3D atlas 對齊。
情境2:
資料記憶體爆炸——4D fMRI、cryo-ET tomogram、whole-slide 影像。
情境3:
要做 physics-aware / PDE 約束——蛋白質動力學、化學反應動力 PDE。
不適合:
(1) 資料天然離散且無連續結構（如 SNP 表）；(2) 每個樣本都要重訓 INR——速度不夠（除非用 hyper-network）。
Case 1:
Need arbitrary resolution — super-resolution, continuous cryo-EM, 3D atlas alignment.
Case 2:
Memory-explosive data — 4D fMRI, cryo-ET tomograms, whole-slide images.
Case 3:
Physics-aware / PDE constraints — protein dynamics, chemical kinetics PDEs.
Not for:
(1) Inherently discrete data with no continuous structure (e.g. SNP table); (2) Per-sample retraining is too slow (unless using a hyper-network).
生物應用

INR 在 cryo-EM、空間轉錄體與分子

CryoDRGN — 連續異質構象

Zhong et al. (2021, Nature Methods) 把每個 cryo-EM particle 視為從 latent z 條件化的 INR 3D 密度：f(x | z) → density。相比 RELION 的 3D classification 必須離散切分，CryoDRGN 學連續構象空間——對 ribosome 動態、剪接體中間態極有用。

Zhong et al. (2021, Nature Methods) treat each cryo-EM particle as an INR-conditioned 3D density: f(x | z) → density. Unlike RELION's discrete 3D classification, CryoDRGN learns continuous conformational space — invaluable for ribosomes, spliceosomes.

iSTAR — spatial transcriptomics super-resolution

Ge et al. (2024, Nature Biotechnology) 把 Visium 的 55 μm spot 結合組織病理 H&E 影像，學 f(x, y) → gene expression 的 INR，重建到 single-cell 解析度——突破光學限制。

Ge et al. (2024, Nature Biotechnology) combine Visium 55 μm spots with H&E histology to learn an INR f(x, y) → gene expression at single-cell resolution — breaking the optical limit.

蛋白質連續構象

Jin et al. (2023, ICML), AlphaFold-3 (Abramson 2024, Nature) 都使用 INR-like 連續結構表示。把 protein 視為「殘基 index → 3D 座標」的函數，可學習 continuous backbone deformation——比離散原子座標 list 更可微、便於分子動力學整合。

Jin et al. (2023, ICML), AlphaFold-3 (Abramson 2024, Nature) use INR-like continuous structural representations. Treating protein as "residue index → 3D coordinate" learns continuous backbone deformation — more differentiable than discrete atom lists, easier MD integration.

4D 醫學影像壓縮

Sun et al. (2022) 與 NeuralCT (Reed 2021) 把 4D cardiac CT / dynamic MRI 表為 f(x, y, z, t) → density。比 H.265 影片壓縮率高、且可任意 time / spatial resampling，對 motion correction、low-dose CT 重建有顯著優勢。

Sun et al. (2022) and NeuralCT (Reed 2021) represent 4D cardiac CT / dynamic MRI as f(x, y, z, t) → density. Higher compression than H.265, arbitrary time/space resampling — significant gains for motion correction and low-dose CT.

NeRF microscopy reconstruction

µ-NeRF / DeepFocus (Park 2022, Nat Methods) 把 light-field / focal-stack 微影像視為視角集合，用 NeRF 重建 3D 樣本（活體胚胎、organoid）。免去機械式 z-stack 掃描，速度與解析度同步提升。

µ-NeRF / DeepFocus (Park 2022, Nat Methods) treat light-field / focal-stack microscopy as a view collection, NeRF-reconstructs 3D samples (live embryos, organoids). Skips mechanical z-stack, faster and higher resolution.

Anatomic atlas 連續對齊

DEMIST / NeuroINR (Liu 2023) 把 atlas 表為 INR f(x) → tissue label，再透過 deformation INR φ(x; subject) 對齊每位 patient——比傳統 ANTs 線性 + B-spline 對齊更平滑、解析度更高。

DEMIST / NeuroINR (Liu 2023) represent atlases as INRs f(x) → tissue label, with deformation INR φ(x; subject) per patient — smoother and higher-res than ANTs linear + B-spline.

連續細胞軌跡 (cell trajectory)

把 scRNA-seq trajectory pseudotime → expression 視為 INR f(t) → gene vector，可平滑插值缺漏 timepoint。Lopez et al. (2023, biorXiv) 結合 INR + VI 學連續細胞發育過程。

Treat scRNA-seq pseudotime → expression as an INR f(t) → gene vector — smoothly interpolates missing timepoints. Lopez et al. (2023, bioRxiv) combine INR + VI to learn continuous development.

Protein INR / SE(3) 表示

RFdiffusion (Watson 2023, Nature)、Chroma (Ingraham 2023, Nature) 把 protein 結構在 SE(3) INR 上表示，配合 diffusion model 做 de novo 設計——把 INR 從「scalar field」推到「SE(3) field」的延伸。

RFdiffusion (Watson 2023, Nature), Chroma (Ingraham 2023, Nature) represent protein structure on SE(3) INR with diffusion for de novo design — extending INR from scalar to SE(3) fields.

深入解析

INR 的進階主題

① Spectral bias 與緩解策略

Rahaman et al. (2019, ICML) 證明 ReLU MLP 收斂高頻訊號比低頻慢很多。對影像、3D scene 都是致命缺陷。三條解方：(1) NeRF positional encoding；(2) SIREN sine activation；(3) hash grid。SIREN 同時解高頻 + 高階導數，但對非 oscillatory 場景（如 SDF 邊界）有時不如 hash grid 穩定。

② Neural Tangent Kernel (NTK) 視角

Tancik et al. (2020) 用 NTK 證明：positional encoding 等價於把網路 NTK 從低通變寬頻，因此學高頻變快。SIREN 的 NTK 更像 sinc——本質上是 ideal low-pass with sharp cutoff，所以高頻 / 低頻同樣學得快。

③ Hash grid vs 純 MLP

Müller (2022) Instant-NGP 用多解析度 (16 levels) hash grid + 2-layer MLP，把 NeRF 訓練時間從 hours → 秒級。代價：模型體積增大（hash table）、稍微 less 連續（hash 衝突）。實務：對速度敏感用 hash grid；對連續性敏感用 SIREN。

④ Generalizable INR (meta-learning)

單一 INR 只代表一個信號；要泛化到「N 個新樣本」需 meta-learn。MetaSDF (Sitzmann 2020)、Functa (Dupont 2022, ICML) 用 MAML / hyper-network 預測 INR 權重，使新樣本只需 few-shot 微調。

⑤ Diffusion + INR

HyperDiffusion (Erkoç 2023, ICCV) 直接在 INR 權重空間做 diffusion——生成新 3D 物體、protein 結構。與「在 voxel grid 上 diffusion」相比，模型更 compact、解析度可任意。

⑥ 解 PDE 的 PINN

Raissi, Perdikaris & Karniadakis (2019, J Comput Phys) 提出 PINN：把 PDE 解 u(x, t) 表為 MLP，loss = data + PDE residual。INR 的可微性讓 ∇u, Δu autodiff 取得。生物應用：reaction-diffusion 模擬 morphogen 梯度、心臟電生理。SIREN 比 ReLU PINN 在高頻 PDE 收斂更好。

⑦ INR for inverse problems

當量測 y = A x + noise，把 x 表為 INR θ，最佳化 ‖A f_θ − y‖² 即可。對 MRI / CT / cryo-EM reconstruction 已成標配——比學習式 (supervised) 方法更具 generalization、無需大量配對訓練資料。

⑧ 缺點與失敗模式

(1) 訓練慢（除非 hash grid）；(2) 每個樣本 separate 訓練——不易 batch；(3) extrapolation 差——在 training coordinate 範圍外給出 unpredictable 值；(4) discrete event（如 cell boundary）邊緣可能模糊化。

① Spectral bias and mitigations

Rahaman et al. (2019, ICML) showed ReLU MLPs converge to high frequencies much slower — fatal for images/3D scenes. Three fixes: (1) NeRF positional encoding; (2) SIREN sine activation; (3) hash grid. SIREN solves high frequencies + high-order derivatives but can be less stable than hash grids on non-oscillatory targets (e.g., SDFs).

② Neural Tangent Kernel (NTK) view

Tancik et al. (2020) prove via NTK that positional encoding broadens the network's NTK from low-pass to broadband — high frequencies learn faster. SIREN's NTK resembles a sinc — essentially an ideal low-pass with sharp cutoff, so high and low frequencies learn at similar rates.

③ Hash grid vs pure MLP

Müller (2022) Instant-NGP uses multi-resolution (16 levels) hash grid + 2-layer MLP, cutting NeRF training from hours to seconds. Trade: larger model (hash table), slight discontinuity (hash collisions). Rule: hash grid for speed; SIREN for continuity.

④ Generalizable INR (meta-learning)

One INR encodes one signal; for N new samples, meta-learn. MetaSDF (Sitzmann 2020), Functa (Dupont 2022, ICML) use MAML / hyper-networks to predict INR weights — new samples need only few-shot fine-tune.

⑤ Diffusion + INR

HyperDiffusion (Erkoç 2023, ICCV) runs diffusion in INR weight space — generates new 3D objects, protein structures. More compact and resolution-free than voxel-grid diffusion.

⑥ PDE-solving with PINN

Raissi, Perdikaris & Karniadakis (2019, J Comput Phys): represent PDE solutions u(x, t) as MLPs, loss = data + PDE residual. INR's differentiability gives ∇u, Δu via autodiff. Bio: morphogen reaction-diffusion, cardiac electrophysiology. SIREN converges better than ReLU PINN on high-frequency PDEs.

⑦ INR for inverse problems

Given y = A x + noise, represent x as INR θ, optimize ‖A f_θ − y‖². Standard for MRI / CT / cryo-EM — better generalization than supervised methods, no paired training data needed.

⑧ Failure modes

(1) Slow training (unless hash grid); (2) per-sample training, hard to batch; (3) poor extrapolation outside training coordinates; (4) discrete events (cell boundaries) may blur.

潛在改良應用

還能改良哪些任務？

🎯 1. INR-based whole-slide image 壓縮 + 病理 AI

Whole-slide image 動輒 100 GB。提案：每張 slide 訓練 INR (≈ 50 MB)，做壓縮儲存、任意縮放查看；下游病理 AI 直接在 INR weight space 做 classification。

🎯 2. 連續多解析度單細胞圖譜

Cell Atlas 累計上百萬細胞。提案：學 f(x, type, age) → expression INR，做連續細胞圖譜——可在任意 cell-type embedding 點查詢預期表現量。

🎯 3. INR + diffusion 做 de novo small molecule

分子座標連續可微，但離散 SMILES 不易直接 diffusion。提案：用 SE(3) INR + diffusion 在連續分子場上生成新藥分子，配合 docking score 條件控制。

🎯 4. 連續 spatial-omics integration

Visium、Xenium、CODEX 解析度不同。提案：用統一 INR 表示 spatial multi-modal，每模態走自己的 head，學連續對齊。

🎯 5. 4D protein dynamics INR

MD trajectories 龐大但連續。提案：INR f(residue, t) → 3D 座標 壓縮 trajectory + 平滑插值，比 cluster 法保留更多連續構象。

🎯 6. INR-driven federated medical imaging

各醫院 imaging 量大且 privacy 敏感。提案：本地訓練 INR、只分享 INR weights（小）做聯邦學習，比共享 raw image 更隱私安全。

🎯 7. INR as biological PDE solver

Reaction-diffusion (Turing pattern)、tumor growth PDE、心臟電生理。提案：SIREN PINN 直接學動態解 + 物理約束，可用 sparse 病人資料微調，比 finite-element solver 更靈活。

🎯 1. INR-based whole-slide compression + pathology AI

WSIs are 100 GB+. Proposal: train per-slide INRs (~50 MB) for compression + arbitrary-zoom viewing; downstream pathology AI runs directly in INR weight space.

🎯 2. Continuous multi-resolution single-cell atlas

Atlases reach millions of cells. Proposal: learn f(x, type, age) → expression INR — query expected expression at any point in cell-type embedding.

🎯 3. INR + diffusion for de novo small molecules

Molecular coordinates are continuous; discrete SMILES is hard to diffuse. Proposal: SE(3) INR + diffusion in continuous molecular fields, with docking-score conditioning.

🎯 4. Continuous spatial-omics integration

Visium, Xenium, CODEX differ in resolution. Proposal: a unified INR for spatial multi-modal, per-modality heads, continuous alignment.

🎯 5. 4D protein dynamics INR

MD trajectories are huge but continuous. Proposal: INR f(residue, t) → 3D coordinate compresses trajectory + smooth interpolation, preserving continuous states.

🎯 6. INR-driven federated medical imaging

Hospital imaging is large and privacy-sensitive. Proposal: train INRs locally, federate only small INR weights — more private than raw image sharing.

🎯 7. INR as biological PDE solver

Reaction-diffusion (Turing), tumor growth, cardiac electrophysiology. Proposal: SIREN PINNs learn dynamics + physics, fine-tuneable from sparse patient data — more flexible than finite-element solvers.

import torch, torch.nn as nn

class FourierFeatures(nn.Module):
    """Tancik 2020: Gaussian RFF"""
    def __init__(self, d_in, n_feat=128, sigma=10.):
        super().__init__()
        self.B = nn.Parameter(torch.randn(d_in, n_feat) * sigma,
                              requires_grad=False)   # fixed!
    def forward(self, x):
        proj = 2 * torch.pi * x @ self.B
        return torch.cat([torch.sin(proj), torch.cos(proj)], -1)

class INR(nn.Module):
    def __init__(self):
        super().__init__()
        self.ff = FourierFeatures(2, 128, sigma=10.)
        self.mlp = nn.Sequential(
            nn.Linear(256, 256), nn.ReLU(),
            nn.Linear(256, 256), nn.ReLU(),
            nn.Linear(256, 3))
    def forward(self, x):
        return self.mlp(self.ff(x))

import torch

def wave_residual(xt):
    xt.requires_grad_(True)
    u = net(xt)
    g = torch.autograd.grad(u.sum(), xt, create_graph=True)[0]
    u_x, u_t = g[..., 0:1], g[..., 1:2]
    u_xx = torch.autograd.grad(u_x.sum(), xt, create_graph=True)[0][..., 0:1]
    u_tt = torch.autograd.grad(u_t.sum(), xt, create_graph=True)[0][..., 1:2]
    c = 1.0
    return (u_tt - c**2 * u_xx)**2      # wave equation residual
