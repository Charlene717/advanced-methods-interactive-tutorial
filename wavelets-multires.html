小波與多解析度分析 (Wavelets & Multi-resolution)

Fourier 變換能告訴你「有哪些頻率」，卻不告訴你「頻率發生在何時」。小波變換同時提供時間與頻率的局部化資訊——這是 Haar (1910)、Daubechies (1988)、Mallat (1989) 等人塑造的多解析度世界。從 EEG spike、ChIP-seq peak 到 circadian rhythm、CNV 偵測，小波是分析多尺度生醫信號最自然的工具。

Fourier transforms tell you which frequencies exist but not when. Wavelet transforms provide simultaneous time-frequency localization — the multi-resolution world built by Haar (1910), Daubechies (1988), Mallat (1989). From EEG spikes to ChIP-seq peaks, circadian rhythms to CNV detection, wavelets are the natural tool for multi-scale biomedical signals.

核心概念

為何 Fourier 不夠？

Fourier 的困境：把信號分解成「無限長正弦波」——你知道頻率，但失去了時間定位。對 EEG 中突然出現的 spike、ChIP-seq 中的 peak、calcium imaging 中的 burst——「事件何時發生」與「事件多寬」同樣重要。

小波的解方：用「短促、波狀、有局部支撐」的基函數 ψ(t) 來分析信號：

W(a, b) = ∫ x(t) · (1/√a) · ψ((t − b)/a) dt

a 是 scale（伸縮——越小越高頻），b 是 translation（時間位置）。每個小波係數同時告訴你「何時」與「哪個尺度的事件」。這就叫 time-frequency localization。

Heisenberg 限制：無法同時無窮精確 localize time 與 frequency。Δt · Δω ≥ ½。小波選擇：高頻時 Δt 小（精確時間）、低頻時 Δω 小（精確頻率）——剛好配合「短促事件常高頻、緩慢趨勢常低頻」的物理直覺。

Mother wavelet：不同任務選不同 mother：

  • Haar (1910)：方波，最簡單，適合階梯狀信號（CNV、binary segmentation）。
  • Daubechies (db4, db8, Daubechies 1988)：正交、緊支撐、平滑——影像/壓縮黃金標準。
  • Morlet：複數 Gaussian × 正弦，最適合振盪信號分析（EEG, calcium imaging）。
  • Mexican hat (DoG 二階導數)：對稱、適合 spot / peak 偵測。

The Fourier problem: decomposes a signal into infinite-length sinusoids — you get frequencies but lose time. For EEG spikes, ChIP-seq peaks, calcium bursts — both when and how wide matter.

The wavelet solution: analyze with short, wavy, locally-supported basis ψ(t):

W(a, b) = ∫ x(t) · (1/√a) · ψ((t − b)/a) dt

a = scale (smaller → higher frequency); b = translation. Each coefficient tells you when and at which scale. This is time-frequency localization.

Heisenberg limit: cannot simultaneously have infinite resolution in time and frequency — Δt · Δω ≥ ½. Wavelets trade off: small Δt at high frequency (precise time), small Δω at low frequency (precise frequency) — matching the physical reality that fast events are high-frequency and slow trends are low-frequency.

Mother wavelets: choose by task:

  • Haar (1910): step function, simplest, great for step-like signals (CNV, binary segmentation).
  • Daubechies (db4, db8, Daubechies 1988): orthogonal, compactly supported, smooth — gold standard for image/compression.
  • Morlet: complex Gaussian × sinusoid — best for oscillations (EEG, calcium imaging).
  • Mexican hat (DoG, 2nd derivative): symmetric, ideal for spot / peak detection.

演算法

從 CWT 到 DWT、wavelet packet、去噪

① CWT (Continuous Wavelet Transform)：連續 (a, b) 雙參數，掃描所有 scale 與 translation。冗餘但可視化清楚——常用於 EEG、生物節律 scalogram。計算 O(N · S)，S = scale 數。

② DWT (Discrete Wavelet Transform)：取 dyadic scale a = 2^j、translation b = k · 2^j。配合 Mallat (1989) 提出的 multi-resolution analysis (MRA)：

V_0 ⊃ V_1 ⊃ V_2 ⊃ ... 每層 V_j = V_{j+1} ⊕ W_{j+1}

V_j 為 scaling space（粗略近似 a_j）、W_j 為 wavelet space（細節 d_j）。Pyramid algorithm：用低通 h 與高通 g 濾波器 + downsample 2，迭代分解，O(N)。

a_{j+1} = (h * a_j) ↓ 2, d_{j+1} = (g * a_j) ↓ 2

反向重建：插零 + 共軛濾波 + 相加，達到完美重建 (perfect reconstruction, PR)。

③ Wavelet packet decomposition (Coifman & Wickerhauser 1992)：標準 DWT 只繼續分解 a_j（低頻）；wavelet packet 對 a_j 與 d_j 都繼續分解，提供更細的時頻 tile。可用 best-basis algorithm 在 packet tree 中選最稀疏表示。

④ Wavelet shrinkage denoising (Donoho & Johnstone 1994, Biometrika)：三步驟——(1) 信號 DWT，(2) 對小波係數做 soft / hard threshold（小於 λ 設零），(3) 反 DWT。原理：信號能量集中於少數大係數，雜訊均勻分散於所有係數；threshold 殺雜訊保信號。Universal threshold λ = σ √(2 log N)。

⑤ Empirical wavelet transform (Gilles 2013, IEEE TSP)：自適應地把頻譜切成 N 個band，再對每 band 設計 wavelet。對非穩態信號（如非典型 EEG）比固定 wavelet 更好。

① CWT (Continuous Wavelet Transform): continuous (a, b), scans all scales and translations. Redundant but visualizable — EEG and biological rhythm scalograms. Cost O(N · S).

② DWT (Discrete Wavelet Transform): dyadic scales a = 2^j, translations b = k · 2^j. Combined with Mallat's (1989) multi-resolution analysis (MRA):

V_0 ⊃ V_1 ⊃ V_2 ⊃ ... each V_j = V_{j+1} ⊕ W_{j+1}

V_j = scaling space (coarse a_j); W_j = wavelet space (detail d_j). Pyramid algorithm: alternate low-pass h and high-pass g filters with downsample-by-2, iterate; O(N).

a_{j+1} = (h * a_j) ↓ 2, d_{j+1} = (g * a_j) ↓ 2

Inverse: upsample + conjugate filters + sum — achieves perfect reconstruction (PR).

③ Wavelet packet decomposition (Coifman & Wickerhauser 1992): standard DWT only splits a_j (low freq); packets split both a_j and d_j, giving finer time-frequency tiling. Best-basis algorithm picks the sparsest packet tree.

④ Wavelet shrinkage denoising (Donoho & Johnstone 1994, Biometrika): (1) DWT the signal, (2) soft/hard-threshold the coefficients (set those below λ to zero), (3) inverse DWT. Signal energy concentrates in few large coefficients; noise spreads — threshold kills noise. Universal threshold λ = σ √(2 log N).

⑤ Empirical wavelet transform (Gilles 2013, IEEE TSP): adaptively splits the spectrum into N bands, designs a wavelet per band — beats fixed wavelets on non-stationary signals (atypical EEG).

Cascade / filter bank 視角：DWT 可視為低通 + 高通 perfect-reconstruction filter bank（quadrature mirror filters, QMF）+ 下採樣。Daubechies 的最大貢獻就是建造出「正交 + 緊支撐 + 給定 vanishing moments」的濾波器組——讓 wavelet 變得實用。
Cascade / filter-bank view: DWT is a low-pass + high-pass perfect-reconstruction filter bank (quadrature mirror filters) with downsampling. Daubechies' breakthrough was constructing orthogonal, compactly supported filters with prescribed vanishing moments — making wavelets practical.

🌳 什麼時候用小波？

情境1:
非穩態信號——EEG、calcium imaging、ChIP-seq、Hi-C 沿染色體位置。Fourier 假設 stationarity，小波不需要。
情境2:
多尺度事件混合——CNV (10kb–100Mb)、circadian + ultradian 節律、TAD + sub-TAD。
情境3:
需要稀疏表示 + 去噪——methylation array、Tiling array、HiC。
不適合:
純穩態、純單頻信號（FFT 或 Lomb-Scargle 更直接）；不規則時間取樣（需先插值或用 lifting scheme）。
Case 1:
Non-stationary signals — EEG, calcium imaging, ChIP-seq, Hi-C along chromosome. Fourier assumes stationarity; wavelets don't.
Case 2:
Mixed multi-scale events — CNV (10kb–100Mb), circadian + ultradian, TADs + sub-TADs.
Case 3:
Need sparse representation + denoising — methylation arrays, tiling arrays, Hi-C.
Not for:
Pure stationary single-frequency signals (use FFT or Lomb-Scargle); irregularly sampled time (need interpolation or lifting scheme).
生物應用

小波在基因體、神經科學與單細胞

MetaCycle / 生物節律偵測

Wu et al. (2016, Bioinformatics) 的 MetaCycle 套件整合 JTK_CYCLE、ARSER、Lomb-Scargle；其中 ARSER 用 wavelet 作為前處理 detrending。Leise (2013, WIREs Dev Biol) 詳述 wavelet 在 circadian 信號（如 PER2::LUC）相位、振幅追蹤的優勢。

Wu et al. (2016, Bioinformatics) MetaCycle integrates JTK_CYCLE, ARSER, Lomb-Scargle; ARSER uses wavelets for detrending. Leise (2013, WIREs Dev Biol) details how wavelets track phase and amplitude in circadian signals (e.g. PER2::LUC).

Calcium imaging spike detection

Two-photon calcium signal 含緩慢漂移 + 快速 spike。Patel et al. (2013) 用 Haar wavelet shrinkage 去 baseline、Mexican hat CWT 識別 spike，現已整合到 CalmAn (Giovannucci 2019, eLife)。

Two-photon calcium signals mix slow drift and fast spikes. Patel et al. (2013) used Haar shrinkage to remove baseline and Mexican-hat CWT to detect spikes; now part of CalmAn (Giovannucci 2019, eLife).

EEG / ERP 神經科學

EEG 既有 alpha (10 Hz)、beta (20 Hz)、gamma (40+ Hz) 振盪、又有突發 spike & sleep spindle。Morlet CWT 是標準工具——Tallon-Baudry & Bertrand (1999, Trends Cog Sci) 是經典。也用於癲癇 ictal 偵測（Adeli 2003）。

EEG has alpha (10 Hz), beta (20 Hz), gamma (40+ Hz) plus spikes and sleep spindles. Morlet CWT is the standard tool — Tallon-Baudry & Bertrand (1999, Trends Cog Sci) is classic. Used in epilepsy detection (Adeli 2003).

ChIP-seq peak detection

Mathelier & Carbone (2013, Bioinformatics) 提出 ChIP-Wave 用 CWT 在不同 scale 同時偵測「窄/寬」peak——比 MACS2 對 H3K4me3（窄）與 H3K27me3（寬）兩種型別都更穩健。Hsu et al. (2005, Genome Res) 更早用於 array CGH。

Mathelier & Carbone (2013, Bioinformatics) proposed ChIP-Wave using CWT to detect narrow + broad peaks simultaneously — more robust than MACS2 for both H3K4me3 (narrow) and H3K27me3 (broad). Hsu et al. (2005, Genome Res) earlier used wavelets for array CGH.

WaveletPCA for single-cell time-series

把 single-cell pseudotime trajectory 視為時間序列，先做 wavelet decomposition、再 PCA——可分離「快速 stochastic 變化」與「緩慢 trajectory drift」。Petukhov et al. (2018, Nat Methods 雛形)、Gut et al. (2018) 應用於 cell-cycle 解析。

Treat single-cell pseudotime trajectories as time series — wavelet decomposition + PCA separates fast stochastic fluctuations from slow trajectory drift. Petukhov et al. (2018, Nat Methods prototype), Gut et al. (2018) applied to cell-cycle resolution.

Spatial multi-scale 結構偵測

2D wavelet 偵測 spatial transcriptomics 中不同尺度的 "hot spot"（單細胞 → tissue domain → tumor zone）。Lo et al. (2024, Nat Commun) WALDO 用 wavelets 整合 Visium / Xenium 跨解析度結構。

2D wavelets find spatial transcriptomics hot spots at multiple scales (single cell → tissue domain → tumor zone). Lo et al. (2024, Nat Commun) WALDO integrates Visium / Xenium cross-resolution structures via wavelets.

Methylation array 平滑

450K / 850K methylation array 沿 genome 排列。Wavelet shrinkage 比 moving average 更能保留 DMR (differentially methylated regions) 邊界——Hansen et al. (2012, Genome Biol) bsseq 提出類似框架。

450K / 850K methylation arrays are arranged along genome. Wavelet shrinkage preserves DMR boundaries better than moving averages — Hansen et al. (2012, Genome Biol) bsseq uses a related framework.

CNV detection with wavelets

Hsu et al. (2005, Genome Res) 與 Wang et al. (2006) 把 CGH / read-depth 視為 piecewise-constant 信號——Haar wavelet shrinkage 是最自然分解工具。後續 GADA、Wavelet-CNVnator 等。

Hsu et al. (2005, Genome Res) and Wang et al. (2006) treat CGH / read-depth as piecewise-constant signals — Haar shrinkage is the natural choice. Successors: GADA, Wavelet-CNVnator.

深入解析

小波的進階主題

① Vanishing moments 與信號平滑度

若 wavelet ψ 有 N 個 vanishing moments（∫ t^k ψ(t) dt = 0, k = 0..N−1），則任何 N 階多項式平滑信號的 wavelet 係數 ≈ 0。Daubechies dbN 有 N 個 vanishing moments——越大越能壓縮平滑信號。

② Lifting scheme (Sweldens 1996, SIAM J Math Anal)

"Second generation wavelets"——不用 Fourier 框架，純用 split / predict / update 步驟構造 wavelet。優點：(1) 可在不規則網格上（如不均勻基因組位置）做小波；(2) integer-to-integer，無浮點誤差；(3) 速度比 cascade 快約 2×。

③ Stationary (Undecimated) Wavelet Transform (SWT)

標準 DWT 因 downsample 而 shift-variant——同樣信號移一位，係數變很多。SWT 不做 downsample，保持 shift-invariant，但代價是冗餘。對 peak detection、changepoint detection 比 DWT 更穩。

④ 2D / 3D wavelets

影像用 2D separable wavelet：HH (對角)、HL (水平)、LH (垂直)、LL (低通)。生醫應用：MRI 壓縮、組織病理 H&E 影像 multi-scale feature。Curvelets (Candès & Donoho 2004) 更適合曲線/邊緣。

⑤ 與 scattering transform 的關係

Mallat (2012, Comm Pure Appl Math) 提出 scattering transform：cascade wavelet + |·|，得到 translation-invariant + stable feature——本質是「凍結」CNN 第一層的 wavelet 替代品。在小資料生醫分類常打敗 CNN。

⑥ Wavelet on graphs / SGWT

Hammond, Vandergheynst & Gribonval (2011, Appl Comput Harmon Anal) 把 wavelet 推廣到任意 graph（基於 graph Laplacian spectral domain）。生醫應用：scRNA-seq KNN graph 上的 multi-scale gene expression smoothing。

⑦ Wavelet coherence

Torrence & Compo (1998, BAMS) 的 wavelet coherence 把 cross-spectrum 推廣到時頻域——量度兩信號何時 / 何頻率耦合。常用於 fMRI connectivity、心率變異與呼吸 coupling。

① Vanishing moments and signal smoothness

If ψ has N vanishing moments (∫ t^k ψ(t) dt = 0, k = 0..N−1), wavelet coefficients of any degree-N polynomial vanish. Daubechies dbN has N vanishing moments — larger N compresses smooth signals more.

② Lifting scheme (Sweldens 1996, SIAM J Math Anal)

"Second generation wavelets" — built via split / predict / update, no Fourier needed. Benefits: (1) wavelets on irregular grids (uneven genomic positions); (2) integer-to-integer, no floating-point error; (3) ~2× faster than cascade.

③ Stationary (Undecimated) Wavelet Transform (SWT)

Standard DWT is shift-variant (downsampling). SWT skips downsampling, preserving shift-invariance at the cost of redundancy. Better than DWT for peak / changepoint detection.

④ 2D / 3D wavelets

Images use 2D separable wavelets: HH (diagonal), HL (horizontal), LH (vertical), LL (low-pass). Biomedical uses: MRI compression, H&E histopathology multi-scale features. Curvelets (Candès & Donoho 2004) suit curves and edges better.

⑤ Connection to scattering transform

Mallat (2012, Comm Pure Appl Math) cascades wavelet + |·| for translation-invariant stable features — essentially a frozen-first-layer CNN replacement. Beats CNNs on small biomedical classification tasks.

⑥ Wavelets on graphs / SGWT

Hammond, Vandergheynst & Gribonval (2011, Appl Comput Harmon Anal) extend wavelets to arbitrary graphs via graph-Laplacian spectrum. Bio use: multi-scale gene-expression smoothing on scRNA-seq KNN graphs.

⑦ Wavelet coherence

Torrence & Compo (1998, BAMS) extend cross-spectra to time-frequency — measures when and at which frequency two signals couple. Used in fMRI connectivity, heart-rate-respiration coupling.

潛在改良應用

還能改良哪些任務？

🎯 1. Wavelet-enhanced spatial transcriptomics

Visium/Xenium 同時有單細胞與組織尺度 pattern。提案：用 2D wavelet 分解 spatial gene expression，分別處理「組織域 LF」與「細胞型別 HF」，比現有 spot-level analysis 更具尺度感知。

🎯 2. Multi-scale ATAC-seq peak calling

ATAC-seq 訊號有 single-nucleosome (~150 bp) 到 super-enhancer (~10 kb) 多尺度。提案：用 SWT 跨尺度 peak calling，比 MACS2 對 broad / narrow 兩類 peak 都穩。

🎯 3. Wavelet + Diffusion model 生醫去噪

單細胞 dropout、低光顯微鏡都需要去噪。提案：把 wavelet shrinkage 當 diffusion 的條件 prior，得到 task-aware denoising，比純 deep denoise 更可解釋。

🎯 4. 時間取樣不規則的 single-cell trajectory

實驗 timepoint 通常稀疏不均勻（0h, 6h, 24h, 72h...）。提案：用 lifting-scheme wavelet 在不規則時間網格上做 trajectory smoothing 與差異偵測。

🎯 5. Graph wavelet for scRNA-seq imputation

MAGIC / DCA 用 diffusion；提案：SGWT 直接在 KNN graph 上做 multi-scale smoothing，可同時保留細胞型別界線與緩解 dropout。

🎯 6. Wavelet 多解析度 deep learning

影像 CNN 用 max-pool 下採樣會丟資訊。提案：把 wavelet pyramid 當 backbone（如 Wavelet U-Net 的延伸），用於組織病理 / cryo-EM——保留多尺度紋理。

🎯 7. Cross-modal wavelet coherence

同時量 EEG + fMRI + behavior。提案：multi-modal wavelet coherence 找跨模態的時頻 coupling 模式，用於神經精神疾病標記發現。

🎯 1. Wavelet-enhanced spatial transcriptomics

Visium/Xenium contain single-cell + tissue patterns. Proposal: 2D wavelet decomposition splitting domain-scale LF from cell-type HF — more scale-aware than spot-level analysis.

🎯 2. Multi-scale ATAC-seq peak calling

ATAC signal spans single-nucleosome (~150 bp) to super-enhancer (~10 kb). Proposal: SWT-based cross-scale peak calling, robust for both narrow and broad peaks.

🎯 3. Wavelet + diffusion biomedical denoising

Single-cell dropout, low-light microscopy. Proposal: wavelet shrinkage as diffusion-prior condition — task-aware, more interpretable than pure deep denoising.

🎯 4. Irregularly sampled single-cell trajectories

Experimental timepoints are sparse, uneven (0h, 6h, 24h, 72h...). Proposal: lifting-scheme wavelets on irregular grids for trajectory smoothing + change detection.

🎯 5. Graph wavelets for scRNA-seq imputation

MAGIC / DCA use diffusion. Proposal: SGWT on KNN graph for multi-scale smoothing — preserves cell-type boundaries while easing dropout.

🎯 6. Wavelet multi-resolution deep learning

Max-pool loses information. Proposal: wavelet-pyramid backbones (Wavelet U-Net extensions) for histopathology / cryo-EM — preserves multi-scale texture.

🎯 7. Cross-modal wavelet coherence

Simultaneous EEG + fMRI + behavior. Proposal: multi-modal wavelet coherence to find time-frequency coupling — biomarker discovery for neuropsychiatric disorders.

import pywt, numpy as np

def wavelet_denoise(y, wavelet='db4', level=5):
    # Estimate noise sigma from finest detail (MAD)
    coeffs = pywt.wavedec(y, wavelet, level=level)
    sigma = np.median(np.abs(coeffs[-1])) / 0.6745
    # Universal threshold (Donoho-Johnstone 1994)
    lam = sigma * np.sqrt(2 * np.log(len(y)))
    # Soft-threshold details (not approximation)
    coeffs[1:] = [pywt.threshold(c, lam, mode='soft') for c in coeffs[1:]]
    return pywt.waverec(coeffs, wavelet)[:len(y)]

clean = np.sin(np.linspace(0,20,1024))
noisy = clean + 0.3*np.random.randn(1024)
denoised = wavelet_denoise(noisy)

# Access approximation and detail coefficients
