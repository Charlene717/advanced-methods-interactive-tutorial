Knockoff Filter（控制 FDR 的變數選擇）

Barber & Candès 2015 (AOS)、Candès 2018 (JRSS-B) 提出的革命性架構：為每個變數 X_j 構造一個「knockoff」X̃_j 作為 negative control，比較 X_j 與 X̃_j 在模型中的重要性，嚴格控 FDR ≤ q。GWAS、單細胞 DE、藥物標靶識別等領域已大量採用。

A revolutionary framework by Barber & Candès 2015 (AoS) and Candès 2018 (JRSS-B): construct a "knockoff" X̃_j as a negative control for each X_j, compare their importance in any model, and strictly control FDR ≤ q. Widely adopted in GWAS, single-cell DE, drug-target identification, etc.

核心概念

什麼是 Knockoff？為何能控 FDR？

問題：高維迴歸（p ≫ n 或 p ~ n）中，LASSO、隨機森林等模型「選出來」的變數可能有大量假陽性。傳統 BH (Benjamini-Hochberg) 需要 p-values，但在 LASSO / DL 模型中 p-value 沒有 closed form。

Knockoff 的天才之處：不需要 p-value！只要為每個 X_j 構造一個 knockoff X̃_j，滿足兩個性質：

  • ① 交換對稱（pairwise exchangeability）：把 (X_j, X̃_j) 對調後，聯合分布不變。直觀：knockoff「長得像」原變數。
  • ② 條件獨立：X̃_j 與 Y 在給定 X 後條件獨立——knockoff 是「假變數」，與 Y 沒有真實連結。

FDR 控制機制：用任何模型（LASSO、RF、XGBoost）算出 importance score Z_j (for X_j) 與 Z̃_j (for X̃_j)，定義對比統計：

W_j = Z_j − Z̃_j

對「null 變數」（X_j 與 Y 真實無關），由對稱性 W_j 有 50% 正、50% 負的機率（sign-flip symmetry）。對「真信號」，W_j 趨向大正值。挑「W_j ≥ τ」即可控 FDR，其中 τ 用「sign-flip 估計」自動決定。

The problem: in high-dimensional regression (p ≫ n or p ~ n), LASSO, random forest, etc. select many false positives. Classical BH (Benjamini-Hochberg) needs p-values, which have no closed form for LASSO / DL.

Knockoff's genius: no p-values needed! Construct a knockoff X̃_j for each X_j satisfying:

  • ① Pairwise exchangeability: swapping (X_j, X̃_j) leaves the joint distribution unchanged. Intuition: knockoff "looks like" the original.
  • ② Conditional independence: X̃_j ⊥ Y | X — knockoffs are "fake" variables, never truly linked to Y.

FDR control mechanism: for any model (LASSO, RF, XGBoost) compute importance Z_j and Z̃_j; define contrast statistic

W_j = Z_j − Z̃_j

For null variables (no true link to Y), symmetry gives W_j positive 50% / negative 50% (sign-flip symmetry). For true signals, W_j skews large positive. Selecting "W_j ≥ τ" controls FDR, with τ auto-chosen by sign-flip estimation.

演算法

從 Fixed-X 到 Model-X 與 Deep Knockoffs

① Fixed-X Knockoff (Barber & Candès 2015, AoS)：假設線性模型 Y = Xβ + ε（高斯噪音）、n ≥ 2p。構造 X̃ 使得 X̃ᵀX̃ = XᵀX 且 X̃ᵀX = XᵀX − diag(s)（s ≥ 0, 2XᵀX − diag(s) ≽ 0）。基本上是矩陣分解 trick，要求 n ≥ 2p 限制了實用性。

② Model-X Knockoff (Candès 2018, JRSS-B)：放鬆條件——只要知道 X 的「邊際分布 p(X)」即可，不限定 n vs p 關係。對 Gaussian X 可用 SDP 或 ASDP 解 s；對非高斯 X 用 Hidden Markov Model 或 mixture。

[X, X̃] ~ joint, swappable pairwise

③ Knockoff Filter 流程：

  1. 構造 knockoff 矩陣 X̃。
  2. 用 [X, X̃] (n × 2p) 跑 LASSO / RF，得 importance Z_j, Z̃_j。
  3. W_j = Z_j − Z̃_j。
  4. 給定 q（如 0.1），找最小 τ 使 #{j: W_j ≤ −τ} / max(#{j: W_j ≥ τ}, 1) ≤ q。
  5. 選 Ŝ = {j: W_j ≥ τ}。

④ Deep Knockoffs (Romano 2020, JASA)：用神經網路（含對抗訓練）學 swappable 的 X̃，不需顯式 p(X)。loss 由 MMD 或 Sinkhorn divergence 度量「swap 後分布是否改變」。可處理任意非線性、非高斯 X。

⑤ KnockoffGAN (Jordon 2019)：用 GAN 架構生成 knockoff——generator 輸出 X̃，discriminator 嘗試區分「swapped vs original」。比 Deep Knockoffs 更靈活，但訓練較不穩。

⑥ Knockoff+：原版 FDR ≤ q + 1/m；Knockoff+ 用「+1」修正使 FDR ≤ q 嚴格成立（代價是 power 略低）。

① Fixed-X Knockoff (Barber & Candès 2015, AoS): assumes linear Y = Xβ + ε (Gaussian noise), n ≥ 2p. Construct X̃ with X̃ᵀX̃ = XᵀX and X̃ᵀX = XᵀX − diag(s) (s ≥ 0, 2XᵀX − diag(s) ≽ 0). Matrix factorization trick; n ≥ 2p limits practicality.

② Model-X Knockoff (Candès 2018, JRSS-B): only needs the marginal p(X), no n-vs-p restriction. For Gaussian X solve s via SDP or ASDP; for non-Gaussian X use HMM or mixture models.

[X, X̃] ~ joint, swappable pairwise

③ Knockoff Filter pipeline:

  1. Construct knockoff matrix X̃.
  2. Run LASSO / RF on [X, X̃] (n × 2p) → importance Z_j, Z̃_j.
  3. W_j = Z_j − Z̃_j.
  4. Given q (e.g. 0.1), find smallest τ such that #{j: W_j ≤ −τ} / max(#{j: W_j ≥ τ}, 1) ≤ q.
  5. Select Ŝ = {j: W_j ≥ τ}.

④ Deep Knockoffs (Romano 2020, JASA): learn swappable X̃ via neural networks (adversarial training); loss is MMD or Sinkhorn divergence checking "swap-invariance". Handles arbitrary non-linear, non-Gaussian X.

⑤ KnockoffGAN (Jordon 2019): generator outputs X̃, discriminator distinguishes "swapped vs original". More flexible than Deep Knockoffs but less stable training.

⑥ Knockoff+: original guarantees FDR ≤ q + 1/m; Knockoff+ adds a "+1" correction giving strict FDR ≤ q (slightly lower power).

為何不直接用 permutation test？Permutation 打亂 Y（或 X 整體），破壞 X 之間的相關結構——對相關性高的變數（如連鎖不平衡的 SNP）會嚴重低估 FDR。Knockoff 保留 X 內部結構，只「對沖」每個變數與 Y 的關係，正是其強大之處。
Why not just permutation test? Permutation shuffles Y (or X entirely), breaking inter-X correlation — for correlated variables (e.g. SNPs in LD) FDR is severely underestimated. Knockoffs preserve X's internal structure and only "cancel out" each variable's link to Y — that's the secret sauce.

🌳 什麼時候用 Knockoff？

情境1:
高維變數選擇 + 需要 FDR 保證——GWAS（百萬 SNP）、單細胞 DE（萬基因 × 千細胞）。
情境2:
變數高度相關——SNP 連鎖不平衡、pathway 基因共表達；permutation 會崩。
情境3:
用 black-box 模型（RF, DL），沒有 p-value 但要 FDR——Knockoff 是少數選擇。
不適合:
n < p 且 X 分布完全未知時——Deep Knockoffs 可能訓練不收斂；建議先做變數預篩。
Case 1:
High-dim selection with FDR guarantee — GWAS (millions of SNPs), scRNA-seq DE (10k genes × 1k cells).
Case 2:
Highly correlated variables — SNP LD, co-expressed pathway genes; permutation collapses.
Case 3:
Black-box models (RF, DL) with no p-value but need FDR — Knockoffs are one of the few options.
Not for:
n < p and X distribution entirely unknown — Deep Knockoffs may not converge; consider pre-screening first.
生物應用

Knockoff 在生物資訊的應用

KnockoffZoom — GWAS

Sesia et al. (2019, Bioinformatics) 用 HMM 對 phased haplotype 建模，產生 SNP knockoffs——在 UK Biobank 上做出嚴格 FDR 控制的 GWAS。可同時做「多解析度定位」：基因區域 / LD block / 單 SNP 三層 FDR 控制。

Sesia et al. (2019, Bioinformatics) model phased haplotypes with HMM to generate SNP knockoffs — strict FDR-controlled GWAS on UK Biobank. Multi-resolution localization: gene-region / LD-block / single-SNP FDR control.

DeepPINK — DL importance

Lu et al. (2018, NeurIPS) 把 knockoff 與深度學習結合：在 NN 第一層用「pairwise-connected」結構（X_j 與 X̃_j 各自接一條 edge），讀取 |w_j| − |w̃_j| 作為 importance。用於 microbiome 與 omics 變數選擇。

Lu et al. (2018, NeurIPS) marry knockoffs with deep learning: NN first layer is "pairwise-connected" (X_j and X̃_j each connect via one edge), use |w_j| − |w̃_j| as importance. Applied to microbiome and omics variable selection.

Genome-wide Knockoffs

Sesia et al. (2021, Nature Communications) 把 KnockoffZoom 擴展為全基因組規模——對 UK Biobank 487k 樣本、9.4M SNP 找出 LDL/HDL/diabetes 等性狀的 candidate gene，FDR 嚴格 ≤ 0.1。多項發現後續被 functional validation 確認。

Sesia et al. (2021, Nature Communications) scale KnockoffZoom genome-wide — 487k UK Biobank samples × 9.4M SNPs, candidate genes for LDL/HDL/diabetes at strict FDR ≤ 0.1. Multiple discoveries since functionally validated.

scKnockoff — scRNA-seq DE

傳統 scRNA-seq DE（Wilcoxon, MAST）忽略基因相關結構，inflated FDR；用 Model-X knockoff 對基因表達建 multivariate Gaussian copula，FDR 控制嚴格。對識別 rare cell-type marker 特別重要（小細胞數 → permutation 不穩）。

Classical scRNA-seq DE (Wilcoxon, MAST) ignore gene-gene correlation → inflated FDR; Model-X knockoffs with a Gaussian copula on gene expression give strict FDR. Critical for rare cell-type markers (low cell count breaks permutation).

Drug-target identification

藥物 high-throughput screening：找出哪些蛋白質目標真正與 phenotype 相關。用 Knockoff filter 對化合物-蛋白親和力矩陣做 FDR controlled selection——避免「能結合 → 認為是 hit」的 multiple testing 災難。已用於 DeepChem、Open Targets pipelines。

High-throughput drug screening: which protein targets really link to phenotype? Knockoff filter on compound-protein affinity matrix gives FDR-controlled selection — avoids the "binds → is a hit" multiple-testing disaster. Used in DeepChem and Open Targets pipelines.

多基因 PRS validation

Polygenic Risk Score 用上百 SNP 加權算疾病風險；要驗證「哪些 SNP 真有貢獻」——用 Knockoff 篩選出 reproducible 的 PRS components。在不同 ancestry cohort 之間做 transfer 時尤其關鍵。

PRS combines hundreds of SNPs into disease risk; need to validate "which SNPs truly contribute" — Knockoffs filter out reproducible PRS components. Especially crucial for cross-ancestry PRS transfer.

Multi-omics integration

整合 RNA、ATAC、protein 預測 phenotype——每個 modality 的特徵相關性結構不同。Multi-Block Knockoff (Liu 2022) 對每個 block 分別構造 knockoff、聯合篩選，避免 single-modality 的 FDR 漏控。

Integrating RNA, ATAC, protein for phenotype prediction — each modality has different correlation structure. Multi-Block Knockoff (Liu 2022) builds knockoffs per block and selects jointly, avoiding single-modality FDR leakage.

HMM Knockoff

Sesia et al. (2018, JRSS-B) 對 phased 基因組資料用 fastPHASE-style HMM 建模——hidden state 是「祖先 haplotype」、emission 是 SNP genotype。在這個 HMM 上 sample knockoff，自動保留 LD 結構，是 GWAS knockoff 的標準作法。

Sesia et al. (2018, JRSS-B) model phased haplotypes with a fastPHASE-style HMM — hidden states are ancestral haplotypes, emissions are SNP genotypes. Sampling knockoffs on this HMM automatically preserves LD, the standard GWAS knockoff recipe.

深入解析

Knockoff 的進階主題

① 為何 sign-flip 推論成立？

對 null 變數 j，由 (X_j, X̃_j) 的 swappability 與 X̃_j 與 Y 的條件獨立，可證 W_j 對稱分布。整個 (W_1, ..., W_p) 中 null 部分的「符號」是 iid Bernoulli(½)——這個 finite-sample 性質讓 FDR 控制不需漸近。詳見 Barber & Candès (2015, AoS) Theorem 1。

② Power 的代價

Knockoff 的代價：對每個變數要加一個 negative control，有效樣本量減半。Power 通常比 oracle 方法低 10-30%。若 X 之間相關性極高，knockoff 可能與原變數「太像」，導致 W_j 接近 0、無法區分 → power 進一步降低。

③ SDP / ASDP solving

Gaussian Model-X 找 s 是個 SDP：maximize Σ s_j subject to 2Σ − diag(s) ≽ 0, s ≥ 0, s ≤ 2 diag(Σ)。當 p 很大時 SDP 慢——ASDP (Approximate SDP) 用 block-coordinate 解，p = 10^5 可行。

④ 統計 power 增強

  • 多套 knockoffs (Gimenez 2019)：用多個獨立 knockoff 投票，提升 stability。
  • Aggregated knockoffs (Nguyen 2020)：每次跑 knockoff filter 結果不同（隨機性），多次平均選頻最高的變數。
  • Knockoff e-values (Ren 2024)：新發展用 e-value 框架統一 knockoff + BH，可結合不同 modality。

⑤ 與 conformal inference 的關係

兩者都用「exchangeability」獲得 finite-sample guarantee；conformal 用 augment test+calibration 做預測區間，knockoff 用 negative control + sign-flip 做變數選擇。Conformalized Knockoff (Saha 2024) 把兩者結合，同時做 selection + prediction interval。

⑥ 因果推論視角

Knockoff 控的是 FDR for conditional independence——選出的變數 X_j 是「在給定其他變數後仍與 Y 相關」。這對應 Markov blanket、d-separation 中的「causal parent」候選。某種意義上 knockoff 是 conditional-independence-based causal discovery 的一個 ingredient。

⑦ 實務 pitfalls

  • p(X) misspecification：若假設 Gaussian 但實際非 Gaussian，FDR 控制失效。對 binary 變數請用 mixture 或 Deep Knockoffs。
  • SNP missing genotype：HMM knockoff 對 missing data 敏感，建議用 IMPUTE2 / Beagle phased 後再做 knockoff。
  • 對非常稀疏的 sparse Y：若 Y 是極不平衡 binary，LASSO importance 可能全為 0，knockoff 完全沒有 power。
  • Q 過小 → 0 variables selected：q = 0.01 在小樣本可能拒絕一切，建議從 q = 0.1 起跳。

① Why does sign-flip work?

For null j, swappability of (X_j, X̃_j) plus X̃_j ⊥ Y | X gives W_j a symmetric distribution. The signs of all null W_j's are jointly iid Bernoulli(½) — this finite-sample property makes FDR control asymptotic-free. See Barber & Candès (2015, AoS) Theorem 1.

② Power cost

Cost of knockoffs: each variable gets a negative control, effective sample size halves. Power is usually 10-30% below oracle. If X's are very correlated, knockoffs may look "too similar" to originals, W_j ≈ 0, killing power.

③ SDP / ASDP solving

Finding s for Gaussian Model-X is an SDP: maximize Σ s_j subject to 2Σ − diag(s) ≽ 0, s ≥ 0, s ≤ 2 diag(Σ). Slow for large p — ASDP (Approximate SDP) uses block coordinate solve, feasible for p = 10^5.

④ Power enhancement

  • Multiple knockoffs (Gimenez 2019): independent knockoffs vote, boosts stability.
  • Aggregated knockoffs (Nguyen 2020): each run varies (randomness); average to pick most-frequent variables.
  • Knockoff e-values (Ren 2024): unify knockoffs + BH in the e-value framework; allows multi-modality combination.

⑤ Relation to conformal inference

Both use "exchangeability" for finite-sample guarantees; conformal uses augment test+calibration for prediction intervals, knockoff uses negative control + sign-flip for variable selection. Conformalized Knockoff (Saha 2024) merges them: joint selection + prediction interval.

⑥ Causal view

Knockoffs control FDR for conditional independence — selected X_j is "still related to Y given others". Maps to Markov blanket / d-separation "causal parent" candidates. In a sense knockoffs are an ingredient of CI-based causal discovery.

⑦ Practical pitfalls

  • p(X) misspecification: assuming Gaussian when X is non-Gaussian breaks FDR. Use mixtures or Deep Knockoffs for binary.
  • SNP missing genotype: HMM knockoff is sensitive — impute first (IMPUTE2 / Beagle).
  • Very sparse / imbalanced Y: LASSO importance may be all zero, knockoff loses power.
  • Tiny q → 0 selected: q = 0.01 may reject everything in small samples — start at q = 0.1.
潛在改良應用

還能改良哪些任務？

🎯 1. scLLM importance interpretation 加 FDR

scGPT / Geneformer 的 attention / SHAP 可給 gene importance，但都沒 FDR 控制。提案：對 transformer hidden representation 應用 Deep Knockoffs，給每個基因 importance「reproducible」標籤。

🎯 2. Cell-type marker discovery 嚴格 FDR

Seurat findMarkers 報出來的 marker 常有大量假陽性。提案：scKnockoff pipeline——對每個 cluster 找出 FDR ≤ 0.1 的 marker，並提供 power calibration 工具。

🎯 3. Spatial GWAS

Spatial transcriptomics + 基因型聯合分析時，多 spatial coordinate 的 spatial autocorrelation 會放大 type-1 error。提案：Spatial Knockoff——把 spatial GP 結構放進 p(X)，自動處理 spatial correlation。

🎯 4. CRISPR screen hit calling

MAGeCK 等工具用 NB 模型挑 hits，但對 perturbation 強相關 sgRNA 控不住 FDR。提案：對 sgRNA-by-gene 計分矩陣做 Knockoff filter，提供「基因-level」FDR 保證。

🎯 5. Drug-drug interaction 的 FDR controlled discovery

Drug pairs interaction matrix 維度爆炸（C(p, 2)）；對 importance 做 Knockoff 可有效控 FDR。提案：應用於 BeatAML、CCLE 等高通量 drug combination screen 資料。

🎯 6. Federated Knockoffs

多醫院聯合 GWAS 時無法分享 raw genotype；用 federated learning 訓 Deep Knockoff generator，再分享 knockoff 與 importance scores。提案：FedKnockoff for multi-site biobank study。

🎯 7. Time-series Knockoff for longitudinal omics

縱貫資料（如 ICU multi-organ time-series）的變數選擇——把 lag/state structure 編進 HMM knockoff。提案：longitudinal-Knockoff for sepsis / 慢性病進展。

🎯 1. FDR-controlled scLLM interpretation

scGPT / Geneformer attention / SHAP give importance but no FDR. Proposal: apply Deep Knockoffs to transformer hidden representations for "reproducible importance".

🎯 2. Strict-FDR cell-type marker discovery

Seurat findMarkers yields many false positives. Proposal: scKnockoff pipeline — per-cluster markers at FDR ≤ 0.1 with power calibration.

🎯 3. Spatial GWAS

Joint spatial transcriptomics + genotype analysis suffers from spatial autocorrelation inflating type-I error. Proposal: Spatial Knockoff embedding spatial GP into p(X).

🎯 4. CRISPR screen hit calling

MAGeCK uses NB models but loses FDR control for correlated sgRNAs. Proposal: Knockoff filter on sgRNA-by-gene matrices → gene-level FDR.

🎯 5. FDR-controlled drug-drug interaction

Drug-pair matrices explode in dimension (C(p, 2)); Knockoffs handle this naturally. Proposal: apply to BeatAML, CCLE drug combination screens.

🎯 6. Federated Knockoffs

Multi-site GWAS cannot share raw genotypes; federated training of Deep Knockoff generators allows sharing only knockoffs + importances. Proposal: FedKnockoff for multi-site biobanks.

🎯 7. Longitudinal Knockoffs

Longitudinal data (ICU multi-organ time-series) variable selection — encode lag/state into HMM knockoffs. Proposal: longitudinal-Knockoff for sepsis / chronic-disease progression.
