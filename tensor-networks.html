張量網路 (Tensor Networks)：MPS / PEPS

物理學家 90 年代用來壓縮量子多體波函數的工具，意外成為高維資料壓縮的利器。MPS（矩陣乘積態）把長度 L 的 d-state 系統從 d^L 壓縮到 L · d · χ²；DMRG (White 1992) 可變分求解；ML 上稱 Tensor Train (Oseledets 2011)。已被用於基因組壓縮、長序列建模、量子啟發 ML。

A tool physicists invented in the 90s to compress quantum many-body wavefunctions, repurposed as a powerful high-dimensional data compressor. MPS (Matrix Product State) compresses an L-site d-state system from d^L to L · d · χ². DMRG (White 1992) provides a variational solver; in ML it appears as Tensor Train (Oseledets 2011). Applied to genome compression, long-sequence modeling, and quantum-inspired ML.

核心概念

維度詛咒與 bond dimension

問題：一個長度 L、每個位置有 d 種狀態的系統，完整描述需要 d^L 個參數（例如 L = 100 個 spin-½ 量子位元需要 2^100 ≈ 10^30 個複數）。不可能存進記憶體。

關鍵洞見：大部分物理上「有意義」的態（gapped Hamiltonian 的 ground state、局部相關性衰減的態）並不在這個 d^L 維空間的隨機角落，而是位於一個「低糾纏 (low-entanglement) 流形」上。 這個流形可以用「張量網路」高效參數化。

MPS (Matrix Product State)：把整個高階張量 Ψ_{s₁s₂...s_L} 寫成一串矩陣連乘：

Ψ_{s₁...s_L} = A^{s₁}_{1,α₁} · A^{s₂}_{α₁,α₂} · ⋯ · A^{s_L}_{α_{L−1},1}

每個 A^{s_k} 是 (χ × d × χ) 的張量，α 是「bond index」，bond dimension χ 控制可表達的糾纏熵上限 S ≤ log χ。總參數量：L · d · χ²（線性而非指數）。

PEPS (Projected Entangled Pair State, Verstraete 2004)：把 MPS 從 1D 鏈推廣到 2D 網格——每個 site 連接 4 個 bond（上下左右）。可表達 2D 量子系統與「面積律」糾纏。代價：精確收縮 PEPS 是 #P-hard，需要近似 algorithm（CTMRG、boundary MPS）。

ML 視角：Tensor Train (Oseledets 2011, SIAM J Sci Comput)。數學上與 MPS 完全等價，但語境是「高維函數 / 張量壓縮」而非量子物理。被用於 neural network weight 壓縮、recommendation system、PDE 求解。

The problem: a length-L system with d states per site needs d^L parameters (L = 100 spin-½ qubits ≈ 2^100 ≈ 10^30 complex numbers). Impossible to store.

Key insight: physically meaningful states (gapped Hamiltonian ground states, locally correlated states) do not occupy random corners of this d^L space — they live on a "low-entanglement manifold" efficiently parameterized by tensor networks.

MPS (Matrix Product State): write the high-order tensor Ψ_{s₁s₂...s_L} as a chain of matrix products:

Ψ_{s₁...s_L} = A^{s₁}_{1,α₁} · A^{s₂}_{α₁,α₂} · ⋯ · A^{s_L}_{α_{L−1},1}

Each A^{s_k} is (χ × d × χ), α is a "bond index", and bond dimension χ caps the entanglement entropy S ≤ log χ. Total parameters: L · d · χ² — linear, not exponential.

PEPS (Projected Entangled Pair State, Verstraete 2004): generalize MPS from a 1D chain to a 2D grid — each site has 4 bonds (up/down/left/right). Captures 2D quantum systems with area-law entanglement. Caveat: exact PEPS contraction is #P-hard — requires approximations (CTMRG, boundary-MPS).

ML view: Tensor Train (Oseledets 2011, SIAM J Sci Comput). Mathematically identical to MPS, but framed as "high-dim function / tensor compression" rather than quantum physics. Used for neural-network weight compression, recommendation systems, PDE solvers.

演算法

DMRG, TEBD, VMPS, Tensor Train

① DMRG (White 1992, Phys Rev Lett)：密度矩陣重整化群——MPS 出現前的偉大發明，事後被 Östlund & Rommer (1995) 證實 DMRG 就是在優化 MPS。演算法：

  • 從隨機 MPS 出發，sweep 過每個 site：固定其他張量，對當前兩個相鄰 site 解 local eigenproblem，得到最佳 (χ × d² × χ) block tensor。
  • 對其做 SVD 並截斷至 χ → 把 block 拆回成兩個 site tensors，bond dimension 控制在 χ。
  • 左右來回 sweep 直到能量收斂。

DMRG 是 1D gapped Hamiltonian ground state 的 gold standard——比 quantum Monte Carlo 對許多模型更精準。

② TEBD (Time-Evolving Block Decimation, Vidal 2003)：用 Trotter 分解時間演化算符 e^{−iHt} 為 local gates，每個 gate 作用在相鄰兩個 site 上 → 再 SVD 截斷。這就是「在 MPS 上跑量子動力學」。

③ VMPS (Variational MPS)：把 MPS 當參數化變分態，直接梯度下降 / Adam 優化（不只 DMRG 的 sweep）。便於整合進現代 ML pipeline（PyTorch、JAX）。

④ Tensor Train (TT) decomposition：對給定的 d^L 維 tensor X，用 TT-SVD 演算法（Oseledets 2011）：

  1. 把 X reshape 為 (d) × (d^{L−1}) 矩陣，做 SVD → 截斷至 χ。
  2. 對 V^T reshape 為 (χd) × (d^{L−2}) 再 SVD → 截斷。
  3. 反覆做 L−1 次 → 得到 L 個 core tensors 即 TT 表示。

⑤ TT-Cross / DMRG-Cross：當完整 X 無法存進記憶體（這正是 TT 的應用情境），TT-Cross 用 maximum-volume sub-matrix 啟發式（Oseledets & Tyrtyshnikov 2010）僅查詢 O(L · χ²) 個 entry 就可重建。

⑥ Tensor Network classifier (Stoudenmire & Schwab 2016, NeurIPS)：把 input image 經 feature map 變成 product state，再用 MPS 做 contraction 輸出 logit。等價於 SVM with implicit kernel；可解釋（每個 bond 是「資訊瓶頸」）。

① DMRG (White 1992, Phys Rev Lett): density matrix renormalization group — a pre-MPS invention, later shown by Östlund & Rommer (1995) to optimize an MPS. Algorithm:

  • Start with a random MPS; sweep across sites — fix all but two adjacent tensors and solve a local eigenproblem on the (χ × d² × χ) block.
  • SVD-truncate to χ → split back into two site tensors.
  • Sweep left-right until energy converges.

DMRG is the gold standard for 1D gapped-Hamiltonian ground states — often more accurate than quantum Monte Carlo.

② TEBD (Time-Evolving Block Decimation, Vidal 2003): Trotterize e^{−iHt} into local two-site gates; apply each gate + SVD truncate. Quantum dynamics on an MPS.

③ VMPS (Variational MPS): treat MPS as a parametric ansatz and optimize by gradient descent / Adam (not only DMRG sweeps). Plays nicely with modern ML stacks (PyTorch, JAX).

④ Tensor Train (TT) decomposition: given a d^L tensor X, apply TT-SVD (Oseledets 2011):

  1. Reshape X as (d) × (d^{L−1}) and SVD → truncate to χ.
  2. Reshape V^T as (χd) × (d^{L−2}) and SVD → truncate.
  3. Repeat L−1 times → L core tensors form the TT.

⑤ TT-Cross / DMRG-Cross: when X is too large to fit in memory (the typical TT use case), TT-Cross (Oseledets & Tyrtyshnikov 2010) uses maximum-volume sub-matrices to reconstruct using only O(L · χ²) queries.

⑥ Tensor Network classifier (Stoudenmire & Schwab 2016, NeurIPS): feature-map the input image into a product state, contract with an MPS to output logits. Equivalent to an SVM with implicit kernel; interpretable (each bond is an information bottleneck).

記憶法：χ = 1 → product state（每個 site 獨立，零糾纏）。χ → ∞ → 完整 d^L 表示（任意糾纏）。中間 χ 是「在多少糾纏上限下的最佳近似」。area-law 系統可用小 χ；critical / random 系統需大 χ。
Mnemonic: χ = 1 → product state (independent sites, zero entanglement). χ → ∞ → exact d^L representation. Intermediate χ → "best approximation under an entanglement budget". Area-law systems work with small χ; critical / random systems need large χ.

1D quantum ground state / dynamics → MPS + DMRG / TEBD. Physics gold standard.
Case 2:
2D / lattice systems (quantum magnets, Hi-C grids, image patches) → PEPS or 2D-MPS (snake ordering).
Case 3:

Not for:

生物應用

Tensor Networks 在生物資訊

基因組壓縮

基因組序列在物種內高度相似（>99% identity）；可看作 4-state（ACGT）長序列張量的低糾纏近似。Cheng et al. (2019) 用 MPS 壓縮人類基因組 reference，與 gzip-like 工具比較壓縮率更好且支援隨機讀取（局部 contraction）。

Genomes within a species are >99% identical — a 4-state (ACGT) long-sequence tensor with low entanglement. Cheng et al. (2019) compress human reference genomes with MPS, achieving better ratios than gzip-like tools while supporting random access via local contraction.

蛋白質結構壓縮

Glasser et al. (2018, Phys Rev X) 把 MPS / TN 用於蛋白質序列 / 結構聯合表示——比 hidden Markov model 表達更豐富，能捕捉長距 contact map 結構。Tensor Network ↔ HMM 等價對應關係也在這篇釐清。

Glasser et al. (2018, Phys Rev X) use MPS / TN for joint protein sequence / structure representation — more expressive than HMMs, captures long-range contact-map structure. They also formalize the TN ↔ HMM correspondence.

量子啟發 ML for 基因組

Liu et al. (2019, PRX Quantum) 用 TN classifier 做基因表達分類——把 RNA-seq sample 經 normalization 變成 product state，MPS 收縮輸出 cancer subtype。性能與 SVM 相當但每個 bond 可解釋為「site-pair 互動」。

Liu et al. (2019, PRX Quantum) classify gene expression with TN — RNA-seq samples become product states; the MPS contraction outputs cancer subtype. Matches SVM performance, but each bond is interpretable as a "site-pair interaction".

基因調控網路壓縮

基因調控網路含 ~20k 個基因、每對基因可能互動——完整聯合分布是 2^20000。用 MPS 把 transcriptome 視為 1D chain，χ ~ 32 即可重建 80%+ 變異——比 PCA 表達能力更高。

A gene regulatory network has ~20k genes with potential pairwise interactions — the full joint distribution is 2^20000. MPS treats the transcriptome as a 1D chain; χ ~ 32 reconstructs 80%+ of variance — more expressive than PCA.

MPS classifier

Stoudenmire & Schwab (2016, NeurIPS) 開創「supervised tensor-network learning」——MNIST classifier 用 MPS bond dimension χ = 10 達到 ~99% 準確率，後續被應用於 small biological tabular datasets（cancer-vs-normal、disease subtyping）。

Stoudenmire & Schwab (2016, NeurIPS) pioneer "supervised tensor-network learning" — MNIST with MPS bond χ = 10 reaches ~99% accuracy. Later applied to small biological tabular datasets (cancer-vs-normal, disease subtyping).

Tensorized RNN

Yang et al. (2017, ICML) 把 RNN 的 weight 用 TT 分解——對 long-context 任務（如全 genome 序列建模）顯著節省參數。可視為「結構化線性 attention」，與 Mamba / S4 等 SSM 不同設計理念但有交集。

Yang et al. (2017, ICML) decompose RNN weights via TT — dramatic parameter savings on long-context tasks (e.g. full-genome modeling). Can be viewed as "structured linear attention", complementary to but distinct from Mamba / S4 SSMs.

多重組學高維張量

病人 × 基因 × organ × time × omic 是天然 5-way tensor，元素數量爆炸。TT 表示可在 O(L · χ²) 參數內近似，配合 PARAFAC / Tucker 比較。已有 prototypes 用於 TCGA 多 modality 整合。

A patient × gene × organ × time × omic dataset is naturally a 5-way tensor with explosive size. TT represents it in O(L · χ²) parameters, complementing PARAFAC / Tucker. Prototypes exist for TCGA multi-modality integration.

MPS for Hi-C / 染色質結構

Hi-C contact map 是 (loci × loci) 矩陣，但更複雜的多體 contact（triple-way contact）需要 high-order tensor。MPS / PEPS 可在 area-law 假設下高效表達——若 chromatin 滿足近似 area-law（TAD-like 局部結構），壓縮比可達 10⁴×。

Hi-C contact maps are (loci × loci); but multi-body contacts (triple-way) need high-order tensors. MPS / PEPS represent them efficiently under area-law assumptions — if chromatin is approximately area-law (TAD-local structure), compression up to 10⁴×.

深入解析

進階主題

① Area-law 是 MPS 可行的關鍵

Hastings (2007, J Stat Mech) 證明：1D gapped Hamiltonian 的 ground state 滿足 area law（糾纏熵 S ~ const，不隨 L 成長），因此 MPS bond dimension χ 不依賴 L。Critical 系統有 log-corrected area law（S ~ log L），需 χ ~ poly(L)。高溫 thermal state / random state 滿足 volume law（S ~ L），MPS 失效。

② Tensor Network 分類學

  • MPS（1D chain）— 量子物理 + Tensor Train
  • PEPS / PEPO（2D / 3D lattice）— 2D 量子相
  • MERA（Multi-scale Entanglement Renormalization Ansatz, Vidal 2007）— 含 disentangler 與 isometry，對 critical systems 表達 log L 糾纏
  • Tree Tensor Network（TTN）— 樹狀拓撲，介於 MPS 與 MERA 之間
  • Branching MERA（Evenbly & Vidal 2014）— 對 fermionic critical systems

③ Canonical form 與數值穩定

MPS 有「測度自由度」（gauge freedom）：對相鄰兩個 site tensors 同時做 A → A G, B → G⁻¹ B 不改變 contraction。實務上需把 MPS 帶到 canonical form（left/right/mixed canonical）以穩定 SVD truncation 與 expectation value 計算。iTensor / TeNPy 自動處理。

④ PEPS contraction 為何 #P-hard

MPS 的 contraction 可線性時間完成（從左到右）；PEPS 必須收縮 2D 網格，contraction order 沒有「天然」的 path。Schuch et al. (2007) 證明精確 contraction 是 #P-hard。實務用 boundary-MPS：把第一行看成 MPS，逐列吸收剩餘 sites（保留 bond dim χ_b）。

⑤ Tensor Network 與 ML 的橋樑

  • TN ≈ structured neural network：MPS 可看作「線性 RNN with structured weight」。
  • TN ≈ kernel method：MPS classifier 等價於「product feature map + linear classifier」。
  • TN ≈ probabilistic graphical model：MPS = HMM 的非負矩陣版本；PEPS = MRF。

⑥ 自動微分與 TN library

現代 TN library（quimb, TensorNetwork, jax-tn）支援 JAX/PyTorch 自動微分——可把 MPS contraction 串進 deep learning pipeline 並用 Adam 訓練。對 PEPS 而言，approximate contraction 也可微分，但梯度可能不穩。

⑦ 量子計算與經典模擬的交界

當前最強的量子模擬器（如 ITensor、TeNPy、quimb 的 tensor network 模擬器）能在筆電上模擬數百 qubit 的「淺電路」——只要糾纏熵不爆。Google Sycamore quantum supremacy 主張被 Pan et al. (2022) 的 TN 模擬挑戰，把預期 10,000 年降到幾小時。

① Area law makes MPS work

Hastings (2007, J Stat Mech) proved: 1D gapped Hamiltonian ground states satisfy area law (S ~ const, independent of L), so MPS bond χ doesn't grow with L. Critical systems have log-corrected area law (S ~ log L), requiring χ ~ poly(L). Thermal / random states satisfy volume law (S ~ L) — MPS fails.

② Tensor Network taxonomy

  • MPS (1D chain) — quantum physics + Tensor Train
  • PEPS / PEPO (2D / 3D lattice) — 2D quantum phases
  • MERA (Multi-scale Entanglement Renormalization Ansatz, Vidal 2007) — disentangler + isometry; expresses log-L entanglement of critical systems
  • Tree Tensor Network (TTN) — tree topology, between MPS and MERA
  • Branching MERA (Evenbly & Vidal 2014) — fermionic critical systems

③ Canonical form and numerical stability

MPS has gauge freedom: A → A G, B → G⁻¹ B leaves contractions invariant. In practice we put MPS in canonical form (left/right/mixed) to stabilize SVD truncation and expectation values. iTensor / TeNPy handle this automatically.

④ Why PEPS contraction is #P-hard

MPS contraction is linear-time (left to right); PEPS lacks a natural contraction order. Schuch et al. (2007) showed exact contraction is #P-hard. Practitioners use boundary-MPS: treat the first row as an MPS, absorb remaining rows column by column (keep χ_b).

⑤ Bridges to ML

  • TN ≈ structured neural network: MPS as "linear RNN with structured weights".
  • TN ≈ kernel method: MPS classifier = "product feature map + linear classifier".
  • TN ≈ probabilistic graphical model: MPS = non-negative-matrix HMM; PEPS = MRF.

⑥ Autodiff and TN libraries

Modern TN libraries (quimb, TensorNetwork, jax-tn) support JAX/PyTorch autodiff — MPS contractions plug into deep-learning pipelines trained with Adam. PEPS approximate contractions are differentiable but gradients can be unstable.

⑦ Quantum computing vs classical simulation

The best modern quantum simulators (ITensor, TeNPy, quimb) simulate hundreds of qubits with "shallow circuits" on a laptop — as long as entanglement doesn't explode. Google's Sycamore quantum-supremacy claim was challenged by Pan et al. (2022) with TN simulation that cut the expected 10,000 years to hours.

潛在改良應用

還能改良哪些任務？

🎯 1. 全基因組長序列建模

HyenaDNA / EVO 用 SSM 處理長基因組——但其實基因組是強局部相關（gene → exon → codon）的 area-law 序列。提案：MPS / TT 替代 SSM 做基因組 LM，χ 控制 receptive field，可解釋為 site-pair 互動。

🎯 2. Hi-C contact map 的 PEPS 模型

2D Hi-C 滿足 area-law（TAD-local 結構）——天然 PEPS 候選。提案：PEPS-based Hi-C reconstruction，從稀疏 contact 還原 dense map；bond dimension 為「TAD structure complexity」。

🎯 3. 多病人多模態 tensor 整合

(病人 × 基因 × organ × time × omic) 5-way tensor。提案：TT 分解 + 缺失填補；每個 core 對應一個 modality 的「latent program」，可比 PARAFAC 更高表達能力。

🎯 4. 蛋白質 conformation ensemble 的 TN 表示

蛋白質結構是 N 個 residue × 3 座標的 high-order tensor，conformations 構成低糾纏流形。提案：MPS-conformer：用 MPS 表示 IDR / 構象集合，自動 enforce 物理約束（bond length、angles）。

🎯 5. CRISPR screen 的 sparse tensor 建模

(guide × gene × cell × time) sparse tensor。提案：TT-Cross 從少數採樣 entry 重建整個 screen 的 perturbation landscape，加速 design-of-experiment。

🎯 6. TN-attention：替代 transformer 的 O(L²)

transformer attention 是 (Q K^T) → O(L²) 矩陣。提案：用 MPS 表達 attention matrix（structured low-rank + locality），對長 context 從 O(L²) 降到 O(L · χ²)。

🎯 7. 量子啟發的 scRNA-seq classifier

提案：MPS-RNAseq classifier — 把單細胞 expression 編碼為 product state，MPS 收縮輸出 cell type。每個 bond 可解釋為「基因 pair interaction」，比 deep learning 更可解釋且訓練快。

🎯 8. PEPS for spatial transcriptomics

Spatial 是 2D lattice 的 cell type 分布。提案：PEPS-based spatial cell-cell interaction modeling，捕捉超越 nearest-neighbor 的 emergent tissue structure。

🎯 1. Whole-genome long-sequence modeling

HyenaDNA / EVO use SSMs for long genomes — but genomes have strong local correlations (gene → exon → codon), an area-law sequence. Proposal: MPS / TT alternatives where χ controls receptive field and bonds are interpretable.

🎯 2. PEPS model for Hi-C contact maps

2D Hi-C satisfies area law (TAD locality) — a natural PEPS candidate. Proposal: PEPS-based Hi-C reconstruction from sparse contacts; bond dimension = "TAD structural complexity".

🎯 3. Multi-patient multi-modal tensor integration

(patient × gene × organ × time × omic) 5-way tensor. Proposal: TT decomposition + missingness; each core is a modality-specific latent program — more expressive than PARAFAC.

🎯 4. Protein conformation-ensemble TN

Protein structure is an N-residue × 3-coord high-order tensor; conformations live on a low-entanglement manifold. Proposal: MPS-conformer for IDR / conformation ensembles, auto-enforcing physical constraints (bond lengths, angles).

🎯 5. Sparse-tensor CRISPR-screen modeling

(guide × gene × cell × time) sparse tensor. Proposal: TT-Cross reconstructs the perturbation landscape from few measurements, accelerating design-of-experiment.

🎯 6. TN-attention as O(L²)→O(L·χ²) substitute

Transformer attention is (Q K^T), O(L²). Proposal: express the attention matrix as an MPS (structured low-rank + locality), dropping cost to O(L · χ²) for long context.

🎯 7. Quantum-inspired scRNA-seq classifier

Proposal: MPS-RNAseq classifier — encode single-cell expression as product state; MPS contraction → cell-type label. Each bond is "gene-pair interaction" — interpretable and fast to train.

🎯 8. PEPS for spatial transcriptomics

Spatial is a 2D cell-type lattice. Proposal: PEPS-based modeling of cell-cell interactions beyond nearest neighbor, capturing emergent tissue structure.

import numpy as np

# Build random MPS (10 sites, physical dim 2, bond dim 8)

print(mps)        # Tensor network with 10 sites

# Build the transverse-field Ising Hamiltonian as an MPO
H = qtn.MPO_ham_ising(L=10, j=1.0, bx=0.5)

# DMRG to find ground state
dmrg = qtn.DMRG2(H, bond_dims=[8, 16, 32])
dmrg.solve(tol=1e-6, verbosity=1)
gs = dmrg.state                 # the ground state MPS

# Expectation value <Z_5>
Z = qtn.spin_operator('Z')
print(gs.local_expectation(Z, where=5))

# Compress an existing MPS to lower bond dim
mps_compressed = mps.compress(max_bond=4, cutoff=1e-8)

import numpy as np

# pip install tensornetwork
import tensornetwork as tn
import numpy as np

# Build a simple MPS by hand
L, d, chi = 6, 2, 4
cores = []
# left edge
cores.append(tn.Node(np.random.randn(1, d, chi)))
for _ in range(L - 2):
    cores.append(tn.Node(np.random.randn(chi, d, chi)))
cores.append(tn.Node(np.random.randn(chi, d, 1)))

# Connect bond edges left-to-right
for i in range(L - 1):
    cores[i][2] ^ cores[i + 1][0]

# Contract everything to get full 2^6 amplitude tensor
result = tn.contractors.greedy(cores, output_edge_order=[
    c[1] for c in cores])
print(result.tensor.shape)      # (2, 2, 2, 2, 2, 2)

📝 自我檢測

MPS 為什麼能把 d^L 維張量壓縮到 L · d · χ² 個參數？

Why can MPS compress a d^L tensor into L · d · χ² parameters?

A. 隨機投影A. By random projection
B. 因為總是無損B. Because it is always lossless
C. 因為「物理上有意義」的態多滿足 area law 糾纏，bond dimension χ 即可上限這個糾纏C. "Physically meaningful" states satisfy area-law entanglement; bond dimension χ caps this entanglement
D. 把 L 改成 log LD. By replacing L with log L

DMRG 與 Tensor Train (TT) 的關係是什麼？

What is the relationship between DMRG and Tensor Train (TT)?

A. 完全不同的演算法A. Completely unrelated algorithms
B. TT 與 MPS 數學上等價；DMRG 是優化 MPS 的演算法，物理與 ML 圈各自命名B. TT and MPS are mathematically equivalent; DMRG is the algorithm to optimize an MPS — different names in physics vs ML
C. TT 只能處理 2D 資料C. TT works only on 2D data
D. DMRG 不再被使用D. DMRG is obsolete

為什麼 PEPS 比 MPS 難算？

Why is PEPS harder to compute than MPS?

A. PEPS 是 2D 網格，contraction order 沒有「天然路徑」，精確 contraction 是 #P-hard，需用 boundary-MPS 近似A. PEPS is on a 2D lattice with no natural contraction order; exact contraction is #P-hard — needs boundary-MPS approximations
B. PEPS 沒有 bond dimensionB. PEPS has no bond dimension
C. PEPS 只能用量子電腦C. PEPS only works on quantum hardware
D. MPS 比 PEPS 多一個維度D. MPS has more dimensions than PEPS
