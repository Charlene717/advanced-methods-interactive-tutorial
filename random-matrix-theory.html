隨機矩陣理論 (Random Matrix Theory)

當 n×p 維度都大、樣本與基因數差不多時，傳統統計直覺崩潰：純噪音矩陣的 covariance 也會出現「看似有結構」的 eigenvalue。Marchenko-Pastur 與 Tracy-Widom 定律給我們一把尺：何處才是真正的訊號？

When n and p are both large and comparable, classical statistics breaks: pure-noise covariances appear "structured". Marchenko-Pastur and Tracy-Widom give a ruler — where does real signal begin and noise end?

核心概念

為什麼需要 RMT？

古典統計直覺的失靈：當樣本數 n 遠大於變數數 p 時（n ≫ p），sample covariance Ŝ → 真實 Σ。然而 omics 資料常見 p ≫ n（基因 20,000 ↔ 樣本數百），或 p, n 同階。此時 Ŝ 的 eigenvalue 散佈大幅扭曲——即使資料是純白噪音，前幾個 PC 也會「看起來顯著」。

Marchenko-Pastur (1967)：若 W 是 n × p iid 噪音矩陣，元素 W_ij ~ N(0, σ²)，當 n, p → ∞ 且 c = p/n 固定，則樣本協方差 (1/n) W^T W 的 eigenvalue 經驗分佈收斂到

ρ(λ) = (1/(2πσ²cλ)) √((λ₊ − λ)(λ − λ₋)) · 𝟙_{[λ₋, λ₊]}

其中 λ₊ = σ²(1 + √c)², λ₋ = σ²(1 − √c)²（若 c < 1）。所有 eigenvalue 集中在 [λ₋, λ₊]！

Tracy-Widom (1994)：最大 eigenvalue λ_max 的精細分佈——非高斯，有特定的左偏型態：

(λ_max − μ_TW) / σ_TW   →   F₁ (TW₁ 分佈)

用來檢驗「最大 eigenvalue 是否顯著超出純噪音 bulk」——這是 RMT-based PCA 顯著性檢驗的核心。

Classical intuition fails: when n ≫ p, sample covariance Ŝ → true Σ. But omics typically has p ≫ n (20,000 genes vs hundreds of samples), or p ≍ n. The eigenvalue spectrum of Ŝ is then heavily distorted — even pure white noise produces "significant-looking" leading PCs.

Marchenko-Pastur (1967): if W is an n × p iid noise matrix with W_ij ~ N(0, σ²), as n, p → ∞ with c = p/n fixed, the empirical eigenvalue distribution of (1/n) W^T W converges to

ρ(λ) = (1/(2πσ²cλ)) √((λ₊ − λ)(λ − λ₋)) · 𝟙_{[λ₋, λ₊]}

where λ₊ = σ²(1 + √c)², λ₋ = σ²(1 − √c)² (if c < 1). All eigenvalues concentrate in [λ₋, λ₊].

Tracy-Widom (1994): the fluctuation of λ_max — non-Gaussian, characteristically left-skewed:

(λ_max − μ_TW) / σ_TW   →   F₁ (TW₁ law)

Used to test whether the largest eigenvalue is significantly above the noise bulk — the core of RMT-based PCA significance testing.

λ₊ 給出「純噪音 eigenvalue 的上限」 — 超過才有可能是訊號。

λ₊ is the upper bound of pure-noise eigenvalues — anything above may be signal.

顯著訊號

Spike model: a few signal eigenvalues stick out above the bulk — identifiable.

📐

維度比 c=p/n

c 越大 → bulk 越寬 → 越難分訊號與噪音。omics p ≫ n 屬高 c。

Larger c → wider bulk → harder to separate signal from noise. Omics p ≫ n is high-c.

演算法

M-P PCA、Spike model、Stieltjes 變換

① M-P based PCA noise threshold（演算法）：

  1. 對中心化、scale 後的資料 X (n × p)，計算 (1/n) X^T X 的 eigenvalue {λ_i}。
  2. 用 robust 估計 σ̂²（例如：取低端 quantile fit M-P）。
  3. 計算 λ₊ = σ̂²(1 + √(p/n))²。
  4. 保留 λ_i > λ₊ 的 PC；其餘視為噪音。

② Spike model (Johnstone 2001)：真實 covariance Σ = σ²(I + Σ_k h_k v_k v_k^T)。BBP 相變 (Baik, Ben Arous, Péché 2005)：

  • 若 h_k > √c → 對應 sample eigenvalue 從 bulk 中分離，PCA 可偵測。
  • 若 h_k < √c → eigenvalue 與 bulk 重疊，無法用 PCA 分離。

這是 PCA 在 high-dimensional 情境的本質限制——訊號太弱時即使無限樣本仍無法分離。

③ Stieltjes transform：RMT 主力工具。對任意 measure ρ 定義

m(z) = ∫ ρ(λ) / (λ − z) dλ,   z ∈ ℂ⁺

M-P 的 m(z) 滿足代數方程 cz m² + (z + c − 1) m + 1 = 0；解出再代回 Stieltjes-Perron inversion 得 ρ(λ)。
應用：計算任意 covariance shrinkage 後的 eigenvalue 分佈（free probability）。

④ Free probability 與 covariance cleaning：Ledoit-Wolf (2004)、Bun-Bouchaud-Potters (2017) RIE estimator 用 Stieltjes / free convolution 推導出最佳「shrinkage 函數」，把 noisy eigenvalue 拉回更接近真值。

⑤ TW 顯著性檢驗：計算統計量 T = (n λ_max / σ̂² − μ_TW) / σ_TW；查 TW₁ 分位數判斷顯著性。Patterson 2006 EIGENSOFT、smartpca 為此設計。

① M-P PCA noise threshold:

  1. Centre and scale X (n × p), compute eigenvalues of (1/n) X^T X.
  2. Estimate σ̂² robustly (e.g. fit M-P to the lower-quantile tail).
  3. Compute λ₊ = σ̂²(1 + √(p/n))².
  4. Keep PCs with λ_i > λ₊; treat the rest as noise.

② Spike model (Johnstone 2001): true covariance Σ = σ²(I + Σ_k h_k v_k v_k^T). BBP phase transition (Baik, Ben Arous, Péché 2005):

  • If h_k > √c → the corresponding sample eigenvalue separates from the bulk; PCA can detect it.
  • If h_k < √c → it overlaps the bulk; undetectable by PCA.

This is an essential PCA limit in high dimensions — weak signals stay hidden no matter how much data.

③ Stieltjes transform: RMT's main tool. For any measure ρ define

m(z) = ∫ ρ(λ) / (λ − z) dλ,   z ∈ ℂ⁺

The M-P m(z) satisfies cz m² + (z + c − 1) m + 1 = 0; solve and invert via Stieltjes-Perron to recover ρ(λ). Used to derive eigenvalue distributions for general covariance shrinkage (free probability).

④ Free probability and covariance cleaning: Ledoit-Wolf (2004), Bun-Bouchaud-Potters (2017) RIE estimator derive the optimal "shrinkage function" using Stieltjes / free convolution — pulling noisy eigenvalues back towards truth.

⑤ TW significance test: compute T = (n λ_max / σ̂² − μ_TW) / σ_TW; compare to TW₁ quantiles. Patterson 2006 EIGENSOFT / smartpca implements this.

BBP 相變的直覺： 在純噪音背景中，真實訊號要強到 h > √(p/n) 才會「冒出 bulk」。omics 常見 c ≈ 1 (p≈n)，需要 h > 1；若 p = 20000, n = 200，則 c = 100、h 必須 > 10。所以高維 PCA 找弱訊號 原理上 不可能——必須先做 feature selection 或正則化。
BBP intuition: in pure noise, a true signal must be strong enough that h > √(p/n) to escape the bulk. Omics often has c ≈ 1 (p ≈ n), requiring h > 1; with p = 20,000, n = 200, c = 100 and we need h > 10. Hence weak-signal PCA is in principle impossible in high dimensions — feature selection or regularisation must come first.

🧭 什麼時候用 RMT？

情境1:
高維 PCA：p, n 相當（gene expression, GWAS, connectome）— 用 M-P / TW 決定保留幾個 PC。
情境2:
Co-expression cleaning：GRN / WGCNA 把 M-P bulk 內的相關係數視為噪音剔除。
情境3:
Covariance shrinkage：portfolio / batch correction 用 RIE / Ledoit-Wolf 改良 covariance 估計。
不適合:
低維 (p ≪ n) 經典統計即可；非 iid 噪音結構（強相關、heavy-tail）需要 generalised RMT。
Case 1:
High-dim PCA: p ≍ n (gene expression, GWAS, connectomes) — use M-P / TW to decide PC count.
Case 2:
Co-expression cleaning: in GRN / WGCNA, drop correlations whose eigenvalue lies in the M-P bulk.
Case 3:
Covariance shrinkage: portfolio / batch correction via RIE / Ledoit-Wolf.
Not for:
Low-dim (p ≪ n) — classical stats suffice; non-iid noise (strong correlation, heavy tails) needs generalised RMT.
生物應用

RMT 在基因組、GRN、連接體

Eigengene 與 M-P 門檻

Alter, Brown & Botstein (2000, PNAS) 提出 SVD 抽取 gene expression 的 "eigengenes"；Kim & Tidor (2003)、Wall (2003) 等明確用 M-P 預測噪音 bulk，只保留超出 λ₊ 的 PC。是 microarray / RNA-seq 的標準前處理之一。

Alter, Brown & Botstein (2000, PNAS) introduced SVD-based "eigengenes" of expression; Kim & Tidor (2003), Wall (2003) used M-P to bound the noise bulk, keeping only PCs with eigenvalue > λ₊. Now a standard microarray / RNA-seq pre-processing step.

WGCNA 模組偵測

WGCNA (Langfelder & Horvath 2008, BMC Bioinformatics) 用 soft thresholding 後的 adjacency matrix 偵測模組；後續變體把 M-P 噪音剔除 (e.g. Luo 2007 RMT-based module finding) 提升模組生物學可解釋性。

WGCNA (Langfelder & Horvath 2008, BMC Bioinformatics) detects modules via soft-thresholded adjacency. RMT-based variants (Luo 2007) remove M-P noise before module discovery, improving biological interpretability.

GRN denoising

基因–基因 correlation 矩陣經常含大量「純隨機」相關。用 M-P 預期 bulk 內 eigenvalue 對應的 modes 視為噪音、扣除——保留 spike modes 為真實 co-expression module。應用於 TCGA、GTEx 表現資料 GRN 重建。

Gene–gene correlation matrices are full of spurious links. Drop modes whose eigenvalues lie in the M-P bulk; keep the spike modes as real co-expression modules. Applied to TCGA, GTEx GRN reconstruction.

GWAS heritability

Patterson, Price & Reich (2006, PLOS Genet) 在 GWAS 用 TW 檢驗 PC 顯著性，避免 population stratification 假陽性；之後 LD score regression 與 RMT-based heritability 估計（如 RHE / SumHer）廣泛採用。

Patterson, Price & Reich (2006, PLOS Genet) used the TW test to assess PC significance in GWAS, controlling stratification false positives. Later LD-score regression and RMT-based heritability (RHE / SumHer) became widespread.

神經連接體噪音過濾

fMRI / DTI / EEG 推導的 functional / structural connectome 含大量短時程隨機相關。MacMahon & Garlaschelli (2015)、Almog 2015 等將 RMT 用於 connectome cleaning，使社群偵測更穩健。

Functional / structural connectomes from fMRI / DTI / EEG carry massive transient correlations. MacMahon & Garlaschelli (2015), Almog 2015 used RMT to clean connectomes, stabilising community detection.

股市 vs 基因組同方法

Laloux et al. (1999, PRL)、Plerou et al. (2002, PRE) 首先把 RMT 用於股市 correlation cleaning；同樣的 Marchenko-Pastur 框架在生物 correlation 矩陣完全成立，是少數可直接「跨領域搬遷」的數理工具。

Laloux et al. (1999, PRL) and Plerou et al. (2002, PRE) first applied RMT to stock-return correlation cleaning. The same Marchenko-Pastur framework applies wholesale to biological correlations — a rare direct cross-domain transfer.

scRNA-seq doublet / 雙倍細胞偵測

scRNA-seq 中 doublet 細胞會引入 spurious high-rank structure。RMT-based denoising（如 ALRA, Linderman 2018; randomly-projected SVD）可去除噪音 PC 同時保留真實 biological variance；下游 cluster 更乾淨。

Doublets in scRNA-seq introduce spurious high-rank structure. RMT-based denoising (ALRA, Linderman 2018; random-projection SVD) removes noise PCs while preserving real biological variance, leading to cleaner clusters.

深度學習 weight matrices 分析

Martin & Mahoney (2021, JMLR) WeightWatcher 用 heavy-tailed RMT 分析 NN weight matrix——好訓練的網路出現 power-law 偏離 M-P 的「heavy tail」，可作為訓練品質指標。已應用於 protein LM 與 cell LM 評估。

Martin & Mahoney (2021, JMLR) WeightWatcher uses heavy-tailed RMT to analyse NN weight matrices — well-trained networks show power-law deviations from M-P. Now used to audit protein and cell LMs.

深入解析

進階主題

① 相關噪音 (correlated noise) 下的 M-P

原始 M-P 假設 iid 噪音；現實 (batch effects、技術 covariates) 不滿足。Generalised M-P / spiked covariance model 允許非單位 noise covariance Ψ，spectrum 變為 Ψ 與 M-P 的 free convolution。批次效應強時 bulk 會「拉長」。

② Heavy-tailed RMT

單細胞 count、scRNA-seq 表現資料常呈 Poisson-like 或 NegBin，非高斯。Burda & Jurkiewicz (2004) 研究 Lévy-stable 元素的 RMT；scRNA-seq 文獻 (Tracy-Widom adjustment for low counts) 已嘗試對應修正。

③ Free probability 簡介

Voiculescu (1980s) 提出的 free independence 概念：兩個「自由獨立」隨機矩陣的 eigenvalue 分佈以 R-transform / S-transform 卷積。對 covariance cleaning（先 PCA 減維、再合併 batch）給出精確分佈計算工具。

④ 信號回收：optimal RIE shrinkage

Bun-Bouchaud-Potters (2017, Phys. Rep.) 推導 rotationally-invariant estimator：

ξ_i^RIE = λ_i / | 1 − c + cz_i m(z_i) |²

把每個 sample eigenvalue 拉回最接近真實的位置——比 Ledoit-Wolf linear shrinkage 更精細，已在生物 / 金融 covariance 估計使用。

⑤ Tracy-Widom 與最大 eigenvalue 假設檢驗

TW₁ (real Wishart) 與 TW₂ (complex) 分佈在資料分析中是標準工具。Patterson 2006 EIGENSOFT 直接內建 TW 顯著性測試；scLLM 的「latent dim 該多少？」也是 TW 問題。

⑥ BBP 相變的實務意涵

若想用 PCA 偵測強度 h 的訊號，至少需要 h > √(p/n)。實務：

  • p = 1,000，n = 100 → c = 10，需要 h > 3.16
  • p = 20,000，n = 200 → c = 100，需要 h > 10

所以生物 PCA 在不做 feature selection 時，弱訊號 幾乎必然遺漏。先用 HVG / variance filtering 把 p 降到 n 量級才合理。

⑦ 與深度學習的接口

NN 訓練後 weight matrix 的 spectrum 可診斷 over-fitting / under-trained。Martin-Mahoney "5+1 phases" 把訓練分為 random → power-law-tailed → bulk-and-spikes 等階段；對 ESM、scGPT 等模型可作為健康度檢測。

① M-P under correlated noise

Vanilla M-P assumes iid noise; real-world batch effects / technical covariates violate this. Generalised M-P / spiked covariance allows non-identity noise covariance Ψ; the spectrum becomes the free convolution of Ψ and M-P. Strong batch effects "stretch" the bulk.

② Heavy-tailed RMT

Single-cell counts are Poisson / negative-binomial, not Gaussian. Burda & Jurkiewicz (2004) studied RMT with Lévy-stable entries; scRNA-seq literature applies Tracy-Widom adjustments for low counts.

③ Free probability primer

Voiculescu's (1980s) free independence: eigenvalue distributions of "freely independent" matrices convolve via the R-transform / S-transform. Gives exact spectral computations for covariance cleaning (PCA + batch merging).

④ Signal recovery: optimal RIE shrinkage

Bun-Bouchaud-Potters (2017, Phys. Rep.) derived the rotationally-invariant estimator:

ξ_i^RIE = λ_i / | 1 − c + cz_i m(z_i) |²

It pulls each sample eigenvalue back to its closest-to-truth value — finer than Ledoit-Wolf linear shrinkage, now used in biological / financial covariance estimation.

⑤ Tracy-Widom tests for largest eigenvalue

TW₁ (real Wishart) and TW₂ (complex) are standard tools. EIGENSOFT (Patterson 2006) ships built-in TW testing. The "what's the right latent dim?" question for scLLMs is a TW problem.

⑥ Practical implication of BBP

To detect a signal of strength h via PCA, you need h > √(p/n). Concretely:

  • p = 1,000, n = 100 → c = 10, need h > 3.16
  • p = 20,000, n = 200 → c = 100, need h > 10

Without prior feature selection, weak biological signals are almost certainly missed. HVG / variance filtering down to p ≍ n is essential.

⑦ Interface with deep learning

Spectra of trained weight matrices diagnose over-fitting / under-training. Martin-Mahoney's "5+1 phases" classify training stages (random → power-law tailed → bulk-and-spikes); applicable as health checks for ESM, scGPT.

程式碼

實作範例

import numpy as np

def mp_threshold(n, p, sigma2=1.0):
    """Returns upper edge λ₊ of Marchenko-Pastur bulk."""
    c = p / n
    return sigma2 * (1 + np.sqrt(c))**2

def mp_density(lam, c, sigma2=1.0):
    lam_p = sigma2*(1+np.sqrt(c))**2
    lam_m = sigma2*(1-np.sqrt(c))**2
    val = np.zeros_like(lam)
    mask = (lam > lam_m) & (lam < lam_p)
    val[mask] = np.sqrt((lam_p - lam[mask])*(lam[mask] - lam_m)) \
                / (2*np.pi*sigma2*c*lam[mask])
    return val

# Demo: simulate vs theory
n, p = 400, 200
W = np.random.randn(n, p)
S = (W.T @ W) / n
ev = np.linalg.eigvalsh(S)
lam_plus = mp_threshold(n, p)
print("M-P upper edge:", lam_plus)
print("# eigenvalues above noise bulk:", (ev > lam_plus).sum())
# Ledoit-Wolf shrinkage (linear) and a sketch of RIE
from sklearn.covariance import LedoitWolf
import numpy as np

lw = LedoitWolf().fit(X)              # X: n × p
Sigma_lw = lw.covariance_
print("shrinkage α =", lw.shrinkage_)

# Sketch of Bun-Bouchaud-Potters RIE (rotationally invariant)
def rie(X, eps=1e-3):
    n, p = X.shape; c = p/n
    S = (X.T @ X) / n
    lam, U = np.linalg.eigh(S)
    # Stieltjes via Cauchy kernel smoothing
    xi = np.empty_like(lam)
    for i, li in enumerate(lam):
        z = li + 1j*eps
        m = np.mean(1/(lam - z))
        denom = abs(1 - c + c*z*m)**2
        xi[i] = li / denom
    return U @ np.diag(xi) @ U.T
# install.packages("RMTstat"); install.packages("RSpectra")
library(RMTstat)

# Tracy-Widom CDF / quantiles (for PCA significance testing)
ptw(2.0, beta=1)         # P(TW1 ≤ 2.0)
qtw(0.95, beta=1)        # 95% TW1 quantile

# Marchenko-Pastur density
dmp(x=1.5, svr=2)         # svr = n/p (sample-to-var ratio)

# Significance of largest PC of a centred matrix X
n <- nrow(X); p <- ncol(X); c <- p/n
S <- crossprod(X)/n
ev <- eigen(S, only.values=TRUE)$values
lam_plus <- (1+sqrt(c))^2          # assuming sigma2 = 1
sig_pcs <- which(ev > lam_plus)

Key libraries: Python — numpy, pyrmt, scikit-learn-extra (LedoitWolf etc.). R — RMTstat, RSpectra, WGCNA. Genomics: EIGENSOFT / smartpca (Patterson 2006).
