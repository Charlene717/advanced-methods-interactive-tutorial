狀態空間模型 / Mamba

線性複雜度 O(n) 取代 Transformer 的 O(n²) — S4、Mamba 等選擇性 SSM 讓「全基因組 1 Mbp 上下文建模」首次成真，正在改寫長序列基因組學的格局。

Linear-time O(n) alternatives to Transformer's O(n²) — S4 and Mamba's selective SSMs make whole-genome 1 Mbp context modeling feasible for the first time, reshaping long-sequence genomics.

核心概念

從連續系統到序列建模

狀態空間模型（State Space Model, SSM）來自控制論。連續形式：

h'(t) = A · h(t) + B · x(t),   y(t) = C · h(t) + D · x(t)

h 是「隱藏狀態」（記憶），A 控制狀態如何隨時間衰減/振盪，B 把輸入注入狀態，C 從狀態讀出輸出。離散化（用 zero-order hold 或 bilinear）後變成：

hk = Ā · hk−1 + B̄ · xk,   yk = C · hk

這個遞迴式長得像 RNN——但因為 (Ā, B̄, C) 是線性常數，可以展開成「卷積核」：y = K * x，其中 K = (C B̄, C Ā B̄, C Ā² B̄, ...)。卷積可平行計算，配合 FFT 可達 O(n log n)；Mamba 更進一步用 hardware-aware scan 做到 O(n)。

關鍵突破：Mamba (Gu & Dao 2024, COLM) 讓 (A, B, C) 依輸入動態變化——「selective SSM」——使模型能像 attention 一樣選擇性記住或忽略 token，但仍維持線性複雜度。

State Space Models (SSMs) come from control theory. Continuous form:

h'(t) = A · h(t) + B · x(t),   y(t) = C · h(t) + D · x(t)

h is the latent state (memory); A governs state decay/oscillation; B injects input; C reads out. After discretization (zero-order hold or bilinear):

hk = Ā · hk−1 + B̄ · xk,   yk = C · hk

This looks like an RNN — but with constant (Ā, B̄, C), it unrolls into a convolution: y = K * x with K = (C B̄, C Ā B̄, C Ā² B̄, ...). Convolution parallelizes, FFT gives O(n log n); Mamba's hardware-aware scan achieves O(n).

The key breakthrough: Mamba (Gu & Dao 2024, COLM) makes (A, B, C) input-dependent — "selective SSM" — letting the model selectively remember or ignore tokens like attention, but still in linear time.

演算法

從 HiPPO 到 Mamba：演進路線

① HiPPO 初始化（Gu 2020, NeurIPS）

SSM 的 A 矩陣怎麼選？HiPPO 理論證明：把 A 初始化為「最佳記憶」狀態（使 h(t) 為 x 的 Legendre 多項式投影）——這給了 SSM 數學基礎，能在連續時間下「無損壓縮」歷史輸入。

② S4：Structured State Space (Gu 2022, ICLR)

HiPPO 的 A 是 normal + low-rank (DPLR) 結構，可用 Cauchy kernel 在 O(n log n) 計算卷積。S4 在 Long Range Arena (LRA) benchmark 上首次擊敗 Transformer，但仍需大量超參數調校。

③ S5、H3、Hyena：架構簡化

S5 (Smith 2023, ICLR) 簡化為對角 A；H3 (Fu 2023, ICLR) 引入 gating；Hyena (Poli 2023, ICML) 把 SSM 拓展為「隱式卷積」+ multiplicative gating，HyenaDNA 在基因組上展示了 1 Mbp 上下文。

④ Mamba：Selective Scan (Gu & Dao 2024)

核心招數：讓 (Δ, B, C) 是輸入 x 的函數——這破壞了 LTI 結構，無法再用卷積。怎麼辦？用 hardware-aware parallel scan：

  1. 把序列分成 block（每個 block 約 1024 token）
  2. 每個 block 用 Blelloch scan 在 GPU SRAM 內並行計算
  3. blocks 之間遞迴串接
  4. 整體在 A100 上達到 5× Transformer 推論速度

Mamba-2 (Dao & Gu 2024, ICML) 統一 SSM 與 attention 為「Structured State Space Duality (SSD)」框架——SSM 其實是 attention 的特殊矩陣形式。

① HiPPO Initialization (Gu 2020, NeurIPS)

How to choose A? HiPPO theory: initialize A so that h(t) is the optimal Legendre-polynomial projection of x's history. Gives SSMs a mathematical basis for lossless continuous-time memory compression.

② S4: Structured State Space (Gu 2022, ICLR)

HiPPO's A is normal + low-rank (DPLR), computable via Cauchy kernel in O(n log n). S4 first beat Transformers on Long Range Arena (LRA), though required heavy hyperparameter tuning.

③ S5, H3, Hyena: simplifications

S5 (Smith 2023, ICLR) uses diagonal A; H3 (Fu 2023, ICLR) adds gating; Hyena (Poli 2023, ICML) extends SSM to "implicit long convolution" + multiplicative gating; HyenaDNA demonstrated 1 Mbp genomic context.

④ Mamba: Selective Scan (Gu & Dao 2024)

Core trick: make (Δ, B, C) functions of input x. This breaks LTI structure — convolution no longer works. Solution: hardware-aware parallel scan:

  1. Chunk sequence into blocks (~1024 tokens each)
  2. Each block: Blelloch scan inside GPU SRAM
  3. Concatenate blocks via recurrence
  4. 5× Transformer inference speed on A100

Mamba-2 (Dao & Gu 2024, ICML) unifies SSM and attention as "Structured State Space Duality (SSD)" — SSMs are special matrix forms of attention.

關鍵直覺：Transformer 的 attention 矩陣是 n×n（O(n²)）；SSM 的 attention 矩陣是「半可分離 (semiseparable)」，可用 O(n) 表達。代價是「結構化」——某些 attention 模式 SSM 學不到（例如需要精確 token 對應的 in-context learning），但對「天然有時間/位置遞迴結構」的訊號（DNA、protein、time series）非常合身。
Key intuition: Transformer attention matrices are n×n dense (O(n²)); SSM attention is "semiseparable", representable in O(n). Trade-off: some patterns SSMs can't learn (e.g. precise-token-matching in-context learning), but for signals with natural temporal/positional recurrence (DNA, protein, time series), SSMs fit beautifully.

生物應用

SSM/Mamba 在生命科學的應用

HyenaDNA

第一個達到 1 Mbp 上下文的 DNA 基礎模型（vs DNABERT 512 bp）。在 Nucleotide Transformer benchmark 上以 1500× 小的模型獲得相當或更好表現。可建模 enhancer–promoter 跨百萬 bp 的長距互動。

First DNA foundation model reaching 1 Mbp context (vs DNABERT 512 bp). On Nucleotide Transformer benchmark, matches or beats with 1500× smaller models. Models enhancer–promoter long-range interactions across megabases.

Caduceus

雙向 Mamba 配上 reverse-complement equivariance（DNA 互補鏈不變性）。在 variant effect prediction、SNP 影響預測上達 SOTA。為「DNA-specific」inductive bias 的範例。

Bidirectional Mamba with reverse-complement equivariance (DNA strand symmetry). SOTA on variant effect prediction and SNP impact prediction — example of DNA-specific inductive bias.

Evo

7B 參數 StripedHyena 在 2.7M 微生物基因組上訓練，可生成 CRISPR-Cas 系統與整個小型基因組（131k bp）。涵蓋 DNA→RNA→protein 統一表徵。

7B-parameter StripedHyena trained on 2.7M microbial genomes — generates entire CRISPR-Cas systems and small genomes (131k bp). Unifies DNA→RNA→protein representation.

ProtMamba

蛋白質序列 Mamba，可處理整個 MSA（多重序列比對）作為單一輸入，把同源蛋白資訊端到端學習。在 contact prediction、fitness landscape 任務上勝過 ESM-2。

Protein-sequence Mamba; ingests entire MSAs as one input, learning homolog information end-to-end. Beats ESM-2 on contact prediction and fitness landscape tasks.

MambaDNA / scMamba

把 Mamba 應用於單細胞表現序列（依基因表現排序的「gene rank sequence」）——學習細胞層級的全基因組表現模式，作為 scGPT/Geneformer 的替代。

Applies Mamba to single-cell expression sequences (gene-rank ordered), learning cell-level genome-wide patterns — an alternative to scGPT/Geneformer.

細胞條碼序列

CRISPR-Cas9 lineage barcoding（如 ScarTrace、GESTALT）產生長 indel 序列。Mamba 適合學習「條碼演化軌跡」的長距依賴，重建細胞家族樹。

CRISPR-Cas9 lineage barcodes (ScarTrace, GESTALT) yield long indel sequences. Mamba captures long-range dependencies in barcode evolution to reconstruct cell lineages.

深入解析

SSM 的限制與最新動向

① SSM 不擅長的事

  • 精確 copy / in-context learning：Jelassi et al. (2024) 證明 Mamba 在「需要精確記住前面 token」的任務（associative recall）上 provably 比 attention 弱。實務上 Mamba 模型在 few-shot prompt 跟 in-context learning 表現不如同規模 Transformer。
  • 固定上下文記憶：SSM 的 hk 維度固定（例如 256），不論序列多長都壓進同一個向量——超長依賴會「壓縮損失」。Transformer 的 KV cache 隨序列線性成長，無此限制（但耗記憶體）。

② Hybrid 架構是未來

Jamba (AI21 2024)、Zamba (Zyphra 2024)、Samba 等模型結合 Mamba 層與 attention 層——Mamba 處理長距記憶，attention 處理精確匹配。MoE-Mamba 進一步加上 Mixture of Experts。生物上 Caduceus + sparse attention 可能成為 DNA 的標準架構。

③ 雙向性對生物序列至關重要

DNA 沒有「方向」——讀正股或反股都合理。原 Mamba 是因果（causal）的，只能由左向右。Caduceus、BiMamba 等變體加入雙向掃描：把序列正向過一次 + 反向過一次，將兩個 hidden state 合併。對 DNA、protein、spatial sequence 都重要。

④ SSM 的數學身分

SSM = linear RNN = linear attention with特定 kernel = Volterra series 一階近似。Mamba-2 的 SSD 框架揭示：SSM 等價於「semiseparable attention」——attention 矩陣有某種低秩 + 三角結構。這給了 SSM 與 Transformer 互通的數學橋樑，未來可能融合為統一架構。

⑤ Hardware-aware 設計的重要性

Mamba 的成功有一半在演算法，一半在工程：parallel scan 必須在 GPU SRAM 內完成，避免 HBM 讀寫。這跟 FlashAttention 的精神類似。沒這個工程實作，Mamba 的線性複雜度只是紙上談兵——實際吞吐量取決於 memory bandwidth。

⑥ 評估：到底比 Transformer 好嗎？

純 Mamba 在 ≤ 3B 參數規模上跟 Transformer 旗鼓相當，某些任務（長序列、time series）更好；但在 ≥ 7B 或需要 in-context learning 的任務上，純 Mamba 通常輸給同等規模 Transformer。結論：不是取代，是新工具——特別適合長序列領域如基因組學、長時間 EHR、protein structure 預測。

① What SSMs Are Bad At

  • Exact copy / in-context learning: Jelassi et al. (2024) proved Mamba is provably weaker than attention on associative recall (precisely remember earlier tokens). Empirically, same-scale Mamba underperforms Transformer on few-shot prompting and ICL.
  • Fixed-size memory: SSM's hk has fixed dimension (e.g. 256) regardless of sequence length — ultra-long dependencies suffer "compression loss." Transformer's KV cache grows linearly (memory-expensive but lossless).

② Hybrid Architectures Are the Future

Jamba (AI21 2024), Zamba (Zyphra 2024), Samba combine Mamba + attention layers — Mamba for long-range memory, attention for precise matching. MoE-Mamba adds Mixture of Experts. In biology, Caduceus + sparse attention may become standard for DNA.

③ Bidirectionality Matters for Biology

DNA has no inherent direction — forward and reverse strands both make sense. Vanilla Mamba is causal (left-to-right). Caduceus, BiMamba scan both ways and merge hidden states. Crucial for DNA, protein, and spatial sequences.

④ Mathematical Identity of SSMs

SSM = linear RNN = linear attention with specific kernel = first-order Volterra series. Mamba-2's SSD framework reveals: SSM equals "semiseparable attention" (low-rank + triangular structure). A mathematical bridge to Transformer — future architectures may unify both.

⑤ The Importance of Hardware-Aware Design

Half of Mamba's success is engineering: parallel scan must stay inside GPU SRAM, avoiding HBM. Same philosophy as FlashAttention. Without this, linear complexity is theoretical — real throughput is memory-bandwidth-bound.

⑥ Evaluation: Is It Actually Better Than Transformer?

Pure Mamba matches Transformer at ≤ 3B scale and excels on long sequences / time series. At ≥ 7B or for in-context learning, pure Mamba underperforms. Bottom line: not a replacement but a new tool — perfect for long-sequence domains like genomics, long EHR, protein structure prediction.

潛在改良應用

SSM 還能改良哪些生物資訊任務？

🎯 1. 全基因組 Epigenome 預測

從 1 Mbp DNA 序列預測 ChIP-seq、ATAC-seq、Hi-C 訊號。Enformer (Avsec 2021) 用 Transformer 只能到 200 kbp，HyenaDNA-based 模型可拓展到全染色體。提案：單一模型輸入整條染色體輸出多軌 epigenomic profile，捕捉 TAD-scale 結構。

🎯 2. 長 Perturbation Lineage 軌跡

Perturb-seq + lineage barcoding 產生「擾動標籤 + 條碼歷史 + 表現序列」的長序列。Mamba 可端到端學習「祖先擾動如何影響後代表現」，跨數十代細胞分裂。

🎯 3. Spatial Gene Panel 序列建模

把 MERFISH、Visium 的 gene panel 視為「依空間軌跡排序的序列」，Mamba 可學「沿組織梯度的長距共表達模式」。優於用 GNN 處理離散 graph。

🎯 4. 長 Single-cell 軌跡建模

scRNA-seq + 時間採樣（如 LARRY、Cospar）構成 cell × time 矩陣。Mamba 把每個細胞的 expression vector 視為 token，沿時間建模狀態演化——比 RNA velocity 的瞬時推估更穩健。

🎯 5. ChIP-seq + ATAC-seq Long Context 整合

多軌 epigenomic data 跨 Mbp 的協同調控。Mamba 可同時吃 DNA 序列 + ChIP-seq peaks + ATAC peaks + Hi-C contacts，學「染色質區域 × 區域」的因果鏈。

🎯 6. 長 Read 基因組組裝

Nanopore、PacBio 長 read（10–100 kbp）的 basecalling 與 assembly。Mamba 取代 RNN-based basecaller 達到更高準確度 + 更快推論。

🎯 7. 全長 RNA isoform 預測

從 DNA 預測所有 isoform 結構（splicing patterns）。需 100 kbp 上下文涵蓋整個 gene locus，Mamba 完美適配。

🎯 8. 長時間病理史 (EHR) 模型

病人數十年的就診紀錄、實驗室數值、藥物史——Mamba 處理長序列同時保留 long-range pattern，比 Transformer 更實用。

🎯 9. 多重序列比對 (MSA) 端到端學習

ProtMamba 已示範：把整個 MSA 當作單一輸入。可拓展至 metagenomics 大型 MSA，找跨菌種保守的調控元件。

🎯 10. Streaming 即時 scRNA-seq 分析

SSM 推論時為 O(1) 記憶體 RNN——適合做「資料一筆筆進來」的線上學習場景，例如 10x Chromium 即時細胞分群、live-cell imaging 時序分析。

🎯 1. Whole-Genome Epigenome Prediction

Predict ChIP-seq, ATAC-seq, Hi-C from 1 Mbp DNA. Enformer (Avsec 2021) caps at 200 kbp with Transformer; HyenaDNA-based models extend to entire chromosomes. Proposal: one model inputs full chromosome, outputs multi-track epigenome — captures TAD-scale structure.

🎯 2. Long Perturbation Lineage Trajectories

Perturb-seq + lineage barcoding gives long sequences of "perturbation labels + barcode history + expression." Mamba learns end-to-end how ancestor perturbations affect descendant expression across dozens of cell generations.

🎯 3. Spatial Gene Panel Sequence Modeling

Treat MERFISH/Visium gene panels as "spatially-ordered sequences"; Mamba learns long-range co-expression along tissue gradients — better than GNNs on discrete graphs.

🎯 4. Long Single-cell Trajectory Modeling

scRNA-seq with time sampling (LARRY, Cospar) forms cell × time matrices. Mamba tokenizes per-cell expression vectors and models state evolution — more robust than RNA velocity's instantaneous estimates.

🎯 5. ChIP-seq + ATAC-seq Long-Context Integration

Multi-track epigenomes with Mbp-scale co-regulation. Mamba simultaneously ingests DNA + ChIP peaks + ATAC + Hi-C, learning region-region causal chains.

🎯 6. Long-Read Genome Assembly

Nanopore / PacBio long reads (10–100 kbp) for basecalling and assembly. Mamba replaces RNN-based basecallers for higher accuracy and faster inference.

🎯 7. Full-Length RNA Isoform Prediction

Predict all isoforms (splicing patterns) from DNA. Needs 100 kbp context to span whole gene loci — Mamba is a perfect fit.

🎯 8. Long Patient History (EHR) Modeling

Decades of patient visits, lab values, medications — Mamba handles long sequences while preserving long-range patterns, more practical than Transformer.

🎯 9. End-to-End MSA Learning

ProtMamba showed: ingest entire MSA as one input. Extendable to metagenomics' giant MSAs to find cross-species conserved regulatory elements.

🎯 10. Streaming Real-Time scRNA-seq

SSM inference is O(1)-memory RNN — ideal for online streaming (live 10x Chromium cell clustering, live-cell imaging temporal analysis).

# pip install mamba-ssm causal-conv1d
import torch
from mamba_ssm import Mamba

# Single Mamba block
model = Mamba(
    d_model=256,        # hidden dim
    d_state=16,          # SSM state dim
    d_conv=4,            # local conv kernel size
    expand=2,            # expansion factor
).cuda()

x = torch.randn(4, 100000, 256).cuda()   # 100k length!
y = model(x)                                # O(n), linear in length
print(y.shape)                              # [4, 100000, 256]
# pip install transformers
from transformers import AutoModel, AutoTokenizer

# HyenaDNA-medium-450k (450k bp context)
model = AutoModel.from_pretrained(
    "LongSafari/hyenadna-medium-450k-seqlen-hf",
    trust_remote_code=True
)
tok = AutoTokenizer.from_pretrained(
    "LongSafari/hyenadna-medium-450k-seqlen-hf",
    trust_remote_code=True
)

seq = "ACGT" * 100000                      # 400k bp
inputs = tok(seq, return_tensors="pt")
out = model(**inputs)
embeddings = out.last_hidden_state            # [1, L, d]
# Minimal SSM (S4-like, diagonal A) in PyTorch
import torch
import torch.nn as nn

class SimpleSSM(nn.Module):
    def __init__(self, d_model, d_state=16):
        super().__init__()
        # Diagonal A initialized via HiPPO
        self.A_log = nn.Parameter(torch.log(torch.arange(1, d_state+1).float()))
        self.B = nn.Parameter(torch.randn(d_model, d_state))
        self.C = nn.Parameter(torch.randn(d_model, d_state))
        self.dt = nn.Parameter(torch.ones(d_model) * 0.01)

    def forward(self, x):                  # x: (B, L, d)
        A = -torch.exp(self.A_log)         # stable, negative
        Ab = torch.exp(self.dt.unsqueeze(-1) * A)   # discretize
        h = torch.zeros(x.shape[0], self.B.shape[1], device=x.device)
        ys = []
        for t in range(x.shape[1]):
            h = Ab.mean(0) * h + (self.B.T @ x[:,t,:].T).T
            y = (self.C @ h.unsqueeze(-1)).squeeze(-1)
            ys.append(y)
        return torch.stack(ys, dim=1)
