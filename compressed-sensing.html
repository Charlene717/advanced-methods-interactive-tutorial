壓縮感知 (Compressed Sensing)

如果信號本質「稀疏」，那麼只用遠少於 Nyquist 取樣定理所要求的量測，就能完整恢復——這就是 Candès, Romberg & Tao (2006) 與 Donoho (2006) 開創的壓縮感知。MERFISH 用 100 個探針偵測 1000+ 基因、Cell Painting 壓縮成像、稀疏 PRS——壓縮感知早已是現代生物資訊的隱形支柱。

If a signal is intrinsically sparse, you can fully recover it from far fewer measurements than Nyquist demands — this is compressed sensing, pioneered by Candès, Romberg & Tao (2006) and Donoho (2006). MERFISH detects 1000+ genes with just 100 probes, compressed Cell Painting, sparse PRS — compressed sensing is a silent backbone of modern bioinformatics.

核心概念

稀疏 + 隨機投影 = 可恢復

傳統取樣的痛點：Shannon–Nyquist 定理說「採樣率必須 ≥ 2×最高頻率」。但對生醫信號——基因表現、影像、磁振造影——大部分維度其實是 0 或近 0。為什麼還要花這麼多 measurement？

壓縮感知三要素：

  1. 稀疏性 (sparsity)：信號 x ∈ ℝ^N 可以寫成 x = Ψ s，其中 s 只有 K ≪ N 個非零項（K-sparse）。Ψ 是某個 dictionary——可以是 wavelet basis、DCT、學到的字典等。
  2. 線性量測 (incoherent sensing)：量測 y = Φ x ∈ ℝ^M，其中 M ≪ N。Φ 通常是隨機矩陣（Gaussian、Bernoulli、subsampled Fourier），重點是要與 Ψ incoherent。
  3. 稀疏恢復 (sparse recovery)：解 min ‖s‖_1 s.t. y = Φ Ψ s（L1 凸鬆弛，等價於 L0 計數最小化）。

RIP (Restricted Isometry Property, Candès & Tao 2005)：若 Φ 對所有 2K-sparse 向量 v 都滿足 (1 − δ) ‖v‖² ≤ ‖Φv‖² ≤ (1 + δ) ‖v‖²（δ < √2 − 1），則 L1 最小化保證唯一恢復原 s。關鍵結論：隨機 Gaussian 矩陣只需 M ≈ K log(N/K) 列就以高機率滿足 RIP——遠低於 Nyquist。

Pain point of classical sampling: Shannon–Nyquist requires sampling at twice the highest frequency. But biomedical signals — gene expression, imaging, MRI — are mostly zero or near-zero. Why pay for so many measurements?

Three ingredients of compressed sensing:

  1. Sparsity: signal x ∈ ℝ^N can be written as x = Ψ s where s has only K ≪ N non-zero entries (K-sparse). Ψ is a dictionary — wavelet basis, DCT, learned dictionary, etc.
  2. Incoherent sensing: measurements y = Φ x ∈ ℝ^M with M ≪ N. Φ is typically a random matrix (Gaussian, Bernoulli, subsampled Fourier), incoherent with Ψ.
  3. Sparse recovery: solve min ‖s‖_1 s.t. y = Φ Ψ s (convex L1 relaxation, equivalent to L0 counting).

RIP (Restricted Isometry Property, Candès & Tao 2005): if Φ satisfies (1 − δ) ‖v‖² ≤ ‖Φv‖² ≤ (1 + δ) ‖v‖² for all 2K-sparse v (δ < √2 − 1), then L1 minimization recovers s uniquely. Key result: random Gaussian matrices satisfy RIP with high probability using only M ≈ K log(N/K) rows — far below Nyquist.

📉

少即是多

M ≈ K log(N/K) 量測就能恢復 K-sparse 信號，K ≪ N。

M ≈ K log(N/K) measurements suffice for K-sparse signals, K ≪ N.

🎲

隨機性是優勢

隨機矩陣以高機率滿足 RIP——比設計確定性矩陣更實用。

Random matrices satisfy RIP w.h.p. — easier than designing deterministic ones.

🧬

L1 = 凸 + 稀疏

L1 是 L0 的最緊凸鬆弛，可用 LASSO 高效求解。

L1 is the tightest convex relaxation of L0 — efficiently solved by LASSO.

演算法

從 LASSO 到 AMP 與 LISTA

① Basis Pursuit / LASSO：標準凸最佳化形式

ŝ = argmin_s ½ ‖y − A s‖² + λ ‖s‖_1 (A = Φ Ψ)

可用 coordinate descent (scikit-learn Lasso)、FISTA、ADMM 求解。λ 控制稀疏程度。當 noise → 0 時退化為 Basis Pursuit (BP)：min ‖s‖_1 s.t. As = y。

② OMP (Orthogonal Matching Pursuit, Pati 1993; Tropp & Gilbert 2007)：貪婪法。每次選擇與當前 residual 最相關的 atom，加入 active set，重解 least squares 更新 residual。O(KMN)，比 L1 求解器快很多。當稀疏度極高（K 很小）時，OMP 已能達到接近最佳的恢復。

③ AMP (Approximate Message Passing, Donoho, Maleki & Montanari 2009)：把 message passing 從 dense graph 簡化為迭代閾值法，並用 Onsager correction 校正自相關：

s^{t+1} = η(A^T r^t + s^t; τ_t), r^{t+1} = y − A s^{t+1} + (1/M) ‖s^{t+1}‖_0 r^t

η 是 soft-thresholding。AMP 在 i.i.d. Gaussian Φ 下，state evolution 可精確預測每步 MSE——是極少數有 sharp phase transition 理論的演算法。

④ Deep CS / LISTA (Gregor & LeCun 2010)：把 ISTA / AMP 的迭代「展開」成神經網路：每層權重學出來，比固定 iteration 快數十倍，且能學到非高斯訊號的 prior。後續 ADMM-Net、LDAMP、Deep K-SVD 等都是這個概念的延伸。

① Basis Pursuit / LASSO: standard convex optimization

ŝ = argmin_s ½ ‖y − A s‖² + λ ‖s‖_1 (A = Φ Ψ)

Solved by coordinate descent (scikit-learn Lasso), FISTA, or ADMM. λ controls sparsity. As noise → 0, reduces to Basis Pursuit (BP): min ‖s‖_1 s.t. As = y.

② OMP (Orthogonal Matching Pursuit, Pati 1993; Tropp & Gilbert 2007): greedy. Pick the atom most correlated with the current residual, add to the active set, re-solve least squares, update residual. O(KMN) — much faster than L1 solvers. For very sparse signals (small K), OMP is near-optimal.

③ AMP (Approximate Message Passing, Donoho, Maleki & Montanari 2009): simplifies dense-graph message passing into iterative thresholding plus an Onsager correction:

s^{t+1} = η(A^T r^t + s^t; τ_t), r^{t+1} = y − A s^{t+1} + (1/M) ‖s^{t+1}‖_0 r^t

η is soft-thresholding. Under i.i.d. Gaussian Φ, state evolution predicts MSE per iteration exactly — one of very few algorithms with a sharp phase-transition theory.

④ Deep CS / LISTA (Gregor & LeCun 2010): unroll ISTA / AMP iterations into a neural network with learnable weights — orders of magnitude faster than fixed iterations and adapts to non-Gaussian priors. Followups: ADMM-Net, LDAMP, Deep K-SVD.

Phase transition：Donoho & Tanner (2009) 證明在大維度 (N → ∞)，恢復成功與否取決於兩個比率：δ = M/N（量測比率）與 ρ = K/M（相對稀疏度）。在 (δ, ρ) 平面有一條 sharp 曲線——上方失敗、下方完美恢復。AMP 演算法達到此理論曲線。
Phase transition: Donoho & Tanner (2009) showed that for large N, recovery success depends on two ratios: δ = M/N (undersampling) and ρ = K/M (relative sparsity). There is a sharp curve in the (δ, ρ) plane — above it, failure; below, exact recovery. AMP attains this theoretical curve.

🌳 什麼時候用壓縮感知？

情境1:
量測昂貴、信號稀疏——MRI、空間轉錄體探針設計、CRISPR 池化篩選。
情境2:
已知或可學 sparsifying basis——影像 wavelet、基因表現 PCA、自編字典。
情境3:
需要解析度高於 Nyquist——super-resolution microscopy (DNA-PAINT, STORM)。
不適合:
信號不稀疏（如連續濃淡的自然影像，且無 wavelet 稀疏先驗），或量測本身已很便宜時——增加量測比學 CS 更直接。
Case 1:
Expensive measurements, sparse signal — MRI, spatial transcriptomics probe design, pooled CRISPR screens.
Case 2:
Known or learnable sparsifying basis — image wavelets, gene-expression PCA, learned dictionaries.
Case 3:
Need super-Nyquist resolution — super-resolution microscopy (DNA-PAINT, STORM).
Not for:
Signals that aren't sparse in any known basis, or when more measurements are cheap — just sample more.
生物應用

壓縮感知在單細胞、影像與基因體

MERFISH / seqFISH

Moffitt et al. (2016, Science / PNAS) 把 1001 個基因編成 16-bit 二元碼字、只用 16 輪雜交（共~100 個探針）即可解碼。本質上就是 CS：基因表現在任一細胞 sparse（只有少量同時高表現），量測矩陣為 binary code book。後續 MERFISH 商業化（Vizgen），seqFISH+ (Eng 2019) 把概念推到 10000 基因。

Moffitt et al. (2016, Science / PNAS) encoded 1001 genes into 16-bit binary codewords, decoded from ~100 probes across 16 hybridization rounds. Pure CS: per-cell expression is sparse, the measurement matrix is the codebook. Commercialized as MERFISH (Vizgen); seqFISH+ (Eng 2019) pushed to 10000 genes.

DNA-PAINT super-resolution

Jungmann et al. (2014, Nature Methods) 用瞬時 DNA 雜交「閃光」單分子定位。CS 重建（Min 2014, Nature Methods）：稀疏 emitter prior + 偽 PSF 量測，比傳統 single-molecule fitting 在高密度區更穩。

Jungmann et al. (2014, Nature Methods) use transient DNA hybridization for single-molecule "blinks". CS reconstruction (Min 2014, Nature Methods): sparse emitter prior + pseudo-PSF measurement — more robust than single-molecule fitting in dense regions.

LASSO-PRS 稀疏多基因評分

PRS 傳統用所有 SNP 加權求和；LASSO-PRS（Mak et al. 2017, Genet Epidemiol; lassosum）把它變成 CS 問題：少量「真因果」SNP 構成稀疏信號，GWAS summary stats 是量測，L1 罰項挑出 active SNP，預測力與可解釋性同時提升。

PRS traditionally sums over all SNPs; LASSO-PRS (Mak et al. 2017, Genet Epidemiol; lassosum) frames it as CS: few causal SNPs form a sparse signal, GWAS summary stats are measurements, L1 picks active SNPs — better prediction and interpretability.

Compressed Cell Painting

Bray et al. (2016, Nature Protocols) 之 Cell Painting 用 5 染料拍 5 通道。Way et al. (2021) 與 Caicedo 後續研究顯示：morphology feature 在 PCA 後極稀疏，可用 CS-like 隨機投影壓縮 50%–80% 仍保留 MoA 訊號——讓 high-content screening 更便宜。

Cell Painting (Bray 2016, Nature Protocols) uses 5 dyes / 5 channels. Way et al. (2021) and Caicedo lab follow-ups show morphology features are highly sparse after PCA — random projections compress by 50–80% while preserving MoA signal, lowering HCS costs.

CITE-seq panel design

CITE-seq 用抗體偵測蛋白質——預算只有 ~100 種抗體，卻想覆蓋 1000+ marker。Cleary et al. (2017, Cell) 提出 composite measurement（多種抗體共 barcode）+ CS 解碼，理論上可從 100 measurements 恢復 1000 protein abundances。

CITE-seq uses antibodies to read proteins — limited to ~100 antibodies but wants to cover 1000+ markers. Cleary et al. (2017, Cell) proposed composite measurements (multi-antibody barcodes) + CS decoding — recovering 1000 protein levels from 100 measurements.

CRISPR pooled screen + CS decoder

Pooled CRISPR screen 用 barcode 把 sgRNA 識別出來。Adamson et al. (2016, Cell) 的 Perturb-seq 與後續 compressed Perturb-seq (Yao 2023, biorXiv) 採用 CS 思想：同時擾動多個基因、用 L1 解碼真正活性基因，降低 library 成本。

Pooled CRISPR screens identify sgRNAs by barcode. Adamson et al. (2016, Cell) Perturb-seq and follow-up compressed Perturb-seq (Yao 2023, bioRxiv) use CS: perturb many genes at once, L1-decode active ones — cuts library cost.

Sparse imaging mass spectrometry

代謝物 MSI 像素 × m/z 維度爆炸——但每像素只有少量 metabolite 顯著。Bemis et al. (2015, Bioinformatics) 用 CS 重建 sparse spatial-spectral map，速度 + 解析度同升。

Metabolite MSI has huge pixel × m/z dimension, yet each pixel has few significant metabolites. Bemis et al. (2015, Bioinformatics) reconstruct sparse spatial-spectral maps with CS — faster and higher-resolution.

Tasic 2018 — sparse marker cell typing

Tasic et al. (2018, Nature) 在 mouse cortex 用稀疏 panel of marker gene 識別 ~100 神經細胞型別。本質上是「在低維 marker 空間中保持階層 cluster 可分」——LASSO + 階層 selection 的 CS 變體。

Tasic et al. (2018, Nature) identify ~100 cortical cell types with a sparse marker panel — keeping clusters separable in low-dim marker space, a CS variant with LASSO + hierarchical selection.

Imaging mass cytometry 重建

IMC 同時量 40+ marker，但取樣慢。Compressed acquisition (Schapiro 2017; Bodenmiller lab) 採用稀疏隨機取樣 + sparse-in-wavelet prior 重建，把成像時間縮短一半。

IMC measures 40+ markers but acquisition is slow. Compressed acquisition (Schapiro 2017; Bodenmiller lab) uses sparse random sampling + wavelet sparsity to halve scan time.

深入解析

壓縮感知的進階主題

① 結構化稀疏 (Structured Sparsity)

實務信號常不只「稀疏」——還有結構（block-sparse、group-sparse、tree-sparse）。Group LASSO（Yuan & Lin 2006）、model-based CS（Baraniuk 2010, IEEE TIT）利用結構先驗，可進一步減少所需量測 M。

② Total Variation (TV) 與 MRI

MRI 影像稀疏於 gradient（鄰像素差很小）——min ‖∇x‖_1 即 TV-CS。Lustig et al. (2007, MRM) 把 CS 引入 MRI 加速 4–6×，現已成商業 MRI 標配（GE/Siemens "Compressed Sensing" mode）。

③ Dictionary Learning

當預設 basis (DCT/Wavelet) 不夠稀疏，可從資料學 dictionary：K-SVD (Aharon 2006, IEEE TSP)、Online dictionary learning (Mairal 2009)。生物應用：Cell Painting feature dictionary、ATAC-seq motif dictionary。

④ Bayesian CS

Ji, Xue & Carin (2008, IEEE TSP) 把 sparse prior 換成 Laplace 或 spike-and-slab，做 MCMC / VB 推論，得到 posterior。優點：自然提供 uncertainty——在臨床預測尤為重要。

⑤ One-bit CS

Boufounos & Baraniuk (2008) 把量測量化為 ±1（只取 sign），仍可恢復方向。對 ADC 受限的硬體（生物感測器、IoT）非常實用。

⑥ Generative / Plug-and-Play Prior

除了 L1，可用 deep generative model（VAE/GAN/Diffusion）當 prior：Bora et al. (2017, ICML) 證明在 m = O(k log n) 量測下，generative prior 比 L1 表現更好。Score-based Diffusion + CS（Chung 2022）成為當前最強 MRI/CT 重建 SOTA。

⑦ 與 sketching / random feature 的關係

CS 的 Φ ≈ 隨機投影，與 dimensionality reduction (Johnson–Lindenstrauss)、kernel sketching、隨機 forest 同源。CS 為這些方法提供 sample complexity 下界。

⑧ 為何 L1 work？

L1 是 L0 (counting) 的最緊凸鬆弛，幾何上「L1 球的角」剛好落在稀疏軸上。Candès 證在 RIP 下，L1 解 = L0 解。

① Structured sparsity

Real signals often have structure beyond mere sparsity — block-sparse, group-sparse, tree-sparse. Group LASSO (Yuan & Lin 2006), model-based CS (Baraniuk 2010, IEEE TIT) exploit structural priors to further reduce M.

② Total Variation (TV) + MRI

MRI images are sparse in gradient — min ‖∇x‖_1 is TV-CS. Lustig et al. (2007, MRM) brought CS to MRI for 4–6× acceleration, now standard in commercial scanners (GE/Siemens "Compressed Sensing" mode).

③ Dictionary learning

When fixed bases (DCT/wavelet) aren't sparse enough, learn a dictionary from data: K-SVD (Aharon 2006, IEEE TSP), online DL (Mairal 2009). Biology: Cell Painting feature dictionaries, ATAC-seq motif dictionaries.

④ Bayesian CS

Ji, Xue & Carin (2008, IEEE TSP) replace the sparse prior with Laplace or spike-and-slab and do MCMC / VB to get a posterior — natural uncertainty quantification, vital for clinical prediction.

⑤ One-bit CS

Boufounos & Baraniuk (2008) quantize measurements to ±1 (sign only) and still recover the direction. Useful for ADC-limited hardware (biosensors, IoT).

⑥ Generative / plug-and-play priors

Beyond L1, use deep generative models (VAE/GAN/Diffusion) as priors: Bora et al. (2017, ICML) showed generative priors beat L1 at m = O(k log n). Score-based diffusion + CS (Chung 2022) is now SOTA for MRI/CT reconstruction.

⑦ Relation to sketching / random features

CS's Φ ≈ random projection, related to JL dimensionality reduction, kernel sketching, random forests. CS gives sample-complexity bounds for these.

⑧ Why does L1 work?

L1 is the tightest convex relaxation of L0; geometrically, the L1 ball's corners lie on sparse axes. Under RIP, L1's solution equals L0's.

潛在改良應用

還能改良哪些任務？

🎯 1. Compressed scRNA-seq 探針設計

目前 single-cell platforms（10x, Visium）量所有基因。提案：設計「composite primer」混合多個 gene 的訊號，用 CS 解碼出單細胞表現，降低 sequencing 成本 5–10×。

🎯 2. CS-based GWAS replication

大型 GWAS 重複常需重新基因型化。提案：只 genotype 隨機選 K log(N/K) 個 SNP，配合 LD reference 用 CS 重建全 panel——大幅降低重現成本。

🎯 3. 動態壓縮感知 single-cell longitudinal

追蹤 patient 動態 transcriptome 時，連續 timepoint 差異稀疏。提案：每個 timepoint 只測 Δ-CS（差異稀疏採樣），重建完整 trajectory。

🎯 4. Compressed Cryo-EM particle picking

Cryo-EM micrograph 大部分是冰、極少 particle。提案：用 CS-like sparse acquisition 加速 particle picking，配合 diffusion prior 做缺失恢復。

🎯 5. 整合 CS + Diffusion model 用於 spatial 重建

Spatial transcriptomics 解析度受限。提案：把 ST 視為 sparse subsampling，用 score-based diffusion prior 做 super-resolution——比現有 deep ST 模型更穩健。

🎯 6. Compressed multiplexed metabolomics

單一細胞代謝體幾百個分子，但每細胞只測 10–20 個。提案：用 CS 設計「複合質譜標籤」，從少量探針恢復數百代謝物濃度。

🎯 7. 壓縮感知 + Federated learning

聯邦學習頻寬瓶頸：gradient 上傳成本高。提案：每 client 對 gradient 做 CS 壓縮，server 端解碼——維持 privacy 又降低通訊（生醫聯邦尤其重要）。

🎯 1. Compressed scRNA-seq probe design

Current platforms (10x, Visium) measure all genes. Proposal: design composite primers mixing many gene signals, CS-decode to per-cell expression — 5–10× cheaper sequencing.

🎯 2. CS-based GWAS replication

Replication often re-genotypes everyone. Proposal: genotype only K log(N/K) random SNPs and CS-reconstruct the full panel via LD reference — slashes replication cost.

🎯 3. Dynamic CS for longitudinal single-cell

For patient time-courses, transcriptome differences between timepoints are sparse. Proposal: Δ-CS sampling per timepoint, reconstruct full trajectory.

🎯 4. Compressed cryo-EM particle picking

Most micrograph pixels are ice. Proposal: CS-like sparse acquisition for picking, with diffusion priors filling missing regions.

🎯 5. CS + diffusion priors for spatial

ST resolution is limited. Proposal: treat ST as sparse subsampling and use a score-based diffusion prior for super-resolution — more robust than current deep-ST models.

🎯 6. Compressed multiplexed metabolomics

Per-cell metabolomes have hundreds of metabolites but few measured. Proposal: CS-designed composite MS tags to recover hundreds from a small panel.

🎯 7. CS + federated learning

Federated learning is bandwidth-limited. Proposal: CS-compress per-client gradients, decode at server — preserves privacy, reduces communication. Especially valuable for biomedical federation.

程式碼

實作範例

# Compressed sensing via LASSO
import numpy as np
from sklearn.linear_model import Lasso

N, K, M = 256, 10, 60
# Sparse signal
s = np.zeros(N)
idx = np.random.choice(N, K, replace=False)
s[idx] = np.random.randn(K) * 3
# Random Gaussian sensing matrix
Phi = np.random.randn(M, N) / np.sqrt(M)
y = Phi @ s + 0.01 * np.random.randn(M)

# Recover with LASSO (L1 regularized regression)
model = Lasso(alpha=0.01, max_iter=5000)
model.fit(Phi, y)
s_hat = model.coef_
err = np.linalg.norm(s - s_hat) / np.linalg.norm(s)
print(f"relative error: {err:.4f}")
# Orthogonal Matching Pursuit (OMP)
from sklearn.linear_model import OrthogonalMatchingPursuit

omp = OrthogonalMatchingPursuit(n_nonzero_coefs=K)
omp.fit(Phi, y)
s_hat = omp.coef_

# Manual OMP (educational)
def omp_manual(Phi, y, K):
    r = y.copy(); support = []
    for _ in range(K):
        j = np.argmax(np.abs(Phi.T @ r))
        support.append(j)
        s_act, *_ = np.linalg.lstsq(Phi[:, support], y, rcond=None)
        r = y - Phi[:, support] @ s_act
    s = np.zeros(Phi.shape[1]); s[support] = s_act
    return s
# Basis Pursuit via cvxpy
import cvxpy as cp

s_var = cp.Variable(N)
problem = cp.Problem(cp.Minimize(cp.norm1(s_var)),
                     [Phi @ s_var == y])
problem.solve(solver=cp.ECOS)
s_hat = s_var.value

# Basis Pursuit Denoising (BPDN) — handles noise
eps = 0.05
problem_bpdn = cp.Problem(
    cp.Minimize(cp.norm1(s_var)),
    [cp.norm(Phi @ s_var - y, 2) <= eps])
problem_bpdn.solve()
# Large-scale CS with pylops (matrix-free linear ops)
import numpy as np
from pylops import MatrixMult
from pylops.optimization.sparsity import FISTA, OMP

Phi_op = MatrixMult(Phi)
# FISTA for L1-regularised problem (fast)
s_hat = FISTA(Phi_op, y, niter=200, eps=0.01)[0]
