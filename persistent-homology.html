持久同調 (Persistent Homology)

拓撲資料分析的核心工具——用「孔洞」、「迴圈」、「連通分支」的數量描述細胞狀態空間的形狀，能偵測 PCA/UMAP 看不見的拓撲特徵。

The core tool of Topological Data Analysis (TDA) — describes the shape of cell-state space via counts of voids, loops, and connected components, revealing topology that PCA/UMAP cannot.

核心概念

同調群與 Betti 數

給定點雲，畫圓（半徑 ε）覆蓋每個點，逐漸放大 ε。在這個過程中：

  • β₀（零維同調）= 連通分支數量
  • β₁（一維同調）= 迴圈、孔洞數量（像甜甜圈中央的洞）
  • β₂（二維同調）= 空腔數量（像球殼裡的空間）

持久的意涵：追蹤這些拓撲特徵在 ε 從 0 增到無窮的過程中「何時誕生、何時消失」。長壽 (long persistence) 的特徵是真實結構，短壽是噪音。

Given a point cloud, draw balls of radius ε around each point and grow ε. Along the way:

  • β₀ (0-dim homology) = # connected components
  • β₁ (1-dim homology) = # loops / holes (like a donut hole)
  • β₂ (2-dim homology) = # voids (like a hollow ball)

Persistence: track when each topological feature is born and dies as ε grows. Long-persistence features are real structure; short ones are noise.

🔵

連通分支 β₀

獨立細胞型別群的數量。

Number of distinct cell-type groups.

🍩

迴圈 β₁

細胞週期、晝夜節律等「循環」軌跡的指標。

Marker of cyclic trajectories (cell cycle, circadian, etc.).

⚽

空腔 β₂

罕見但有意義——例如分化分支中的「空白」狀態。

Rare but meaningful — e.g. "empty" states between differentiation branches.

演算法

Filtration、Persistence Diagram、Barcode

① Vietoris-Rips Filtration：用 ε 為閾值建立單純複形——點與點距離 ≤ ε 連邊，三點兩兩相連時填三角形，依此類推。隨 ε 增加，複形逐漸長大。

② Persistence Diagram：把每個拓撲特徵畫成 2D 點 (birth, death)。對角線附近 = 噪音；遠離對角線 = 顯著結構。

③ Barcode：每個特徵畫一條水平線（從誕生到死亡的 ε 區間）。長條 = 重要、短條 = 噪音。

常用工具：Ripser (C++, 最快)、GUDHI (Python)、giotto-tda (機器學習友善)、TDAstats (R)。

① Vietoris-Rips filtration: threshold by ε — connect points within ε; fill triangles when triples connect; etc. Complex grows with ε.

② Persistence diagram: each topological feature is a 2D point (birth, death). Near the diagonal = noise; far away = significant structure.

③ Barcode: each feature gets a horizontal bar from birth to death ε. Long bars = important; short = noise.

Tools: Ripser (C++, fastest), GUDHI (Python), giotto-tda (ML-friendly), TDAstats (R).

互動模擬

掃描 ε，看 Betti 數變化

選擇點雲形狀（雙團、圓圈、8 字形、隨機），調整 ε 觀察連通分支與迴圈的變化。圓圈形狀在某個 ε 範圍內 β₁ = 1，這就是「迴圈持久」的具體表現。

Pick a point cloud shape (two blobs, circle, figure-8, random); sweep ε to watch components and loops. A circle yields β₁ = 1 over a wide ε range — that's a persistent loop.

生物應用

TDA 在單細胞與多體學

細胞狀態空間拓撲

scTDA、TopoMap 等：直接刻畫細胞狀態空間的全局拓撲——分支點、迴圈、孔洞，補足 UMAP 容易扭曲的部分。

scTDA, TopoMap, etc.: characterize global topology of cell-state space — branch points, loops, voids — complementing UMAP's distortions.

循環過程偵測

細胞週期、晝夜節律基因群的軌跡是迴圈 → 持久 β₁ 為 1 是強訊號。比手動指定 marker 更客觀。

Cell cycle and circadian gene trajectories are loops → persistent β₁ = 1 is a strong signal. More objective than hand-picked markers.

Mapper 演算法

將高維資料投影到 1D 函數（如假時間）→ 沿 cover 分群 → 建構圖。在乳癌 (Nicolau 2011) 等研究中找到新次型。

Project high-dim data along a 1D function (e.g. pseudotime) → cluster within cover → build graph. Discovered new breast cancer subtypes (Nicolau 2011).

空間轉錄體拓撲

TopACT (2023) 用 TDA 偵測組織內的局部結構（小型免疫聚集、血管環等）。

TopACT (2023) uses TDA to detect local tissue structures (immune aggregates, vascular loops, etc.).

基因組學

Hi-C 染色質互動圖、序列保守區的拓撲分析——TDA 是少數同時處理「形狀」與「尺度」的工具。

Hi-C chromatin maps, conserved sequence topology — TDA uniquely captures both "shape" and "scale."

深度學習特徵

把 persistence diagram 向量化（persistence images / landscapes）→ 直接作為 ML 特徵餵入分類器。

Vectorize persistence diagrams (persistence images / landscapes) → use as ML features.

深入解析

深入解析 TDA：filtration、穩定性、向量化、可微化

① Filtration 種類比較：Rips、Čech、Alpha、Witness

不同 filtration 在「計算成本」與「topological fidelity」上有顯著取捨：

  • Vietoris–Rips：最常用——閾值 ε 內所有點對相連即組成 simplex。優點：定義簡單、只需距離矩陣；缺點：simplex 數量在維度 d 是 O(nd+1)，n=10,000 時 H2 就會爆。
  • Čech complex：幾何上「正確」（Nerve theorem 保證），但需測試 (d+1) 顆球是否共交，計算昂貴。
  • Alpha complex：Delaunay 三角化的子複形，simplex 數量 O(n⌈d/2⌉)——在 ℝ³ 中遠少於 Rips，是低維點雲（蛋白質結構、空間轉錄體座標）的首選。
  • Witness complex (de Silva–Carlsson 2004)：選 ℓ 個 landmark，其他點作為「見證」。把 100k 細胞 → 500 landmark 可以把計算量降 4–5 個數量級，是 scTDA 必用工具。

② 穩定性定理（Cohen-Steiner, Edelsbrunner, Harer 2007）

TDA 之所以「敢用」的數學基石：若兩個 filtration f, g 的 sup-norm 距離 ≤ ε，則對應 persistence diagram 的 bottleneck distance ≤ ε。換句話說，對輸入的小擾動只會造成輸出的小擾動——這在 scRNA-seq 等高噪音資料上非常關鍵。實務含意：少量 dropout 不會把真實的 H1 loop 殺掉，只會讓它在 persistence diagram 上「往對角線稍微移」。

③ 向量化方法：把 PD 變成 ML feature

Persistence diagram 是「multi-set of points in ℝ²」，無法直接餵 sklearn。三種主流向量化：

  • Persistence Landscape (Bubenik 2015)：把每個 (b, d) 點轉成「三角函數」λk(t)，得到 ℓ² Hilbert space 上的函數——可以做 mean、PCA、t-test。穩定且有統計理論。
  • Persistence Image (Adams 2017)：用 Gaussian kernel 把 PD smoothing 成固定大小的圖像（如 50×50）——適合 CNN 處理。權重函數通常選 persistence 本身（讓長壽特徵更亮）。
  • Betti Curve：對每個 ε 數 simplex 拓撲群的維度——資訊量較少但快速。

④ 計算複雜度與 sparse Rips

標準 Rips 計算 Hk 是 O(nk+2)。對 n=5000、k=1 約 6×1010 次運算。解法：

  • Sparse Rips (Sheehy 2013)：用 (1+ε)-近似，把多餘 simplex 砍掉——對 ε=0.1 可減 90% 計算量，理論誤差有界。
  • Ripser (Bauer 2021)：透過 cohomology + clearing optimization，比 GUDHI 快 10–40 倍，是目前 H1 計算的事實標準。
  • GPU Ripser (Pérez 2022)：把 reduction 並行化，再快 5–10 倍。

⑤ 噪音與信號分離：bootstrap confidence band、persistent entropy

怎麼知道 PD 上的點是「真實 topology」還是「採樣雜訊」？

  • Bootstrap confidence band (Fasy 2014)：重抽樣 B 次計算 PD，求 bottleneck distance 的 (1-α) quantile cα。距離對角線 > cα 的點即為顯著拓撲特徵。
  • Persistent Entropy (Rucco 2017)：E = −Σ (lᵢ/L) log(lᵢ/L)，其中 lᵢ 是第 i 個特徵的 persistence、L 為總和。低熵 = 一兩個明顯特徵；高熵 = 雜訊或多尺度結構。
  • 實務技巧：把 PD 上每個點的 persistence 排序，找「gap」——通常前 k 個點是訊號，其後是噪音。

⑥ Mapper 演算法的詳細步驟

Mapper 是 TDA 的「資料探索」端，輸出一張圖而非 PD：

  1. Lens function：選一個 f: X → ℝd（d 通常 1 或 2）。常用 lens：eccentricity（離中心距離）、density estimate、PC1、pseudotime、Laplacian eigenmap。
  2. Cover：把 f(X) 的範圍切成重疊的小區間 {Ui}（resolution 與 overlap 是超參數）。
  3. Cluster：在每個 f−1(Ui) 內跑 clustering（DBSCAN、單連結等）。
  4. Nerve：每個 cluster 是節點，重疊區域內共享元素則連邊。

Mapper 的優勢是「同時顯示 branching、loops、群集」——這是 UMAP 做不到的。但 lens/resolution/overlap 三超參數讓結果敏感，通常要做 grid sweep。

⑦ 拓撲機器學習：TDA × GNN

近年熱門方向是把 TDA 訊號注入 GNN：

  • TopoGNN (Horn 2022)：每個節點周圍取 ego-graph 算 PH，把 PD 向量化後作為 node feature 串接到 GNN 隱藏層。在分子分類上 SOTA。
  • Persistent Weisfeiler-Lehman kernel (Rieck 2019)：把 WL 顏色 refinement 配合 filtration，得到考慮拓撲的 graph kernel。
  • 單細胞應用：在 cell-cell graph 上跑 TopoGNN 可同時捕捉「局部鄰域結構」與「全域 cycle / branch」，比純 GCN 更能保留發育拓撲。

⑧ Differentiable TDA（Carrière 2021、Leygonie 2022）

傳統 PD 計算是組合性的（不可微）；近年發展出「以 persistence 為 loss」的方法：對 birth-death 配對，把 bi, di 視為輸入點座標的函數，可在 simplex matching 不變時做 chain rule。應用：

  • Topological autoencoder (Moor 2020)：強制 latent space 保留輸入 PD——比普通 AE 更能保留拓撲。
  • Topology-aware GAN：產生器加 persistence 正則項，迫使生成樣本保留目標 topology（例如「連通且有一個 loop」）。
  • 在 scLLM 上：用 differentiable TDA 當 pretraining auxiliary loss，可讓 latent 空間更貼近細胞發育的真實拓撲。

① Filtration comparison: Rips, Čech, Alpha, Witness

Different filtrations trade compute against topological fidelity:

  • Vietoris–Rips: all pairs within ε form a simplex. Simple, distance-matrix only; but simplex count is O(nd+1) — H2 on n=10,000 explodes.
  • Čech complex: geometrically correct (Nerve theorem), but requires testing if (d+1) balls intersect — expensive.
  • Alpha complex: subcomplex of Delaunay triangulation; O(n⌈d/2⌉) simplices. In ℝ³ much smaller than Rips — first choice for low-dim point clouds (protein structure, spatial coords).
  • Witness complex (de Silva–Carlsson 2004): pick ℓ landmarks, use the rest as witnesses. 100k cells → 500 landmarks drops compute 4–5 orders of magnitude — essential for scTDA.

② Stability theorem (Cohen-Steiner, Edelsbrunner, Harer 2007)

TDA's mathematical bedrock: if two filtrations f, g satisfy ‖f−g‖∞ ≤ ε, the corresponding persistence diagrams' bottleneck distance ≤ ε. Small input perturbations cause only small output perturbations — critical for noisy scRNA-seq. Practical implication: mild dropout won't kill a real H1 loop, only shift it slightly toward the diagonal.

③ Vectorization: turning PDs into ML features

Persistence diagrams are multi-sets of points in ℝ² — can't go directly into sklearn. Three main vectorizations:

  • Persistence Landscape (Bubenik 2015): turn each (b, d) into triangular functions λk(t) in ℓ² Hilbert space — supports mean, PCA, t-tests; well-grounded statistically.
  • Persistence Image (Adams 2017): Gaussian-smooth the PD into a fixed-size grid (e.g. 50×50) — feeds CNNs. Weight function usually scales with persistence.
  • Betti Curve: count topology dimension at each ε — less info, but fast.

④ Complexity and sparse Rips

Standard Rips for Hk is O(nk+2). For n=5000, k=1 ≈ 6×1010 ops. Solutions:

  • Sparse Rips (Sheehy 2013): (1+ε)-approximation drops 90% simplices at ε=0.1 with bounded error.
  • Ripser (Bauer 2021): cohomology + clearing optimization, 10–40× faster than GUDHI — the de facto H1 standard.
  • GPU Ripser (Pérez 2022): parallel reduction, another 5–10× speedup.

⑤ Noise vs. signal: bootstrap bands and persistent entropy

How do we know a PD point is real topology, not sampling noise?

  • Bootstrap confidence band (Fasy 2014): resample B times, compute the (1−α) quantile cα of bottleneck distances. Points farther than cα from the diagonal are significant.
  • Persistent Entropy (Rucco 2017): E = −Σ (lᵢ/L) log(lᵢ/L). Low E = one or two clear features; high E = noise or multi-scale.
  • Trick: sort all PD points by persistence — find the gap; everything above is signal.

⑥ Mapper algorithm in detail

Mapper is the "exploration" side of TDA, producing a graph rather than a PD:

  1. Lens function: f: X → ℝd (usually d=1 or 2). Common: eccentricity, density, PC1, pseudotime, Laplacian eigenmaps.
  2. Cover: chop f(X) range into overlapping intervals {Ui} (resolution + overlap are hyperparams).
  3. Cluster: cluster within each f−1(Ui) (DBSCAN, single-linkage, etc).
  4. Nerve: each cluster is a node; share elements → edge.

Mapper shows branching, loops, and clusters together — UMAP cannot. But lens/resolution/overlap make results sensitive; grid sweeps are essential.

⑦ Topological ML: TDA × GNN

A hot recent direction is injecting TDA into GNNs:

  • TopoGNN (Horn 2022): compute PH on each node's ego-graph, vectorize, concatenate into GNN hidden layers. SOTA on molecular classification.
  • Persistent WL kernel (Rieck 2019): couple WL color refinement with filtrations → topology-aware graph kernels.
  • Single-cell use: TopoGNN on the cell-cell graph captures both local neighborhoods and global cycles/branches — better than plain GCN for preserving developmental topology.

⑧ Differentiable TDA (Carrière 2021, Leygonie 2022)

Classical PD computation is combinatorial (non-differentiable); recent work treats persistence as a loss: birth–death pairs (bi, di) are functions of input coordinates, so chain rule applies while the simplex matching is locally constant. Applications:

  • Topological autoencoder (Moor 2020): forces latent space to preserve input PD — better topology than plain AE.
  • Topology-aware GAN: persistence regularizer makes the generator produce target topologies (e.g. "connected with one loop").
  • scLLM: use differentiable TDA as auxiliary pretraining loss, aligning latent space with developmental topology.
詳細已知應用

TDA 在生物資訊的具體案例

🧬 案例 1：Nicolau, Levine, Carlsson (2011, PNAS) — Mapper 找乳癌新亞型

TDA 的里程碑論文。用 Mapper 對 NKI 乳癌 microarray 資料（295 病人 × 24,000 基因）建圖，lens 函數為 DSGA（disease-specific genomic analysis）的 L¹ deviation norm。發現一個獨特的「分支」——所有屬於該分支的病人都是 ER⁺、有 c-MYB 高表現、且 10 年內存活率 100%。這個亞型用傳統 hierarchical clustering 完全看不到，因為它在主結構之外「岔出去」。後續 GLAS 等獨立資料集驗證。

🧬 案例 2：scTDA — Rizvi et al. (2017, Nature Biotechnology)

第一篇將 Mapper 系統化應用到 scRNA-seq 的論文。對小鼠胚胎肝臟發育（E12–E18）資料建 Mapper graph，lens 為「擬時間 + density」。發現原本被認為線性的肝細胞 vs 紅血球分化其實有「branching point + 再匯聚」的拓撲——這個 loop 在 PCA/diffusion map 上是看不到的（它們會「展開」loops）。該論文也提出 multi-scale persistence 量化發育關鍵節點。

🧬 案例 3：TopACT (Benjamin et al. 2023, Nature Communications)

把 PH 用在空間轉錄體：對每個 spot 周圍的「局部基因表現分布」算 H0、H1，找出「拓撲不尋常」的區域——例如小型免疫聚集 (tertiary lymphoid structures) 通常呈現異常 H1 訊號（圓環狀組織）。在乳癌 Visium 資料中找到 SpatialDE/Hotspot 漏掉的微結構，並經免疫螢光驗證。

🧬 案例 4：Camara et al. (2016, Cell Systems) — HIV reassortment 的拓撲

傳統演化樹假設「樹狀結構」，但病毒重組會破壞樹狀性。Camara 用 PH 計算 HIV 序列的 H1——若有 reassortment，會出現非零 H1 loops，loop 的「壽命」對應重組頻率。在 HIV-1 group M 上發現比傳統 phylogeny 多 30% 的 reassortment 事件。

🧬 案例 5：染色質拓撲 — Lesne et al. (2014), Boninsegna et al. (2017)

把 Hi-C 接觸矩陣轉成距離矩陣後算 PH，發現 TAD (topologically associated domains) 與 H1 loops 的對應——特別是在 CTCF binding site 集中的區域出現高 persistence loop。Boninsegna 進一步在 3D polymer model 上跑 TDA，找到傳統 contact-frequency 看不到的「染色質結構模塊」。

🧬 案例 6：蛋白質結構的 PH (Edelsbrunner, Xia et al.)

用 alpha complex 對蛋白質 Cα 原子建 filtration，得到的 PD 可作為 protein folding 分類特徵。在 SCOP 分類上 PD-based feature 比 contact-map 更穩健於不同物種的同源蛋白。CIST 演算法（Xia & Wei 2014, Bioinformatics）是其中代表。

🧬 案例 7：癌症演化的拓撲 loops — Smith et al. (2024)

用 multi-region tumor sequencing 資料的 SNV profile 跑 PH，發現 convergent evolution（不同 region 獨立演化到相似 genotype）會在 PD 上呈現特定 H1 pattern——這在傳統 phylogenetic tree 上會被「強制成樹」而看不到。在 TRACERx 肺癌資料上識別出 30% 病人有 topological convergence。

🧬 案例 8：神經迴路拓撲 — Singh et al. (2008), Giusti et al. (2015)

對 V1 視覺皮層的神經元發放率資料跑 PH，發現「pinwheel-like」拓撲（torus + Klein bottle 混合）——這對應於 orientation tuning 的拓撲結構。Giusti 2015 進一步用「order-based」filtration（取代距離）對 hippocampal place cells 跑 TDA，重建出 spatial environment 的真實拓撲（圓環 vs 平面），不需要任何 calibration。

🧬 案例 9：免疫 TCR repertoire 的多樣性 — Reeves et al. (2018)

把 TCR 序列空間建為點雲（用 BLOSUM-based distance），用 PH 量「repertoire 形狀」。發現 chronic infection 與 healthy 個體在 H1 persistent entropy 上有顯著差異——chronic 個體 repertoire 呈現更多「loops」（clonal expansion + somatic hypermutation 形成的環狀拓撲）。

🧬 Case 1: Nicolau, Levine, Carlsson (2011, PNAS) — Mapper Finds New Breast Cancer Subtype

The landmark TDA paper. Applied Mapper to NKI breast cancer microarray data (295 patients × 24,000 genes) with DSGA L¹ deviation norm as the lens. Discovered a distinct "branch" — all patients in it were ER⁺, c-MYB high, with 100% 10-year survival. This subtype was invisible to classical hierarchical clustering because it branched off the main structure. Independently validated on the GLAS cohort.

🧬 Case 2: scTDA — Rizvi et al. (2017, Nature Biotechnology)

The first systematic Mapper application to scRNA-seq. On mouse embryonic liver (E12–E18) data, lens = pseudotime + density. Found that liver vs. erythrocyte differentiation, thought linear, actually has a "branch + re-merge" topology — this loop is invisible in PCA/diffusion maps (which unfold loops). The paper also introduced multi-scale persistence to quantify developmental decision points.

🧬 Case 3: TopACT (Benjamin et al. 2023, Nature Communications)

PH applied to spatial transcriptomics: compute H0, H1 on local gene-expression distributions per spot to flag topologically unusual regions — e.g. tertiary lymphoid structures show anomalous H1 (ring-like). On breast cancer Visium, recovered microstructures missed by SpatialDE/Hotspot, validated by immunofluorescence.

🧬 Case 4: Camara et al. (2016, Cell Systems) — Topology of HIV Reassortment

Standard phylogenetics assumes a tree, but viral reassortment breaks tree-ness. Camara used PH on HIV sequences — reassortments give non-zero H1 loops, with loop lifetime ∝ reassortment frequency. On HIV-1 group M, found 30% more reassortment events than classical phylogeny.

🧬 Case 5: Chromatin topology — Lesne et al. (2014), Boninsegna et al. (2017)

Transform Hi-C contact matrices to distances, then PH — TADs correspond to H1 loops, especially in CTCF-rich regions. Boninsegna further ran TDA on 3D polymer models, recovering "chromatin structure modules" invisible to contact-frequency alone.

🧬 Case 6: Protein structure PH (Edelsbrunner, Xia et al.)

Alpha complex on Cα atoms gives a PD that classifies protein folds — more robust to cross-species homologs than contact maps. CIST (Xia & Wei 2014, Bioinformatics) is a representative algorithm.

🧬 Case 7: Cancer evolution loops — Smith et al. (2024)

PH on multi-region SNV profiles uncovers convergent evolution — different regions reaching similar genotypes — which produces a specific H1 pattern. Standard phylogenetic trees force these into trees and miss it. On TRACERx lung cancer data, 30% of patients showed topological convergence.

🧬 Case 8: Neural circuit topology — Singh et al. (2008), Giusti et al. (2015)

PH on V1 cortex firing rates revealed pinwheel-like topology (torus + Klein bottle hybrid), matching orientation tuning. Giusti 2015 used order-based filtrations (no metric) on hippocampal place cells to recover the spatial environment's true topology (ring vs. plane) without calibration.

🧬 Case 9: Immune TCR repertoire — Reeves et al. (2018)

Build TCR sequence space as a point cloud (BLOSUM-based distance), run PH on repertoire "shape." Chronic infection vs. healthy donors differ significantly in H1 persistent entropy — chronic patients have more loops (clonal expansion + somatic hypermutation forming ring topologies).

潛在改良應用

TDA 還能改良 / 新解決哪些任務？

🎯 1. 空間轉錄體的「真實 niche」偵測

CellCharter、Banksy 用 GMM 或 graph clustering 找 niche，但對「形狀」（環狀 vs 線狀 vs 散點）沒有區分能力。提案：對每個 spot 的 k-hop neighborhood gene expression 算 local PH，把 H1 persistence 當作「niche shape feature」串接到 Banksy 的 SVD step。預期效果：能區分「球狀腫瘤巢」vs「沿血管的線狀浸潤」vs「免疫環狀聚集」。對 Stereo-seq、CosMx、Xenium 高解析度資料特別有效。

🎯 2. RNA velocity 與 TDA 整合找「拓撲匯聚 attractors」

scVelo 給出每個細胞的速度向量，但「終點」（fate attractor）的判斷靠 CellRank 的 absorbing state 計算。提案：把 velocity field 視為向量場，對其「散度」（divergence）的負區域做 sublevel-set filtration——negative divergence ≡ sink region ≡ attractor。用 PH 找「H0 連通分量數量隨閾值變化」，得到 robust 的 attractor 數量。對 hematopoiesis 等多分支系統可避免人工指定 endpoint。

🎯 3. 多時間點軌跡的拓撲變化追蹤

同一個系統在 Day 0, 3, 7, 14 採樣，傳統 trajectory inference 把所有 timepoint 合在一起跑 pseudotime——這假設「軌跡形狀不變」。提案：對每個時間點分別跑 TDA，計算 PD0 → PD3 → PD7 → PD14 的 bottleneck distance 序列。「pothole」（突然的大跳）對應 fate decision moment。應用：早期胚胎發育（gastrulation 前後）、re-programming（iPSC 階段）、疾病進展（pre-cancer → cancer）。

🎯 4. scLLM 嵌入的拓撲可解釋性 audit

scGPT、Geneformer 預訓練後得到的 latent space 號稱「保留生物學結構」，但沒人嚴格驗證。提案：對 ground-truth dataset（如有完整 lineage 的 mouse atlas）的原始空間與 scLLM latent 空間分別跑 PH，計算 PD bottleneck distance。若 distance 大 = scLLM 「重塑」了拓撲（可能丟失 fate branches）。可以作為 scLLM 的 benchmark metric，補充傳統的 batch correction / cell-type ARI。

🎯 5. ChIP-seq peak cluster 的拓撲

傳統 ChIP-seq 分析只看 peak 位置；但 peaks 在基因組上的「空間分布形狀」（成簇 vs 分散 vs 週期）也帶資訊。提案：把每條染色體的 peak 位置視為 1D 點雲，跑 PH 算 H0 persistence——short-persistence H0 = 緊密 peak cluster (super-enhancer 候選)；long-persistence = 分散 peak。比傳統 ROSE 演算法（單一閾值）更穩健，並能跨樣本比較 cluster 結構演化。

🎯 6. 細胞分群的「真實 k 值」決定

Leiden/Louvain resolution 是人工選的，常用 silhouette、modularity 等指標——但這些對「形狀不規則」的群集（如 T cell 漸層）不友善。提案：對 PCA-reduced 表現矩陣跑 PH，看 H0 persistence diagram 的「明顯點數量」——對應「天然分離的群數」。比 silhouette analysis 更穩定，且能識別「連續變化」（H0 點靠對角線）vs「真實離散群」（H0 點遠離對角線）。

🎯 7. Tumor heterogeneity 的拓撲量化

單一腫瘤內的細胞異質性目前用 Shannon entropy、ITH score 量化，但這些都是「點估計」沒有結構資訊。提案：對每個 patient 的 tumor scRNA-seq 跑 PH，計算 persistent entropy + 多尺度 Betti numbers，作為「tumor topological signature」。在 TCGA 或 single-cell pan-cancer atlas 上跑，預期能找到傳統 ITH 看不到的「拓撲亞型」，並可能預測 immunotherapy 反應（複雜拓撲 → 多 fate paths → 更可能 resistance）。

🎯 8. 跨物種 GRN 拓撲保守性

跨物種比較 GRN 時，邊權重往往差異很大但「網路骨架」（哪些 hub、哪些 module）類似。提案：把 GRN 視為加權圖，用 graph filtration（依邊權重 sublevel）算 PH。H0 persistence = module 形成順序、H1 persistence = feedback loops。比較人 vs 鼠 vs 斑馬魚的 GRN PD 找 conserved topology。應用：找跨物種保守的調控核心、識別人類特有 regulatory innovations。

🎯 9. 免疫 TCR/BCR repertoire 的多尺度拓撲

Reeves 2018 已示範可行性，但只用 H1。提案：結合 H0（clonal cluster 數）、H1（somatic hypermutation cycles）、H2（cavity = 抗原識別 hole？）跑 multi-dim PH。應用：(a) COVID-19 病人 vs vaccinated vs naive 的 BCR 拓撲區分；(b) lupus 等自體免疫病的 TCR 拓撲簽名；(c) CAR-T therapy 反應預測。

🎯 10. 多體學 latent 拓撲對齊

MOFA、totalVI 等多體學整合方法 align 各 modality 的 latent，但「對齊」標準通常是 reconstruction loss 或 mutual nearest neighbor。提案：用 bottleneck distance 量「RNA latent PD」與「ATAC latent PD」的拓撲一致性，作為新的對齊目標函數。優勢：(a) 對 cell numbers 差異 robust；(b) 不需 1-to-1 cell matching；(c) 能保留 fate branches 而非只保 cluster 結構。

🎯 1. True spatial-niche detection

CellCharter and Banksy use GMM / graph clustering for niches but ignore shape (ring vs. line vs. scattered). Proposal: compute local PH on each spot's k-hop neighborhood expression, use H1 persistence as a "niche shape" feature concatenated with Banksy's SVD step. Expected: distinguish spherical tumor nests, linear perivascular infiltration, and ring-like immune aggregates. Especially useful for Stereo-seq, CosMx, Xenium.

🎯 2. RNA velocity + TDA — topological attractors

scVelo gives velocity vectors per cell; CellRank uses absorbing states to find fates. Proposal: treat velocity as a vector field; sub-level filter the negative divergence (sinks = attractors). PH on H0 connected-component changes gives a robust attractor count. Avoids manually specifying endpoints in branching systems like hematopoiesis.

🎯 3. Topological tracking across timepoints

Sampling at Day 0, 3, 7, 14 — standard pseudotime pools all timepoints, assuming a fixed trajectory shape. Proposal: compute a PD per timepoint, then bottleneck distance sequence PD0→PD3→PD7→PD14. "Potholes" (sudden jumps) flag fate decisions. Applications: gastrulation, reprogramming, pre-cancer → cancer.

🎯 4. Topological audit of scLLM embeddings

scGPT, Geneformer claim to preserve biological structure — rarely tested rigorously. Proposal: on a ground-truth lineage-labeled dataset, compute PD on raw vs. scLLM latent; bottleneck distance quantifies topological distortion. Use as a benchmark metric alongside batch correction / ARI.

🎯 5. Topology of ChIP-seq peak clusters

Standard ChIP-seq pipelines only look at peak positions; the genomic spatial distribution shape (clustered vs. dispersed vs. periodic) also carries information. Proposal: treat per-chromosome peak positions as a 1D point cloud, compute H0 persistence — short = tight clusters (super-enhancer candidates), long = dispersed. More robust than ROSE's single-threshold approach and supports cross-sample structural comparison.

🎯 6. Choosing the "real" k for clustering

Leiden/Louvain resolution is manual; silhouette/modularity fails on shape-irregular clusters (e.g. T-cell gradients). Proposal: compute PH on PCA-reduced expression; the count of "above-noise" H0 points = natural cluster count. Identifies continuous gradients (points near diagonal) vs. true discrete clusters (far from diagonal).

🎯 7. Topological quantification of tumor heterogeneity

Current ITH metrics are point estimates without structure. Proposal: per-patient PH on tumor scRNA-seq gives persistent entropy + multi-scale Betti numbers — a "topological tumor signature." On TCGA / pan-cancer atlases, likely uncovers topology subtypes invisible to ITH, and may predict immunotherapy response (complex topology → many fates → resistance).

🎯 8. Cross-species GRN topological conservation

Cross-species GRN comparisons: edge weights differ but the "skeleton" is similar. Proposal: sub-level filter weighted GRN edges, compute PH. H0 = module formation order; H1 = feedback loops. Compare human vs. mouse vs. zebrafish PDs to find conserved cores and human-specific innovations.

🎯 9. Multi-scale topology of TCR/BCR repertoires

Reeves 2018 used only H1. Proposal: combine H0 (clonal clusters), H1 (SHM cycles), H2 (antigen-recognition cavities?) for multi-dim PH. Applications: (a) COVID-19 BCR topology vs vaccinated vs naive; (b) autoimmune TCR signatures; (c) CAR-T response prediction.

🎯 10. Multi-omics latent topological alignment

MOFA, totalVI align modality latents by reconstruction loss or MNN. Proposal: use bottleneck distance between RNA-latent PD and ATAC-latent PD as the alignment objective. Advantages: robust to differing cell counts, no 1-to-1 matching, preserves fate branches rather than just cluster structure.
