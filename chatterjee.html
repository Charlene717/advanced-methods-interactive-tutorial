Chatterjee 相關係數 (ξ)

2021 年 Sourav Chatterjee 提出的新型秩相關係數——能偵測 Pearson 與 Spearman 看不到的非單調依賴關係，O(n log n) 時間複雜度，已成為 scRNA-seq GRN 推論的新利器。

A 2021 rank-based correlation by Sourav Chatterjee — detects non-monotonic dependencies that Pearson/Spearman miss, runs in O(n log n), and has become a new tool for scRNA-seq GRN inference.

動機

為什麼 Pearson / Spearman 不夠用？

傳統相關係數對 單調關係（Spearman）或 線性關係（Pearson）才敏感。但 scRNA-seq 裡基因–基因關係常常是非單調的——例如某個轉錄因子在中等表現量時最強、過高反而抑制下游基因；或細胞週期相關基因隨假時間呈正弦波振盪。Pearson/Spearman 對這類關係幾乎是「盲的」（係數接近 0）。

Chatterjee 的 ξ（讀作 xi）填補了這個缺口：它 對任意函數依賴都敏感，且只比排序多花一點時間。

Classical correlation responds only to monotonic (Spearman) or linear (Pearson) relationships. But gene–gene relationships in scRNA-seq are often non-monotonic — a TF may peak at moderate expression and repress downstream targets at high levels; cell-cycle genes oscillate sinusoidally along pseudotime. Pearson/Spearman are essentially blind to such patterns (≈ 0).

Chatterjee's ξ (xi) fills this gap: it is sensitive to any functional dependency, with only slightly more cost than sorting.

不對稱

ξ(X→Y) ≠ ξ(Y→X) provides directional information — useful for regulatory direction.

演算法

ξ 怎麼算？

  1. 排序：把 (Xi, Yi) 依 X 由小到大排序，得到對應的 Y 序列。
  2. 取秩：計算 Y 的秩 r1, r2, ..., rn（依排序後的順序）。
  3. 相鄰差：計算 |ri+1 − ri| 的總和。
  4. 標準化：ξn(X,Y) = 1 − 3 · Σ|ri+1 − ri| / (n² − 1)

解讀：如果 X 完全決定 Y（Y = f(X)），把資料依 X 排序後，相鄰的 Y 秩會非常接近 → Σ|Δr| 很小 → ξ ≈ 1。反之獨立時相鄰秩跳躍劇烈 → ξ ≈ 0。

  1. Sort: Order (Xi, Yi) pairs by X ascending; record the corresponding Y values.
  2. Rank: Compute ranks r1, r2, ..., rn of the sorted Y sequence.
  3. Adjacent diffs: Sum |ri+1 − ri|.
  4. Normalize: ξn(X,Y) = 1 − 3 · Σ|ri+1 − ri| / (n² − 1)

Intuition: If X fully determines Y (Y = f(X)), sorting by X makes adjacent Y ranks close → Σ|Δr| small → ξ ≈ 1. If independent, adjacent ranks jump wildly → ξ ≈ 0.

數學性質：當 n → ∞，若 X、Y 獨立則 ξ → 0；若 Y 為 X 的可測函數則 ξ → 1。對連續變數，獨立檢定的漸近分布為 √n · ξ → N(0, 2/5)，可直接做假設檢定。
Theoretical properties: As n → ∞, ξ → 0 iff X, Y are independent; ξ → 1 iff Y is a measurable function of X. For continuous variables, the null asymptotic distribution is √n · ξ → N(0, 2/5), enabling direct hypothesis testing.

🌳 什麼時候用 ξ？

情境1:
非線性 GRN 推論——轉錄因子–標靶基因常有非單調關係，ξ 能找到 Pearson 漏掉的真實調控。
情境2:
振盪基因偵測——細胞週期、晝夜節律基因隨時間呈正弦變化，ξ 直接捕捉。
情境3:
調控方向推估——比較 ξ(TF→target) 與 ξ(target→TF)，較大者可能為「因」端（謹慎解釋）。
注意:
對單調訊號 ξ 的「功率」（statistical power）比 Spearman 略低——所以不是要取代，是要 互補。常見做法是 max(|Pearson|, |Spearman|, ξ) 或加權整合。
Case 1:
Non-linear GRN inference — TF–target relationships are often non-monotonic; ξ finds true regulations missed by Pearson.
Case 2:
Oscillatory gene detection — cell cycle and circadian genes vary sinusoidally; ξ captures them directly.
Case 3:
Direction inference — compare ξ(TF→target) vs ξ(target→TF); larger value suggests the "cause" side (interpret with care).
Caveat:
For monotonic signals, ξ has slightly less power than Spearman — it's complementary, not a replacement. Combine via max(|Pearson|, |Spearman|, ξ) or ensemble.
生物應用

在 scRNA-seq / 多體學的應用

GRN 推論 (2025)

Liu 等人 (2025, bioRxiv) 提出基於 Chatterjee ξ 的 GRN 推論框架，利用 ξ 的不對稱性建立 有方向 的調控網路，並設計多重檢定框架處理 scRNA-seq 非獨立樣本的統計問題。在模擬與真實資料上勝過多個 state-of-the-art 方法。

Liu et al. (2025, bioRxiv) introduce a GRN framework using Chatterjee's ξ, exploiting its asymmetry to build directed regulatory networks. They develop a multiple-testing framework for the non-IID nature of scRNA-seq, outperforming SOTA methods on simulated and real data.

細胞–細胞通訊

在 ligand–receptor 共表達分析中，ξ 對「閾值依賴」型的訊號（受體只在配體達某閾值才啟動）比 Pearson 更敏感。

For ligand–receptor co-expression with threshold-dependent signaling (receptor activates only above a ligand threshold), ξ outperforms Pearson.

批次效應檢測

用 ξ 檢驗 batch label 與 PC 的依賴性——即使是非線性偏移也能偵測，比經典 ANOVA 更靈敏。

Test dependency between batch labels and PCs with ξ — catches non-linear shifts that classical ANOVA misses.

多體學模態間關聯

在 multi-omics 整合（RNA × ATAC × protein）中，ξ 適合衡量染色質開放性與基因表現的非單調對應關係。

In multi-omics integration (RNA × ATAC × protein), ξ captures non-monotonic correspondence between chromatin accessibility and gene expression.

深入解析

深入解析 ξ：陷阱、變體、效應量解讀

① 效應量怎麼讀？

Chatterjee ξ 的範圍是 [0, 1]（樣本估計值可能略小於 0 或在獨立時呈 N(0, 2/(5n))，但通常以 0 為參考）。和 Pearson r 不同的是，ξ 沒有「正/負相關」之分——它只回答「Y 在多大程度上由 X 決定？」這個尺度的經驗詮釋：

  • ξ < 0.05：幾乎獨立；可能因樣本噪音呈現非零。
  • 0.05 ≤ ξ < 0.15：弱依賴；scRNA-seq 中對應「微弱共表達」。
  • 0.15 ≤ ξ < 0.4：中等依賴；很多真實 TF–target 關係落在這個區間（受 dropout、bursty expression 影響）。
  • 0.4 ≤ ξ < 0.7：強依賴；具有明確函數結構，值得進一步驗證。
  • ξ ≥ 0.7：近乎決定性關係；在 scRNA-seq 罕見，常為共因（同一複合體成員）或技術 artefact。

② Tie（相同值）的處理

原始 ξ 在 Y 有大量平局時偏低估。scRNA-seq 因 dropout 與整數 UMI counts 平局極多，建議使用 Chatterjee–Lin (2022) 提出的「ties-aware」變體：用 mid-rank 並調整正則項 ℓᵢ = Σⱼ I(Yⱼ ≥ Yᵢ)。R 套件 XICOR 從 v0.4.1 起內建 ties=TRUE 旗標處理。

③ p-value 與多重檢定

連續變數無 ties 時：√n · ξ → N(0, 2/5)，可直接用 z 檢定。

有 ties 時：變異數變大（最壞時可達 0.5），須用 permutation。實務上，做基因–基因 GRN 推論時樣本數 n 大（細胞數通常 ≥ 500），permutation 太貴，建議 (a) 用「Asymptotic with ties correction」(b) 對所有 pair 跑 Benjamini-Hochberg FDR 控制（q < 0.05），不只是單一 p-value。

④ 不對稱性的真實意涵

ξ(X→Y) 衡量「給定 X，Y 有多確定」；ξ(Y→X) 衡量「給定 Y，X 有多確定」。理論上：若 Y 是 X 的函數但 X 不是 Y 的函數（例如多對一映射），ξ(X→Y) > ξ(Y→X)。但 不對稱性≠因果方向——它只反映「資訊量」的不對稱。要嚴謹推因果還是要回到 Ch 14 的干預資料。

⑥ Power-vs-noise 圖譜（ξ 的弱點）

Chatterjee ξ 的統計檢力（power）在純線性 + 高斯噪音情境下「不如」Pearson/Spearman——這是已被 Shi, Drton, Han (2022) 數學嚴格證明的。實務上 ξ 在「弱線性 + 大樣本」的情境會給出偏小的值，並非 ξ 算錯，而是它根本不為這個情境設計。建議：用 max(|r|, ξ) 或加權整合作為 robust 量測。

① How to read the effect size?

Chatterjee ξ ranges [0, 1] (sample estimate may slightly dip below 0; under independence it is ~N(0, 2/(5n))). Unlike Pearson r there is no positive/negative direction; ξ only answers "to what extent does X determine Y?" Empirical interpretation:

  • ξ < 0.05: nearly independent; may appear non-zero due to noise.
  • 0.05 ≤ ξ < 0.15: weak dependence; in scRNA-seq this often reflects mild co-expression.
  • 0.15 ≤ ξ < 0.4: moderate; many real TF–target relationships fall here (limited by dropout, bursty expression).
  • 0.4 ≤ ξ < 0.7: strong; well-defined functional structure, worth follow-up.
  • ξ ≥ 0.7: nearly deterministic; rare in scRNA-seq, often complex co-membership or technical artefact.

② Handling ties

The original ξ underestimates when Y has many ties — scRNA-seq has plenty due to dropouts and integer UMIs. Use the Chatterjee–Lin (2022) ties-aware variant with mid-rank and adjusted ℓᵢ = Σⱼ I(Yⱼ ≥ Yᵢ). R package XICOR ≥ v0.4.1 supports ties=TRUE.

③ p-value and multiple testing

No ties + continuous: √n · ξ → N(0, 2/5), z-test directly.

With ties: variance grows (up to 0.5 worst case); use permutation. For pair-wise GRN over many cells, permutation is too expensive — use asymptotic-with-ties-correction and apply BH-FDR (q < 0.05) across all gene pairs.

④ What asymmetry really means

ξ(X→Y) measures "how determined is Y given X"; ξ(Y→X) the reverse. If Y is a many-to-one function of X, ξ(X→Y) > ξ(Y→X). But asymmetry ≠ causal direction — it only reflects information asymmetry. Strict causal inference still requires interventional data (see Ch 14).

⑥ The power-vs-noise trade-off (ξ's weakness)

ξ's statistical power in pure linear + Gaussian noise is provably lower than Pearson/Spearman (Shi, Drton, Han 2022). For "weak linear + large sample," ξ may produce values that look small — not wrong, but ξ wasn't designed for that. Recommendation: use max(|r|, ξ) or weighted ensemble as a robust measure.

詳細已知應用

ξ 在生物資訊的具體案例

🧬 案例 1：Chatterjee-GRN（Liu et al. 2025, bioRxiv）

這是目前最完整的 scRNA-seq ξ 應用。他們的貢獻：

  • 核心創新：利用 ξ 的不對稱性建構「有方向」的 GRN，而 Pearson/MI/dCor 都只能給出無向邊。
  • 多重檢定框架：處理 scRNA-seq 非獨立觀測（細胞共享技術因子），提出「resampling-based calibration」估計實際 null 分布——避免標準 BH-FDR 在相關樣本下失效。
  • 實證結果：在 BEELINE benchmark（涵蓋 7 個模擬與 5 個真實資料集）上 AUPR/AUROC 都勝過 GENIE3、PIDC、SCODE 等 state-of-the-art。在 mouse embryonic stem cell (mESC) 資料上發現多個被傳統方法漏掉的真實 TF–target 對，後續經 Perturb-seq 驗證。
  • 計算可行：33,000 細胞 × 5000 基因約 8 分鐘（24 cores）。

🧪 案例 2：Damásio et al. (2024) 神經學基因網路

用 ξ 找 Alzheimer's brain 中與認知功能相關的非線性 gene–gene 關係。Pearson 找出 ~300 個 hub gene；ξ 額外找出 ~80 個 hub，多數為早期警告基因（與 tau 蛋白聚集相關但表現非單調）。

📊 案例 3：Lin & Han (2023)：spike-and-slab gene selection

把 ξ 整合進高維貝氏變數選擇——以 ξ 的單調轉換作為先驗的權重。在乳癌微陣列資料上比 Lasso、Elastic Net 選出更少但更具生物意義的基因。

🌍 案例 4：跨領域驗證

  • 氣候科學：Mehta et al. (2023) 用 ξ 找 ENSO 與全球溫度的非單調滯後關係，揭示 Pearson 看不到的非線性遠端關聯。
  • 金融：Wang & Tsay (2022) 用 ξ 偵測股票回報的「閾值依賴」（在某波動度以上才出現相關）。
  • 單細胞 fLuc 螢光：2024 年數個論文用 ξ 在 live-cell imaging 軌跡資料找週期性表現基因，比傳統 cosinor analysis 更穩健。

🧬 Case 1: Chatterjee-GRN (Liu et al. 2025, bioRxiv)

The most complete scRNA-seq ξ application to date:

  • Core innovation: exploit ξ's asymmetry to build directed GRNs — Pearson/MI/dCor only give undirected edges.
  • Multiple-testing framework: resampling-based calibration handles scRNA-seq's non-IID observations (cells share technical factors), avoiding standard BH-FDR's failure under dependence.
  • Empirical results: on BEELINE benchmark (7 simulated, 5 real datasets), AUPR/AUROC beat GENIE3, PIDC, SCODE. In mESC data, recovered TF–target pairs missed by prior methods — later validated by Perturb-seq.
  • Tractable: 33,000 cells × 5000 genes in ~8 min (24 cores).

🧪 Case 2: Damásio et al. (2024) Neural Gene Networks

Used ξ to find non-linear gene–gene relationships tied to cognition in Alzheimer's brain. Pearson found ~300 hubs; ξ added ~80, most being early-warning genes (non-monotonic with tau aggregation).

📊 Case 3: Lin & Han (2023) — spike-and-slab gene selection

Embedded ξ into high-dim Bayesian variable selection as prior weight. On breast cancer microarray data, selected fewer but more biologically meaningful genes than Lasso / Elastic Net.

🌍 Case 4: Cross-domain validations

  • Climate science: Mehta et al. (2023) used ξ for non-monotonic lagged ENSO–temperature relations, revealing nonlinear teleconnections.
  • Finance: Wang & Tsay (2022) detected threshold-dependent stock returns (correlation appears only above some volatility level).
  • Live-cell imaging: several 2024 papers used ξ on fluorescent live-cell trajectories to find oscillatory genes, more robust than classical cosinor.
潛在改良應用

ξ 還能改良哪些單細胞與多體學任務？

🎯 1. 找到傳統 DE 漏掉的「非線性 marker gene」

Seurat::FindMarkers 用 Wilcoxon 假設「兩群之間表現分布有 shift」。但有些 marker 在某 cluster 內部呈雙峰（subpopulation）、在其他 cluster 呈單峰——這種「distribution shape 差異」Wilcoxon 看不到。提案：用 ξ(cluster_label, gene_expression) 找出與群集 label「強函數依賴」的基因，能補足傳統 DE 漏掉的 marker。

🎯 2. 軌跡 (pseudotime) 中的振盪基因偵測

細胞週期、晝夜節律、發炎週期的基因表現沿 pseudotime 呈正弦曲線。Pearson(pseudotime, gene) ≈ 0 因為正負相消；ξ(pseudotime, gene) 反而看得很清楚。提案：取代或補充 tradeSeq/Monocle 的線性 GAM-based 振盪偵測，特別是在 cycle following 章節描述的循環軌跡上。

🎯 3. Ligand–Receptor 閾值偵測

CellChat、CellPhoneDB 用 expression product 或 mean 篩 L–R 對。但很多 L–R 關係是「閾值型」（受體只在 ligand 達某水準才啟動），用 ξ 量「ligand → downstream signaling gene」的依賴比平均值乘法更敏感。提案：建構 ξ-based L–R 偵測流程，與 CellChat 整合作為「下游驗證」階段。

🎯 4. 多體學「鬆耦合」對應偵測

scRNA + scATAC 配對細胞時，染色質開放性與表現量常呈「on/off」型關係（peak 開了 ⇒ 表現可能上升也可能不變）。Pearson 看不到，但 ξ(ATAC peak, gene expression) 能找到這類「條件式」對應。提案：取代/補充 Cicero、ArchR 的線性 link 偵測。

🎯 5. 空間轉錄體的非線性空間模式

SpatialDE 等假設 Gaussian Process kernel；ξ(spatial_coord_x, gene) 與 ξ(spatial_coord_y, gene) 能直接偵測「依距離有結構但非高斯」的基因（例如多焦點 hotspot）。提案：新的 SVG 偵測管線，特別適合 Stereo-seq 等高解析度資料。

🎯 6. scLLM embedding 評估

scGPT/Geneformer 預訓練後產生的 latent 維度有不同「語意」。用 ξ(latent_dim_i, gene_j) 可量化每個 latent 維度與「具體基因表現」的依賴強度，比 Pearson 更能找到非線性編碼的維度。提案：scLLM 可解釋性分析工具。

🎯 7. 藥物劑量–反應曲線分析

藥物在低劑量可能上調基因 A，高劑量轉為下調（U 型或鐘型曲線）。傳統劑量–效應分析用 Hill function fit，要先選模型；ξ(dose, gene) 對任何非線性形狀都敏感，可作為「篩選哪些基因需要進一步劑量曲線建模」的步驟。提案：整合進 Connectivity Map / LINCS L1000 pipeline。

🎯 8. Perturb-seq 效應分析

Perturb-seq 通常看「擾動 A 後 B 表現是否改變」。但有些擾動效應是「重塑表現變異度」而非「改變均值」——例如敲除某 chromatin remodeler 後，某基因表現分布變寬。提案：ξ(perturbation_label, gene) 對均值與變異度的改變都敏感，可作為「broader」的擾動效應偵測。

🎯 9. 細胞型別決定基因（fate-determining genes）

分化過程中，「決定終態」的基因在初期表現可能與終態 label 不單調相關（先升後降）。ξ(early_expression, final_celltype) 比 Pearson/Spearman 更能找到這類 fate-determining 基因。提案：與 RNA velocity / Waddington-OT 整合，找出真實的 fate gates。

🎯 10. 跨物種保守關係偵測

跨物種比較基因網路時，物種間的調控強度可能變但「結構」（依賴形式）保留。用 ξ 量「人 vs 鼠」中同源基因對的依賴一致性，比 Pearson 更穩健於物種間 scale 差異。提案：跨物種 GRN 保守模塊分析。

🎯 1. Find non-linear marker genes missed by classical DE

Seurat::FindMarkers (Wilcoxon) assumes a shift between groups. But some markers are bimodal in one cluster, unimodal in others — Wilcoxon misses such distribution-shape differences. Proposal: use ξ(cluster_label, gene_expression) to find genes with strong functional dependence on group label, complementing standard DE.

🎯 2. Detect oscillatory genes along pseudotime

Cell-cycle, circadian, inflammatory cycle genes are sinusoidal along pseudotime → Pearson ≈ 0 (cancels). ξ catches them. Proposal: replace or complement tradeSeq/Monocle's GAM-based oscillator detection, especially on the circular trajectories from Ch 8.

🎯 3. Threshold-dependent L–R detection

CellChat/CellPhoneDB use expression products. But many L–R interactions are threshold-type (receptor activates only above some ligand level). ξ on "ligand → downstream signaling gene" is more sensitive than mean × mean. Proposal: ξ-based L–R pipeline, integrated with CellChat as downstream validation.

🎯 4. Loosely-coupled multi-omics correspondences

For paired RNA + ATAC, chromatin accessibility and expression often have "on/off" relationships (peak open ⇒ expression may or may not rise). Pearson misses this; ξ(ATAC peak, gene expression) catches it. Proposal: replace/complement linear link detection in Cicero, ArchR.

🎯 5. Non-linear spatial patterns

SpatialDE assumes a Gaussian Process kernel; ξ(spatial_x, gene) and ξ(spatial_y, gene) directly detect "structured but non-Gaussian" genes (e.g. multi-focal hotspots). Proposal: a new SVG detection pipeline well-suited to Stereo-seq's high resolution.

🎯 6. scLLM embedding evaluation

scGPT/Geneformer latents have different "semantics" per dimension. ξ(latent_dim_i, gene_j) quantifies the dependence of each latent on specific genes — finds non-linearly encoded dimensions better than Pearson. Proposal: scLLM interpretability tool.

🎯 7. Drug dose–response analysis

A drug may upregulate gene A at low dose, downregulate at high dose (U- or bell-shaped). Classical analysis fits Hill functions, requiring model selection upfront. ξ(dose, gene) is sensitive to any non-linear shape — good first-pass screen. Proposal: integrate into Connectivity Map / LINCS L1000 pipelines.

🎯 8. Perturb-seq effect analysis

Perturb-seq usually asks "after perturbing A, did B's expression change?" But some effects reshape variance, not mean (e.g. knocking out a chromatin remodeler widens a gene's distribution). ξ(perturbation_label, gene) catches both kinds. Proposal: broader perturbation effect detector.

🎯 9. Fate-determining gene discovery

During differentiation, fate-determining genes may have non-monotonic relationships with terminal labels (rise then fall). ξ(early_expression, final_celltype) finds them better than Pearson/Spearman. Proposal: integrate with RNA velocity / Waddington-OT to find true fate gates.

🎯 10. Cross-species conserved relationships

Strength of regulation varies across species but the structure (dependence form) is conserved. ξ between homolog pairs in human vs mouse is more robust to scale differences than Pearson. Proposal: conserved module analysis in cross-species GRNs.

# Pure NumPy — for understanding
import numpy as np
from scipy.stats import rankdata

def chatterjee_xi(x, y):
    n = len(x)
    # 1. Sort by x; keep corresponding y
    order = np.argsort(x)
    y_sorted = y[order]
    # 2. Compute ranks of y in sorted order
    r = rankdata(y_sorted, method="average")
    # 3. Sum adjacent absolute differences
    s = np.sum(np.abs(np.diff(r)))
    # 4. Normalize
    xi = 1 - 3 * s / (n**2 - 1)
    return xi
