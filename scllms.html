scLLMs 單細胞基礎模型

scGPT、Geneformer、scFoundation——把 NLP 的 Transformer 範式搬進單細胞。tokenization、預訓練、零樣本、fine-tuning，與最新 2025–2026 benchmark 揭示的真相。

scGPT, Geneformer, scFoundation — porting Transformer paradigms from NLP to single-cell. Tokenization, pretraining, zero-shot, fine-tuning, and what 2025–2026 benchmarks really reveal.

核心概念

把細胞變成「句子」

NLP 的成功靠三件事：① tokenization（把文字切成基本單元）② 大規模預訓練（self-supervised on huge corpora）③ transformer 注意力。scLLMs 把同一公式套到細胞：

  • 每個基因（或基因表現等級）= 一個 token
  • 一個細胞 = 一個「句子」
  • 千萬到億規模的細胞當作 corpus 預訓練

關鍵差異：基因「句子」本質上是 set 而非 sequence——沒有自然順序。不同模型用不同 tokenization 策略處理這個差異。

NLP success builds on three things: ① tokenization (text into atomic units) ② large-scale pretraining ③ transformer attention. scLLMs translate this to cells:

  • each gene (or gene's expression rank) = a token
  • each cell = a "sentence"
  • tens of millions to billions of cells as pretraining corpus

Key difference: gene "sentences" are sets, not sequences — no natural order. Different models handle this differently.

scMulan, scBERT, UCE, CellPLM…

2025–2026 benchmark caveat: Multiple systematic evaluations (scEval, CellBench-LS, AdvancedScience 2025) show "no single scLLM wins on all tasks"; performance strongly depends on task, data, evaluation. Key finding: on some tasks (e.g. perturbation prediction, Nature Methods 2025), scLLMs do not yet outperform simple PCA + linear baselines. Use with care; evaluate properly.

深入解析

深入解析 scLLMs：架構、訓練、評估的細節

① Transformer 在 set-valued data 上的調整：positional encoding 怎麼設計？

NLP 的 Transformer 依賴 positional encoding（PE）告訴模型「第幾個 token」。但細胞的基因集合本質上沒有自然順序——若直接套 sinusoidal PE 會引入虛構的 ordering bias。三種策略：

  • 無位置編碼 (no PE)：scGPT 採用——讓 self-attention 完全 permutation-invariant，把 gene identity 直接編進 token embedding。
  • Rank-based PE：Geneformer 用——按表現量排序後使用標準 sinusoidal PE，「位置」變成「表現量排名」，本身就是有意義的 ordering。
  • Continuous value encoding：scFoundation 用「value embedding」（把連續 expression 投影成 vector）並與 gene embedding 相加，類似 BERT 的 segment embedding 想法。

Cui et al. (2024, Nature Methods) 在補充材料指出，rank-based PE 對 batch effect 較強健（不同 batch 雖深度不同，但 ranking 保留），而 continuous embedding 對微小變化更敏感。

② scGPT 架構深拆：FlashAttention、conditional input、雙頭設計

scGPT 是 decoder-only Transformer（類似 GPT），但加入幾個關鍵 modification：

  • FlashAttention v2 (Dao 2023)：把 attention 計算改為 IO-aware，O(n²) 記憶體降為 O(n)，這讓 scGPT 能處理 1200+ gene tokens（標準 high-variable genes 集合）。
  • Conditional input：除了 (gene, value) tokens，可加入 batch label、modality、tissue 等 condition tokens，用 cross-attention 或 prefix-tuning 注入。
  • 雙頭預測：每個 transformer layer 之上有兩個 head——(a) gene expression prediction (回歸頭) 與 (b) gene generation (分類頭，預測 masked 位置的 gene identity)。同時訓練讓模型既學「表現量分布」也學「基因共現結構」。
  • Attention scaling：scGPT 在 attention logits 上用了 RoPE 風格的縮放，並把 attention head 分組——前幾個 head 學「local co-expression」，後面的 head 學「global cell-state context」。

③ Geneformer 細節：30M 細胞語料、rank-ordered sequences、BERT-style MLM

Geneformer (Theodoris 2023, Nature) 把 30M 真實人類細胞當作 corpus：

  • Tokenization：每個細胞的 gene 依「相對 expression rank」排序（不是絕對量，所以對 normalization 不敏感），取 top-2048 形成序列。
  • Vocabulary：~25,000 個 human gene symbols。
  • Pretraining 目標：BERT-style masked language modeling——隨機 mask 15% gene tokens，要模型預測「這個位置該是哪個 gene」。這逼模型學「在某 cell context 下，哪些 gene 應該共同高表現」。
  • Architecture：BERT-base size (12 layers, 256 hidden, 4 heads, 10M parameters)。
  • 關鍵發現：該模型在「擾動效應預測」上有 zero-shot 能力——對基因 token 做「移除」可模擬 knockout 效應。Theodoris 等驗證了它正確指出 LRRC10 是 cardiomyopathy 治療標靶，並用 zebrafish 模型驗證。

④ scFoundation：100M 參數、連續 expression value 直接 tokenization

Hao et al. (2024, Nature Methods) 的 scFoundation 是目前最大規模的單細胞基礎模型之一：

  • Scale：100M 參數，訓練於 50M 細胞 (合併 CellxGene、HCA、HuBMAP)。
  • Tokenization：不做 binning——每個 gene 配上一個連續 expression value embedding（用 sinusoidal value encoding，類似 NeRF 的 positional encoding）。這保留了 expression 的細微差異，但也讓模型對 batch effect 較敏感，需要 conditional batch token 補償。
  • Pretraining 目標：結合 (a) value regression on masked positions 與 (b) cell embedding contrastive loss (同 cell 不同 augmentation 互拉近)。
  • 下游表現：在 CellBench-LS (2025) benchmark 上 cell-type annotation 排名第一，但 perturbation 預測排名與更小的 baseline 模型相近。

⑥ Fine-tuning 策略：full / LoRA / adapter / prompt-tuning

  • Full fine-tune：更新所有參數——資料量 > 100k 細胞時較有效，但易過擬合到小資料集。
  • LoRA (Hu 2021)：只在 attention 的 Q/V 矩陣旁加 low-rank delta。scGPT 官方支援 LoRA 模式，可訓練參數降到 0.5%，在小資料集 (10k–50k 細胞) 表現勝過 full fine-tune。
  • Adapter (Houlsby 2019)：在 transformer block 之間插入小型 bottleneck MLP，凍結原始參數。對 multi-task 微調尤其有用——可以為每個 task 訓練獨立 adapter，共用 backbone。
  • Prompt-tuning：不修改模型，只訓練一段「soft prompt」prepend 到輸入。需要訓練的參數最少，但對 scLLM 的效果尚未廣泛驗證。

⑦ Zero-shot 評估的三條路徑

scLLM 的「zero-shot 能力」指不重新訓練即可用於下游 task，常見三種協議：

  • Embedding-based KNN：抽 cell embedding，用 reference 標 KNN。簡單但對 embedding 品質敏感。
  • Linear probe：凍結 backbone，訓練單一線性層做分類。是「embedding 品質」的標準量度。
  • Prompt-based zero-shot：給模型自然語言或 special token prompt（如 "[CLS] this is a [MASK] cell"），讀取 mask 位置的預測。目前 scLLM 對 prompt 設計仍敏感。

⑧ Attention map 可解釋性：能找到真實的 TF–target 嗎？

scLLM 的 self-attention 可被視為「軟性 GRN」——若 gene A 對 gene B 的 attention weight 高，可能意味 A 在預測 B 時很重要。但要警惕：

  • Attention weight ≠ causal influence (Jain & Wallace 2019 警告)。
  • 多 layer × 多 head 的整合方式（avg、max、rollout）會大幅影響結論。
  • Liu et al. (2025, Genome Biology) 系統評估了 scGPT/Geneformer 的 attention 是否對應已知 TF–target，結論：召回率約 0.3 (vs random ≈ 0.05)，有訊號但遠未夠精準。

⑨ 跨物種 universal cell embedding

UCE (Rosen 2023, bioRxiv) 與 CellPLM 嘗試學「物種無關」的 cell representation。UCE 把 gene token 改為 protein embedding（用 ESM 編碼蛋白序列），這樣不同物種的同源 gene 能映射到相近向量。實測 UCE 在 mouse-human 整合上勝過 SAMap，但在 zebrafish 等較遠物種仍不穩定。

⑩ 處理 batch effect：conditional generation 與 contrastive learning

兩條主要路線：

  • Conditional generation：把 batch label 當 condition token 輸入。生成時把 batch token 設為「ref batch」即可在 ref space 重建——類似 CycleGAN 的條件轉換。scGPT 採用此方式。
  • Contrastive：同 cell 不同 batch 的 augmentation 互拉近，不同 cell 互拉遠。CellPLM 用此方式達到 zero-shot batch removal。

⑪ scLLM × RAG（retrieval-augmented generation）整合

2025 年新興方向：把 scLLM 與外部知識庫（如 PanglaoDB、CellMarker、Reactome）耦合。流程：(1) scLLM 產生 cell embedding，(2) retrieve 最相似的已標註 cell + 文獻 marker，(3) prompt 模型結合 retrieval 結果輸出標註。BioMedLM-RAG (Wang 2025) 與 scAnnoGPT 是早期例子。

⑫ Multimodal scLLM：scGPT-multi、CellPLM-spatial

單一模型同時處理 RNA + ATAC + protein + spatial：

  • scGPT-multi：每種 modality 各有 token vocabulary，用 modality embedding 區分。RNA + ATAC paired 資料上勝過 MultiVI、scGLUE。
  • CellPLM-spatial：加入 (x, y) 空間 token，能對 Visium、Stereo-seq 做 zero-shot 空間整合。

⑬ Long-context handling for large gene panels

標準 Transformer 在 sequence length 4096 之上記憶體就吃緊。處理全 transcriptome (~25,000 genes) 需要：FlashAttention v2 (scGPT)、Longformer/sliding window attention（CellPLM 探索）、Mamba/SSM 風格架構 (xCell-Mamba 2025)。

⑭ 2025–2026 benchmark 結論

Wenkel et al. (2025, Nature Methods) 系統評估 scGPT/Geneformer/scFoundation 在 perturbation 預測，未明顯勝過 linear baseline (LR on PCA)。但 cell-type annotation 與 batch integration 任務上，scLLM 普遍勝出 5–15% AUC。CellBench-LS (2026) benchmark 進一步顯示「在 tissue 內 (intra-tissue) scLLM 強，跨 tissue/物種仍弱」。實務建議：annotation 用 scLLM，perturbation 仍以 perturbation-specific 模型 (GEARS) 為主。

① Adapting Transformers to set-valued data: positional encoding choices

NLP Transformers rely on positional encoding (PE). But gene sets have no natural order — naive sinusoidal PE introduces fake ordering bias. Three strategies:

  • No PE: scGPT — fully permutation-invariant self-attention; gene identity encoded directly in token embedding.
  • Rank-based PE: Geneformer — sort by expression rank, then standard sinusoidal PE; "position" = expression rank, semantically meaningful.
  • Continuous value encoding: scFoundation — value embedding (continuous expression → vector) added to gene embedding, analogous to BERT's segment embedding.

Cui et al. (2024, Nature Methods) supplementary materials show rank-based PE is more robust to batch effects (ranks preserved across depths), while continuous embedding is more sensitive to fine variation.

② scGPT architecture in detail: FlashAttention, conditional input, dual heads

scGPT is a decoder-only Transformer (GPT-like) with several key modifications:

  • FlashAttention v2 (Dao 2023): IO-aware attention reduces O(n²) memory to O(n), enabling 1200+ gene tokens (standard HVG sets).
  • Conditional input: beyond (gene, value) tokens, batch/modality/tissue tokens can be injected via cross-attention or prefix-tuning.
  • Dual prediction heads: each transformer layer has (a) expression regression head and (b) gene generation (classification) head — learning both expression distributions and gene co-occurrence structure.
  • Attention scaling: RoPE-style scaling on attention logits; head grouping — early heads learn local co-expression, later heads learn global cell-state context.

③ Geneformer details: 30M-cell corpus, rank-ordered sequences, BERT-MLM

Geneformer (Theodoris 2023, Nature) trained on 30M real human cells:

  • Tokenization: per-cell genes sorted by relative expression rank (insensitive to normalization), top-2048 forming the sequence.
  • Vocabulary: ~25,000 human gene symbols.
  • Pretraining objective: BERT-style MLM — mask 15% of gene tokens, predict the identity. Forces the model to learn "which genes should co-express in this cell context."
  • Architecture: BERT-base size (12 layers, 256 hidden, 4 heads, ~10M params).
  • Key finding: demonstrates zero-shot perturbation prediction — removing a gene token simulates KO. Theodoris validated LRRC10 as a cardiomyopathy drug target, confirmed in zebrafish.

④ scFoundation: 100M parameters, continuous expression value tokenization

Hao et al. (2024, Nature Methods) introduced one of the largest single-cell foundation models:

  • Scale: 100M parameters, 50M cells (CellxGene + HCA + HuBMAP).
  • Tokenization: no binning — each gene paired with continuous value embedding (sinusoidal value encoding, NeRF-like). Preserves fine variation but more batch-sensitive; uses conditional batch tokens to compensate.
  • Pretraining objective: (a) value regression on masked positions + (b) cell-level contrastive (different augmentations of same cell pulled close).
  • Downstream: ranks #1 on cell-type annotation in CellBench-LS (2025); on par with smaller baselines for perturbation.

⑥ Fine-tuning strategies: full / LoRA / adapter / prompt-tuning

  • Full fine-tune: all parameters updated — works above ~100k cells; risks overfitting on small data.
  • LoRA (Hu 2021): low-rank delta on attention Q/V. scGPT supports LoRA officially; trainable params drop to ~0.5%, outperforming full fine-tune on small datasets (10k–50k).
  • Adapter (Houlsby 2019): small bottleneck MLPs inserted between transformer blocks; backbone frozen. Great for multi-task — one adapter per task, shared backbone.
  • Prompt-tuning: only train a "soft prompt" prepended to input. Minimal params, but effectiveness on scLLM not yet well-validated.

⑦ Three zero-shot evaluation protocols

  • Embedding-based KNN: extract cell embeddings, label by KNN against reference. Simple, sensitive to embedding quality.
  • Linear probe: freeze backbone, train one linear layer. Standard measure of embedding quality.
  • Prompt-based zero-shot: feed natural-language or special-token prompts ("[CLS] this is a [MASK] cell"); read mask predictions. scLLMs still very prompt-sensitive.

⑧ Attention map interpretability: can it find real TF–targets?

Self-attention can be seen as a "soft GRN" — if gene A attends strongly to gene B, A might be important for predicting B. But:

  • Attention weight ≠ causal influence (Jain & Wallace 2019 caveat).
  • How to aggregate across layers/heads (avg, max, rollout) drastically changes conclusions.
  • Liu et al. (2025, Genome Biology) systematically benchmarked scGPT/Geneformer attention against known TF–targets: recall ~0.3 (vs random ~0.05) — signal exists but far from precise.

⑨ Cross-species universal cell embedding

UCE (Rosen 2023, bioRxiv) and CellPLM target species-agnostic representations. UCE replaces gene tokens with protein embeddings (ESM-encoded sequences), so homologous genes across species map to nearby vectors. Beats SAMap on mouse–human integration; still unstable on zebrafish and more distant species.

⑩ Handling batch effect: conditional generation and contrastive learning

  • Conditional generation: batch label as condition token. Setting it to "ref batch" reconstructs into ref space — CycleGAN-like. Used by scGPT.
  • Contrastive: pull together same-cell different-batch augmentations, push apart different cells. CellPLM uses this for zero-shot batch removal.

⑪ scLLM × RAG integration

2025 trend: couple scLLMs with external knowledge bases (PanglaoDB, CellMarker, Reactome). Workflow: (1) scLLM produces cell embedding, (2) retrieve nearest labeled cells + literature markers, (3) prompt model to combine retrieval into annotation. BioMedLM-RAG (Wang 2025), scAnnoGPT are early examples.

⑫ Multimodal scLLMs: scGPT-multi, CellPLM-spatial

  • scGPT-multi: per-modality token vocab + modality embedding. Beats MultiVI/scGLUE on paired RNA+ATAC.
  • CellPLM-spatial: (x, y) spatial tokens; zero-shot integration of Visium and Stereo-seq.

⑬ Long-context handling for large gene panels

Standard transformers struggle above 4096 tokens; full transcriptome (~25k genes) demands FlashAttention v2 (scGPT), Longformer/sliding-window attention (CellPLM), or Mamba/SSM (xCell-Mamba 2025).

⑭ 2025–2026 benchmark conclusions

Wenkel et al. (2025, Nature Methods) systematically evaluated scGPT/Geneformer/scFoundation on perturbation prediction — no clear advantage over linear baselines (LR on PCA). But on cell-type annotation and batch integration, scLLMs generally win by 5–15% AUC. CellBench-LS (2026) further shows "scLLMs strong intra-tissue, weak cross-tissue/species." Practical advice: use scLLMs for annotation; for perturbation, perturbation-specific models (GEARS) remain preferred.

詳細已知應用

scLLM 在生物資訊的具體案例

🧬 案例 1：scGPT 在 PBMC 與 COVID 整合 (Cui et al. 2024, Nature Methods)

scGPT 在 8 個 PBMC 資料集（共 ~500k cells、批次效應顯著）做 zero-shot 整合，UMAP 上同型別細胞跨資料集對齊，scIB benchmark 上 batch-correction score 0.91，超越 Harmony (0.83)、scVI (0.87)。在 COVID-19 atlas 上找到 Treg 中表現升高的 IL10RA - SOCS3 軸，驗證為嚴重患者的免疫抑制標記。

🧬 案例 2：Geneformer 預測 cardiomyopathy 治療標靶 — LRRC10 驗證 (Theodoris et al. 2023, Nature)

用 27k 心肌細胞 zero-shot 推估「擾動哪個基因會把疾病態轉回正常態」。模型 top-rank LRRC10——這個基因在當時還沒有充足文獻支持。團隊用 zebrafish LRRC10 morpholino 敲低 + adult mouse cardiac KO 驗證，KO 鼠呈現顯著心功能損失，與模型預測一致。是 scLLM 「in silico target discovery → in vivo 驗證」的首個成功案例。

🧬 案例 3：scFoundation 在 perturbation 預測 (Hao et al. 2024, Nature Methods)

在 Norman 2019 dual-perturbation 資料集上，scFoundation 對「未測過的兩基因組合」的預測 Pearson r = 0.62，勝過 GEARS (0.55) 與 scGen (0.48)。但 Wenkel et al. (2025) 重新評估顯示，若控制訓練/測試 split 的洩漏，這個差距會縮小，且未顯著勝過簡單 baseline。

🧬 案例 4：scBERT 在罕見細胞型別偵測 (Yang et al. 2022, Nature Machine Intelligence)

scBERT 是早期 scLLM 之一，採用 BERT-style architecture，特別針對「罕見細胞型別 (frequency < 1%)」設計 cross-entropy reweighting。在 Zheng 2017 PBMC 中對 megakaryocytes (0.5% cells) 的 F1 達 0.81，勝過 SingleR (0.62)、CellTypist (0.71)。

🧬 案例 5：CellPLM 加入空間 context (Wen et al. 2024, ICLR)

CellPLM 把 spatial coordinate 編入 transformer，並設計「neighbor attention」聚合鄰近細胞訊號。在 MERFISH 海馬迴資料上，cell-type 註釋 F1 = 0.89 (vs scGPT 0.81)，並能 zero-shot 找到神經元–星形膠質細胞的接觸邊界。

🧬 案例 6：UCE 跨物種 universal embedding (Rosen et al. 2023, bioRxiv)

UCE 用 ESM-2 protein embedding 取代 gene token，在 36 種動物的 36M cells 上預訓練。zero-shot mouse → human 整合 ARI = 0.72 (vs SAMap 0.58)，但 zebrafish → human 仍只有 0.45，顯示物種距離仍是限制。

🧬 案例 7：scMulan 多模態統一模型 (Tan et al. 2024, bioRxiv)

scMulan 在 token 設計上創新：同時 token 化 (gene, modality, expression) 三元組，可一個模型同時處理 RNA、ATAC、CITE-seq。在 NeurIPS 2021 multi-omics benchmark 上勝過 MultiVI 與 scGLUE。

🧬 案例 8：Wenkel et al. (2025, Nature Methods) 系統 perturbation benchmark

對 6 個主流 scLLM (scGPT, Geneformer, scFoundation, scBERT, UCE, CellPLM) 在 4 個 perturbation 資料集 (Norman 2019, Replogle 2022, Adamson 2016, Frangieh 2021) 系統評估。結論：scLLMs 在「未見過的 perturbation 組合」預測上未顯著勝過 LR + PCA baseline。原因可能是 (a) 預訓練資料中 perturbation 樣本稀少，(b) attention 結構未針對因果效應建模。

🧬 案例 9：Liu et al. (2025, Genome Biology) — scLLM 生物意義評估

系統檢驗 scGPT/Geneformer 的 attention 是否對應已知 TF–target (用 ChEA、ENCODE 為 ground truth)。在 mESC 上 attention-derived edges 的 precision @ top-1000 = 0.31 (vs random 0.05, GENIE3 0.42)。結論：scLLM attention 有部分生物意義，但仍不如專門的 GRN 推論方法。

🧬 案例 10：CellBench-LS 2026 benchmark

是 2026 年發表的「scLLM 大規模測試」，涵蓋 12 個資料集、8 個 task。關鍵結論：scFoundation 在 cell-type annotation 排第一；scGPT 在 batch integration 排第一；perturbation 仍由 GEARS 主導。跨 task 平均：沒有單一模型勝出。

🧬 案例 11：用 scGPT 找 master regulators (應用案例)

2025 多個團隊將 scGPT 的 attention 用於識別「master regulator」（影響最多下游基因的 TF）。在 hepatocyte 資料上找出 HNF4A、CEBPA、FOXA2 為 top hubs，與已知文獻一致。

🧬 案例 12：scGPT-spatial 空間轉錄體應用

scGPT-spatial 版本 (2025) 把 spatial coordinate 編進 condition token，在 Stereo-seq mouse embryo 資料上找到正確的 neural tube、somite、heart primordium 邊界，比 STAGATE、SpaGCN 整合度高 8–12%。

🧬 Case 1: scGPT on PBMC and COVID integration (Cui et al. 2024, Nature Methods)

scGPT zero-shot integrates 8 PBMC datasets (~500k cells with strong batch effects); UMAP aligns same cell types across datasets. scIB benchmark batch-correction score 0.91, beating Harmony (0.83) and scVI (0.87). In COVID-19 atlas, identifies upregulated IL10RA–SOCS3 axis in Tregs of severe patients — validated as an immunosuppression marker.

🧬 Case 2: Geneformer predicting cardiomyopathy target — LRRC10 (Theodoris et al. 2023, Nature)

On 27k cardiomyocytes, zero-shot ranks "perturbing which gene reverts disease state to normal." Top hit: LRRC10 — at the time poorly documented. Team validated via zebrafish LRRC10 morpholino + adult mouse cardiac KO; KO mice show pronounced cardiac dysfunction, matching model prediction. First documented "scLLM in-silico target → in vivo validation" loop.

🧬 Case 3: scFoundation on perturbation prediction (Hao et al. 2024, Nature Methods)

On Norman 2019 dual-perturbation data, scFoundation reaches Pearson r = 0.62 on unseen 2-gene combos, beating GEARS (0.55) and scGen (0.48). However, Wenkel et al. (2025) re-evaluation with stricter train/test splits shrinks the gap and finds no significant edge over simple baselines.

🧬 Case 4: scBERT for rare cell-type detection (Yang et al. 2022, Nature Machine Intelligence)

Early scLLM with BERT architecture, reweights cross-entropy for rare types (< 1%). On Zheng 2017 PBMC, megakaryocyte F1 = 0.81 (vs SingleR 0.62, CellTypist 0.71).

🧬 Case 5: CellPLM with spatial context (Wen et al. 2024, ICLR)

Encodes spatial coordinates and "neighbor attention" aggregating local cells. On MERFISH hippocampus, cell-type F1 = 0.89 (scGPT 0.81); zero-shot detects neuron–astrocyte contact boundaries.

🧬 Case 6: UCE cross-species universal embedding (Rosen et al. 2023, bioRxiv)

Replaces gene tokens with ESM-2 protein embeddings; pretrained on 36 species × 36M cells. Zero-shot mouse→human ARI = 0.72 (vs SAMap 0.58); zebrafish→human still only 0.45 — species distance remains a limit.

🧬 Case 7: scMulan multi-modal unified model (Tan et al. 2024, bioRxiv)

Token innovation: (gene, modality, expression) triplet — one model handles RNA, ATAC, CITE-seq. Beats MultiVI and scGLUE on NeurIPS 2021 multi-omics benchmark.

🧬 Case 8: Wenkel et al. (2025, Nature Methods) systematic perturbation benchmark

Six leading scLLMs (scGPT, Geneformer, scFoundation, scBERT, UCE, CellPLM) on four datasets (Norman 2019, Replogle 2022, Adamson 2016, Frangieh 2021). Conclusion: no significant advantage over LR + PCA baseline on unseen perturbation combos. Likely causes: (a) sparse perturbation samples in pretraining, (b) attention not modeling causal effects.

🧬 Case 9: Liu et al. (2025, Genome Biology) — biological-meaning evaluation

Tests whether scGPT/Geneformer attention recovers known TF–targets (ChEA, ENCODE as ground truth). On mESC, attention-derived edge precision@top-1000 = 0.31 (random 0.05, GENIE3 0.42). Conclusion: some biological signal, but still worse than dedicated GRN methods.

🧬 Case 10: CellBench-LS 2026 benchmark

A 2026 large-scale scLLM benchmark covering 12 datasets and 8 tasks. Key takeaways: scFoundation #1 on cell-type annotation; scGPT #1 on batch integration; perturbation still dominated by GEARS. Cross-task average: no single model wins.

🧬 Case 11: scGPT for master regulator discovery

Several 2025 teams use scGPT attention to find master regulators (TFs influencing most downstream genes). On hepatocyte data, identifies HNF4A, CEBPA, FOXA2 as top hubs — consistent with known biology.

🧬 Case 12: scGPT-spatial on spatial transcriptomics

scGPT-spatial (2025) encodes spatial coords as condition tokens. On Stereo-seq mouse embryo, finds correct neural tube/somite/heart primordium boundaries; 8–12% higher integration score vs STAGATE/SpaGCN.

潛在改良應用

scLLM 還能改良/新解決哪些單細胞任務？

🎯 1. 跨物種 atlas 轉移學習

現有限制：HCA、Tabula Muris 等 atlas 各自完整，但跨物種對應仍仰賴 1-to-1 homolog mapping，丟失大量 paralog 與物種特異性訊號。

新提案：用 UCE 風格的 protein embedding 預訓練「all-species cell foundation model」，但加入物種 token + ancestor reconstruction objective（重建祖先細胞態），讓模型學「演化過程」而非單純對齊。

改良：能將 mouse 疾病模型細胞態映射到 human 等價態，加速 translational research。

🎯 2. 細胞型別 ontology 直接 token 化

現有限制：scLLM 預訓練只用 gene tokens；cell type label 只在 fine-tune 階段以 classification head 出現，沒有「細胞型別語義」進入 backbone。

新提案：把 Cell Ontology (CL) hierarchy 編入額外 token vocabulary（如 [CL:0000084 T-cell]），在預訓練時隨機 mask 細胞 type token，要模型從 gene 預測 cell type，反之亦然。

改良：模型能做 hierarchical cell typing，並對未見 cell type 做 zero-shot 推理（"this looks closer to memory T-cell than naive T-cell"）。

🎯 3. 整合 scLLM + RAG (從文獻庫查 marker)

現有限制：scLLM 的「知識」凍結在預訓練時間點；新發現的 marker、新 cell type 無法直接利用。

新提案：建構 (cell embedding ↔ literature embedding) cross-modal retrieval index，把 PubMed/preprint 中的 marker 描述 chunked + embedded。Inference 時模型對每個 cell embedding 做 RAG，retrieve top-k 相關文獻 chunks，作為 context 輸入 LLM 產生 annotation + 證據。

改良：annotation 帶 citation，研究者可驗證；模型「持續更新」不需重新訓練。

🎯 4. 用 attention 改良 GRN 推論

現有限制：單看 attention map 訊號雜，precision 不如 GENIE3。但 attention 含有資訊（precision@1000 ≈ 0.3 vs random 0.05）。

新提案：用 attention 作為 GRN 的「prior weights」初始化 GENIE3 / SCENIC++ 的隨機森林，讓 scLLM 知識與機制方法互補。或用 attention rollout (Abnar 2020) 把多層整合，再用 causal calibration 過濾。

改良：結合兩家所長，benchmark 預期可達 GENIE3 + 10% AUPR。

🎯 5. scLLM in-context learning (few-shot annotation)

現有限制：scLLM zero-shot 註釋對「reference 中未見」cell type 失效；fine-tune 又要大量標註資料。

新提案：模仿 GPT-4 in-context learning：把 5–10 個「示範細胞 + 正確 label」直接拼成 prompt 輸入 scLLM，讓模型對新細胞做類比。需要在預訓練時加入「demonstration-based prediction」objective。

改良：遇到罕見 cell type 或新疾病，只需手工標 10 個示範細胞即可大規模 annotate。

🎯 6. Drug response prediction with conditional scLLM

現有限制：傳統 drug response 預測模型（DeepCDR、CDRscan）對單一藥物–細胞配對訓練，難 generalize 到新藥。

新提案：條件化 scLLM——把 drug 表示為 (SMILES embedding from ChemBERTa) condition token，與 cell embedding 透過 cross-attention 結合，預測擾動後 transcriptome。預訓練資料：LINCS L1000 (~1M perturbation profiles)。

改良：對未測試藥物 (zero-shot drug) 做 cell-level response 預測，加速 drug repurposing。

🎯 7. 結合 scLLM + protein language model (ESM)

現有限制：scLLM 的 gene token 只是 ID，沒有蛋白序列訊號；蛋白功能變異 (e.g. cancer mutations) 無法表示。

新提案：把每個 gene token 替換為「ESM-2 protein embedding + gene ID」concatenated representation；mutation 透過序列改動直接影響 embedding。

改良：能對 missense mutation 做「轉錄組級」效應預測——對 cancer somatic mutation 的 functional impact 提供新工具。

🎯 8. Synthetic data generation with scLLM

現有限制：單細胞資料生成 (scGAN、scVI sampling) 對 rare cell type 樣本太少，合成資料 mode-collapse。

新提案：scLLM 作為條件生成模型——給 cell type token 與 batch token，模型自迴歸式生成完整 gene expression profile。可加入 RLHF 用「生物學家評分」reward 模型生成更逼真細胞。

改良：解決資料不平衡，supplement 罕見細胞型別訓練。

🎯 9. 個人化醫療：fine-tune 到單一病人 atlas

現有限制：單一病人 scRNA-seq 細胞數有限 (5k–50k)，難訓練客製模型。

新提案：從預訓練 scLLM 出發，用 LoRA 在病人自己的 healthy + tumor cells 上做極輕量 fine-tune（< 100k 可訓練參數），產生「病人專屬」cell state model。可用於：(a) 預測藥物個別反應、(b) 識別病人特異的 escape pathway。

改良：真正的「N-of-1」精準醫療模型。

🎯 10. scLLM 的不確定性量化 (uncertainty quantification)

現有限制：大多數 scLLM 給 point prediction，沒有 calibrated confidence。對醫療決策不負責任。

新提案：整合 Monte Carlo dropout、deep ensemble、或 conformal prediction (Vovk)，對 cell-type prediction 給「90% confidence interval」或「abstain if uncertain」選項。對 rare cell type 與 out-of-distribution cells 特別重要。

改良：讓 scLLM 在 clinical workflow 可信賴。

🎯 11. 多模態 prompt engineering（給 cell type，生成 expression）

現有限制：scLLM 的生成方向通常是 cell → label；反向 (label → cell) 缺乏好的 prompt 設計。

新提案：類似 Stable Diffusion 的 text-to-image，建立「text prompt → cell expression」管線。Prompt 例：「a CD8+ T cell from a COVID-19 severe patient with exhaustion phenotype」→ 模型生成對應 expression profile。需要 cell description 語料庫 (用 PanglaoDB + CellMarker 描述 + 自動生成 prompt)。

改良：為 in-silico 實驗設計與假設生成提供新工具。

🎯 1. Cross-species atlas transfer learning

Current limit: HCA, Tabula Muris are each complete; cross-species mapping still relies on 1-to-1 homologs, losing paralogs and species-specific signals.

Proposal: UCE-style protein-embedding pretraining for an all-species foundation model, plus species tokens and an ancestor-reconstruction objective — model learns evolutionary trajectories, not just alignment.

Improvement: maps mouse disease cells to human equivalents, accelerating translational research.

🎯 2. Tokenize cell-type ontology directly

Current limit: scLLMs pretrain with gene tokens only; cell-type labels appear only at fine-tune via classification head — no semantic in backbone.

Proposal: embed Cell Ontology (CL) hierarchy as extra token vocab (e.g. [CL:0000084 T-cell]); during pretraining randomly mask cell-type tokens, force model to predict cell type from genes and vice versa.

Improvement: hierarchical cell typing, zero-shot reasoning on novel types ("looks closer to memory than naive T cell").

🎯 3. scLLM + RAG (retrieve markers from literature)

Current limit: scLLM knowledge is frozen at pretraining time; new markers/cell types are inaccessible.

Proposal: cross-modal (cell embedding ↔ literature embedding) retrieval index; chunk + embed PubMed/preprints. At inference, retrieve top-k literature chunks per cell, feed as context to an LLM that outputs annotation + citations.

Improvement: citation-backed annotations; continuous knowledge updates without retraining.

🎯 4. Use attention to improve GRN inference

Current limit: attention precision is below GENIE3 alone, but it carries signal (precision@1000 ≈ 0.3 vs random 0.05).

Proposal: use attention as "prior weights" initializing GENIE3 / SCENIC++ random forests, or combine layers via attention rollout (Abnar 2020) followed by causal calibration.

Improvement: ensemble of scLLM knowledge + mechanistic methods; expected +10% AUPR over GENIE3.

🎯 5. scLLM in-context learning (few-shot annotation)

Current limit: zero-shot fails on novel types; fine-tune needs lots of labeled data.

Proposal: GPT-4-style in-context learning — prepend 5–10 "demo cells + labels" to the prompt; model reasons by analogy. Requires "demonstration-based prediction" objective during pretraining.

Improvement: with only 10 hand-labeled demos, annotate at scale for rare types or novel diseases.

🎯 6. Drug-response prediction with conditional scLLM

Current limit: DeepCDR, CDRscan train per drug-cell pair; poor zero-shot generalization to new drugs.

Proposal: condition scLLM on drug — drug as SMILES embedding (ChemBERTa) injected via cross-attention; predict post-perturbation transcriptome. Pretrain on LINCS L1000 (~1M profiles).

Improvement: zero-shot drug response prediction → drug repurposing.

🎯 7. scLLM + protein language model (ESM)

Current limit: gene tokens are IDs only; can't represent protein variants (cancer mutations).

Proposal: replace each gene token with concat(ESM-2 protein embedding, gene ID); mutations alter sequence → embedding shifts.

Improvement: transcriptome-level functional impact for missense mutations — new tool for cancer somatic variant interpretation.

🎯 8. Synthetic data generation with scLLM

Current limit: scGAN, scVI sampling face mode collapse on rare types.

Proposal: scLLM as conditional generator — given (cell type, batch) tokens, autoregressively generate expression profiles. Optional RLHF with biologist-quality scoring.

Improvement: address data imbalance; augment training for rare cell types.

🎯 9. Personalized medicine: fine-tune to one patient's atlas

Current limit: per-patient scRNA-seq (5k–50k cells) is too small to train a custom model from scratch.

Proposal: from pretrained scLLM, LoRA fine-tune on patient's healthy + tumor cells (< 100k trainable params) → patient-specific cell state model. Uses: (a) per-patient drug response, (b) patient-specific escape pathways.

Improvement: true N-of-1 precision medicine.

🎯 10. Uncertainty quantification for scLLMs

Current limit: most scLLMs give point predictions; no calibrated confidence — irresponsible for clinical decisions.

Proposal: integrate Monte Carlo dropout, deep ensembles, or conformal prediction (Vovk) for "90% CI" or "abstain if uncertain" outputs. Critical for rare types and OOD cells.

Improvement: makes scLLMs trustworthy in clinical workflows.

🎯 11. Multimodal prompt engineering (cell type → expression)

Current limit: scLLM generation is usually cell → label; the reverse (label → cell) lacks good prompt design.

Proposal: Stable-Diffusion-style "text → cell expression" pipeline. Prompt example: "a CD8+ T cell from a COVID-19 severe patient with exhaustion phenotype" → model generates corresponding expression profile. Requires a cell-description corpus (PanglaoDB + CellMarker auto-templated prompts).

Improvement: new tool for in-silico experimental design and hypothesis generation.
