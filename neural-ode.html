Neural ODE 連續時間神經網路

把 ResNet 的離散層數 → 連續時間，把神經網路變成一個 ODE。Chen et al. NeurIPS 2018 最佳論文，已成為 single-cell 動力學、藥物動力學、time-series imputation 的核心工具。

Turn ResNet's discrete layers into continuous time — a neural network becomes an ODE. Chen et al. (NeurIPS 2018 best paper) — now a core tool for single-cell dynamics, pharmacokinetics, and time-series imputation.

核心概念

從 ResNet 到 Neural ODE

觀察：ResNet 的核心更新為 h_{t+1} = h_t + f(h_t, θ_t)——這恰好是 ODE dh/dt = f(h, t, θ) 的 Euler 步進（步長 = 1）。Chen et al. (2018, NeurIPS) 直接取極限：把離散層替換為 ODE：

dh(t)/dt = f(h(t), t, θ)，   h(t₁) = h(t₀) + ∫_{t₀}^{t₁} f(h(s), s, θ) ds

前向就是「對 ODE 求解」——可用任何 ODE solver（Euler、Runge-Kutta、Dopri5），讓步長 自動適應（誤差大時細分、誤差小時粗略）。

核心優勢：

  1. 連續深度：沒有「層數」這個離散超參——用 solver tolerance 控制精度/速度。
  2. 連續時間序列：處理 irregular sampling（資料不規則時間採樣）——傳統 RNN 必須補零或內插，ODE 自然處理任意時間點。
  3. O(1) 記憶體訓練：用 adjoint method 反傳，記憶體不隨層數成長。
  4. 可逆性：ODE 軌跡可逆，天然適合 normalizing flows。

Observation: ResNet's core update h_{t+1} = h_t + f(h_t, θ_t) is exactly an Euler step of the ODE dh/dt = f(h, t, θ) with step size 1. Chen et al. (2018, NeurIPS) took the limit, replacing discrete layers with an ODE:

dh(t)/dt = f(h(t), t, θ),   h(t₁) = h(t₀) + ∫_{t₀}^{t₁} f(h(s), s, θ) ds

The forward pass becomes "solve an ODE" — any solver (Euler, Runge-Kutta, Dopri5) works, with adaptive step size (refine where error is large, coarsen where it's small).

Key advantages:

  1. Continuous depth: no discrete "number of layers" — use solver tolerance to trade accuracy vs speed.
  2. Continuous-time series: handles irregular sampling natively — classical RNNs must pad or interpolate; ODEs work at any time point.
  3. O(1)-memory training: via the adjoint method; memory doesn't grow with depth.
  4. Invertibility: ODE trajectories are reversible — ideal for normalizing flows.

演算法

Adjoint Method：透過 ODE solver 反傳

問題：前向是 h(t₁) = ODESolve(h(t₀), f, θ, t₀, t₁)。要算 ∂L/∂θ，怎麼透過 solver 反傳？

直觀做法：把 solver 所有中間步驟存下來，當作計算圖反傳——但記憶體會隨 solver 步數線性增長，與 ResNet 一樣。

Adjoint method（Pontryagin 1962、Chen 2018 重新引入）：定義「adjoint」狀態 a(t) = ∂L/∂h(t)。它本身滿足另一條 ODE：

da(t)/dt = −a(t)ᵀ ∂f/∂h   (在反向時間積分)

所以反傳是「再解一次 ODE」，從 t₁ 倒退到 t₀，記憶體只需 O(1)（不需存中間 state）。同時可由

∂L/∂θ = −∫_{t₁}^{t₀} a(t)ᵀ ∂f/∂θ dt

得到參數梯度。注意：adjoint 假設 f 是「可逆」前向——若 ODE 有混沌或硬性 (stiff) 行為，adjoint 數值不穩；實務常用 「checkpointing + adjoint」 折衷。

Problem: Forward is h(t₁) = ODESolve(h(t₀), f, θ, t₀, t₁). How to compute ∂L/∂θ through the solver?

Naïve approach: store every intermediate solver step and backprop — but memory grows linearly with solver steps, just like deep ResNets.

Adjoint method (Pontryagin 1962, revived by Chen 2018): define the adjoint state a(t) = ∂L/∂h(t). It satisfies another ODE:

da(t)/dt = −a(t)ᵀ ∂f/∂h   (integrated backward in time)

So backprop = solve another ODE from t₁ back to t₀, with O(1) memory (no need to store states). Parameter gradients are:

∂L/∂θ = −∫_{t₁}^{t₀} a(t)ᵀ ∂f/∂θ dt

Caveat: the adjoint assumes the forward ODE is well-reversible. For chaotic or stiff systems, adjoint is numerically unstable; in practice "checkpointing + adjoint" is a common compromise.

實務細節：選 solver 時，平滑問題用 Dopri5（adaptive RK45）；硬性問題用 implicit solver（如 Radau）。容差通常 rtol = 1e-3, atol = 1e-4 起步；scRNA-seq 動力學一般 rtol = 1e-5 才夠穩。
In practice: for smooth dynamics use Dopri5 (adaptive RK45); for stiff problems use implicit solvers (e.g. Radau). Start with rtol = 1e-3, atol = 1e-4; scRNA-seq dynamics typically need rtol = 1e-5 for stability.

生物應用

Neural ODE 在單細胞 / 藥物 / 時序

scTour

用 Neural ODE 學「latent 細胞動力學」——同時推估 pseudotime、cell-fate probability、vector field。比 scVelo（依賴 spliced/unspliced ratio）更穩健，且能在沒有 velocity 資訊時也運作。

Uses Neural ODE to learn latent cell dynamics — jointly infers pseudotime, fate probability, and vector field. More robust than scVelo (which depends on spliced/unspliced ratios) and works without RNA velocity inputs.

TrajectoryNet

把 single-cell time-course 資料視為 distributional sequence，用 continuous normalizing flow + OT 學細胞「沿時間的最可能軌跡」。可預測中間時間點的細胞分布。

Treats single-cell time-course as a distributional sequence; uses CNF + OT to learn most-likely trajectories. Predicts cell distributions at unsampled time points.

PRESCIENT

用 NODE-style 向量場學「藥物擾動後的細胞軌跡」——給定一個未受擾動的細胞 state，預測加藥後它的演化路徑。可指導 dose 篩選。

Learns NODE-style vector fields for drug-perturbed cell trajectories — given an unperturbed cell state, predicts its trajectory under treatment. Guides dose screening.

Dynamo 向量場重建

Qiu et al. (2022, Cell) 用 sparse VFK（reproducing kernel Hilbert space）重建連續向量場——本質與 NODE 殊途同歸。可做 cell fate prediction、in silico perturbation。

Qiu et al. (2022, Cell) reconstruct continuous vector fields via sparse VFK (RKHS) — same spirit as NODE. Enables fate prediction and in silico perturbation.

PK/PD 藥物動力學

傳統 PK/PD 用 compartment model（隔室模型）+ 手調 ODE。Neural ODE 從資料學動力學，再用機制 ODE 校驗——已用於 FDA 提交的某些藥物模型（如 Janssen 2023 的 small-molecule PK）。

Classical PK/PD uses compartment models with hand-tuned ODEs. Neural ODE learns dynamics from data, then validates against mechanistic ODEs — used in some FDA submissions (e.g. Janssen 2023 small-molecule PK).

Latent ODE

Variational autoencoder + Neural ODE，用於 EHR（電子病歷）irregular time-series——處理不規則就診時間的疾病軌跡建模。

VAE + Neural ODE for irregular EHR time-series — handles disease trajectory modeling with irregular clinical visits.

深入解析

Neural ODE 的進階主題

① Continuous Normalizing Flow (CNF)

結合 NODE 與 normalizing flow：z₁ = z₀ + ∫ f(z, t) dt，log-density 變化滿足 瞬時 Jacobian trace 公式：

d log p(z(t))/dt = −tr(∂f/∂z)

不需要計算 full Jacobian determinant（O(d³)），只需要 trace（O(d)）。是 FFJORD (Grathwohl 2019) 的基石，把生成式建模從「離散層 flow」推進到「連續時間 flow」。

② FFJORD：Free-form Jacobian

用 Hutchinson trace estimator 把 trace(∂f/∂z) 估為 𝔼[ε ᵀ (∂f/∂z) ε]，可任意設計 f（沒有 invertibility 限制）。但訓練慢、需大量 ODE solver call——後續被 flow matching 取代。

③ Augmented Neural ODE (ANODE)

Dupont et al. (2019, NeurIPS) 證明：標準 NODE 的軌跡 不能交叉（因為 ODE 解的唯一性），這限制了它能表示的函數類別。ANODE 解法：把 state 擴增 a 個維度（h, h_aug），讓軌跡有「升維」空間。實證上訓練更穩、表示能力強。

④ Stiff ODE 與生物動力學

很多生物動力學是 stiff（時間尺度差異大，如轉錄秒級 vs 分化天級）。Stiff ODE 需要 implicit solver（Radau, BDF）；adjoint 在 stiff 下不穩。實務：用 forward AD（如 JAX）或 Gauss-Newton 替代 adjoint。Symbolic regression 也是一條路（見 Ch 24）。

⑤ Neural Controlled Differential Equation (CDE)

Kidger et al. (2020) 推廣：dz/dt = f(z, t) · dX(t)/dt——讓 ODE 對 input X 連續響應（不只是初始條件）。比 NODE 更適合「隨時間有 input 注入」的場景，如 medication dosing 軌跡。

⑥ 與 Diffusion / Flow Matching 的關係

Diffusion model 反向過程可寫為 SDE，其 deterministic flow（probability flow ODE）就是 Neural ODE 的特例。Flow matching（見 Ch 17）跳過「先 score 後積分」，直接學 velocity field——可視為「沒有 score 的 NODE」。

⑦ 訓練陷阱

  • Solver 步數爆炸：訓練到 ODE 變太「複雜」，adaptive solver 需要極小步長——監控 NFE (number of function evaluations)，加 regularization term ‖dh/dt‖²。
  • 梯度爆炸/消失：對非常長的時間積分（t₁ - t₀ > 5）易爆——把 t scale 到 [0, 1] 是常見技巧。
  • Initialization：把 f 的最後 layer weight 初始化為 0，讓 NODE 一開始是恆等映射，訓練更穩。

① Continuous Normalizing Flow (CNF)

Combines NODE with normalizing flows: z₁ = z₀ + ∫ f(z, t) dt, with log-density satisfying the instantaneous trace formula:

d log p(z(t))/dt = −tr(∂f/∂z)

No need to compute the full Jacobian determinant (O(d³)) — only the trace (O(d)). The foundation of FFJORD (Grathwohl 2019), extending generative modeling from discrete-layer flows to continuous-time flows.

② FFJORD: free-form Jacobian

Estimates tr(∂f/∂z) via Hutchinson trace: 𝔼[εᵀ (∂f/∂z) ε], allowing any f (no invertibility restriction). But training is slow with many ODE solver calls — superseded by flow matching.

③ Augmented Neural ODE (ANODE)

Dupont et al. (2019, NeurIPS) showed that standard NODE trajectories cannot cross (by ODE uniqueness), limiting expressiveness. Fix: augment state with extra dimensions (h, h_aug) — trajectories can "lift" through the extra dim. More stable training and stronger representation.

④ Stiff ODEs and biological dynamics

Many biological dynamics are stiff (timescales differ widely — transcription in seconds vs differentiation in days). Stiff ODEs need implicit solvers (Radau, BDF); adjoint is unstable. In practice: use forward AD (JAX) or Gauss-Newton instead. Symbolic regression is another path (see Ch 24).

⑤ Neural Controlled Differential Equations (CDE)

Kidger et al. (2020) generalize: dz/dt = f(z, t) · dX(t)/dt — the ODE responds continuously to input X (not only initial conditions). Better for "input injected over time" scenarios like medication dosing.

⑥ Relation to Diffusion / Flow Matching

The reverse diffusion process can be written as an SDE; its deterministic flow (probability flow ODE) is a special Neural ODE. Flow matching (Ch 17) skips "score, then integrate" and learns velocity directly — a "score-less NODE".

⑦ Training pitfalls

  • Step-count blow-up: adaptive solvers may need extreme refinement when f gets too complex — monitor NFE (function evaluations); add a regularizer on ‖dh/dt‖².
  • Gradient explosion/vanishing: long integration windows (t₁ − t₀ > 5) tend to blow up — rescale t to [0, 1].
  • Initialization: set the last layer of f to zero so NODE starts as identity — much more stable training.
潛在改良應用

還能改良哪些任務？

🎯 1. 把 PK/PD 與 scRNA-seq 整合

目前藥物動力學（血液濃度）與單細胞反應分開建模。提案：聯合 NODE：血藥濃度 c(t) → 細胞 state h(t) 的 controlled ODE，從少量 in vivo + 大量 in vitro 資料學跨尺度動力學。

🎯 2. 跨時間點 imputation

單細胞 time-course 常常採樣稀疏（每 12 hr 或更少）。NODE 學到 vector field 後可在任何時間點 simulate cell state。提案：NODE-based imputation pipeline，可整合進 Scanpy。

🎯 3. Continuous-time perturbation modeling

Perturb-seq 通常假設「擾動結束時的最終 state」。但很多生物效應是動態的（如轉錄因子 binding 後分鐘級反應）。提案：Neural CDE 模型，input = 連續 perturbation signal、output = 細胞軌跡。

🎯 4. Stiff biological ODE 學習

細胞訊號傳導常是 stiff（如 MAPK cascade）。提案：Implicit Neural ODE + Gauss-Newton 訓練，比 baseline NODE 更穩，可用於 Boolean → 連續訊號網路推估。

🎯 5. EHR 疾病進程模型

慢性病 EHR 軌跡（如糖尿病、CKD）非常 irregular。Latent ODE + 介入學習 → 個人化疾病進程預測 + counterfactual treatment simulation。提案：臨床決策支援工具。

🎯 6. 微生物群動力學

16S/metagenomic time-series（gut microbiome）有 stiff、multi-timescale 動力學。NODE 學跨菌種交互作用網路。提案：用於 antibiotic recovery 預測或 probiotic 設計。

🎯 7. Spatial transcriptomics 時空動力學

Spatial + temporal（如組織再生過程）需同時建模位置與時間。提案：PDE-Net 或 graph-NODE，把 spatial dependency 與時間動力學聯合學習。

🎯 1. Integrate PK/PD with scRNA-seq

Currently drug PK (blood concentration) and single-cell responses are modeled separately. Proposal: joint NODE with c(t) → h(t) as a controlled ODE — learn cross-scale dynamics from sparse in vivo + abundant in vitro data.

🎯 2. Time-point imputation

Single-cell time-courses are usually sparsely sampled (12 hr+). Once NODE learns the vector field, it can simulate cell state at any time. Proposal: NODE-based imputation pipeline integrated with Scanpy.

🎯 3. Continuous-time perturbation modeling

Perturb-seq usually assumes "final state after perturbation". But many effects are dynamic (e.g. minute-scale TF binding response). Proposal: Neural CDE with continuous perturbation signal as input and cell trajectory as output.

🎯 4. Stiff biological ODE learning

Cell signaling (MAPK cascade) is often stiff. Proposal: implicit Neural ODE + Gauss-Newton training, more stable than baseline NODE, useful for Boolean → continuous signaling network inference.

🎯 5. EHR disease progression

Chronic disease EHRs (diabetes, CKD) are highly irregular. Latent ODE + intervention learning → personalized progression prediction + counterfactual treatment simulation. Proposal: clinical decision support tool.

🎯 6. Microbiome dynamics

16S/metagenomic time-series have stiff multi-timescale dynamics. NODE learns inter-species interaction networks. Proposal: antibiotic recovery prediction or probiotic design.

🎯 7. Spatiotemporal transcriptomics

Spatial + temporal (e.g. tissue regeneration) requires joint spatial-temporal modeling. Proposal: PDE-Net or graph-NODE that learns spatial dependency + temporal dynamics jointly.

import torch
import torch.nn as nn
from torchdiffeq import odeint_adjoint as odeint

class ODEFunc(nn.Module):
    def __init__(self, d=32):
        super().__init__()
        self.net = nn.Sequential(
            nn.Linear(d, 64), nn.Tanh(),
            nn.Linear(64, d))
    def forward(self, t, h):
        return self.net(h)            # dh/dt = f(h)

f = ODEFunc()
h0 = torch.randn(256, 32)
t  = torch.linspace(0., 1., 10)
h_traj = odeint(f, h0, t, method="dopri5", rtol=1e-5)
# h_traj : (10, 256, 32) — state at 10 timestamps

import diffrax
