決定性點過程 (DPP)

Macchi (1975) 起源於量子物理 fermion 統計，Kulesza & Taskar (2012, FnT-ML) 帶入機器學習：用 det(L_Y) 對「多元 (diverse)」子集賦予高機率。Marker gene 選取、藥篩 panel、active learning 抽樣都用得到。

Originating from fermion statistics (Macchi 1975), Kulesza & Taskar (2012, FnT-ML) brought DPPs to ML: assign high probability to "diverse" subsets via det(L_Y). Used in marker gene selection, drug-screening panels, and active-learning sampling.

核心概念

從 row-similarity matrix 到「多元抽樣」

問題：從 N 個候選 item（genes、分子、cells、ROI）中挑 k 個，要求 (i) 每個都「重要」、(ii) 彼此「不太像」。Naive top-k 只滿足 (i)；隨機抽樣滿足 (ii) 但浪費 quality 資訊。DPP 把兩者寫在同一個機率模型裡。

定義：給定 PSD kernel matrix L ∈ ℝ^{N×N}，子集 Y ⊆ {1,…,N} 的機率：

P(Y) = det(L_Y) / det(L + I)，   L_Y = L 取 Y 對應 row/column 的 sub-matrix

幾何直覺：det(L_Y) 等於「L_Y 對應的特徵向量 span 的平行體 (parallelepiped) 體積平方」。Y 內 item 越「正交（不像）」、體積越大、機率越大。相似 item 等於 row 共線，行列式趨近 0，幾乎不會被一起抽中。

Quality-Diversity 分解（Kulesza 2012）：

L_ij = q_i · (φ_i^T φ_j) · q_j

q_i ≥ 0 是 item i 的 quality（marker importance, drug efficacy…），φ_i 是 unit-norm feature。這樣 P(Y) 可拆成「個別 quality 連乘 × feature 子空間體積平方」——清晰分離兩個訴求。

Problem: from N candidate items (genes, molecules, cells, ROIs) pick k such that (i) each is "important" and (ii) they are mutually "dissimilar". Top-k satisfies (i) only; uniform sampling satisfies (ii) but wastes quality info. DPP encodes both in a single probability model.

Definition: given a PSD kernel matrix L ∈ ℝ^{N×N}, the probability of subset Y ⊆ {1,…,N}:

P(Y) = det(L_Y) / det(L + I),   L_Y = sub-matrix of L on rows/cols Y

Geometric intuition: det(L_Y) is the squared volume of the parallelepiped spanned by L_Y's eigenvectors. The more orthogonal (dissimilar) the items in Y, the larger the volume, the higher the probability. Similar items make collinear rows, the determinant collapses to 0, and they almost never co-appear.

Quality-Diversity decomposition (Kulesza 2012):

L_ij = q_i · (φ_i^T φ_j) · q_j

q_i ≥ 0 is item-i's quality (marker importance, drug efficacy…), φ_i is a unit-norm feature vector. Then P(Y) factorises into "product of qualities × squared subspace volume" — cleanly separating the two desiderata.

📐

det = 平行體體積；item 越正交體積越大，所以「彼此最不像」的子集機率最高。

det equals parallelepiped volume; the more orthogonal the items, the higher the probability of the subset.

🎲

精確採樣

透過 eigendecomposition，O(N³) 後 O(Nk²) 即可精確抽 — 不需 MCMC。

Eigendecompose once (O(N³)); exact sampling in O(Nk²) — no MCMC needed.

演算法

DPP Sampling、k-DPP、MAP Greedy

① 精確採樣（Hough et al. 2006, Kulesza 2012）：分兩階段。

  1. Eigendecompose L = Σλ_n v_n v_n^T。
  2. 每個 eigen-index n 以 prob λ_n/(λ_n+1) 獨立採樣，得到 V = {v_n}。
  3. 在 V span 的子空間做「Gram-Schmidt 抽取」：每一步用條件機率挑下一個 item、projection 出該 item 方向，剩餘 |V|−1 維子空間繼續抽，直到子空間為 0 維。

結果 |Y| 是隨機（期望 Σλ_n/(λ_n+1)）；想固定 |Y|=k 就用 k-DPP（Kulesza & Taskar 2011, ICML）：第二步改成「從所有 size-k 子集 of eigen-indices」依 elementary symmetric polynomial 比重抽。

② MAP（最可能子集）是 NP-hard。實務上用 greedy（Chen et al. 2018, NeurIPS）：每步加入「marginal gain log det 最大」的 item，可達 1−1/e 近似（因 log det 為 submodular）。

③ Dual representation：若 L = B^T B 是 low-rank（B ∈ ℝ^{D×N}, D ≪ N），改用 D×D dual kernel，計算降到 O(D³)——大規模 (N ≫ 10⁴) 必備。

④ Fixed-size k-DPP 近似：實務若 N、k 太大，可用 Nyström approximation 或 random projection 把 L 壓到 low-rank 後做 dual sampling。`dppy` 內建。

① Exact sampling (Hough et al. 2006; Kulesza 2012): two phases.

  1. Eigendecompose L = Σλ_n v_n v_n^T.
  2. For each eigen-index n, include it with probability λ_n/(λ_n+1) → set V = {v_n}.
  3. In subspace span(V) perform "Gram-Schmidt sampling": at each step pick an item with conditional probability, project out that direction, and continue in the remaining |V|−1 dim until empty.

|Y| is random with mean Σλ_n/(λ_n+1). For fixed |Y|=k use k-DPP (Kulesza & Taskar 2011, ICML): replace step 2 by sampling size-k eigen-subsets weighted by an elementary symmetric polynomial.

② MAP (most probable Y) is NP-hard. In practice use the greedy algorithm (Chen et al. 2018, NeurIPS): add the item with largest marginal log-det gain — guarantees 1 − 1/e thanks to submodularity of log det.

③ Dual representation: if L = B^T B is low-rank (B ∈ ℝ^{D×N}, D ≪ N), use the D×D dual kernel — cost drops to O(D³). Essential for N ≫ 10⁴.

④ Fixed-size k-DPP approximations: for large N, k use Nyström or random projection to compress L, then dual-sample. Built into `dppy`.

為什麼 DPP 比「先 cluster 再每群挑一個」好？ Cluster-and-pick 對「正確 k」很敏感、丟失軟邊界訊息、亦無法兼顧 quality。DPP 把 quality + diversity 寫進同一個 prob model，邊際 inclusion probability 對 quality 單調、聯合 inclusion probability 對 similarity 單調遞減——數學乾淨、實證也通常更好。
Why is DPP better than "cluster then pick one per cluster"? Cluster-and-pick is sensitive to the chosen k, ignores soft boundaries, and cannot trade off quality. DPP encodes quality + diversity in a single probability model: marginal inclusion is monotone in quality, joint inclusion is monotone-decreasing in pairwise similarity. Clean math and usually better empirically.

生物應用

DPP 在生物資訊

Marker gene 多元選取

Top-k DE genes 常擠在同一通路（例如全是 ribosome genes）。對 cell-type marker selection，Kulesza 風格的 quality-diversity DPP（q = DE score、φ = expression profile）可挑出「同一細胞型內、彼此功能不重複」的 marker。已被 CITE-seq panel 設計、Visium target-panel 工具採用。

Top-k DE genes often cluster in one pathway (e.g. all ribosomal). For cell-type markers, a Kulesza-style quality-diversity DPP (q = DE score, φ = expression profile) selects markers that are functionally non-redundant. Adopted by CITE-seq and Visium target-panel design tools.

高內容藥篩多元 hit panel

從 10⁵ 化合物庫挑 1k 進入第二輪 screen——pure top-by-score 易擠在同一 scaffold。k-DPP（q = predicted activity, φ = Morgan/ECFP fingerprint）給「最多元、活性高」的子集，提升 follow-up hit diversity。Bouchard et al. 2018 (J Chem Inf Model) 系統驗證。

Picking 1k from 10⁵ compounds for round-2 screening — top-by-score crowds the same scaffold. k-DPP (q = predicted activity, φ = Morgan/ECFP fingerprint) yields the most diverse hit panel; systematically validated by Bouchard et al. 2018 (J Chem Inf Model).

CITE-seq antibody panel

100+ surface protein panel 預算下，要選出能 maximise 細胞型分辨力的 subset。DPP-objective：q = protein-level cell-type marker score、φ = 候選 antibody binding-profile correlation。比 forward-selection 一致地多 +5% balanced accuracy（Stoeckius 2017 啟發、後續 panel-design 工具普遍採用）。

Under a 100+ surface-protein budget, pick the subset that maximises cell-type discrimination. DPP objective: q = cell-type marker score, φ = antibody binding correlation. Beats forward-selection by ~5% balanced accuracy (inspired by Stoeckius 2017; standard in modern panel-design tools).

TCR/BCR 多樣性抽樣

TCR repertoire 通常有 10⁶ 以上唯一 clones，downstream 分析（如 epitope 預測、AlphaFold 預測）僅能跑 1k 個。DPP 用 TCRdist/CDR3-embedding 當 kernel，挑出 maximally diverse clones 做 representative subset——比 frequency-top-k 多覆蓋 30% epitope cluster（Mayer-Blackwell 2021 啟發）。

A TCR repertoire often contains 10⁶+ unique clones; downstream analyses (epitope prediction, AlphaFold-Multimer) can only run on ~1k. DPP with TCRdist / CDR3-embedding kernels picks maximally diverse clones — ~30% more epitope-cluster coverage than top-by-frequency (inspired by Mayer-Blackwell 2021).

Active learning：稀有細胞發掘

迭代式 scRNA-seq 採樣（如 oracle 顯微鏡選取要 sort 的細胞）。每一輪選「不確定 × 與已標細胞不像」的子集——典型 DPP-style batch active learning（Bıyık 2019, ICML）。對 rare cell type 偵測比 entropy-only 提升 2–3x recall。

Iterative scRNA-seq sampling (e.g. oracle-guided cell sorting). Each batch picks "uncertain × dissimilar to labelled" — classic DPP batch active learning (Bıyık 2019, ICML). For rare cell discovery, 2–3× the recall of entropy-only baselines.

Spatial ROI 抽樣

GeoMx / Stereo-seq 預算限定 30 個 ROI，希望兼顧「組織標記陽性」與「空間覆蓋」。DPP kernel = spatial-distance Gaussian × marker-intensity → 自動避開重疊 ROI 同時集中在感興趣區域。Visium HD 與 MERSCOPE 採後分析常用。

GeoMx / Stereo-seq budgets ~30 ROIs and wants "marker-positive" plus "spatial coverage". DPP kernel = spatial-distance Gaussian × marker intensity → avoids overlapping ROIs while focusing on regions of interest. Standard in Visium HD / MERSCOPE post-hoc analysis.

Fragment library design

FBDD (fragment-based drug discovery) 要求 1k–5k 片段 maximally cover chemical space。DPP（q = synthetic accessibility + drug-likeness、φ = Morgan ECFP）給出最多元庫——比 MaxMin（Kennard-Stone）少 ~10% 重複 scaffold（Pucci 2021）。

FBDD requires a 1k–5k fragment library that maximally covers chemical space. DPP (q = synthetic accessibility + drug-likeness, φ = Morgan ECFP) yields the most diverse library — ~10% less scaffold duplication than MaxMin (Kennard-Stone), per Pucci 2021.

Cell line panel

CCLE 1000+ cell lines 中選 60 個 NCI-60 風格 panel：要 cover lineage、driver mutation、drug-response phenotype。Multi-feature DPP（concatenated CNV、expression、dependency profile）——比 lineage-stratified random 多 +12% 預測力（Ghandi 2019 啟發）。

From CCLE 1000+ cell lines pick a 60-line NCI-60-style panel covering lineage, driver mutations, and drug-response phenotypes. Multi-feature DPP (concatenated CNV + expression + dependency) — +12% predictive power vs lineage-stratified random (Ghandi 2019 inspired).

深入解析

DPP 進階主題

① L-ensemble vs marginal kernel K

L 是「elementary」kernel：P(Y) ∝ det(L_Y)。但「marginal inclusion probability」P(A ⊆ Y) = det(K_A)，其中 K = L(L+I)⁻¹。兩種等價，但 marginal kernel 給「i 被選中的機率」、L-kernel 給「整體子集」聯合分布。建模時通常給 L，分析時用 K。

② Continuous DPP & non-symmetric kernel

DPP 也定義在連續空間（如 spatial point processes，see Ch 10）。非對稱 DPP（Brunel 2018）允許 L 不對稱、可建模「order-sensitive diversity」——例如 sequential drug treatment、recommendation 時序。

③ Strongly Rayleigh measures & negative dependence

DPP 是 strongly Rayleigh distribution 一種。SR 性質保證 negative association（i ∈ Y 降低 j ∈ Y 機率）—— concentration inequality 與 mixing time bound 都很強。Anari 等人 2016 證明 MCMC for k-DPP 的 mixing 是 polynomial。

④ Learning the kernel（Kulesza 2012）

給定 training 子集 {Y_t}，學 q_i, φ_i 使 likelihood Π det(L_{Y_t}) / det(L+I) 最大。Φ 用 deep net 參數化 → Deep DPP（Gartrell 2017, KDD）：可從 raw feature 端到端學 diversity 度量。

⑤ Conditional & constrained DPP

實務常加硬限制：必須包含某些 item（partition matroid）、預算總和 ≤ B（knapsack）。Kulesza 2012 給出 condition on inclusion 的 closed form；knapsack-DPP 需要近似（Celis 2018, ICML）。

⑥ Submodularity, 與 Maximum Volume Sampling

log det(L_Y) 是 submodular function；DPP MAP 就是 submodular maximisation。同時 det(L_Y) 對應 "maximum volume design" 的 D-optimal experimental design — DPP 與 optimal design 是同一個數學物件不同視角。

⑦ DPP 與 Volume Sampling, Determinantal Sampling for ML

近年 (Derezinski & Mahoney 2020+) volume sampling 在 randomised linear algebra 出現：用 DPP 抽 columns 做 subspace approximation，比 leverage-score sampling 給出 unbiased & tighter bound。生物資訊潛在：scRNA cell-subset selection 做 fast PCA / NMF。

⑧ 與 Repulsive Point Processes 的關係

Spatial statistics 的 Gibbs hardcore、Matérn 都是 repulsive；DPP 是「最 tractable」repulsive process（精確採樣 + closed-form likelihood）。對空間轉錄體 cell-cell repulsion 建模，DPP 比 Strauss 更可學習。

① L-ensemble vs marginal kernel K

L is the "elementary" kernel: P(Y) ∝ det(L_Y). Marginal inclusion is P(A ⊆ Y) = det(K_A) with K = L(L+I)⁻¹. The two are equivalent: K gives "prob item i is in Y", L gives the joint over Y. Model with L; analyse with K.

② Continuous & non-symmetric DPPs

DPPs are also defined on continuous spaces (spatial point processes, see Ch 10). Non-symmetric DPPs (Brunel 2018) allow asymmetric L, capturing "order-sensitive diversity" — useful for sequential drug treatment, time-series recommendation.

③ Strongly Rayleigh measures & negative dependence

DPPs belong to the strongly-Rayleigh class. SR implies negative association (i ∈ Y lowers P(j ∈ Y)) — yielding strong concentration and mixing-time bounds. Anari et al. (2016) proved polynomial MCMC mixing for k-DPP.

④ Learning the kernel (Kulesza 2012)

Given training subsets {Y_t}, learn q_i, φ_i to maximise Π det(L_{Y_t}) / det(L+I). Parameterise Φ with a deep network → Deep DPP (Gartrell 2017, KDD): end-to-end diversity learning from raw features.

⑤ Conditional & constrained DPPs

In practice we often add hard constraints: must include certain items (partition matroid), total budget ≤ B (knapsack). Kulesza 2012 gives closed-form conditioning; knapsack-DPP needs approximation (Celis 2018, ICML).

⑥ Submodularity & Maximum Volume Sampling

log det(L_Y) is submodular; DPP-MAP is submodular maximisation. Equivalently, det(L_Y) corresponds to D-optimal design ("maximum volume"). DPP and optimal design are two views of the same object.

⑦ Volume sampling & determinantal RandNLA

Recent work (Derezinski & Mahoney 2020+) uses DPP-style volume sampling in randomised linear algebra: pick columns via DPP for subspace approximation — unbiased and tighter bounds than leverage-score sampling. Bio potential: fast PCA / NMF on scRNA cell subsets.

⑧ Repulsive point processes

Gibbs hardcore, Matérn point processes are repulsive; DPP is the most tractable repulsive process (exact sampling + closed-form likelihood). For modelling cell-cell repulsion in spatial transcriptomics, DPP is more learnable than Strauss.

潛在改良應用

還能改良哪些任務？

🎯 1. Pre-training data curation

scLLM (e.g. Geneformer, scGPT) 訓練資料來自 CELLxGENE 5 千萬 cells。提案：用 cell-embedding DPP 挑 5M「最多元」cells 做 high-quality pretraining；比 random subsample 可期 +1–2% downstream zero-shot accuracy。

🎯 2. CRISPR sgRNA library

10k gene × 4–6 sgRNA library 設計：要避免 sgRNA 間靶序列重疊、最大化 efficiency。提案：k-DPP（q = sgRNA efficiency predictor、φ = on-target sequence + flanking PAM context）。

🎯 3. AlphaFold-Multimer ensemble selection

AlphaFold 給出 5 models × 多 seed → 數十預測結構。提案：DPP（q = pLDDT、φ = RMSD-based distance）挑 K 個 maximally diverse 結構做 conformation ensemble。對 IDR、loop region 特別有用。

🎯 4. Perturb-seq 擾動目標選取

欲設計 Perturb-seq library 預算 ~1000 gene KO。提案：q = expected phenotype magnitude（DepMap prior）、φ = pathway embedding；DPP 挑出 maximally informative 擾動組——比 essential-gene cutoff 多覆蓋 +25% pathway。

🎯 5. 抗體庫多元抽樣（VDJ）

從 10⁷ heavy + light chain combinations 抽 10⁴ 做 expression + binding screen。提案：DPP kernel based on CDR-H3 embedding（IgFold/AbLang）；比 random 多 30% 「unique epitope coverage」。

🎯 6. Single-cell sketching for atlas integration

Hemberg-style "Geometric sketching" 用 grid-based sample 子集；提案：替代為 DPP-sketch，依 expression-embedding density 自動 down-sample——更精確地保留 rare type。

🎯 7. Multi-modal panel co-design

CITE-seq 同時挑 RNA panel + antibody panel；兩者要互補 (joint). 提案：Joint DPP on stacked feature space，引入 modality-cross block 的 quality-diversity 結構 → 自動避免「兩 panel 都只測 T cell」。

🎯 8. Active learning for spatial omics

Slide-seq / Stereo-seq 後續迭代——next batch ROI 應補當前 model 不確定且空間覆蓋差的區域。提案：DPP-batch active learning，q = predictive entropy、φ = spatial coordinate + cell-type composition。

🎯 9. Decoupled drug-target diversity

Polypharmacology drug 篩：要找「打多個 target、彼此 mechanism 不同」化合物組合。提案：DPP over (drug, target-profile) pair，可促進 combination 多樣性（如不同 kinase family）。

🎯 1. Pre-training data curation

scLLM (Geneformer, scGPT) pretrain on ~50M cells from CELLxGENE. Proposal: embedding-DPP picks the 5M most diverse cells for high-quality pretraining; expect +1–2% downstream zero-shot accuracy vs random subsample.

🎯 4. Perturb-seq target selection

🎯 5. Antibody library diversity (VDJ)

程式碼

實作範例

# pip install dppy
import numpy as np
from dppy.finite_dpps import FiniteDPP

# Build L = q q^T * phi phi^T (quality-diversity)
N, D, k = 200, 32, 15
phi = np.random.randn(N, D); phi /= np.linalg.norm(phi, axis=1, keepdims=True)
q   = np.random.gamma(2, 1, size=N)
L   = (q[:,None] * phi) @ (phi.T * q[None,:])

dpp = FiniteDPP('likelihood', L=L)

# k-DPP exact sample of size k
dpp.sample_exact_k_dpp(size=k)
Y = dpp.list_of_samples[-1]
print("Selected items:", Y)

# MAP (greedy)
from dppy.utils import dpp_map_greedy
Y_map = dpp_map_greedy(L, k)
# Exact DPP sampling — pure NumPy (Hough et al. 2006)
import numpy as np
def sample_dpp(L):
    eigvals, eigvecs = np.linalg.eigh(L)
    # Phase 1: pick eigen-index n with prob lambda/(lambda+1)
    probs = eigvals / (1 + eigvals)
    keep  = np.random.rand(len(probs)) < probs
    V = eigvecs[:, keep]                       # (N, |V|)
    Y = []
    while V.shape[1] > 0:
        # pick item i ~ ||V[i,:]||^2 / ||V||_F^2
        p = (V**2).sum(axis=1)
        i = np.random.choice(len(p), p=p/p.sum())
        Y.append(i)
        # project out direction e_i in V's column-space
        col = V[i] / np.linalg.norm(V[i])
        V = V - np.outer(V @ col, col)
        # drop zero column
        _, s, Vt = np.linalg.svd(V, full_matrices=False)
        V = Vt[s > 1e-10].T
    return sorted(Y)
# Marker-gene selection with quality-diversity DPP
import scanpy as sc, numpy as np
from dppy.finite_dpps import FiniteDPP

adata = sc.read_h5ad("pbmc.h5ad")
sc.tl.rank_genes_groups(adata, "cell_type", method="wilcoxon")

# For one cell type, build L
ct = "T_cell"
de = adata.uns['rank_genes_groups']
genes  = list(de['names'][ct][:200])
scores = np.array([de['scores'][ct][i] for i in range(200)])
q = np.clip(scores, 0, None) + 1e-3

X = adata[:, genes].X.toarray().T            # (200, n_cells)
phi = X / (np.linalg.norm(X, axis=1, keepdims=True) + 1e-9)
L = (q[:,None] * phi) @ (phi.T * q[None,:])

dpp = FiniteDPP('likelihood', L=L)
dpp.sample_exact_k_dpp(size=12)
panel = [genes[i] for i in dpp.list_of_samples[-1]]
