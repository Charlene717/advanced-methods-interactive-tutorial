保適預測 (Conformal Prediction)

給任何模型加上「有限樣本、無分布假設」的覆蓋率保證。輸出不是單一預測，而是「預測集合」——保證真實答案以 95% 機率落在裡面。是 scGPT 等 black-box 模型走入臨床的關鍵橋梁。

Adds finite-sample, distribution-free coverage guarantees to any model. Output isn't a point but a "prediction set" — guaranteed to contain the truth with 95% probability. The key bridge from black-box models like scGPT to clinical use.

核心概念

為什麼需要 Conformal？

傳統 ML 模型給 point prediction：「這個細胞是 T cell」。但是有多確定？softmax 信心值惡名昭彰地校準不良——0.99 信心可能只有 70% 正確率。在醫療、藥物開發、CRISPR target prioritization 這類高風險任務上，這不夠用。

Conformal prediction 由 Vovk, Gammerman, Shafer (1999, 2005) 創立，核心 promise：

P( Ytest ∈ C(Xtest) ) ≥ 1 − α

其中 C(X) 是「prediction set」（分類）或「prediction interval」（回歸）。關鍵特性：

  • Distribution-free：不需要對資料分布做任何假設（除了 exchangeability，即可交換性）。
  • Finite-sample：不是漸進保證，是有限樣本嚴格成立。
  • Model-agnostic：包在任何模型外面，包括 scGPT、Geneformer、神經網路、Random Forest。

交換在 5%（α=0.05）的成本是「prediction set 變大」——比如某個細胞 conformal 可能輸出 {T cell, NK cell}，承認不確定性，但保證真實答案 95% 機率在裡面。

Classical ML gives point predictions: "this cell is a T cell." But how sure? Softmax confidence is notoriously poorly calibrated — 0.99 confidence may correspond to 70% accuracy. For high-stakes tasks (clinical, drug development, CRISPR target prioritization), this isn't enough.

Conformal prediction (Vovk, Gammerman, Shafer 1999, 2005) makes this promise:

P( Ytest ∈ C(Xtest) ) ≥ 1 − α

where C(X) is a prediction set (classification) or interval (regression). Key properties:

  • Distribution-free: no assumptions on data distribution (only exchangeability).
  • Finite-sample: not asymptotic — strictly holds for finite n.
  • Model-agnostic: wraps any model — scGPT, Geneformer, NN, Random Forest.

The cost of 5% guarantee (α=0.05) is "larger prediction sets" — e.g. conformal output may be {T cell, NK cell}, honestly acknowledging uncertainty while guaranteeing 95% probability of containing the truth.

演算法

Split Conformal：最簡單也最常用

  1. 切資料：把訓練資料切成 train set（訓模型）+ calibration set（校準，~1000 個 sample 即可）。
  2. 訓練：在 train set 上訓任意模型 f̂（不需要修改模型）。
  3. 算 non-conformity score：對 calibration set 每個樣本 (Xi, Yi)，計算 si = score(Xi, Yi)。最簡單選擇：分類用 1 − f̂(Xi)[Yi]（真實類別的「不像」程度）；回歸用 |f̂(Xi) − Yi|。
  4. 取分位數：計算 q̂ = ⌈(n+1)(1−α)⌉/n 分位數 of {s1, ..., sn}。
  5. 預測：對新樣本 Xtest，prediction set = {y : score(Xtest, y) ≤ q̂}。

分類：C(X) = {y : 1 − f̂(X)[y] ≤ q̂} = {y : f̂(X)[y] ≥ 1 − q̂}。可能 1 個類、可能多個類（不確定），有時甚至 ∅（極罕見，需要報 abstain）。

回歸：C(X) = [f̂(X) − q̂, f̂(X) + q̂]，等寬區間。

關鍵保證（Vovk 2005）：若 calibration + test set 互可交換（IID 即可），則 P(Ytest ∈ C(Xtest)) ≥ 1 − α 嚴格成立——不依賴模型是否準確！

  1. Split data: Partition training data into train set (model fitting) + calibration set (~1000 samples is plenty).
  2. Train: Fit any model f̂ on train set (no modifications needed).
  3. Non-conformity scores: For each (Xi, Yi) in calibration, compute si = score(Xi, Yi). Simplest: classification 1 − f̂(Xi)[Yi]; regression |f̂(Xi) − Yi|.
  4. Take quantile: q̂ = ⌈(n+1)(1−α)⌉/n quantile of {s1, ..., sn}.
  5. Predict: For new Xtest, prediction set = {y : score(Xtest, y) ≤ q̂}.

Classification: C(X) = {y : f̂(X)[y] ≥ 1 − q̂}. Single label, multiple labels, or rarely ∅ (abstain).

Regression: C(X) = [f̂(X) − q̂, f̂(X) + q̂], equal-width interval.

Key guarantee (Vovk 2005): If calibration and test are exchangeable (IID suffices), P(Ytest ∈ C(Xtest)) ≥ 1 − α holds strictly — independent of model accuracy!

改良版本

  • Adaptive Prediction Sets (APS, Romano 2020, NeurIPS)：讓 set 大小依「條件不確定性」自動調整——簡單樣本一個 class，邊界樣本給多個 class。score = 累積機率到真實類別為止。
  • Conformalized Quantile Regression (CQR, Romano 2019)：回歸版本，先學 q_α/2(X), q_1−α/2(X) 的條件分位數，再校準——得到「異質寬度」prediction interval。
  • Jackknife+ (Barber 2021, AOS)：leave-one-out 版本，不需切 calibration set。
  • Cross-conformal、CV+：K-fold 版本，效率與準確之間的折衷。
  • Adaptive Prediction Sets (APS, Romano 2020, NeurIPS): Set size adapts to conditional uncertainty — easy samples → one class, ambiguous → many. Score = cumulative probability up to true class.
  • Conformalized Quantile Regression (CQR, Romano 2019): Regression variant — fit conditional quantiles q_α/2(X), q_1−α/2(X) then calibrate; produces heteroscedastic intervals.
  • Jackknife+ (Barber 2021, AOS): Leave-one-out version, no calibration split needed.
  • Cross-conformal, CV+: K-fold variants, trade-off efficiency vs accuracy.

Conformal 在生命科學的應用

Conformal Cell-Type 註釋

把 scGPT/Geneformer 包成 conformal predictor — 對不確定細胞輸出「可能 cell type 集合」而非強行單一答案。對於 rare cell types、跨組織泛化的情境特別重要。已有多個 2024 預印本（bioRxiv）展示在 Human Cell Atlas 與 Tabula Sapiens 上的應用。

Wrap scGPT/Geneformer as conformal predictors — uncertain cells get "possible cell type sets" instead of forced single labels. Critical for rare types and cross-tissue generalization. Multiple 2024 bioRxiv preprints on Human Cell Atlas and Tabula Sapiens.

CrepeS

用 conformal 給 ESM-2 / ProtT5 的蛋白質功能預測加 prediction sets。輸出「最可能的幾個 GO term」而非單一 top-1 預測。對 metagenomics 中的「dark proteins」特別有用。

Conformalized ESM-2 / ProtT5 protein function predictions — outputs "top probable GO terms" rather than single top-1. Especially useful for metagenomic "dark proteins".

藥物反應不確定性

在患者分層 (patient stratification) 中，conformal 給每個病人「可能的藥物反應強度區間」。Sundin et al. (2024) 用 CQR 在 GDSC、CCLE 上展示——對 outlier 病人特別誠實。

For patient stratification, conformal outputs per-patient drug response intervals. Sundin et al. (2024) used CQR on GDSC, CCLE — especially honest about outlier patients.

基因組風險預測

Polygenic risk score (PRS) 用 conformal 給「個人化風險區間」而非單一值。Marcus et al. (2024) 在 UK Biobank 上展示，跨人種的 PRS calibration 大幅改善。

Conformalized polygenic risk scores produce personalized risk intervals. Marcus et al. (2024) on UK Biobank — significantly improves cross-ethnicity PRS calibration.

Perturb-seq 預測

scGen、scFoundation 預測「擾動後表現變化」的不確定性——對 CRISPR 設計優先級至關重要。Cleary et al. (2024) 提出 conformal-wrapped foundation models 用於 in silico screen 的可靠性。

Quantify uncertainty of perturbation effect predictions from scGen, scFoundation — critical for CRISPR design prioritization. Cleary et al. (2024) on conformal-wrapped foundation models for in silico screen reliability.

深入解析

Conformal 的陷阱、進階變體與最新動態

① Exchangeability 是真假設

「Distribution-free」常被誤解為「沒有任何假設」。實際上 conformal 需要 calibration 與 test 互可交換（exchangeable）——對 IID 資料天然成立，但對時間序列、空間資料、共變量偏移會失敗。例如用 2019 訓練的模型對 2024 病人做 conformal，因為 distribution shift 覆蓋率會掉。

② Marginal vs Conditional Coverage

標準 conformal 保證 marginal coverage：P(Y ∈ C(X)) ≥ 1−α，跨所有 X 平均。但對特定子群（rare cell type、minority population）覆蓋率可能 < 1−α！這是 Vovk 2012 的著名 impossibility result：嚴格 conditional coverage 對連續 X 不可能達成（只能 trivial intervals）。實務折衷：group-conditional conformal — 對重要子群分別校準。

③ Distribution Shift 怎麼辦？

多種變體應對：

  • Weighted conformal (Tibshirani 2019, NeurIPS)：已知 likelihood ratio 時，重新加權 calibration scores。
  • Adaptive conformal inference (Gibbs 2021, NeurIPS)：online 設定下，自動調整 α 維持長期覆蓋率。
  • Conformalized cross-covariate shift (Lei 2021)：已知特徵分布變但 P(Y|X) 不變時。

④ Set Size 怎麼最小化？

不同 non-conformity score 給不同 set size。重要選擇：

  • 1 − f̂(X)[y]：最簡單，但 set 不夠 adaptive。
  • APS (Romano 2020)：累積機率，set 大小依條件不確定性自動調整。
  • RAPS (Angelopoulos 2021)：regularized APS，懲罰 large set，set 平均小 30–50%。
  • Conformal score based on energy / likelihood：對生成模型友善。

⑤ Conformal 與 Bayesian 的關係

表面上類似（都量化不確定性），但本質不同：

  • Bayesian：posterior credible interval — 給「在我先驗下，Y 在這區間的機率」。需要 prior + 模型正確。
  • Conformal：frequentist coverage — 給「重複實驗下，這區間包含 Y 的長期頻率」。不需 prior，模型可錯但保證仍成立。

實務上常結合：用 Bayesian NN 的 posterior std 作為 conformal 的 non-conformity score，得到 calibrated Bayesian credible intervals (Stanton 2023)。

⑥ Multi-label 與 Structured Prediction

對 GO term 等 multi-label 任務，conformal 可給「label set 集合」而非 label。對序列、樹結構的 structured conformal 是 active research area (Cauchois 2021, AOS)。

⑦ 與 LLM 的結合

2024 大爆發：對 LLM 輸出做 conformal — 給「可能正確答案的集合」。對 medical QA、CRISPR target 推薦特別重要。Quach et al. (2024, ICLR) 提出 conformal language modeling，把 LLM 包成可靠 advisor。

⑧ Conditional → Localized Conformal

Guan (2023, Biometrika) 與 LCV (Locally Adaptive Conformal Validation) 用 kernel 或 NN 局部校準——對 conditional coverage 的最佳實務逼近。

① Exchangeability Is a Real Assumption

"Distribution-free" is often misread as "assumption-free." Conformal needs calibration and test to be exchangeable — natural for IID but fails under time series, spatial data, covariate shift. A model trained on 2019 data run conformal on 2024 patients loses coverage due to distribution shift.

② Marginal vs Conditional Coverage

Standard conformal guarantees marginal coverage: P(Y ∈ C(X)) ≥ 1−α averaged over X. But specific subgroups (rare cell types, minority populations) may have coverage < 1−α. Vovk 2012's famous impossibility result: strict conditional coverage for continuous X is impossible (only trivial intervals). Practical compromise: group-conditional conformal — calibrate per important subgroup.

③ What About Distribution Shift?

Several variants:

  • Weighted conformal (Tibshirani 2019, NeurIPS): reweight calibration scores by known likelihood ratio.
  • Adaptive conformal inference (Gibbs 2021, NeurIPS): online setting — auto-adjust α to maintain long-term coverage.
  • Conformalized covariate shift (Lei 2021): for known shift in P(X) but stable P(Y|X).

④ Minimizing Set Size

Different scores give different set sizes:

  • 1 − f̂(X)[y]: simplest, not adaptive.
  • APS (Romano 2020): cumulative probability, set size adapts.
  • RAPS (Angelopoulos 2021): regularized APS — 30–50% smaller average sets.
  • Energy/likelihood scores: friendly for generative models.

⑤ Conformal vs Bayesian

Superficially similar (both quantify uncertainty), fundamentally different:

  • Bayesian: credible interval — "given my prior, probability Y in this interval". Needs prior + correct model.
  • Conformal: frequentist coverage — "long-run frequency this interval contains Y under repeated sampling". No prior; model can be wrong, guarantee still holds.

Often combined: use Bayesian NN's posterior std as conformal non-conformity score — get calibrated Bayesian credible intervals (Stanton 2023).

⑥ Multi-label and Structured Prediction

For GO terms (multi-label), conformal can output "set of label sets". Structured conformal for sequences/trees is active research (Cauchois 2021, AOS).

⑦ Conformal Meets LLMs

2024 explosion: conformalized LLM outputs — "set of possibly-correct answers". Critical for medical QA, CRISPR target recommendation. Quach et al. (2024, ICLR) proposed conformal language modeling, turning LLMs into reliable advisors.

⑧ Conditional → Localized Conformal

Guan (2023, Biometrika) and LCV (Locally Adaptive Conformal Validation) use kernel or NN local calibration — best practical approximation to conditional coverage.

Conformal 還能改良哪些任務？

🎯 1. 給 scGPT/Geneformer 預測加 Conformal Sets

foundation model 的細胞分群預測常過度自信。Wrapping 成 conformal classifier → 對不確定細胞輸出 prediction set，正確處理 rare cell types 與 OOD samples。提案：scGPT-Conformal 作為 atlas-level annotation pipeline。

conformal 的 ∅（empty prediction set）= 「沒有任何已知類別合適」= 潛在新 cell type。可作為 rare/novel cell type discovery 的自動 flag。

在 Visium / MERFISH 每個 spot 給 prediction set — 視覺化「不確定區域」（通常是 cell type boundary）。對組織病理 hot-zones 偵測有用。

# Pure NumPy — for clarity
import numpy as np

def split_conformal_classification(probs_cal, y_cal, probs_test, alpha=0.05):
    """
    probs_cal: (n_cal, n_classes) — model softmax on calibration
    y_cal:     (n_cal,) — true labels
    probs_test: (n_test, n_classes)
    Returns: prediction_sets as (n_test, n_classes) bool array
    """
    n = len(y_cal)
    # 1. Non-conformity scores: 1 - prob of true class
    scores = 1 - probs_cal[np.arange(n), y_cal]
    # 2. Quantile threshold (finite-sample correction)
    q_level = np.ceil((n+1) * (1-alpha)) / n
    q_hat = np.quantile(scores, q_level, method="higher")
    # 3. Prediction set = classes with high enough probability
    pred_sets = probs_test >= (1 - q_hat)
    return pred_sets
