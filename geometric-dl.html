幾何深度學習 (Geometric Deep Learning)

Bronstein 等人 (2021) 的「Erlangen Program for ML」——把資料的對稱性 (symmetry) 與不變性 (invariance) 編碼進架構，是 AlphaFold、RFdiffusion、DiffDock 共同的數學基礎。

Bronstein et al. (2021)'s "Erlangen Program for ML" — encoding data symmetries and invariances directly into architecture. The mathematical foundation behind AlphaFold, RFdiffusion, and DiffDock.

核心概念

不變性 (invariance) vs 等變性 (equivariance)

對一群「對稱操作」G（旋轉、平移、置換⋯）：

  • 不變性 (invariant): f(g·x) = f(x)。函數輸出對群操作完全不變。例：分子能量對旋轉不變。
  • 等變性 (equivariant): f(g·x) = g·f(x)。輸出隨輸入「同步變化」。例：原子座標的 force vector，輸入旋轉後 force 也應隨著旋轉。

Bronstein 等人 (2021) 在 Geometric Deep Learning 一書中提出「5G」架構分類——所有 DL 模型其實都是在不同 domain (geometry) 上做 invariant / equivariant 操作：

  1. Grids → CNN：2D 影像的平移對稱性。
  2. Graphs → GNN：節點置換對稱性。
  3. Manifolds → Riemannian nets：曲面上的測地不變性。
  4. Groups → Equivariant nets：一般 Lie group 對稱性 (SE(3) for 3D)。
  5. Gauges → Gauge-equivariant nets：局部坐標選取無關性。

For a symmetry group G (rotations, translations, permutations, ...):

  • Invariant: f(g·x) = f(x). Output unchanged by group action. Example: molecular energy is rotation-invariant.
  • Equivariant: f(g·x) = g·f(x). Output "co-transforms" with input. Example: a force-vector field — rotating the molecule rotates the forces.

Bronstein et al. (2021), in Geometric Deep Learning, propose the "5G" taxonomy — every modern DL model is some invariance/equivariance on some domain:

  1. Grids → CNN: translation symmetry of 2D images.
  2. Graphs → GNN: node-permutation symmetry.
  3. Manifolds → Riemannian nets: geodesic invariance on surfaces.
  4. Groups → Equivariant nets: general Lie-group symmetry (SE(3) for 3D).
  5. Gauges → Gauge-equivariant nets: independence from local-frame choice.
🧲

SE(3) 等變

3D 旋轉 + 平移；蛋白質、分子的天然對稱。

3D rotation + translation — natural symmetry of proteins and molecules.

📊

樣本效率

內建對稱性 → 不需 data augmentation 學旋轉，少 10–100× 資料。

Built-in symmetry → no rotation augmentation needed; 10–100× less data.

🛡️

可信外推

物理對稱保證 → 模型不會因「視角旋轉」而給出矛盾答案。

Physical guarantees → outputs are never contradicted by a rotated view.

演算法

主要架構家族

① Tensor Field Networks (Thomas et al. 2018)

第一個 SE(3)-equivariant 神經網路。每個原子上放「球諧 (spherical harmonics) 張量」，用 Clebsch-Gordan 係數做張量積——使得旋轉作用於 input 等於同一個旋轉作用於 output。

② SE(3)-Transformer (Fuchs et al. 2020, NeurIPS)

把 self-attention 推廣到 SE(3) 等變：query/key/value 都是 SE(3) 不可約表示 (irreducible representations)。attention 權重對旋轉不變，輸出 vector field 對旋轉等變。應用：AlphaFold2 EvoFormer 的結構模組。

③ EGNN — E(n)-Equivariant GNN (Satorras et al. 2021, ICML)

非常輕量的設計：訊息傳遞只用 scalar features（不變量）和 position（等變量），不需 spherical harmonics。每層更新： mij = φ_e(hi, hj, ‖xi − xj‖²)；xi ← xi + Σj(xi − xj)·φ_x(mij)。比 SE(3)-Transformer 快 3–10×，準確度相近。

④ MACE (Batatia et al. 2022, NeurIPS)

高階 message passing + ACE basis，目前 force field 預測的 SOTA。MACE-OFF 與 MACE-MP 是 chemistry 社群的 foundation model。

⑤ Riemannian SGD on manifolds

當參數空間是 manifold（hyperbolic space, Stiefel manifold, SPD matrices）時，普通 gradient step 會跳出 manifold。Riemannian gradient descent：先計算 Riemannian gradient，再用 exponential map 做 retraction 回 manifold。geomstats、geoopt 是常用工具。

① Tensor Field Networks (Thomas et al. 2018)

The first SE(3)-equivariant network. Spherical-harmonics tensors live on each atom; Clebsch-Gordan tensor products keep rotations applied to input equal to the same rotation applied to output.

② SE(3)-Transformer (Fuchs et al. 2020, NeurIPS)

Self-attention generalized to SE(3) equivariance: Q/K/V are irreducible representations of SE(3). Attention weights are rotation-invariant; output vector fields are rotation-equivariant. Used in AlphaFold2's EvoFormer structure module.

③ EGNN — E(n)-Equivariant GNN (Satorras et al. 2021, ICML)

Lean design using only scalar features (invariants) and positions (equivariants), no spherical harmonics needed. Per-layer: mij = φ_e(hi, hj, ‖xi − xj‖²); xi ← xi + Σj(xi − xj)·φ_x(mij). 3–10× faster than SE(3)-Transformer at comparable accuracy.

④ MACE (Batatia et al. 2022, NeurIPS)

Higher-order message passing + ACE basis, current SOTA for force-field prediction. MACE-OFF and MACE-MP are foundation models for the chemistry community.

⑤ Riemannian SGD on manifolds

When the parameter space is a manifold (hyperbolic space, Stiefel, SPD matrices), ordinary gradient steps leave the manifold. Riemannian gradient descent: take the Riemannian gradient, then retract via the exponential map. Tools: geomstats, geoopt.

為什麼蛋白質非要等變不可？ 蛋白質的功能不會因為你把它在實驗室裡轉一圈就改變——但 naive 神經網路會給出完全不同的預測。等變網路在架構上排除這種荒謬，且不需要透過 rotation augmentation「死背」對稱性。
Why must proteins be equivariant? A protein's function doesn't change when you rotate it in the lab — yet a naive net will produce a different prediction. Equivariant networks rule this out by architecture, without needing rotation augmentation to "memorize" the symmetry.

生物應用

GDL 在結構生物學的革命

AlphaFold2

EvoFormer + Structure Module 含 SE(3)-equivariant 注意力——對胺基酸殘基的 3D 座標做等變更新。把蛋白結構預測準確度從 ~50% 提升到 ~90%，獲 2024 Nobel Chemistry。

EvoFormer + Structure Module include SE(3)-equivariant attention on residue coordinates. Lifted protein-structure prediction from ~50% to ~90% accuracy; awarded 2024 Nobel Chemistry.

AlphaFold3

用 SE(3)-equivariant diffusion 直接生成蛋白-蛋白、蛋白-小分子、蛋白-核酸複合物——把 GDL 與 diffusion model 結合，可預測 ligand binding pose。

Uses SE(3)-equivariant diffusion to generate protein-protein, protein-ligand, and protein-nucleic-acid complexes — fuses GDL with diffusion to predict binding poses.

RFdiffusion

SE(3)-equivariant diffusion 用於蛋白質 de novo 設計：給定一個結合 motif，生成圍繞它的蛋白骨架。已用於設計 binder、酵素、自組裝 nano-cage。

SE(3)-equivariant diffusion for de novo protein design: given a binding motif, generate a backbone around it. Used to design binders, enzymes, self-assembling nanocages.

DiffDock

蛋白-配體 docking：用 SE(3)-equivariant diffusion 在 ligand pose 空間上採樣，比傳統 AutoDock 在 PDBBind benchmark 上 top-1 RMSD < 2Å 達 38%（vs Glide 22%）。

Protein-ligand docking via SE(3)-equivariant diffusion on pose space. On PDBBind, 38% top-1 RMSD < 2Å (vs Glide 22%).

Boltz / EquiDock

Boltz (Wohlwend 2024) 為開源 AlphaFold3 等價物，含 SE(3)-equivariant blocks。EquiDock (Ganea 2022) 為純等變的 rigid-body 蛋白docking——比 traditional ZDOCK 快 800×。

Boltz (Wohlwend 2024) is an open-source AlphaFold3-equivalent with SE(3)-equivariant blocks. EquiDock (Ganea 2022) is rigid-body protein docking that is 800× faster than ZDOCK.

GeoMol / MACE-OFF

GeoMol (Ganea 2021)：等變生成 3D conformer。MACE-OFF (Kovács 2023)：等變 force field foundation model，用於 MD 模擬，比 ANI/SchNet 準確且快。

GeoMol (Ganea 2021): equivariant 3D conformer generation. MACE-OFF (Kovács 2023): equivariant force-field foundation model — used in MD simulations, faster and more accurate than ANI/SchNet.

ESMFold / OmegaFold

用語言模型 embedding 替換 MSA 步驟，再接上 SE(3)-equivariant decoder 出 3D 結構——速度比 AlphaFold2 快 60×。

Replace the MSA step with a language-model embedding, then pipe into an SE(3)-equivariant decoder for 3D structure — 60× faster than AlphaFold2.

Cryo-EM reconstruction

Levy 等 (2022) 用 SO(3)-equivariant nets 從 2D 投影直接推 3D 結構，省去傳統「pose alignment」步驟。對 conformational heterogeneity 處理更強。

Levy et al. (2022) used SO(3)-equivariant nets to recover 3D from 2D projections directly, skipping pose alignment — handles conformational heterogeneity better.

潛在改良應用

GDL 還能應用在哪？

🎯 1. SE(3)-equivariant 細胞位置 atlas

Spatial transcriptomics atlas 在 3D 重建後是「點雲 + 表現向量」；用 EGNN 學「組織型別」對任意旋轉/翻轉的 atlas 都給一致答案，比 Visium 的 raw 座標更穩健。

🎯 2. Equivariant 多模態整合

scRNA + spatial 整合時，spatial 部分有 SE(2) 對稱性（切片旋轉）；用 equivariant encoder 可在不同切片方向都得到一致 latent。

🎯 3. Riemannian scLLM on hyperbolic manifold

結合 Ch 15 hyperbolic embeddings 與 Riemannian SGD——scLLM 在 hyperbolic 空間 fine-tune 時用 Riemannian optimizer，比 Euclidean 收斂快且樹狀結構更清楚。

🎯 4. 蛋白質構象動態軌跡生成

結合 SE(3)-equivariant + flow matching (Ch 17)：生成蛋白質從一個構象到另一個的等變軌跡——比 MD 快 1000×，可用於 allosteric mechanism 研究。

🎯 5. 等變細胞形狀分析

對細胞形狀 mesh 用 gauge-equivariant nets，對形狀分類（apoptotic / mitotic / motile）做出對視角不變的判斷——適合 high-content imaging。

🎯 6. 3D 染色體結構建模

從 Hi-C 矩陣推 3D 染色體結構：用 SE(3)-equivariant 生成模型，對 TAD 邊界與 chromatin loop 的物理對稱保證更穩。

🎯 1. SE(3)-equivariant cell positioning atlas

Reconstructed 3D spatial-transcriptomics atlases are point-clouds + expression vectors. EGNN learns tissue type with outputs invariant to arbitrary atlas rotations/flips — more robust than raw Visium coordinates.

🎯 2. Equivariant multi-modal integration

For scRNA + spatial integration, the spatial side has SE(2) symmetry (slice rotation). An equivariant encoder gives the same latent under different slice orientations.

🎯 3. Riemannian scLLM on hyperbolic manifold

Combine Ch 15 hyperbolic embeddings with Riemannian SGD — fine-tune scLLMs in hyperbolic space with Riemannian optimizers, faster convergence and clearer hierarchical structure.

🎯 4. Protein conformational trajectory generation

Combine SE(3)-equivariant + flow matching (Ch 17) to generate equivariant trajectories between protein conformations — 1000× faster than MD, useful for allosteric mechanism studies.

🎯 5. Equivariant cell-shape analysis

Gauge-equivariant nets on cell-shape meshes give view-invariant shape classification (apoptotic / mitotic / motile) — ideal for high-content imaging.

🎯 6. 3D chromosome structure modeling

From Hi-C to 3D chromosome structure via SE(3)-equivariant generative models — physically consistent TAD boundaries and chromatin loops.

實作範例

# Minimal E(n)-Equivariant GNN layer (Satorras 2021)
import torch
import torch.nn as nn

class EGNNLayer(nn.Module):
    def __init__(self, in_dim, hidden):
        super().__init__()
        self.phi_e = nn.Sequential(
            nn.Linear(2*in_dim + 1, hidden), nn.SiLU(),
            nn.Linear(hidden, hidden),
        )
        self.phi_x = nn.Sequential(nn.Linear(hidden, 1))
        self.phi_h = nn.Sequential(
            nn.Linear(in_dim + hidden, hidden), nn.SiLU(),
            nn.Linear(hidden, in_dim),
        )

    def forward(self, h, x, edge_index):
        i, j = edge_index
        diff = x[i] - x[j]
        d2 = (diff**2).sum(-1, keepdim=True)
        m = self.phi_e(torch.cat([h[i], h[j], d2], -1))
        # Equivariant position update
        x_upd = x + torch.zeros_like(x).index_add_(
            0, i, diff * self.phi_x(m))
        # Invariant feature update
        h_upd = h + self.phi_h(torch.cat(
            [h, torch.zeros_like(h).index_add_(0, i, m)], -1))
        return h_upd, x_upd
# pip install e3nn — for full SE(3)-equivariant networks
import torch
from e3nn import o3
from e3nn.nn import Gate

# Irreducible representations: scalars (0e) + vectors (1o)
irreps_in  = o3.Irreps("16x0e + 8x1o")
irreps_out = o3.Irreps("32x0e + 16x1o")

tp = o3.FullyConnectedTensorProduct(
    irreps_in, irreps_in, irreps_out
)
# Forward call respects SE(3) equivariance by construction
x = irreps_in.randn(10, -1)
y = tp(x, x)                       # shape (10, irreps_out.dim)
# pip install geomstats — Riemannian SGD
import geomstats.backend as gs
from geomstats.geometry.hyperbolic import Hyperbolic

manifold = Hyperbolic(dim=10)

def riem_sgd_step(x, grad, lr=0.01):
    # 1. project Euclidean grad to tangent space
    tangent_grad = manifold.to_tangent(grad, base_point=x)
    # 2. exponential map step
    x_new = manifold.metric.exp(
        tangent_vec=-lr * tangent_grad, base_point=x)
    return x_new
