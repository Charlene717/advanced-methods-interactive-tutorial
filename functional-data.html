泛函資料分析 (Functional Data Analysis)

把每筆資料看成一條「函式」而非數列——Ramsay & Silverman (2005) 經典框架。連續血糖、24h gene expression、fMRI BOLD、wearable signals 都是「曲線資料」，FPCA、functional regression 是處理它們的天然語言。

Treat each observation as a function rather than a sequence — the Ramsay & Silverman (2005) framework. Continuous glucose, 24h gene expression, fMRI BOLD, wearables — all are "curve data". FPCA and functional regression are their native language.

核心概念

為什麼把資料看成函式？

傳統 multivariate 視角的問題：若把 24h 內每小時測量的血糖視為 24 維向量，PCA 完全忽略「相鄰小時應該相關、應該光滑」的事實。同樣的數據如果採樣時間錯位（patient A 整點測、patient B 半點測），向量根本無法對齊。

FDA 的核心觀點 (Ramsay & Silverman 2005)：每筆觀測是一條函式 x_i(t)，t ∈ [a, b] 是連續參數（時間、波長、空間位置）。資料的 dimension 是「無限」（function space），但實際只在有限離散點觀測 + 帶雜訊。

基底函式展開：用 K 個基底函式 φ_k(t)（splines、Fourier、wavelets）表示函式：

x_i(t) = Σ_{k=1}^K c_{ik} φ_k(t)

於是「無限維函式」變成「K 維係數向量 c_i」。關鍵差別：基底是光滑的，所以 c_i 自然編碼了 smoothness 結構；標準 multivariate 做不到這點。

常用基底：

  • B-splines：局部支撐、可控光滑度 — 最常用，適合非周期資料。
  • Fourier：適合周期資料（circadian、cell cycle）。
  • Wavelets：適合多尺度 / 局部峰值（NMR spectra、ECG）。
  • RBF / Gaussian kernels：適合 scattered 不規則觀測（與 GP 等價）。

The multivariate problem: Treating 24h hourly glucose as a 24-D vector ignores the fact that adjacent hours should be correlated and the function should be smooth. If patient A measures on the hour and patient B on the half-hour, the vectors don't even align.

FDA's core view (Ramsay & Silverman 2005): each observation is a function x_i(t), t ∈ [a, b] is a continuous parameter (time, wavelength, space). Data is infinite-dimensional (function space), observed only at finite, noisy points.

Basis expansion: represent each function with K basis functions φ_k(t) (splines, Fourier, wavelets):

x_i(t) = Σ_{k=1}^K c_{ik} φ_k(t)

The "infinite-dim function" becomes a "K-D coefficient vector c_i". Key difference: the basis is smooth, so c_i naturally encodes smoothness — standard multivariate analysis cannot.

Common bases:

  • B-splines: compact support, controlled smoothness — most common, for non-periodic data.
  • Fourier: periodic data (circadian, cell cycle).
  • Wavelets: multiscale / local peaks (NMR spectra, ECG).
  • RBF / Gaussian kernels: scattered irregular sampling (equivalent to GP).

Each patient's 24h glucose curve = 1 FDA datum (not 24).

基底展開天然光滑，無需事後 smoothing；可控連續可微階數。

Basis expansion is intrinsically smooth — no post-hoc smoothing needed; differentiability controlled.

🔬

FPCA 找主模態

解 covariance operator 的 eigenfunctions — 第 1 主成分 = 「最主要的曲線變異」。

Eigenfunctions of the covariance operator — PC1 = "dominant curve-shape variation".

演算法

從 raw curves 到 FPCA 與 functional regression

① Smoothing 與基底擬合：給定 noisy observations {(t_{ij}, y_{ij})}，用 penalized least squares 擬合每條 x_i(t)：

argmin_{c_i} Σ_j (y_{ij} − Σ_k c_{ik} φ_k(t_{ij}))² + λ ∫ (D²x_i(t))² dt

第二項是roughness penalty（二階導數平方積分），λ 控制光滑度。最佳 λ 用 generalized cross-validation (GCV) 選。

② FPCA (Functional PCA, Ramsay 2005)：函式版 PCA。先計算 sample covariance function

v(s, t) = (1/n) Σ_i (x_i(s) − x̄(s))(x_i(t) − x̄(t))

解 eigenvalue problem ∫ v(s, t) ξ_k(t) dt = λ_k ξ_k(s)，得到一系列 eigenfunctions ξ_k(t)。第 k 個 FPC score 為 s_{ik} = ∫(x_i(t) − x̄(t)) ξ_k(t) dt。一個典型結果是前 3–5 個 FPC 解釋 > 95% variance。

③ Sparse FPCA (Yao et al. 2005, JASA)：當每位 subject 只有少數 irregular sampling 點（如 longitudinal study），不能逐 subject smoothing。改為 borrow strength across subjects：把所有 (i, j) 點當 scattered observation，估計 mean function μ(t) 與 covariance function v(s, t)，然後對每位 subject 用 conditional expectation 預測 FPC score（PACE algorithm）。單細胞 longitudinal 資料的核心方法。

④ Functional Linear Regression：

  • Scalar-on-function：y_i (scalar) = α + ∫ β(t) x_i(t) dt + ε_i。係數函式 β(t) 用 basis 展開，係數 λ-penalize 估計。
  • Function-on-scalar：x_i(t) = α(t) + β(t) z_i + ε_i(t)（每位 subject 的曲線是 covariate 的函式）。
  • Function-on-function：y_i(s) = α(s) + ∫ β(s, t) x_i(t) dt + ε_i(s)。最一般、但需更多資料。

⑤ Multivariate / Multi-dimensional FDA：每筆資料是「多條曲線」（如 multi-channel ECG）→ 用 product basis 或共同 covariance。Chen et al. (2017) 提 functional manifold learning，允許函式分布在 manifold 上。

⑥ Deep FDA：近年用 neural network 學 functional representation：NN 從 dense observations 預測 mean + variance + warping (curve registration)。比 splines 更靈活，但需要更多資料。

① Smoothing and basis fitting: Given noisy observations {(t_{ij}, y_{ij})}, fit each x_i(t) by penalized least squares:

argmin_{c_i} Σ_j (y_{ij} − Σ_k c_{ik} φ_k(t_{ij}))² + λ ∫ (D²x_i(t))² dt

The second term is the roughness penalty (integrated squared second derivative); λ controls smoothness. Choose λ by generalized cross-validation (GCV).

② FPCA (Functional PCA, Ramsay 2005): Function-valued PCA. Compute sample covariance function

v(s, t) = (1/n) Σ_i (x_i(s) − x̄(s))(x_i(t) − x̄(t))

Solve ∫ v(s, t) ξ_k(t) dt = λ_k ξ_k(s) for eigenfunctions ξ_k(t). The k-th FPC score is s_{ik} = ∫(x_i(t) − x̄(t)) ξ_k(t) dt. Typically the first 3–5 FPCs explain > 95% of variance.

③ Sparse FPCA (Yao et al. 2005, JASA): When each subject has only a few irregular observations (longitudinal studies), per-subject smoothing fails. Instead borrow strength across subjects: treat all (i, j) as scattered points; estimate μ(t) and v(s, t); predict each subject's FPC scores via conditional expectation (PACE algorithm). The core method for single-cell longitudinal data.

④ Functional Linear Regression:

  • Scalar-on-function: y_i (scalar) = α + ∫ β(t) x_i(t) dt + ε_i. Coefficient function β(t) is basis-expanded and λ-penalized.
  • Function-on-scalar: x_i(t) = α(t) + β(t) z_i + ε_i(t) (each subject's curve is a function of a covariate).
  • Function-on-function: y_i(s) = α(s) + ∫ β(s, t) x_i(t) dt + ε_i(s). Most general but data-hungry.

⑤ Multivariate / multi-dimensional FDA: Each datum is multiple curves (e.g. multi-channel ECG) → product basis or joint covariance. Chen et al. (2017) propose functional manifold learning when functions lie on a manifold.

⑥ Deep FDA: Recent NN models learn functional representations: a network predicts mean + variance + warping (curve registration) from dense observations. More flexible than splines, more data-hungry.

Curve registration (時間對齊) 的重要性：不同 subject 的「事件」可能在不同時間發生（如 menstrual cycle 不同 phase；藥物 onset 早晚不一）。Amplitude variation（曲線形狀差異）與 phase variation（時間錯位）若不分開，FPCA 第一主成分常常只是「時間錯位」。Marron et al. (2015) 推薦先做 warping（如 SRSF, Srivastava 2011）再做 FPCA。
Curve registration matters: events may occur at different times across subjects (menstrual phase; drug onset). Without separating amplitude vs phase variation, FPC1 often just captures time misalignment. Marron et al. (2015) recommend warping first (e.g. SRSF, Srivastava 2011) then FPCA.

情境2:
Longitudinal 稀疏資料——每 subject 只 5–10 個不規則時間點：Sparse FPCA (Yao 2005) 借用 cross-subject 資訊。

Spectroscopy / 連續波長資料——NMR / Raman / mass spec 譜線天然是函式。

Case 1:
Dense continuous sampling — CGM every 5 min, 24h gene expression, fMRI BOLD: FDA is the default.
Case 2:
Sparse longitudinal — only 5–10 irregular timepoints per subject: Sparse FPCA (Yao 2005) borrows across subjects.

Spectroscopy / continuous wavelength — NMR / Raman / mass spec are intrinsically functional.
Not for:
Truly discrete categorical sequences (amino acids) — use sequence models, not FDA.
生物應用

FDA 在生物與醫學的應用

連續血糖監測 (CGM)

CGM 每 5 min 一筆血糖，每位糖尿病患者一天 288 點。Wang et al. (2018, Diabetes Care) 用 FPCA 把 24h 曲線壓縮成 3–4 個 FPC scores，預測 HbA1c、低血糖風險、用藥反應，比 mean / time-in-range 更敏感。

CGM samples every 5 min, 288 points/day per diabetic. Wang et al. (2018, Diabetes Care) compress 24h curves to 3–4 FPCs, predicting HbA1c, hypoglycemia risk, drug response — more sensitive than mean / time-in-range.

24h Circadian gene expression

JTK_CYCLE、MetaCycle 等檢定週期性的工具用 Fourier basis FDA 處理時序 microarray / RNA-seq——找出 BMAL1, PER, CRY 等核心 clock gene 與下游 oscillating genes。

JTK_CYCLE, MetaCycle and related rhythm-detection tools use Fourier-basis FDA on time-series microarray / RNA-seq — identifying BMAL1, PER, CRY and downstream oscillating genes.

fMRI BOLD 訊號

每體素的 BOLD 時序是一條 functional curve。Functional regression 把 task condition (covariate) 對應到 BOLD 響應（function-on-scalar），優於傳統 GLM 點對點分析。Lindquist (2008, Stat Sci) 是綜述。

Each voxel's BOLD time-course is a functional curve. Function-on-scalar regression maps task condition to BOLD response, beating pointwise GLM. Lindquist (2008, Stat Sci) reviews this.

動態心電圖 / Holter

24h Holter 監護生成連續 ECG 曲線。FDA 用於 QT interval dynamics、HRV functional analysis，預測心血管事件比單點測量更早。已嵌入部分 ICU 風險評分。

24h Holter monitoring yields continuous ECG. FDA analyzes QT-interval dynamics, functional HRV — predicting cardiovascular events earlier than single measurements. Embedded in some ICU risk scores.

Wearable activity curves

Fitbit / Apple Watch 每分鐘 step count / heart rate 構成 24h 活動曲線。Goldsmith et al. (2015, JASA) 用 multilevel FDA 找「日間活動 pattern」對應 BMI、抑鬱症狀的關聯。

Fitbit / Apple Watch generate 24h step / HR curves. Goldsmith et al. (2015, JASA) use multilevel FDA to associate daytime activity patterns with BMI, depression symptoms.

連續單細胞軌跡

Pseudotime 上的 gene expression 是 t 的函式。Yao 2005 sparse FPCA 是 pseudotime-by-gene matrix 的天然分析框架——每個 gene 是一條曲線，可分群（waves）、做 functional regression 找 driver。Campbell & Yau (2018) 早期工作示範這個方向。

Gene expression along pseudotime is a function of t. Yao 2005 sparse FPCA is the native framework for pseudotime-by-gene matrices — each gene is a curve, can be clustered (waves), regression finds drivers. Campbell & Yau (2018) demonstrated this direction.

連續藥物動力學 PK/PD

血藥濃度 vs time 是 functional curve。NONMEM、Phoenix 等 PK 軟體本質都在做 functional model fitting。FDA framework 允許 cross-patient 借用統計力量，比個體 NLME 更穩。

Plasma concentration vs time is a functional curve. NONMEM, Phoenix essentially perform functional model fitting. FDA enables cross-patient borrowing of statistical power — more stable than per-subject NLME.

蛋白動力學模擬

MD trajectory 中每殘基 RMSF/距離曲線是 functional data。Krivobokova et al. (2012) 提 functional Procrustes 對齊 MD trajectory，做 trajectory-level clustering 找 conformational families。

Per-residue RMSF / distance curves in MD trajectories are functional data. Krivobokova et al. (2012) propose functional Procrustes alignment of MD trajectories for trajectory-level clustering into conformational families.

FACS / 流式細胞

流式細胞每細胞的 marker intensity 經 kinetic stimulation 後是時序——FDA 把 receptor signaling kinetics 視為曲線，FPCA 找響應亞群。Pyne et al. (2009) 是早期應用。

Flow cytometry marker kinetics post-stimulation are time-series — FDA treats receptor signaling kinetics as curves; FPCA finds responder subpopulations. Pyne et al. (2009) is an early application.

細胞代謝率 Seahorse

Seahorse XF 連續測 OCR / ECAR，每 well 一條代謝曲線。FDA 用於比較不同 perturbation 下的 metabolic phenotype curves，比 endpoint analysis 信息更豐富。

Seahorse XF measures OCR / ECAR continuously — one metabolic curve per well. FDA compares perturbation-induced metabolic phenotype curves, far richer than endpoint analysis.

深入解析

FDA 進階主題

① Mercer 定理與 Karhunen-Loève 展開

FDA 的數學基礎：若 covariance function v(s, t) 連續且 positive semi-definite，則 Mercer 定理保證 v(s, t) = Σ_k λ_k ξ_k(s) ξ_k(t)。任意樣本可寫為 x_i(t) = μ(t) + Σ_k s_{ik} ξ_k(t)（Karhunen-Loève expansion）。這就是 FPCA 的數學基礎——它跟矩陣 PCA 一模一樣，只是把離散 eigenvalue problem 換成連續 integral equation。

② Curve registration: SRSF framework

Srivastava et al. (2011, JRSSB) 的 Square Root Slope Function (SRSF) 把曲線 x(t) 映成 q(t) = sign(x'(t)) √|x'(t)|，使「amplitude–phase 分離」變成 SRSF 空間中的 Procrustes 對齊問題。比 dynamic time warping 更有理論保證、更穩定。是現代 curve registration 的標準。

③ Functional manifold learning

Chen et al. (2017, JMLR) 的 functional manifold learning：函式不一定分布在 linear function space，可能在 manifold 上（例如蛋白動力學 trajectory 限制在 conformational sub-manifold）。在 SRSF 空間做 Isomap / diffusion map 萃取低維結構。

④ Functional graphical model

把 multivariate functional data 的依賴結構視為「函式版 graph」——Zhu et al. (2016) 的 functional graphical lasso 估計 functional precision operator。對 multi-channel ECG / EEG / multi-region fMRI 都適用。

⑤ Density-valued FDA

當每筆觀測本身是一個機率密度（如每個 subject 的 cell-type composition、microbiome distribution），它在 Bayes-Hilbert space 上是 functional data。Petersen et al. (2022, Stat Sinica) 提供完整 FDA on densities 框架，已用於 microbiome longitudinal 與 connectome density。

⑥ 高維 FDA 的挑戰

「many curves, many time points, few subjects」是現代生物資料的典型問題（如 scRNA pseudotime + 20,000 gene）。標準 FPCA 在 high-dim 失效——需要稀疏化（sparse FPCA）、low-rank + sparse decomposition (e.g. FunRPCA)，或先用 NMF / autoencoder 降維。

⑦ 與 Gaussian process 的關係

GP 是「函式上的機率分布」，本質上就是 FDA 的貝氏版。GP regression 等價於 RBF basis FDA + 特定先驗。實務差異：FDA 偏向 spline / Fourier basis（解釋性好、計算快），GP 偏向 kernel ridge（不需指定 basis、自動 length-scale）。對小資料 GP 更穩；對大資料 FDA 更可擴展。

⑧ Deep functional models

近年 Deep Functional Factor Model (DF2M)、Neural Functional Encoders 用 NN 學 latent function。其中 Neural Process 與 Functional Neural Networks（Rao 2023）把 NN input 視為函式空間映射，可處理 irregular sampling 與 multi-resolution data。

① Mercer's theorem and Karhunen-Loève expansion

FDA's math foundation: if covariance v(s, t) is continuous and PSD, Mercer guarantees v(s, t) = Σ_k λ_k ξ_k(s) ξ_k(t). Any sample expands as x_i(t) = μ(t) + Σ_k s_{ik} ξ_k(t) (Karhunen-Loève). This is the math behind FPCA — identical to matrix PCA but with integral equations replacing discrete eigenproblems.

② Curve registration: SRSF framework

Srivastava et al. (2011, JRSSB) Square Root Slope Function: q(t) = sign(x'(t)) √|x'(t)| turns amplitude–phase separation into a Procrustes alignment in SRSF space. Theoretically grounded and more stable than DTW; the modern standard for curve registration.

③ Functional manifold learning

Chen et al. (2017, JMLR): functions may live on a manifold rather than a linear function space (e.g. protein dynamics confined to a conformational sub-manifold). Use Isomap / diffusion maps in SRSF space.

④ Functional graphical model

View multivariate functional dependence as a "functional graph" — Zhu et al. (2016) functional graphical lasso estimates a functional precision operator. Applies to multi-channel ECG / EEG / multi-region fMRI.

⑤ Density-valued FDA

When each observation is itself a probability density (subject's cell-type composition, microbiome distribution), it lives in Bayes-Hilbert space — functional data. Petersen et al. (2022, Stat Sinica) give a complete framework, applied to longitudinal microbiome and connectome densities.

⑥ High-dim FDA challenges

"Many curves, many points, few subjects" is typical of modern bio data (scRNA pseudotime × 20,000 genes). Standard FPCA breaks down — use sparse FPCA, low-rank + sparse decomposition (FunRPCA), or pre-reduce with NMF / autoencoder.

⑦ Relation to Gaussian processes

A GP is a probability distribution over functions — essentially Bayesian FDA. GP regression equals RBF-basis FDA with a specific prior. In practice: FDA favors splines / Fourier (interpretable, fast); GP favors kernel ridge (no explicit basis, auto length-scale). GP better for small data; FDA more scalable for large data.

⑧ Deep functional models

Recent Deep Functional Factor Model (DF2M), Neural Functional Encoders use NN to learn latent functions. Neural Processes and Functional Neural Networks (Rao 2023) treat NN inputs as function-space maps, handling irregular sampling and multi-resolution data.

潛在改良應用

還能改良哪些任務？

🎯 1. scRNA-seq pseudotime gene profile clustering

目前對 pseudotime expression matrix 通常用 hierarchical clustering on gene profiles。提案：用 functional clustering（k-medoids on SRSF distance）把 gene 分群為「early peak」「late wave」「bimodal」等 functional motif，比 Pearson clustering 更解釋性。

🎯 2. CGM-Diet-MetaG 三模態 functional 整合

連續血糖 + 飲食日誌 + 腸道微生物 → 三條同步 functional series。提案：multivariate FDA (joint FPCA + functional CCA) 找「微生物–血糖–飲食」三方耦合 pattern，幫助精準營養。

🎯 3. Wearable longitudinal mental-health monitoring

HR、HRV、sleep、activity 每日曲線 × 數週 × 抑鬱 / 焦慮 scale。提案：multilevel functional regression（subject-level + day-level）預測情緒 episode onset，比 daily aggregate 早 2–3 天。

🎯 4. 連續單細胞時序 perturbation atlas

對 perturb-seq + 時序 sampling，每 gene × 每 perturbation 構成 functional curve。提案：functional ANOVA 找「peak time / amplitude / phase」對 perturbation 的依賴，自動分類 immediate-early vs late-response genes。

🎯 5. AlphaFold ensemble functional analysis

AlphaFlow 生成 conformational ensemble 沿 sample-index 是「sample 軸 functional data」。提案：FPCA 對 ensemble 找主要 conformational mode，做 allostery prediction 與 binding pocket prediction。

🎯 6. 抗體成熟動力學

長期免疫追蹤：每位 donor 在數月內 BCR repertoire 與 antibody titer 形成 functional curve。提案：functional regression of titer curve on BCR diversity curve，預測疫苗加強針反應。

🎯 7. Spatial transcriptomics 跨 slice 連續重建

Visium / Slide-seq 連續切片是 spatial functional data。提案：functional spatial PCA 把切片視為空間 + 切片號雙參數的函式 x(s, z)，重建連續 3D 表達場。

🎯 8. EHR longitudinal lab 整合

EHR 中每位患者多個 lab values × 不規則時間。提案：sparse FPCA + functional graphical model 整合 longitudinal labs，預測急性事件（如 AKI、sepsis）比 endpoint snapshot 更穩。

🎯 9. 蛋白質 NMR / Raman 光譜分群

NMR 化學位移、Raman 譜帶都是函式。提案：functional clustering + functional regression to bioactivity，可作為傳統 chemometrics 的升級。

🎯 1. scRNA pseudotime gene-profile clustering

Standard practice clusters pseudotime expression matrices with hierarchical clustering. Proposal: functional clustering (k-medoids on SRSF distance) groups genes into "early peak", "late wave", "bimodal" functional motifs — more interpretable than Pearson clustering.

🎯 2. CGM + diet + microbiome trimodal functional integration

Continuous glucose + diet log + gut microbiome → 3 synchronized functional series. Proposal: multivariate FDA (joint FPCA + functional CCA) finds tri-coupled "microbiome–glucose–diet" patterns for precision nutrition.

🎯 3. Wearable longitudinal mental-health monitoring

Daily HR, HRV, sleep, activity × weeks × depression / anxiety scales. Proposal: multilevel functional regression (subject + day level) predicts mood episodes 2–3 days earlier than daily aggregates.

🎯 4. Continuous scRNA time-series perturbation atlas

For Perturb-seq + time sampling, every (gene, perturbation) is a functional curve. Proposal: functional ANOVA tests dependence of "peak time / amplitude / phase" on perturbation, auto-classifying immediate-early vs late-response genes.

🎯 5. AlphaFold ensemble functional analysis

                 + 0.2*np.random.randn(len(t))
                 for _ in range(20)])

scores = fpca.fit_transform(fd_smooth)
print("Explained variance ratio:", fpca.explained_variance_ratio_)
fpca.components_.plot()           # plot eigenfunctions
